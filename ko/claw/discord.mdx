Claw 인스턴스를 Discord에 연결하고 서버 또는 DM에서 AI와 직접 채팅하세요.

## 사전 요구 사항

- [Discord](https://discord.com/) 계정
- **서버 관리** 권한이 있는 Discord 서버
- Claw 인스턴스 ([여기에서 생성](https://lemondata.cc/dashboard/claw))

## 설정 가이드

<Steps>
  <Step title="Discord 애플리케이션 생성">
    1. [Discord Developer Portal](https://discord.com/developers/applications)로 이동합니다
    2. **New Application**을 클릭합니다
    3. 이름(예: "My AI Bot")을 입력하고 **Create**를 클릭합니다
  </Step>
  <Step title="봇 생성">
    1. 애플리케이션에서 **Bot** 탭으로 이동합니다
    2. **Add Bot** → **Yes, do it!**을 클릭합니다
    3. **Privileged Gateway Intents** 아래에서 다음을 활성화합니다:
       - **Message Content Intent** (필수)
       - **Server Members Intent** (권장)
    4. **Reset Token**을 클릭하여 봇 token을 생성합니다
    5. token을 복사합니다. 다음 단계에서 필요합니다
    <Warning>
    token은 한 번만 표시됩니다. 분실 시 다시 재설정해야 합니다.
    </Warning>
  </Step>
  <Step title="서버에 봇 초대">
    1. **OAuth2** 탭 → **URL Generator**로 이동합니다
    2. **Scopes** 아래에서 `bot`, `applications.commands`를 선택합니다
    3. **Bot Permissions** 아래에서 `Send Messages`, `Read Message History`를 선택합니다
    4. 생성된 URL을 복사하여 브라우저에서 엽니다
    5. 서버를 선택하고 **Authorize**를 클릭합니다
  </Step>
  <Step title="LemonData 대시보드에서 설정">
    1. [Claw Dashboard](https://lemondata.cc/dashboard/claw)로 이동합니다
    2. 인스턴스의 **설정 아이콘**을 클릭합니다
    3. **IM Integrations** 탭을 엽니다
    4. **Discord** 섹션을 확장합니다
    5. **Bot Token**을 붙여넣습니다
    6. **Save**를 클릭합니다
  </Step>
  <Step title="채팅 시작">
    1. Discord 서버에서 봇을 멘션하거나 DM을 보냅니다
    2. **페어링** 과정을 완료합니다 (Claw WebUI에 코드를 입력하세요)
    3. AI와 채팅을 시작하세요!
  </Step>
</Steps>

## 문제 해결

<AccordionGroup>
  <Accordion title="봇이 온라인이지만 응답하지 않음">
    - Developer Portal에서 **Message Content Intent**가 활성화되어 있는지 확인하세요
    - 봇 token이 올바른지 확인하세요
    - 봇이 채널에서 메시지를 읽고 보낼 수 있는 권한이 있는지 확인하세요
  </Accordion>
  <Accordion title="봇이 오프라인으로 표시됨">
    - 대시보드에서 인스턴스 상태를 확인하세요 (녹색/실행 중이어야 함)
    - 인스턴스를 재시작해 보세요
    - Developer Portal에서 token이 재설정되지 않았는지 확인하세요
  </Accordion>
  <Accordion title="로그에 409 Conflict 오류 발생">
    - 두 개의 인스턴스가 동일한 봇 token을 사용하려고 할 때 발생합니다
    - 하나의 Claw 인스턴스만 이 token으로 설정되어 있는지 확인하세요
  </Accordion>
</AccordionGroup>
