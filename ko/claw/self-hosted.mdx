<Note>
**셀프 호스팅**은 OpenClaw를 자체 VPS 또는 머신에서 직접 실행하는 것을 의미합니다. LemonData 잔액으로 AI 모델 사용료만 지불하면 됩니다 — 인스턴스 비용은 없습니다.
</Note>

## 빠른 설치

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash 비대화식
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

설치 프로그램이 **LemonData API Key** 입력을 요청합니다. [대시보드](https://lemondata.cc/dashboard)에서 발급받으세요.

## 설치 내용

<Steps>
  <Step title="Node.js 22+">
    설치되어 있지 않으면 자동으로 설치됩니다 (Linux는 NodeSource, macOS는 Homebrew 사용)
  </Step>
  <Step title="OpenClaw">
    npm의 공식 OpenClaw 패키지, LemonData API 사전 구성 완료
  </Step>
  <Step title="15개 AI 모델">
    Claude, GPT-5, Gemini, DeepSeek 등 — 바로 사용 가능
  </Step>
  <Step title="IM 플러그인">
    DingTalk, WeCom, QQ 플러그인 사전 설치
  </Step>
  <Step title="시스템 서비스">
    부팅 시 자동 시작 (Linux는 systemd, macOS는 launchd)
  </Step>
</Steps>

## 관리 명령어

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # 상태 확인
    journalctl --user -u openclaw -f    # 로그 보기
    systemctl --user restart openclaw   # 재시작
    systemctl --user stop openclaw      # 중지
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # 로그 보기
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # 재시작
    launchctl stop cc.lemondata.openclaw  # 중지
    ```
  </Tab>
</Tabs>

## 설정

설정 파일 위치: `~/.openclaw/openclaw.json`

편집 후 변경 사항을 적용하려면 서비스를 재시작하세요.

### 기본 모델 변경

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

사용 가능한 모델 접두사:
- `lemondata-claude/` — Claude 모델 (anthropic-messages API)
- `lemondata-minimax/` — MiniMax 모델 (anthropic-messages API)
- `lemondata/` — 기타 모든 모델 (openai-completions API)

## 업그레이드

```bash
npm update -g openclaw
# 서비스 재시작
systemctl --user restart openclaw  # Linux
# 또는
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## 제거

```bash
# 서비스 중지 및 비활성화
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# OpenClaw 제거
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
