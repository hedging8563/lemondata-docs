## 개요

[Claude Code](https://docs.anthropic.com/en/docs/claude-code)는 Anthropic의 공식 AI 보조 코딩 CLI 도구입니다. LemonData를 API 제공업체로 사용하도록 설정할 수 있습니다.

## 설정

### 환경 변수

다음 환경 변수를 설정하세요:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

지속적인 적용을 위해 이를 셸 프로필(`~/.bashrc`, `~/.zshrc` 등)에 추가하세요.

### 설정 파일

또는 `~/.claude/settings.json` 파일을 생성하거나 수정하세요:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## 사용법

설정 후, 평소와 같이 Claude Code를 사용하세요:

```bash
# 대화형 세션 시작
claude

# 단일 명령 실행
claude "Explain this code" < main.py

# 채팅 모드
claude chat
```

## 사용 가능한 모델

LemonData는 모든 Claude 모델을 지원합니다:

| 모델 | 설명 |
|-------|-------------|
| `claude-opus-4-5` | 가장 강력한 성능, 복잡한 추론 |
| `claude-sonnet-4-5` | 균형 잡힌 성능, 코딩에 탁월 |
| `claude-haiku-4-5` | 가장 빠르고 비용 효율적 |

## 모델 선택

`--model` 플래그를 사용하여 모델을 지정하세요:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

또는 설정에서 기본값을 지정할 수 있습니다:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Extended Thinking

복잡한 추론 작업의 경우, extended thinking 기능이 있는 모델을 사용하세요:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## 문제 해결

<AccordionGroup>
  <Accordion title="연결 오류">
    기본 URL이 정확히 `https://api.lemondata.cc`인지 확인하세요 (Anthropic SDK의 경우 `/v1` 접미사 없음).
  </Accordion>

  <Accordion title="유효하지 않은 API 키">
    LemonData API 키가 `sk-`로 시작하고 잔액이 충분한지 확인하세요.
  </Accordion>

  <Accordion title="모델을 찾을 수 없음">
    모델 이름이 정확한지 확인하세요. `claude-sonnet-4-5` 형식을 사용하세요.
  </Accordion>
</AccordionGroup>
