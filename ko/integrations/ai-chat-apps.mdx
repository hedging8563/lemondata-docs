## 개요

LemonData는 모든 OpenAI 호환 채팅 애플리케이션과 연동됩니다. 이 가이드는 인기 있는 데스크톱 및 모바일 채팅 앱의 설정 방법을 다룹니다.

## 지원되는 애플리케이션

| 애플리케이션 | 플랫폼 | 기능 |
|-------------|----------|----------|
| LobeChat | 웹, 데스크톱, 셀프 호스팅 | 멀티 프로바이더, 플러그인, RAG |
| CherryStudio | Windows, macOS, Linux | 풍부한 플러그인, 다중 프로바이더 |
| Chatbox | Windows, macOS, Linux | 가볍고 빠름 |
| BotGem | iOS, macOS | 네이티브 Apple 경험 |
| TypingMind | 웹, 데스크톱 | 커스텀 페르소나, 플러그인 |
| OpenCat | iOS, macOS | 네이티브, 오프라인 지원 |
| ChatWise | Windows, macOS | 팀 협업 |

## 설정 단계

### 1단계: API 키 가져오기

1. [LemonData 대시보드](https://lemondata.cc/dashboard)에 로그인합니다
2. [API Keys](https://lemondata.cc/dashboard/api)로 이동합니다
3. API 키를 생성하고 복사합니다 (형식: `sk-...`)

### 2단계: 앱 설정하기

<Tabs>
  <Tab title="LobeChat">
    **셀프 호스팅 LobeChat의 경우:**

    환경 변수를 설정합니다:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **LobeChat Cloud의 경우:**
    1. **설정** → **AI 서비스 프로바이더**를 엽니다
    2. **OpenAI**를 선택합니다
    3. LemonData API 키를 입력합니다
    4. 커스텀 엔드포인트를 설정합니다: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. **설정** → **모델 프로바이더**를 엽니다
    2. **프로바이더 추가**를 클릭합니다
    3. **OpenAI API 호환**을 선택합니다
    4. 다음을 입력합니다:
       - **이름**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API 키**: `sk-your-lemondata-key`
    5. 모델 추가: `gpt-4o`, `claude-sonnet-4-5` 등
    6. 저장 및 테스트
  </Tab>
  <Tab title="Chatbox">
    1. **설정** → **AI 모델**을 엽니다
    2. **OpenAI API 호환**을 선택합니다
    3. 다음을 입력합니다:
       - **API 호스트**: `https://api.lemondata.cc/v1`
       - **API 키**: `sk-your-lemondata-key`
       - **모델**: `gpt-4o`
    4. **연결 테스트**를 위해 확인을 클릭합니다
  </Tab>
  <Tab title="BotGem">
    1. **설정** → **프로바이더**를 엽니다
    2. **커스텀 프로바이더 추가**를 탭합니다
    3. 다음을 입력합니다:
       - **이름**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API 키**: `sk-your-lemondata-key`
    4. 선호하는 모델을 추가합니다
    5. 설정 저장
  </Tab>
  <Tab title="TypingMind">
    1. **설정** (톱니바퀴 아이콘)을 클릭합니다
    2. **커스텀 엔드포인트**를 선택합니다
    3. 다음을 입력합니다:
       - **엔드포인트 URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API 키**: `sk-your-lemondata-key`
    4. 모델을 설정하고 저장합니다
  </Tab>
</Tabs>

### 3단계: 모델 추가

프로바이더를 설정한 후, 사용하려는 모델을 추가합니다:

| 카테고리 | 모델 |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## 설정 참조

| 필드 | 값 |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API Type | OpenAI Compatible |

**JSON 설정 (지원하는 앱의 경우):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## 문제 해결

<AccordionGroup>
  <Accordion title="연결 오류">
    - Base URL이 정확히 `https://api.lemondata.cc/v1`인지 확인합니다
    - 끝에 슬래시(/)가 있는지 확인합니다 (있는 경우 제거)
    - 인터넷 연결 및 방화벽 설정을 확인합니다
  </Accordion>

  <Accordion title="401 권한 없음">
    - API 키를 다시 확인합니다
    - [LemonData 대시보드](https://lemondata.cc/dashboard/api)에서 키가 활성 상태인지 확인합니다
    - 키에 불필요한 공백이 포함되지 않았는지 확인합니다
  </Accordion>

  <Accordion title="402 잔액 부족">
    - 대시보드에서 계정 잔액을 확인합니다
    - 계정에 크레딧을 충전합니다
  </Accordion>

  <Accordion title="404 모델을 사용할 수 없음">
    - 모델 이름의 철자가 정확한지 확인합니다
    - [lemondata.cc/en/models](https://lemondata.cc/ko/models)에서 모델 가용성을 확인합니다
    - 다른 모델을 시도해 보세요
  </Accordion>

  <Accordion title="느린 응답">
    - 더 빠른 모델을 시도해 보세요 (예: `gpt-4o` 대신 `gpt-4o-mini`)
    - 가능한 경우 `max_tokens` 파라미터를 줄입니다
    - 네트워크 연결을 확인합니다
  </Accordion>
</AccordionGroup>
