이 스킬을 사용하면 필요한 내용을 자연어로 설명하는 것만으로 LemonData의 **수백 가지 AI API**를 통합할 수 있습니다. API는 **Agent-First** 설계 — Claude가 모델 이름을 잘못 추측하더라도, 에러 응답이 정확한 수정 방법을 알려줍니다.

## 이 스킬로 무엇을 할 수 있나요?

코드에서 AI 기능(GPT-4, 이미지 생성, 음성 인식 등)을 사용하고 싶을 때, 이 스킬은 다음을 수행합니다:

1. ✅ **API를 직접 시도** — 검색 불필요
2. ✅ **에러에서 자동 수정** — 구조화된 에러 힌트가 다음 시도를 안내
3. ✅ 원하는 언어로 **완전하고 실행 가능한 코드 생성**
4. ✅ 실시간 가격과 가용성을 기반으로 **최적의 모델 선택**
5. ✅ 네이티브 엔드포인트 제안으로 **통합 최적화**

**요약: 필요한 것을 설명하기만 하면 작동하는 API 통합 코드를 얻을 수 있습니다 — 첫 번째 시도가 완벽하지 않더라도, API가 Claude를 올바른 답으로 안내합니다.**

### 예시 1: GPT-4 사용하기

**사용자:**
```
I want to use GPT-4 in my Python project
```

**Claude Code의 동작:**
1. 최적의 모델 추측으로 API를 직접 호출
2. 모델 이름이 틀린 경우, 에러에서 `did_you_mean`을 읽고 자동 수정
3. 올바른 모델로 완전한 Python 코드 생성

**결과물:**
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

## Agent-First API 기능

LemonData의 API는 AI 에이전트를 위해 설계되었습니다. 모든 에러 응답에 구조화된 힌트가 포함됩니다:

### 스마트 에러 복구

| 에러 | API가 반환하는 내용 | Claude의 동작 |
|-------|---------------------|-----------------|
| 잘못된 모델 이름 | `did_you_mean` + `suggestions` | 자동 수정 후 재시도 |
| 잔액 부족 | `balance_usd` + 더 저렴한 `suggestions` | 저렴한 모델로 전환 |
| 모델 사용 불가 | `alternatives` + `retry_after` | 사용 가능한 모델로 전환 |
| 속도 제한 | `retry_after` (정확한 초 단위) | 대기 후 재시도 |
| 컨텍스트 초과 | 더 큰 모델의 `suggestions` | 더 큰 컨텍스트 모델로 전환 |

### 모델 디스커버리

문서를 검색할 필요가 없습니다. 다음 API 엔드포인트를 사용하세요:

```bash
# 기계 판독 가능한 API 개요 읽기
curl https://lemondata.cc/llms.txt

# 카테고리별 모델 목록
curl "https://api.lemondata.cc/v1/models?category=chat" -H "Authorization: Bearer sk-KEY"
curl "https://api.lemondata.cc/v1/models?category=image" -H "Authorization: Bearer sk-KEY"

# 기능별 필터링
curl "https://api.lemondata.cc/v1/models?tag=coding" -H "Authorization: Bearer sk-KEY"
```

### 네이티브 엔드포인트 최적화

Claude가 `/v1/chat/completions`에서 Claude 또는 Gemini 모델을 사용하면, API가 최적화 헤더를 반환합니다:

```
X-LemonData-Hint: This model supports native Anthropic format. Use POST /v1/messages for better performance.
X-LemonData-Native-Endpoint: /v1/messages
```

Claude Code는 자동으로 네이티브 엔드포인트로 전환하여 성능을 향상시킵니다.

### 비용 관리

- 각 API 호출 시 크레딧이 소모됩니다
- [대시보드](https://lemondata.cc/dashboard)에서 사용량 제한을 설정하세요
- 테스트 시에는 적은 양의 데이터를 사용하세요
- `error.balance_usd`와 `error.estimated_cost_usd`로 프로그래밍 방식으로 비용을 모니터링할 수 있습니다

### 에러 처리

생성된 코드에는 Agent-First 에러 처리가 포함되어 있습니다:
- 구조화된 에러 힌트(`did_you_mean`, `suggestions`, `retry_after`)를 읽음
- 일반적인 에러에서 자동 복구
- 필요 시 대체 모델로 폴백
