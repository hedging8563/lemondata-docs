## 개요

[Cursor](https://cursor.sh)는 AI 기반 코드 에디터입니다. LemonData를 커스텀 API 제공업체로 사용하여 300개 이상의 모델을 사용할 수 있습니다.

## 설정

1. Cursor 설정 열기 (`Cmd/Ctrl + ,`)
2. 왼쪽 사이드바에서 **Models**로 이동
3. **OpenAI API Key** 섹션으로 스크롤 다운
4. LemonData API key 입력
5. **Override OpenAI Base URL** 활성화
6. Base URL을 다음과 같이 설정:

```
https://api.lemondata.cc/v1
```

## 모델 선택

설정 후 Cursor에서 모든 LemonData 모델을 선택할 수 있습니다:

### 코딩 추천 모델

| 모델 | 용도 |
|-------|----------|
| `claude-opus-4-5` | 복잡한 코딩, 아키텍처 |
| `claude-sonnet-4-5` | 일반적인 코딩, 디버깅 |
| `gpt-4o` | 코드 생성, 설명 |
| `deepseek-r1` | 추론 중심 작업 |
| `gpt-4o-mini` | 빠른 완성, 단순 작업 |

## 기능

### Chat

모든 LemonData 모델과 함께 AI 채팅 패널(`Cmd/Ctrl + L`)을 사용하세요:

- 코드 설명
- 버그 수정
- 리팩토링 제안
- 문서 생성

### Composer

Composer 기능(`Cmd/Ctrl + I`)은 LemonData 모델과 함께 다음 작업에 사용할 수 있습니다:

- 다중 파일 편집
- 기능 구현
- 코드 마이그레이션

### Tab Completion

탭 완성의 경우, 빠른 모델 사용을 권장합니다:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
