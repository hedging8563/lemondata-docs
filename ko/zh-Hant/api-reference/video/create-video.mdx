## 개요

비디오 생성은 비동기 방식으로 이루어집니다. 요청을 제출하면 `task_id`를 받게 되며, 이후 결과를 폴링(polling)하여 확인합니다.

## 요청 본문

<ParamField body="model" type="string" default="kling-video">
  비디오 모델 (예: `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  생성할 비디오에 대한 텍스트 설명입니다.
</ParamField>

<ParamField body="image_url" type="string">
  이미지 투 비디오(image-to-video) 생성을 위한 선택 사항인 시작 이미지입니다.
</ParamField>

<ParamField body="duration" type="integer">
  비디오 길이, 초 단위 (모델에 따라 다름).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  종횡비 (예: `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  비디오 해상도 (예: `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  초당 프레임 수 (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  비디오 생성 시 제외할 내용입니다.
</ParamField>

<ParamField body="seed" type="integer">
  재현 가능한 생성을 위한 랜덤 시드 값입니다.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Classifier-Free Guidance 스케일 (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  모션 강도 (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  시작 프레임 이미지의 URL.
</ParamField>

<ParamField body="end_image" type="string">
  종료 프레임 이미지의 URL.
</ParamField>

<ParamField body="user" type="string">
  최종 사용자의 고유 식별자입니다.
</ParamField>

## 응답

<ResponseField name="task_id" type="string">
  폴링에 사용되는 고유 작업 식별자입니다.
</ResponseField>

<ResponseField name="status" type="string">
  초기 상태: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  사용된 모델.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/videos/generations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-video",
    "prompt": "A cat walking through a garden, cinematic lighting",
    "duration": 5
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```


## 이미지 투 비디오 (Image-to-Video)

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```

## 사용 가능한 모델

| 모델 | 유형 | 설명 |
|-------|------|-------------|
| `kling-video` | T2V/I2V | 사실적인 움직임, 여러 버전 지원 (v1.0-v2.6) |
| `veo3.1` | T2V | Google의 최신 모델 |
| `veo3.1-pro` | T2V | 전문가급 품질 |
| `minimax/video-01` | T2V | 높은 리듬감과 안정성 |
| `wan2.6-i2v` | I2V | 이미지 투 비디오 |
