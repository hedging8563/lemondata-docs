## 개요

[Claude Code](https://docs.anthropic.com/en/docs/claude-code)는 Anthropic에서 공식적으로 출시한 AI 보조 프로그래밍 CLI 도구입니다. LemonData를 API 제공업체로 사용하도록 구성할 수 있습니다.

## 구성

### 환경 변수

다음 환경 변수를 설정하십시오:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

이 내용을 셸 프로필(`~/.bashrc`, `~/.zshrc` 등)에 추가하여 영구적으로 적용하십시오.

### 구성 파일

또는 `~/.claude/settings.json` 파일을 생성하거나 편집하십시오:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## 사용 방법

구성이 완료되면 Claude Code를 정상적으로 사용할 수 있습니다:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## 사용 가능한 모델

LemonData는 모든 Claude 모델을 지원합니다:

| 모델 | 설명 |
|-------|-------------|
| `claude-opus-4-5` | 복잡한 추론에 적합한 가장 강력한 모델 |
| `claude-sonnet-4-5` | 성능 균형이 뛰어나며 프로그래밍에 매우 적합함 |
| `claude-haiku-4-5` | 가장 빠른 속도와 비용 효율성 제공 |

## 모델 선택

`--model` 플래그를 사용하여 모델을 지정하십시오:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

또는 구성에서 기본값을 설정하십시오:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## 확장 사고

복잡한 추론 작업의 경우 확장 사고(Extended Thinking) 기능이 있는 모델을 사용하십시오:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## 문제 해결

<AccordionGroup>
  <Accordion title="연결 오류">
    기본 URL이 정확히 `https://api.lemondata.cc`인지 확인하십시오 (Anthropic SDK는 `/v1` 접미사가 필요하지 않습니다).
  </Accordion>

  <Accordion title="유효하지 않은 API 키">
    LemonData API 키가 `sk-`로 시작하고 잔액이 충분한지 확인하십시오.
  </Accordion>

  <Accordion title="모델을 찾을 수 없음">
    모델 이름이 정확한지 확인하십시오. `claude-sonnet-4-5` 형식을 사용하십시오.
  </Accordion>
</AccordionGroup>
