## 개요

OpenCode는 터미널에서 실행되는 오픈 소스 AI 코딩 어시스턴트입니다. 여러 LLM 제공업체를 지원하며, 300개 이상의 모델에 액세스하기 위해 LemonData의 API를 사용하도록 설정할 수 있습니다.

```bash
opencode --version
```

## 설정

### 1단계: 환경 변수 설정

```bash
export OPENAI_API_KEY="sk-your-lemondata-key"
export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"
```

영구적인 설정을 위해 `~/.bashrc` 또는 `~/.zshrc`에 다음을 추가하세요:

```bash
echo 'export OPENAI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
echo 'export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"' >> ~/.zshrc
source ~/.zshrc
```

### 2단계: OpenCode 설정

프로젝트 루트 디렉토리에 `~/.config/opencode/opencode.json`(전역) 또는 `opencode.json` 파일을 생성하거나 편집하세요:

```json
{
  "provider": {
    "openai": {
      "options": {
        "apiKey": "sk-your-lemondata-key",
        "baseURL": "https://api.lemondata.cc/v1"
      }
    }
  },
  "model": "gpt-4o",
  "agent": {
    "coder": {
      "model": "gpt-4o"
    }
  }
}
```

## 기본 사용법

**대화형 모드 시작:**

```bash
opencode
```

**프롬프트를 사용하여 실행:**

```bash
opencode "Explain this codebase"
```

**모델 지정:**

```bash
opencode --model gpt-4o "Fix the bugs in main.py"
```

## 사용 가능한 모델

| 모델 | 최적 용도 |
|-------|----------|
| `gpt-4o` | 복잡한 작업, 코드 아키텍처 |
| `gpt-4o-mini` | 빠른 수정, 간단한 질의 |
| `claude-sonnet-4-5` | 코드 리뷰, 문서 작성 |
| `claude-opus-4-5` | 복잡한 추론 |
| `gemini-2.5-flash` | 빠른 응답 |
| `deepseek-r1` | 알고리즘 설계 |

## 주요 명령어

**코드 분석:**

```bash
opencode "What does this function do?" < src/utils.ts
```

**코드 생성:**

```bash
opencode "Create a REST API with Express"
```

**변경 사항 리뷰:**

```bash
git diff | opencode "Review these changes"
```

**오류 수정:**

```bash
opencode "Fix the TypeScript errors in this project"
```

## 대화형 명령어

| 명령어 | 설명 |
|---------|-------------|
| `/help` | 사용 가능한 명령어 표시 |
| `/model <name>` | 다른 모델로 전환 |
| `/clear` | 대화 기록 삭제 |
| `/exit` | OpenCode 종료 |

## 문제 해결

<AccordionGroup>
  <Accordion title="연결 오류">
    - `LOCAL_ENDPOINT`가 `https://api.lemondata.cc/v1`으로 설정되었는지 확인하세요.
    - 네트워크 연결을 확인하세요.
    - `curl https://api.lemondata.cc/v1/models`를 사용하여 테스트해 보세요.
  </Accordion>

  <Accordion title="인증 실패">
    - `OPENAI_API_KEY` 환경 변수가 설정되었는지 확인하세요.
    - 키가 `sk-`로 시작하는지 확인하세요.
    - LemonData 대시보드에서 키가 활성화 상태인지 확인하세요.
  </Accordion>

  <Accordion title="모델을 찾을 수 없음">
    - 설정의 모델 이름이 정확히 일치하는지 확인하세요.
    - [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)에서 모델 가용성을 확인하세요.
  </Accordion>
</AccordionGroup>
