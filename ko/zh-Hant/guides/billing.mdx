## 개요

LemonData는 **종량제 (pay-as-you-go)** 방식을 채택하고 있습니다. 구독이나 최소 사용 약정 없이 사용한 만큼만 비용을 지불하면 됩니다.

## 결제 방식

1. 계정에 **크레딧 충전**
2. **API 사용** - 요청에 따라 비용 차감
3. 대시보드에서 **사용량 모니터링**
4. **잔액 부족 시 충전**

## Token 계산

Tokens는 텍스트 처리의 기본 단위입니다:

- 약 4자 = 1 token (영어)
- 약 1-2자 = 1 token (한글/중국어)
- 이미지 1장 = 크기 및 세부 사항에 따라 다름

### Token 추정

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English
```
