## 개요

LemonData는 **종량제(pay-as-you-go) 요금제**를 사용합니다. 구독료나 최소 약정 없이 사용한 만큼만 비용을 지불하면 됩니다.

## 결제 방식

1. 계정에 **크레딧 추가**
2. **API 사용** - 요청당 비용이 차감됩니다
3. 대시보드에서 **사용량 모니터링**
4. 잔액이 부족할 때 **충전**

### 비동기 작업 과금 (비디오/음악/3D)

<Warning>
  **중요**: 비디오, 음악 및 3D 생성 작업은 완료 시점이 아닌 **제출 시점에 과금**됩니다.
</Warning>

비동기 생성 작업(비디오, 음악, 3D 모델)의 경우:

1. **작업 제출 시 즉시 과금**됩니다
2. 상태를 폴링할 수 있는 `task_id`를 받습니다
3. **작업 실패 시 자동 환불되지 않습니다**

이는 업스트림 제공업체가 결과에 관계없이 작업 제출 시점에 비용을 청구하기 때문입니다. 작업 실패가 발생하면 support@lemondata.cc로 문의해 주세요.

```python
# 예시: 비디오 생성 과금
response = client.post("/v1/video/generations", json={
    "model": "kling-v1",
    "prompt": "바다 위의 일몰"
})
# ⚠️ 비디오가 완성될 때가 아니라 지금 과금됩니다

task_id = response.json()["task_id"]
# 상태 폴링...
```

## 토큰 계산

토큰은 텍스트 처리의 기본 단위입니다:

- 약 4자 = 1 토큰 (영어)
- 약 1-2자 = 1 토큰 (중국어)
- 이미지 1개 = 크기 및 세부 사항에 따라 다름

### 토큰 추정

```python
# 대략적인 추정
def estimate_tokens(text):
    return len(text) / 4  # 영어 기준 근사치

# 실제 카운트 (OpenAI 모델용)
import tiktoken
encoder = tiktoken.encoding_for_model("gpt-4o")
tokens = encoder.encode("Your text here")
print(f"Token count: {len(tokens)}")
```

## 사용량 추적

### 대시보드

[대시보드](https://lemondata.cc/dashboard)에서 사용량을 모니터링하세요:

- 실시간 잔액
- 모델별 사용 기록

- API 키 사용량

### API 응답

각 응답에는 사용량 정보가 포함됩니다:

```json
{
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 100,
    "total_tokens": 150
  }
}
```

## 잔액 부족 알림

잔액이 떨어졌을 때 알림을 받도록 설정하세요:

1. **대시보드 → 설정 → 알림**으로 이동
2. 임계값 금액 설정
3. 이메일 알림 수신

## 크레딧 추가

### 결제 수단

- Alipay (支付宝)
- WeChat Pay (微信支付)
- Stripe (Visa, Mastercard)

### 단계

1. [대시보드](https://lemondata.cc/dashboard)에 로그인
2. **크레딧 추가** 클릭
3. 금액 및 결제 수단 선택
4. 결제 완료

결제 확인 후 크레딧이 즉시 추가됩니다.

## API 키 제한

개별 API 키에 지출 제한을 설정할 수 있습니다:

1. **대시보드 → API 키**로 이동
2. 편집할 키 클릭
3. **사용량 제한** 설정

제한에 도달하면 해당 키를 사용한 요청은 `402 Payment Required`를 반환합니다.

## 인보이스(송장)

비즈니스 계정의 경우 인보이스를 사용할 수 있습니다:

1. **대시보드 → 결제**로 이동
2. 거래 내역 보기
3. 인보이스를 PDF로 다운로드
