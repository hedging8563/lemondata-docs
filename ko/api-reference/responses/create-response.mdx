## 인증

<ParamField header="Authorization" type="string" required>
  API 인증을 위한 Bearer 토큰. 형식: `Bearer sk-your-api-key`
</ParamField>

Responses API는 OpenAI의 새로운 상태 유지형(stateful) 대화 API입니다. LemonData는 호환되는 모델에 대해 이 형식을 지원합니다.

## 요청 본문

<ParamField body="model" type="string" required>
  사용할 모델의 ID입니다. 사용 가능한 옵션은 [모델](https://lemondata.cc/ko/models)을 참조하세요.
</ParamField>

<ParamField body="input" type="array" required>
  대화를 구성하는 입력 항목 목록입니다.

  각 항목은 다음과 같을 수 있습니다:
  - `message`: 역할(role)과 내용(content)이 포함된 대화 메시지
  - `function_call`: 함수 호출 요청
  - `function_call_output`: 함수 호출 결과물
</ParamField>

<ParamField body="instructions" type="string">
  모델을 위한 시스템 지침입니다 (시스템 메시지와 동일).
</ParamField>

<ParamField body="max_output_tokens" type="integer">
  생성할 최대 토큰 수입니다.
</ParamField>

<ParamField body="temperature" type="number" default="1">
  0에서 2 사이의 샘플링 온도입니다.
</ParamField>

<ParamField body="tools" type="array">
  모델이 호출할 수 있는 도구 목록입니다.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  true인 경우, 이벤트 스트림을 반환합니다.
</ParamField>

<ParamField body="previous_response_id" type="string">
  대화를 이어가기 위한 이전 응답의 ID입니다.
</ParamField>

<ParamField body="store" type="boolean" default="true">
  나중에 조회할 수 있도록 응답을 저장할지 여부입니다.
</ParamField>

<ParamField body="metadata" type="object">
  추적 목적으로 응답에 첨부할 메타데이터입니다.
</ParamField>

<ParamField body="text" type="object">
  텍스트 생성 구성 옵션입니다.
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  여러 도구 호출을 병렬로 허용할지 여부입니다.
</ParamField>

<ParamField body="top_p" type="number">
  Nucleus 샘플링 파라미터 (0-1)입니다.
</ParamField>

<ParamField body="reasoning" type="object">
  o1/o3 모델을 위한 추론(Reasoning) 구성입니다.

  - `effort` (string): 추론 노력 수준 (`low`, `medium`, `high`)
</ParamField
