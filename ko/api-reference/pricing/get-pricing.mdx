## 쿼리 파라미터

<ParamField query="model" type="string">
  가격을 조회할 모델 ID입니다. 지정하지 않으면 모든 모델의 가격을 반환합니다.
</ParamField>

## 응답

<ResponseField name="object" type="string">
  항상 `list`입니다.
</ResponseField>

<ResponseField name="data" type="array">
  모델 가격 객체의 배열입니다.

  각 객체는 다음을 포함합니다:
  - `model` (string): 모델 식별자
  - `pricing` (object): 가격 정보
    - `input_per_1m` (string | null): 100만 입력 토큰당 USD
    - `output_per_1m` (string | null): 100만 출력 토큰당 USD
    - `per_request` (string | null): 요청당 USD (고정 가격 모델의 경우)
    - `is_lock_price` (boolean): 요청당 가격이 고정되어 있는지 여부
    - `currency` (string): 항상 `USD`
  - `provider` (string): 모델 제공자
</ResponseField>

## 가격 유형

| 유형 | 설명 | 예시 모델 |
|------|-------------|----------------|
| **토큰 기반** | 입력/출력 토큰당 과금 | GPT-4o, Claude, Gemini |
| **요청당 과금** | 요청당 고정 가격 | DALL-E 3, Sora, Suno |
