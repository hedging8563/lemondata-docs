## LemonData とは？

LemonData は、OpenAI、Anthropic、Google、DeepSeek などの主要プロバイダーが提供する **300 以上の AI モデル** への統合アクセスを提供する AI API アグリゲータープラットフォームです。

<CardGroup cols={2}>
  <Card title="OpenAI 互換" icon="plug">
    OpenAI API を直接置き換え可能です。`base URL` を変更するだけです。
  </Card>
  <Card title="300 以上のモデル" icon="robot">
    GPT-4o、Claude、Gemini、DeepSeek、Midjourney、Sora などにアクセスできます。
  </Card>
  <Card title="従量課金制" icon="coins">
    サブスクリプションは不要です。使用した分だけ支払う、非常に競争力のある価格設定です。
  </Card>
  <Card title="高可用性" icon="shield-check">
    自動フェイルオーバーと負荷分散により、信頼性の高いサービスを保証します。
  </Card>
</CardGroup>

## サポートされている機能

| 機能 | 説明 | モデル例 |
|------------|-------------|----------------|
| **チャット** | テキスト生成と対話 | GPT-4o, Claude 3.5, Gemini 2.0 |
| **ビジョン** | 画像理解と分析 | GPT-4o, Claude 3.5 Sonnet, Gemini |
| **画像生成** | テキストから画像を生成 | DALL-E 3, Midjourney, Flux, Ideogram |
| **動画生成** | テキスト/画像から動画を生成 | Sora, Runway Gen-3, Kling |
| **オーディオ** | テキスト読み上げと文字起こし | Whisper, TTS-1, MiniMax |
| **Embeddings** | テキストのベクトル化 | text-embedding-3-small |
| **音楽** | AI 音楽生成 | Suno |
| **3D 生成** | 3D モデルの作成 | Tripo3D |

## クイックサンプル

<CodeGroup>

```bash cURL
curl https://api.lemondata.cc/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"

    client := openai.NewClientWithConfig(config)

    resp, _ := client.CreateChatCompletion(
        context.Background(),
        openai.ChatCompletionRequest{
            Model: "gpt-4o",
            Messages: []openai.ChatCompletionMessage{
                {Role: openai.ChatMessageRoleUser, Content: "Hello!"},
            },
        },
    )
    fmt.Println(resp.Choices[0].Message.Content)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/chat/completions');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'gpt-4o',
        'messages' => [
            ['role' => 'user', 'content' => 'Hello!']
        ]
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['choices'][0]['message']['content'];
```
