## 概要

LemonData は **pay-as-you-go (従量課金制)** を採用しています。使用した分だけお支払いいただく形式で、サブスクリプションや最低利用料金の制限はありません。

## お支払いの仕組み

1. アカウントに **クレジットをチャージ** します
2. **API を使用** します - 料金はリクエストごとに差し引かれます
3. コントロールパネルで **使用状況を監視** します
4. **残高が不足した際にチャージ** します

## token 数をカウントする

token はテキスト処理の基本単位です：

- 約4文字 = 1 token (英語)
- 約1-2文字 = 1 token (中国語)
- 画像1枚 = サイズと詳細度によります

### token 数を推定する

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English

# Actual count (for OpenAI models)
```
