---
title: "ドキュメントのリランク"
openapi: "POST /v1/rerank"
description: "クエリとの関連性に基づいてドキュメントをリランクします"
---

セマンティック類似性モデルを使用してドキュメントをリランクします。検索結果の最適化や RAG アプリケーションに適しています。

## リクエストボディ

<ParamField body="model" type="string" required>
  使用するリランクモデルの ID（例：`BAAI/bge-reranker-v2-m3`、`qwen3-rerank`）。
</ParamField>

<ParamField body="query" type="string" required>
  ドキュメントのランキングに使用するクエリ文字列。
</ParamField>

<ParamField body="documents" type="array" required>
  リランク対象のドキュメントリスト（文字列）。
</ParamField>

<ParamField body="top_n" type="integer">
  返される上位結果の数。デフォルトはすべてのドキュメントです。
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  レスポンスに元のドキュメントテキストを含めるかどうか。
</ParamField>

## レスポンス

<ResponseField name="results" type="array">
  スコア付きのソート済みドキュメントリスト。

  各結果には以下が含まれます：
  - `index` (integer)：元のドキュメントのインデックス
  - `relevance_score` (number)：関連性スコア (0-1)
  - `document` (string)：元のテキスト（`return_documents=true` の場合）
</ResponseField>

<ResponseField name="model" type="string">
  リランクに使用されたモデル。
</ResponseField>

<ResponseField name="usage" type="object">
  Token 使用量の統計。
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/rerank" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "BAAI/bge-reranker-v2-m3",
    "query": "What is machine learning?",
    "documents": [
      "Machine learning is a subset of AI",
      "The weather is nice today",
      "Deep learning uses neural networks"
    ],
    "top_n": 2,
    "return_documents": true
  }'
```
