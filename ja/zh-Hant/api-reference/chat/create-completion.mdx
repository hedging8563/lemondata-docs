---

## リクエストボディ (Request Body)

<ParamField body="model" type="string" required>
  使用するモデル ID。利用可能なオプションについては [Models](https://lemondata.cc/zh-TW/models) を参照してください。
</ParamField>

<ParamField body="messages" type="array" required>
  対話を構成するメッセージのリスト。

  各メッセージオブジェクトには以下が含まれます：
  - `role` (string)：`system`、`user` または `assistant`
  - `content` (string | array)：メッセージの内容
</ParamField>

<ParamField body="temperature" type="number" default="1">
  0 から 2 の間のサンプリング温度。値が高いほど、出力のランダム性が増します。
</ParamField>

<ParamField body="max_tokens" type="integer">
  生成する最大 token 数。
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  true の場合、メッセージの一部が SSE イベントとして逐次送信されます。
</ParamField>

<ParamField body="stream_options" type="object">
  ストリームオプション。`include_usage: true` を設定すると、ストリームチャンク内で token の使用状況を受け取ることができます。
</ParamField>

<ParamField body="top_p" type="number" default="1">
  核サンプリング (Nucleus sampling) パラメータ。このパラメータまたは `temperature` のいずれか一方のみを調整することをお勧めします。
</ParamField>

<ParamField body="frequency_penalty" type="number" default="0">
  -2.0 から 2.0 の間の数値。正の値は、繰り返される token にペナルティを課します。
</ParamField>

<ParamField body="presence_penalty" type="number" default="0">
  -2.0 から 2.0 の間の数値。正の値は、テキスト内に既に出現した token にペナルティを課します。
</ParamField>

<ParamField body="stop" type="string | array">
  最大 4 つのシーケンス。API はこれらのシーケンスで token の生成を停止します。
</ParamField>

<ParamField body="tools" type="array">
  モデルが呼び出す可能性のあるツールのリスト (function calling)。
</ParamField>

<ParamField body="tool_choice" type="string | object">
  モデルがツールをどのように使用するかを制御します。オプション：`auto`、`none`、`required` または特定のツールオブジェクト。
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  並列関数呼び出しを有効にするかどうか。false に設定すると、関数は順番に呼び出されます。
</ParamField>

<ParamField body="max_completion_tokens" type="integer">
  補完の最大 token 数。`max_tokens` の代替案であり、o1/o3 などの新しいモデルでの使用が推奨されます。
</ParamField>

<ParamField body="reasoning_effort" type="string">
  o1/o3 モデルの推論強度。オプション：`low`、`medium`、`high`。
</ParamField>

<ParamField body="seed" type="integer">
  決定論的サンプリングに使用されるランダムシード。
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成する補完の数 (1-128)。
</ParamField>

<ParamField body="logprobs" type="boolean">
  対数確率 (log probabilities) を返すかどうか。
</ParamField>

<ParamField body="top_logprobs" type="integer">
  返される上位の対数確率の数 (0-20)。`logprobs: true` を設定する必要があります。
</ParamField>

<ParamField body="top_k" type="integer">
  Top-K サンプリングパラメータ（Anthropic/Gemini モデルに適用）。
</ParamField>

<ParamField body="response_format" type="object">
  レスポンス形式の指定。`{"type": "json_object"}` を使用して JSON モードを有効にするか、`{"type": "json_schema", "json_schema": {...}}` を使用して構造化出力を行います。
</ParamField>

<ParamField body="logit_bias" type="object">
  特定の token が出現する可能性を修正します。token ID（文字列形式）を -100 から 100 の間のバイアス値にマッピングします。
</ParamField>

<ParamField body="user" type="string">
  エンドユーザーを表す一意の識別子。不正利用の監視に使用されます。
</ParamField>

<ParamField body="cache_control" type="object">
  LemonData キャッシュ制御オプション。

  - `type` (string)：キャッシュ戦略 - `default`、`no_cache`、`no_store`、`response_only`、`semantic_only`
  - `max_age` (integer)：キャッシュの生存期間 (TTL)、秒単位（最大 86400）
</ParamField>

## レスポンス (Response)

<ResponseField name="id" type="string">
  補完の一意の識別子。
</ResponseField>

<ResponseField name="object" type="string">
  常に `chat.completion`。
</ResponseField>

<ResponseField name="created" type="integer">
  補完が作成された時の Unix タイムスタンプ。
</ResponseField>

<ResponseField name="model" type="string">
  補完に使用されたモデル。
</ResponseField>

<ResponseField name="choices" type="array">
  補完オプションのリスト。

  各オプションには以下が含まれます：
  - `index` (integer)：オプションのインデックス
  - `message` (object)：生成されたメッセージ
  - `finish_reason` (string)：モデルが停止した理由 (`stop`、`length`、`tool_calls`)
</ResponseField>

<ResponseField name="usage" type="object">
  Token 使用統計。

  - `prompt_tokens` (integer)：プロンプト内の token 数
  - `completion_tokens` (integer)：補完内の token 数
  - `total_tokens` (integer)：使用された合計 token 数
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"}
    ],
    temperature=0.7,
    max_tokens=1000
)

print(response.choices[0].message.content)
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateChatCompletion(
        context.Background(),
        openai.ChatCompletionRequest{
            Model: "gpt-4o",
            Messages: []openai.ChatCompletionMessage{
                {Role: openai.ChatMessageRoleSystem, Content: "You are a helpful assistant."},
                {Role: openai.ChatMessageRoleUser, Content: "Hello!"},
            },
            Temperature: 0.7,
            MaxTokens:   1000,
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Choices[0].Message.Content)
}
```
