## リクエストボディ

<ParamField body="file" type="file" required>
  文字起こしする音声ファイル。サポートされている形式：flac, mp3, mp4, mpeg, mpga, m4a, ogg, wav, webm。
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  使用するモデル。現在、`whisper-1` のみがサポートされています。
</ParamField>

<ParamField body="language" type="string">
  音声の言語。ISO-639-1 形式（例：`en`、`zh`、`ja`）。
</ParamField>

<ParamField body="prompt" type="string">
  モデルのスタイルをガイドしたり、前のセグメントを継続したりするためのオプションのテキスト。
</ParamField>

<ParamField body="response_format" type="string" default="json">
  出力形式：`json`、`text`、`srt`、`verbose_json`、`vtt`。
</ParamField>

<ParamField body="temperature" type="number" default="0">
  サンプリング温度（0 から 1）。
</ParamField>

<ParamField body="timestamp_granularities" type="array">
  タイムスタンプの粒度：`word` および/または `segment`。`verbose_json` が必要です。
</ParamField>

## レスポンス

<ResponseField name="text" type="string">
  文字起こしされたテキスト。
</ResponseField>

`verbose_json` の場合：

<ResponseField name="task" type="string">
  `transcribe` に固定されます。
</ResponseField>

<ResponseField name="language" type="string">
  検出された言語。
</ResponseField>

<ResponseField name="duration" type="number">
  音声の長さ（秒）。
</ResponseField>

<ResponseField name="segments" type="array">
  タイムスタンプを含む文字起こしセグメント。
</ResponseField>

<ResponseField name="words" type="array">
  単語レベルのタイムスタンプ（リクエストされた場合）。
</ResponseField>
