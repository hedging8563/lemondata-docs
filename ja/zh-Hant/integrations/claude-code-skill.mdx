## ステップ 2：API Keyの取得

このスキルを使用する前に、LemonData API Keyが必要です。

<Tip>
**事前にAPI Keyを設定する必要はありません！** このスキルを使用すると、Claude Codeが自動的にAPI Keyを尋ね、生成されたコードに含めます。
</Tip>

<Steps>
  <Step title="LemonDataにアクセス">
    [lemondata.cc](https://lemondata.cc) に移動
  </Step>
  <Step title="ログイン">
    アカウントを作成またはログイン
  </Step>
  <Step title="API Keyを取得">
    [Dashboard → API Keys](https://lemondata.cc/dashboard/api) に移動し、新しいキーを作成
  </Step>
  <Step title="キーをコピー">
    キーは `sk-...` で始まります。安全に保存してください
  </Step>
</Steps>

<Warning>
**重要な注意事項：**
- すべての LemonData API で同じ API Key を使用します
- キーを他人に共有しないでください
- キーを GitHub にコミットしないでください
</Warning>

## ステップ 3：使い始める

このスキルの使い方はチャットのように簡単です！

### 例 1：GPT-4を使用する

**あなたの入力：**
```
I want to use GPT-4 in my Python project
```

**Claude Codeの動作：**
1. API Keyを尋ねる
2. GPT-4に関連するAPIを検索
3. 利用可能なAPIオプションを表示
4. 完全なPythonコードを生成
5. 使いかたを説明

**出力結果：**
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

### 例 2：画像を生成する

**あなたの入力：**
```
How do I generate images with Flux in Node.js?
```

**Claude Codeの動作：**
- 画像生成APIを検索
- JavaScriptコードを生成
- 画像のダウンロードと保存コードを含む

### 例 3：音声認識

**あなたの入力：**
```
Integrate a speech-to-text API
```

**Claude Codeの動作：**
- 使用するプログラミング言語を尋ねる
- 音声認識APIを検索
- 選択した言語でコードを生成
- オーディオファイル処理の例を提供
