## 概要

[Cursor](https://cursor.sh) は、AI 搭載のコードエディタです。LemonData をカスタム API プロバイダーとして使用することで、300 以上のモデルにアクセスできます。

## 設定

1. Cursor の設定（`Cmd/Ctrl + ,`）を開きます
2. 左サイドバーの **Models** に移動します
3. **OpenAI API Key** セクションまでスクロールします
4. LemonData の API key を入力します
5. **Override OpenAI Base URL** を有効にします
6. Base URL を以下に設定します：

```
https://api.lemondata.cc/v1
```

## モデルの選択

設定が完了すると、Cursor で任意の LemonData モデルを選択できるようになります：

### 推奨されるコーディングモデル

| モデル | 用途 |
|-------|----------|
| `claude-opus-4-5` | 複雑なコーディング、アーキテクチャ設計 |
| `claude-sonnet-4-5` | 一般的なコーディング、デバッグ |
| `gpt-4o` | コード生成、解説 |
| `deepseek-r1` | 高度な推論タスク |
| `gpt-4o-mini` | 高速な補完、シンプルなタスク |

## 機能

### Chat

任意の LemonData モデルを使用して AI チャットパネル（`Cmd/Ctrl + L`）を利用できます：

- コードの解説
- バグの修正
- リファクタリングの提案
- ドキュメントの生成

### Composer

Composer 機能（`Cmd/Ctrl + I`）を LemonData モデルと組み合わせて以下の用途で使用できます：

- 複数ファイルの編集
- 機能の実装
- コードの移行

### Tab 補完

Tab 補完には、高速なモデルの使用を推奨します：

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
