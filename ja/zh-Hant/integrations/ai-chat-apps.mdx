## 概要

LemonDataは、OpenAI互換のあらゆるチャットアプリケーションで使用できます。このガイドでは、人気のデスクトップおよびモバイル向けチャットアプリケーションの設定方法について説明します。

## サポートされているアプリケーション

| アプリケーション | プラットフォーム | 機能 |
|-------------|----------|----------|
| LobeChat | ウェブ、デスクトップ、セルフホスト | マルチプロバイダー、プラグイン、RAG |
| CherryStudio | Windows, macOS, Linux | 豊富なプラグイン、マルチプロバイダー |
| Chatbox | Windows, macOS, Linux | 軽量、高速 |
| BotGem | iOS, macOS | ネイティブなApple体験 |
| TypingMind | ウェブ、デスクトップ | カスタムペルソナ、プラグイン |
| OpenCat | iOS, macOS | ネイティブ、オフラインサポート |
| ChatWise | Windows, macOS | チームコラボレーション |

## 設定手順

### ステップ 1：API Keyの取得

1. [LemonData コンソール](https://lemondata.cc/dashboard)にログインします
2. [API Keys](https://lemondata.cc/dashboard/api)に移動します
3. API Keyを作成してコピーします（形式：`sk-...`）

### ステップ 2：アプリケーションの設定

<Tabs>
  <Tab title="LobeChat">
    **セルフホスト型LobeChatの場合：**

    環境変数を設定します：
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **LobeChat Cloudの場合：**
    1. **Settings** → **AI Service Provider**を開きます
    2. **OpenAI**を選択します
    3. LemonDataのAPI Keyを入力します
    4. カスタムエンドポイントを設定します：`https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. **Settings** → **Model Provider**を開きます
    2. **Add Provider**をクリックします
    3. **OpenAI API Compatible**を選択します
    4. 以下を入力します：
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. モデルを追加します：`gpt-4o`, `claude-sonnet-4-5` など
    6. 保存してテストします
  </Tab>
  <Tab title="Chatbox">
    1. **Settings** → **AI Model**を開きます
    2. **OpenAI API Compatible**を選択します
    3. 以下を入力します：
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. **Verify**をクリックして接続をテストします
  </Tab>
  <Tab title="BotGem">
    1. **Settings** → **Providers**を開きます
    2. **Add Custom Provider**をクリックします
    3. 以下を入力します：
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. お好みのモデルを追加します
    5. 設定を保存します
  </Tab>
  <Tab title="TypingMind">
    1. **Settings**（歯車アイコン）をクリックします
    2. **Custom Endpoint**を選択します
    3. 以下を入力します：
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. モデルを設定して保存します
  </Tab>
</Tabs>

### ステップ 3：モデルの追加

プロバイダーを設定した後、使用したいモデルを追加します：

| カテゴリ | モデル |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## 設定リファレンス

| フィールド | 値 |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| APIタイプ | OpenAI Compatible |

**JSON設定（サポートされているアプリケーション用）：**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## トラブルシューティング

<AccordionGroup>
  <Accordion title="接続エラー">
    - Base URLが正確に `https://api.lemondata.cc/v1` であることを確認してください
    - 末尾にスラッシュが含まれていないか確認してください（含まれている場合は削除してください）
    - ネットワーク接続とファイアウォールの設定を確認してください
  </Accordion>

  <Accordion title="401 Unauthorized">
    - API Keyを再確認してください
    - [LemonData コンソール](https://lemondata.cc/dashboard/api) でKeyが有効な状態であることを確認してください
    - Keyに余計なスペースが含まれていないことを確認してください
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - コンソールでアカウント残高を確認してください
    - アカウントにチャージしてください
  </Accordion>

  <Accordion title="404 Model Not Available">
    - モデル名のスペルが正しいか確認してください
    - [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models) でモデルの利用可能性を確認してください
    - 別のモデルを試してください
  </Accordion>

  <Accordion title="レスポンスが遅い">
    - より高速なモデルを試してください（例：`gpt-4o` の代わりに `gpt-4o-mini` を使用）
    - 可能であれば、`max_tokens` パラメータを減らしてください
    - ネットワーク接続を確認してください
  </Accordion>
</AccordionGroup>
