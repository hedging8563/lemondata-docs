Telegramは、接続が**最も簡単**なプラットフォームです。BotFatherからボットトークンを取得するだけで、2分以内にAIとのチャットを開始できます。

## 前提条件

- [Telegram](https://telegram.org/)アカウント
- Clawインスタンス（[こちらで作成](https://lemondata.cc/dashboard/claw)）

## セットアップガイド

<Steps>
  <Step title="BotFatherでボットを作成する">
    1. Telegramを開き、**@BotFather**を検索します
    2. `/newbot`を送信します
    3. ボットの**表示名**を選択します（例：「My AI Assistant」）
    4. `bot`で終わる**ユーザー名**を選択します（例：`my_ai_assistant_bot`）
    5. BotFatherが**ボットトークン**を返信します。以下のような形式です：
    ```
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
    ```
    <Warning>
    ボットトークンは秘密にしてください！このトークンを知っている人は誰でもあなたのボットを操作できてしまいます。
    </Warning>
  </Step>
  <Step title="LemonDataダッシュボードで設定する">
    1. [Clawダッシュボード](https://lemondata.cc/dashboard/claw)に移動します
    2. インスタンスの**設定アイコン**をクリックします
    3. **IM Integrations**タブを開きます
    4. **Telegram**セクションを展開します
    5. **ボットトークン**を貼り付けます
    6. **Save**をクリックします
  </Step>
  <Step title="チャットを開始する">
    1. Telegramを開き、ユーザー名でボットを検索します
    2. 任意のメッセージを送信します（例：「Hello!」）
    3. ボットが**ペアリングコード**を返信します
    4. ClawのWebUIにコードを入力してペアリングを完了させます
    5. これで準備完了です。AIとのチャットを始めましょう！
  </Step>
</Steps>

## オプション：ボットプロフィールのカスタマイズ

ボットを作成した後、BotFatherを使用してカスタマイズできます：

| コマンド | 目的 |
|---------|---------|
| `/setdescription` | ボットのバイオ/説明を設定する |
| `/setabouttext` | 「About」セクションを設定する |
| `/setuserpic` | プロフィール画像をアップロードする |
| `/setcommands` | コマンドの候補を追加する |

## トラブルシューティング

<AccordionGroup>
  <Accordion title="ボットが応答しない">
    - ボットトークンが正しいか確認してください（余計なスペースがないかなど）
    - インスタンスが実行中であることを確認してください（ダッシュボードで緑色のステータス）
    - ダッシュボードからインスタンスを再起動してみてください
  </Accordion>
  <Accordion title="「ペアリングコード」が表示されない">
    - グループではなく、ボットに**ダイレクトメッセージ**を送信していることを確認してください
    - ペアリングシステムは、新しいユーザーからの最初のメッセージで有効になります
  </Accordion>
  <Accordion title="ボットの応答が遅い">
    - 応答時間は選択されたAIモデルによって異なります
    - 大規模なモデル（GPT-4, Claude）は数秒かかる場合があります
    - LemonDataの残高を確認してください。クレジットが不足しているとエラーが発生します
  </Accordion>
</AccordionGroup>
