<Note>
**セルフホスト**とは、OpenClaw をご自身の VPS やマシンで直接実行することを意味します。LemonData の残高から AI モデルの使用料のみお支払いいただきます — インスタンス料金はかかりません。
</Note>

## クイックインストール

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash 非対話式
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

インストーラーが **LemonData API Key** の入力を求めます。[ダッシュボード](https://lemondata.cc/dashboard)から取得してください。

## インストール内容

<Steps>
  <Step title="Node.js 22+">
    未インストールの場合は自動的にインストールされます（Linux は NodeSource 経由、macOS は Homebrew 経由）
  </Step>
  <Step title="OpenClaw">
    npm からの公式 OpenClaw パッケージ、LemonData API 設定済み
  </Step>
  <Step title="15 の AI モデル">
    Claude、GPT-5、Gemini、DeepSeek など — すぐに利用可能
  </Step>
  <Step title="IM プラグイン">
    DingTalk、WeCom、QQ プラグインがプリインストール済み
  </Step>
  <Step title="システムサービス">
    起動時に自動開始（Linux は systemd、macOS は launchd）
  </Step>
</Steps>

## 要件

| 要件 | 詳細 |
|------|------|
| **OS** | Linux（Ubuntu/Debian/CentOS/RHEL）または macOS |
| **アーキテクチャ** | x64 または arm64 |
| **メモリ** | 最低 1 GB、推奨 2 GB |
| **API Key** | [こちらから取得](https://lemondata.cc) |

## 管理コマンド

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # ステータス確認
    journalctl --user -u openclaw -f    # ログ表示
    systemctl --user restart openclaw   # 再起動
    systemctl --user stop openclaw      # 停止
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # ログ表示
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # 再起動
    launchctl stop cc.lemondata.openclaw  # 停止
    ```
  </Tab>
</Tabs>

## 設定

設定ファイルの場所：`~/.openclaw/openclaw.json`

編集後、変更を適用するためにサービスを再起動してください。

### デフォルトモデルの変更

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

利用可能なモデルプレフィックス：
- `lemondata-claude/` — Claude モデル（anthropic-messages API）
- `lemondata-minimax/` — MiniMax モデル（anthropic-messages API）
- `lemondata/` — その他すべてのモデル（openai-completions API）

## アップグレード

```bash
npm update -g openclaw
# サービスを再起動
systemctl --user restart openclaw  # Linux
# または
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## アンインストール

```bash
# サービスの停止と無効化
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# OpenClaw の削除
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
