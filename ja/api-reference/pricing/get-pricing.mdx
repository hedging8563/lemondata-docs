---
title: "料金情報の取得"
openapi: "GET /v1/pricing"
description: "モデルの料金情報を取得します"
---

## 認証

<ParamField header="Authorization" type="string" required>
  API認証用のBearerトークン。形式：`Bearer sk-your-api-key`
</ParamField>

## クエリパラメータ

<ParamField query="model" type="string">
  料金を取得するモデルのID。指定しない場合、すべてのモデルの料金を返します。
</ParamField>

## レスポンス

<ResponseField name="object" type="string">
  常に `list` です。
</ResponseField>

<ResponseField name="data" type="array">
  モデル料金オブジェクトの配列。

  各オブジェクトには以下が含まれます：
  - `model` (string): モデル識別子
  - `pricing` (object): 料金情報
    - `input_per_1m` (string | null): 100万入力トークンあたりのUSD
    - `output_per_1m` (string | null): 100万出力トークンあたりのUSD
    - `per_request` (string | null): リクエストあたりのUSD（固定料金モデルの場合）
    - `is_lock_price` (boolean): リクエストごとに料金が固定されているかどうか
    - `currency` (string): 常に `USD`
  - `provider` (string): モデルプロバイダー
</ResponseField>

```python Python
import requests

response = requests.get(
    "https://api.lemondata.cc/v1/pricing",
    headers={"Authorization": "Bearer sk-your-api-key"},
    params={"model": "gpt-4o"}
)

print(response.json())
```
