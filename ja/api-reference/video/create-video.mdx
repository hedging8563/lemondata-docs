---
title: "動画作成"
openapi: "POST /v1/videos/generations"
description: "動画生成タスクを作成します"
---

## 概要

動画生成は非同期で行われます。リクエストを送信してタスクIDを受け取り、その後結果をポーリングして取得します。

## リクエストボディ

<ParamField body="model" type="string" default="sora-2">
  動画モデルID。11のプロバイダーにわたる80以上のモデルの全リストについては、[Video Generation Guide](/guides/video-generation)を参照してください。
</ParamField>

<ParamField body="prompt" type="string" required>
  生成する動画のテキスト説明。
</ParamField>

<ParamField body="image_url" type="string">
  image-to-video (I2V) 生成の開始画像のURL。
</ParamField>

<ParamField body="image" type="string">
  データURLプレフィックス（例：`data:image/jpeg;base64,...`）を含むBase64エンコードされた画像。I2V生成における `image_url` の代替手段。
</ParamField>

<ParamField body="duration" type="integer">
  動画の長さ（秒単位、モデルに依存）。
</ParamField>

<ParamField body="aspect_ratio" type="string">
  アスペクト比（例：`16:9`、`9:16`、`1:1`）。
</ParamField>

<ParamField body="resolution" type="string">
  動画の解像度（例：`1080p`、`720p`）。
</ParamField>

<ParamField body="fps" type="integer">
  フレームレート (1-120)。
</ParamField>

<ParamField body="negative_prompt" type="string">
  動画生成で避けるべき内容。
</ParamField>

<ParamField body="seed" type="integer">
  再現可能な生成のためのランダムシード。
</ParamField>

<ParamField body="cfg_scale" type="number">
  Classifier-free guidance スケール (0-20)。
</ParamField>

<ParamField body="motion_strength" type="number">
  動きの強さ (0-1)。
</ParamField>

<ParamField body="start_image" type="string">
  開始フレーム画像のURL。
</ParamField>

<ParamField body="end_image" type="string">
  終了フレーム画像のURL。
</ParamField>

<ParamField body="user" type="string">
  エンドユーザーの一意識別子。
</ParamField>

## レスポンス

<ResponseField name="task_id" type="string">
  ポーリング用の一意のタスク識別子。
</ResponseField>
