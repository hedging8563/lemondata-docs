## 概要

LemonDataは**従量課金制（pay-as-you-go pricing）**を採用しています。サブスクリプションや最低利用料金はなく、使用した分だけお支払いいただきます。

## 請求の仕組み

1. アカウントに**クレジットを追加**する
2. **APIを使用する** - リクエストごとに費用が差し引かれます
3. ダッシュボードで**使用状況を監視**する
4. 残高が少なくなったら**チャージ**する

### リクエスト単位の料金

画像、動画、音声モデルはリクエストごとに課金されます。

| モデル | リクエストあたりの料金 |
|-------|-------------------|
| DALL-E 3 (1024x1024) | $0.04 |
| Midjourney | $0.05 |
| Sora Video (5s) | $0.20 |
| Whisper (1分あたり) | $0.006 |
| TTS-1 | $0.015 |

### 非同期タスクの課金（動画/音楽/3D）

<Warning>
  **重要**：動画、音楽、3D生成タスクは、完了時ではなく**送信時に課金**されます。
</Warning>

非同期生成タスク（動画、音楽、3Dモデル）の場合：

1. **タスク送信時に即座に課金**されます
2. ステータスをポーリングするための `task_id` を受け取ります
3. **タスクが失敗しても自動返金はありません**

これは、上流プロバイダーが結果に関係なくタスク送信時に課金するためです。タスクの失敗が発生した場合は、support@lemondata.cc までお問い合わせください。

```python
# 例：動画生成の課金
response = client.post("/v1/video/generations", json={
    "model": "kling-v1",
    "prompt": "海に沈む夕日"
})
# ⚠️ 動画が完成した時ではなく、今課金されます

task_id = response.json()["task_id"]
# ステータスをポーリング...
```

## トークンのカウント

トークンはテキスト処理の基本単位です。

- 約4文字 = 1トークン（英語）
- 約1〜2文字 = 1トークン（中国語）
- 画像1枚 = サイズと詳細度によって異なります

### トークン数の見積もり

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English

# Actual count (for OpenAI models)
import tiktoken
encoder = tiktoken.encoding_for_model("gpt-4o")
tokens = encoder.encode("Your text here")
print(f"Token count: {len(tokens)}")
```

## 使用状況の追跡

### ダッシュボード

[ダッシュボード](https://lemondata.cc/dashboard)で使用状況を確認できます。

- リアルタイムの残高
- モデル別の使用履歴
- 費用の内訳
- APIキーの使用状況

### APIレスポンス

各レスポンスには使用状況の情報が含まれています。

```json
{
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 100,
    "total_tokens": 150
  }
}
```

## 残高不足のアラート

残高が低下した際のアラートを設定します。

1. **ダッシュボード → 設定 → 通知**に移動します
2. しきい値を設定します
3. メール通知を受け取ります

## クレジットの追加

### 支払い方法

- Alipay (支付宝)
- WeChat Pay (微信支付)
- Stripe (Visa, Mastercard)

### 手順

1. [ダッシュボード](https://lemondata.cc/dashboard)にログインします
2. **クレジットを追加**をクリックします
3. 金額と支払い方法を選択します
4. 支払いを完了します

クレジットは支払い確認後、即座に追加されます。

## APIキーの制限

個々のAPIキーに使用制限を設定できます。

1. **ダッシュボード → APIキー**に移動します
2. 編集するキーをクリックします
3. **使用制限（Usage Limit）**を設定します

制限に達すると、そのキーを使用したリクエストは`402 Payment Required`を返します。

## 請求書

ビジネスアカウントでは、請求書を利用できます。

1. **ダッシュボード → 請求**に移動します
2. 取引履歴を表示します
3. 請求書をPDFとしてダウンロードします
