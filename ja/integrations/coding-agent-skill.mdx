## このスキルでできること

コード内で AI 機能（GPT-4、画像生成、音声認識など）を使用したい場合、このスキルは以下のことを行います：

1. ✅ LemonData の数百の API を**自動的に検索**
2. ✅ ニーズに**最適な API を特定**
3. ✅ **完全で実行可能なコードを生成**
4. ✅ **API Key を設定**
5. ✅ **使用例**とベストプラクティスを提供

**要約：自然言語でニーズを伝えるだけで、完全な API 統合コードを取得できます！**

## ステップ 2：API Key の取得

スキルを使用する前に、LemonData の API Key が必要です。

<Tip>
**事前に API Key を設定する必要はありません！** スキルを使用すると、Claude Code が自動的に API Key を尋ね、生成されたコードに含めてくれます。
</Tip>

<Steps>
  <Step title="LemonData にアクセス">
    [lemondata.cc](https://lemondata.cc) に移動します
  </Step>
  <Step title="サインイン">
    アカウントを作成するか、ログインします
  </Step>
  <Step title="API Key を取得">
    [ダッシュボード → API Keys](https://lemondata.cc/dashboard/api) に移動し、新しいキーを作成します
  </Step>
  <Step title="キーをコピー">
    キーは `sk-...` で始まります。安全に保存してください
  </Step>
</Steps>

<Warning>
**重要：**
- すべての LemonData API は同じ API Key を使用します
- キーを他人に教えないでください
- キーを GitHub にコミットしないでください
</Warning>

## ステップ 3：使用を開始する

スキルの使い方は、チャットするのと同じくらい簡単です！

### 例 1：GPT-4 を使用する

**ユーザー：**
```
I want to use GPT-4 in my Python project
```

**Claude Code の動作：**
1. API Key を尋ねる
2. GPT-4 関連の API を検索する
3. 利用可能な API オプションを表示する
4. 完全な Python コードを生成する
5. 使用方法を説明する

**取得結果：**
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

### 例 2：画像を生成する

## 重要な注意事項

### API Key のセキュリティ

<CardGroup cols={2}>
  <Card title="❌ 絶対にしないでください" icon="xmark" color="#ef4444">
    - フロントエンドコードに API Key を直接記述する
    - GitHub に API Key をコミットする
    - API Key を他人に共有する
  </Card>
  <Card title="✅ ベストプラクティス" icon="check" color="#22c55e">
    - 環境変数を使用する
    - Web サイトの場合はバックエンドから API を呼び出す
    - Next.js や Express などのフレームワークを使用する
  </Card>
</CardGroup>

### コスト管理

- 各 API 呼び出しでクレジットが消費されます
- [ダッシュボード](https://lemondata.cc/dashboard)で利用制限を設定してください
- テストには少量のデータを使用してください

### エラー処理

生成されたコードには基本的なエラー処理が含まれていますが、以下の対応を検討してください：
- リトライメカニズムの追加
- エラーのログ記録
- ユーザーフレンドリーなエラーメッセージの提供
