## 概要

LemonDataは、OpenAI互換のあらゆるチャットアプリケーションで動作します。このガイドでは、人気のデスクトップおよびモバイルチャットアプリの設定方法について説明します。

## サポートされているアプリケーション

| アプリケーション | プラットフォーム | 機能 |
|-------------|----------|----------|
| LobeChat | Web, デスクトップ, セルフホスト | マルチプロバイダー、プラグイン、RAG |
| CherryStudio | Windows, macOS, Linux | 豊富なプラグイン、マルチプロバイダー |
| Chatbox | Windows, macOS, Linux | 軽量、高速 |
| BotGem | iOS, macOS | ネイティブなApple体験 |
| TypingMind | Web, デスクトップ | カスタムペルソナ、プラグイン |
| OpenCat | iOS, macOS | ネイティブ、オフラインサポート |
| ChatWise | Windows, macOS | チームコラボレーション |

## 設定手順

### ステップ 1: APIキーの取得

1. [LemonDataダッシュボード](https://lemondata.cc/dashboard)にログインします
2. [APIキー](https://lemondata.cc/dashboard/api)に移動します
3. APIキーを作成してコピーします（形式: `sk-...`）

### ステップ 2: アプリの設定

<Tabs>
  <Tab title="LobeChat">
    **セルフホスト型LobeChatの場合:**

    環境変数を設定します:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **LobeChat Cloudの場合:**
    1. **設定** → **AIサービスプロバイダー**を開きます
    2. **OpenAI**を選択します
    3. LemonDataのAPIキーを入力します
    4. カスタムエンドポイントを設定します: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. **設定** → **モデルプロバイダー**を開きます
    2. **プロバイダーを追加**をクリックします
    3. **OpenAI API互換**を選択します
    4. 以下を入力します:
       - **名前**: LemonData
       - **APIベースURL**: `https://api.lemondata.cc/v1`
       - **APIキー**: `sk-your-lemondata-key`
    5. モデルを追加します: `gpt-4o`, `claude-sonnet-4-5`など
    6. 保存してテストします
  </Tab>
  <Tab title="Chatbox">
    1. **設定** → **AIモデル**を開きます
    2. **OpenAI API互換**を選択します
    3. 以下を入力します:
       - **APIホスト**: `https://api.lemondata.cc/v1`
       - **APIキー**: `sk-your-lemondata-key`
       - **モデル**: `gpt-4o`
    4. **検証**をクリックして接続をテストします
  </Tab>
  <Tab title="BotGem">
    1. **設定** → **プロバイダー**を開きます
    2. **カスタムプロバイダーを追加**をタップします
    3. 以下を入力します:
       - **名前**: LemonData
       - **ベースURL**: `https://api.lemondata.cc/v1`
       - **APIキー**: `sk-your-lemondata-key`
    4. お好みのモデルを追加します
    5. 設定を保存します
  </Tab>
  <Tab title="TypingMind">
    1. **設定**（歯車アイコン）をクリックします
    2. **カスタムエンドポイント**を選択します
    3. 以下を入力します:
       - **エンドポイントURL**: `https://api.lemondata.cc/v1/chat/completions`
       - **APIキー**: `sk-your-lemondata-key`
    4. モデルを設定して保存します
  </Tab>
</Tabs>

### ステップ 3: モデルの追加

プロバイダーを設定した後、使用したいモデルを追加します:

| カテゴリ | モデル |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## 設定リファレンス

| フィールド | 値 |
|-------|-------|
| ベースURL | `https://api.lemondata.cc/v1` |
| APIキー | `sk-your-lemondata-key` |
| APIタイプ | OpenAI互換 |

**JSON設定（サポートしているアプリの場合）:**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## トラブルシューティング

<AccordionGroup>
  <Accordion title="接続エラー">
    - ベースURLが正確に `https://api.lemondata.cc/v1` であることを確認してください
    - 末尾のスラッシュを確認してください（存在する場合は削除してください）
    - インターネット接続とファイアウォールの設定を確認してください
  </Accordion>

  <Accordion title="401 Unauthorized">
    - APIキーを再確認してください
    - [LemonDataダッシュボード](https://lemondata.cc/dashboard/api)でキーが有効であることを確認してください
    - キーに余分なスペースが含まれていないことを確認してください
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - ダッシュボードでアカウント残高を確認してください
    - アカウントにクレジットを追加してください
  </Accordion>

  <Accordion title="404 Model Not Available">
    - モデル名の綴りが正しいか確認してください
    - [lemondata.cc/en/models](https://lemondata.cc/ja/models) でモデルの利用可能性を確認してください
    - 別のモデルを試してください
  </Accordion>

  <Accordion title="レスポンスが遅い">
    - より高速なモデルを試してください（例：`gpt-4o` の代わりに `gpt-4o-mini`）
    - 利用可能な場合は `max_tokens` パラメータを減らしてください
    - ネットワーク接続を確認してください
  </Accordion>
</AccordionGroup>

## ヒント
