---
title: "Obtenir les tarifs"
openapi: "GET /v1/pricing"
description: "Obtenir les informations de tarification pour les modèles"
---

## Paramètres de requête

<ParamField query="model" type="string">
  ID du modèle pour lequel obtenir les tarifs. Si non spécifié, retourne les tarifs pour tous les modèles.
</ParamField>

## Réponse

<ResponseField name="object" type="string">
  Toujours `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Tableau d'objets de tarification de modèle.

  Chaque objet contient :
  - `model` (string) : Identifiant du modèle
  - `pricing` (object) : Informations de tarification
    - `input_per_1m` (string | null) : USD pour 1M de tokens d'entrée
    - `output_per_1m` (string | null) : USD pour 1M de tokens de sortie
    - `per_request` (string | null) : USD par requête (pour les modèles à prix fixe)
    - `is_lock_price` (boolean) : Indique si le tarif est fixe par requête
    - `currency` (string) : Toujours `USD`
  - `provider` (string) : Fournisseur du modèle
</ResponseField>

## Types de tarification

| Type | Description | Exemples de modèles |
|------|-------------|----------------|
| **Basé sur les tokens** | Facturé par tokens d'entrée/sortie | GPT-4o, Claude, Gemini |
| **Par requête** | Prix fixe par requête | DALL-E 3, Sora, Suno |
