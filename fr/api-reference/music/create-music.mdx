---
title: "Créer de la musique"
openapi: "POST /v1/music/generations"
description: "Crée une tâche de génération de musique à l'aide de Suno"
---

## Autorisation

<ParamField header="Authorization" type="string" required>
  Jeton Bearer pour l'authentification API. Format : `Bearer sk-your-api-key`
</ParamField>

Générez de la musique et des paroles à l'aide de l'IA. Il s'agit d'une API asynchrone - vous recevrez un ID de tâche pour interroger l'état d'avancement.

## Corps de la requête

<ParamField body="model" type="string" default="suno_music">
  Modèle à utiliser : `suno_music` pour la génération de musique, `suno_lyrics` pour les paroles uniquement.
</ParamField>

<ParamField body="prompt" type="string" required>
  Description de la musique à générer. Peut inclure le style, l'ambiance, les instruments, etc.
</ParamField>

<ParamField body="title" type="string">
  Titre de la chanson générée.
</ParamField>

<ParamField body="tags" type="string">
  Tags de style (ex : "pop, upbeat, electronic").
</ParamField>

<ParamField body="action" type="string">
  Type de génération : `MUSIC` (par défaut) ou `LYRICS`.
</ParamField>

<ParamField body="mv" type="string">
  Version du modèle à utiliser.
</ParamField>

<ParamField body="continue_clip_id" type="string">
  ID d'un clip précédent à partir duquel continuer.
</ParamField>

<ParamField body="continue_at" type="number">
  Horodatage (en secondes) à partir duquel continuer.
</ParamField>

<ParamField body="user" type="string">
  Un identifiant unique pour l'utilisateur final.
</ParamField>

## Réponse

<ResponseField name="id" type="string">
  ID de la tâche pour interroger le statut.
</ResponseField>

<ResponseField name="status" type="string">
  Statut de la tâche : `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Horodatage Unix de la création de la tâche.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "suno_music",
        "prompt": "An upbeat electronic dance track with heavy bass",
        "title":  "Night Drive",
        "tags":   "electronic, EDM, energetic",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/music/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```
