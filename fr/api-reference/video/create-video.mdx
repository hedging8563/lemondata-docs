---
title: "Créer une vidéo"
api: "POST /v1/videos/generations"
description: "Crée une tâche de génération de vidéo"
---

## Aperçu

La génération de vidéo est asynchrone. Vous soumettez une requête et recevez un ID de tâche, puis vous effectuez un polling pour obtenir le résultat.

## Corps de la requête

<ParamField body="model" type="string" default="kling-video">
  Modèle vidéo (ex. : `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Description textuelle de la vidéo à générer.
</ParamField>

<ParamField body="image_url" type="string">
  Image de départ optionnelle pour la génération image-vers-vidéo.
</ParamField>

<ParamField body="duration" type="integer">
  Durée de la vidéo en secondes (dépend du modèle).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Format d'image (ex. : `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Résolution vidéo (ex. : `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Images par seconde (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Ce qu'il faut éviter dans la génération de la vidéo.
</ParamField>

<ParamField body="seed" type="integer">
  Graine aléatoire (`seed`) pour une génération reproductible.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Échelle de guidage sans classificateur (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Intensité du mouvement (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL de l'image de la frame de départ.
</ParamField>

<ParamField body="end_image" type="string">
  URL de l'image de la frame de fin.
</ParamField>

<ParamField body="user" type="string">
  Un identifiant unique pour l'utilisateur final.
</ParamField>

## Réponse

<ResponseField name="task_id" type="string">
  Identifiant de tâche unique pour le polling.
</ResponseField>

<ResponseField name="status" type="string">
  Statut initial : `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Modèle
