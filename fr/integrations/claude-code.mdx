## Aperçu

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) est l'outil CLI officiel d'Anthropic pour le codage assisté par IA. Vous pouvez le configurer pour utiliser LemonData comme fournisseur d'API.

## Configuration

### Variables d'environnement

Définissez les variables d'environnement suivantes :

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Ajoutez-les à votre profil de shell (`~/.bashrc`, `~/.zshrc`, etc.) pour la persistance.

### Fichier de configuration

Alternativement, créez ou modifiez `~/.claude/settings.json` :

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Utilisation

Après la configuration, utilisez Claude Code normalement :

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Modèles disponibles

LemonData prend en charge tous les modèles Claude :

| Modèle | Description |
|-------|-------------|
| `claude-opus-4-5` | Le plus performant, raisonnement complexe |
| `claude-sonnet-4-5` | Performance équilibrée, excellent pour le codage |
| `claude-haiku-4-5` | Le plus rapide, rentable |

## Sélection du modèle

Spécifiez un modèle avec le flag `--model` :

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Ou définissez une valeur par défaut dans votre configuration :

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Pensée étendue (Extended Thinking)

Pour les tâches de raisonnement complexes, utilisez des modèles avec la pensée étendue :

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Dépannage

<AccordionGroup>
  <Accordion title="Erreurs de connexion">
    Vérifiez que l'URL de base est exactement `https://api.lemondata.cc` (pas de suffixe `/v1` pour le SDK Anthropic).
  </Accordion>

  <Accordion title="Clé API invalide">
    Assurez-vous que votre clé API LemonData commence par `sk-` et dispose d'un solde suffisant.
  </Accordion>

  <Accordion title="Modèle non trouvé">
    Vérifiez que le nom du modèle est correct. Utilisez le format `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
