## Aperçu

[Cursor](https://cursor.sh) est un éditeur de code propulsé par l'IA. Vous pouvez utiliser LemonData comme fournisseur d'API personnalisé pour accéder à plus de 300 modèles.

## Configuration

1. Ouvrez les paramètres de Cursor (`Cmd/Ctrl + ,`)
2. Accédez à **Models** dans la barre latérale gauche
3. Faites défiler jusqu'à la section **OpenAI API Key**
4. Saisissez votre clé API LemonData
5. Activez **Override OpenAI Base URL**
6. Définissez l'URL de base sur :

```
https://api.lemondata.cc/v1
```

## Sélection de modèles

Après la configuration, vous pouvez sélectionner n'importe quel modèle LemonData dans Cursor :

### Modèles recommandés pour le codage

| Modèle | Idéal pour |
|-------|----------|
| `claude-opus-4-5` | Codage complexe, architecture |
| `claude-sonnet-4-5` | Codage général, débogage |
| `gpt-4o` | Génération de code, explications |
| `deepseek-r1` | Tâches nécessitant un raisonnement poussé |
| `gpt-4o-mini` | Complétions rapides, tâches simples |

## Fonctionnalités

### Chat

Utilisez le panneau de chat IA (`Cmd/Ctrl + L`) avec n'importe quel modèle LemonData :

- Explications de code
- Correction de bugs
- Suggestions de refactorisation
- Génération de documentation

### Composer

La fonctionnalité Composer (`Cmd/Ctrl + I`) fonctionne avec les modèles LemonData pour :

- Modifications multi-fichiers
- Implémentation de fonctionnalités
- Migrations de code

### Complétion par tabulation

Pour les complétions par tabulation, nous recommandons d'utiliser des modèles rapides :

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
