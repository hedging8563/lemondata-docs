## Aperçu

LemonData fonctionne avec n'importe quelle application de chat compatible OpenAI. Ce guide couvre la configuration des applications de chat populaires pour bureau et mobile.

## Applications prises en charge

| Application | Plateforme | Fonctionnalités |
|-------------|----------|----------|
| LobeChat | Web, Bureau, Auto-hébergé | Multi-fournisseur, plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Plugins riches, fournisseurs multiples |
| Chatbox | Windows, macOS, Linux | Léger, rapide |
| BotGem | iOS, macOS | Expérience Apple native |
| TypingMind | Web, Bureau | Personas personnalisés, plugins |
| OpenCat | iOS, macOS | Natif, support hors ligne |
| ChatWise | Windows, macOS | Collaboration d'équipe |

## Étapes de configuration

### Étape 1 : Obtenir votre clé API

1. Connectez-vous au [Tableau de bord LemonData](https://lemondata.cc/dashboard)
2. Accédez aux [Clés API](https://lemondata.cc/dashboard/api)
3. Créez et copiez votre clé API (format : `sk-...`)

### Étape 2 : Configurer votre application

<Tabs>
  <Tab title="LobeChat">
    **Pour LobeChat auto-hébergé :**

    Définissez les variables d'environnement :
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Pour LobeChat Cloud :**
    1. Ouvrez **Settings** → **AI Service Provider**
    2. Sélectionnez **OpenAI**
    3. Entrez votre clé API LemonData
    4. Définissez le point de terminaison personnalisé : `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Ouvrez **Settings** → **Model Provider**
    2. Cliquez sur **Add Provider**
    3. Sélectionnez **OpenAI API Compatible**
    4. Entrez :
       - **Name** : LemonData
       - **API Base URL** : `https://api.lemondata.cc/v1`
       - **API Key** : `sk-your-lemondata-key`
    5. Ajoutez des modèles : `gpt-4o`, `claude-sonnet-4-5`, etc.
    6. Enregistrez et testez
  </Tab>
  <Tab title="Chatbox">
    1. Ouvrez **Settings** → **AI Model**
    2. Sélectionnez **OpenAI API Compatible**
    3. Entrez :
       - **API Host** : `https://api.lemondata.cc/v1`
       - **API Key** : `sk-your-lemondata-key`
       - **Model** : `gpt-4o`
    4. Cliquez sur **Verify** pour tester la connexion
  </Tab>
  <Tab title="BotGem">
    1. Ouvrez **Settings** → **Providers**
    2. Appuyez sur **Add Custom Provider**
    3. Entrez :
       - **Name** : LemonData
       - **Base URL** : `https://api.lemondata.cc/v1`
       - **API Key** : `sk-your-lemondata-key`
    4. Ajoutez vos modèles préférés
    5. Enregistrez la configuration
  </Tab>
  <Tab title="TypingMind">
    1. Cliquez sur **Settings** (icône d'engrenage)
    2. Sélectionnez **Custom Endpoint**
    3. Entrez :
       - **Endpoint URL** : `https://api.lemondata.cc/v1/chat/completions`
       - **API Key** : `sk-your-lemondata-key`
    4. Configurez les modèles et enregistrez
  </Tab>
</Tabs>

### Étape 3 : Ajouter des modèles

Après avoir configuré le fournisseur, ajoutez les modèles que vous souhaitez utiliser :

| Catégorie | Modèles |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Référence de configuration

| Champ | Valeur |
|-------|-------|
| URL de base | `https://api.lemondata.cc/v1` |
| Clé API | `sk-your-lemondata-key` |
| Type d'API | Compatible OpenAI |

**Configuration JSON (pour les applications qui le prennent en charge) :**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Dépannage

<AccordionGroup>
  <Accordion title="Erreur de connexion">
    - Vérifiez que l'URL de base est exactement `https://api.lemondata.cc/v1`
    - Vérifiez les barres obliques de fin (supprimez-les si présentes)
    - Vérifiez la connectivité Internet et les paramètres du pare-feu
  </Accordion>

  <Accordion title="401 Non autorisé">
    - Vérifiez à nouveau votre clé API
    - Vérifiez que la clé est active dans le [Tableau de bord LemonData](https://lemondata.cc/dashboard/api)
    - Assurez-vous qu'il n'y a pas d'espaces supplémentaires dans la clé
  </Accordion>

  <Accordion title="402 Solde insuffisant">
    - Vérifiez le solde de votre compte dans le tableau de bord
    - Ajoutez des crédits à votre compte
  </Accordion>

  <Accordion title="404 Modèle non disponible">
    - Vérifiez que le nom du modèle est correctement orthographié
    - Vérifiez la disponibilité du modèle sur [lemondata.cc/en/models](https://lemondata.cc/fr/models)
    - Essayez un autre modèle
  </Accordion>

  <Accordion title="Réponses lentes">
    - Essayez un modèle plus rapide (par ex., `gpt-4o-mini` au lieu de `gpt-4o`)
    - Réduisez le paramètre `max_tokens` si disponible
    - Vérifiez votre connexion réseau
  </Accordion>
</AccordionGroup>

## Conseils

<AccordionGroup>
  <Accordion title="Utilisez les modèles appropriés">
    Adaptez le modèle à votre tâche - utilisez GPT-4o-mini pour les discussions rapides, Claude ou GPT-4o pour les tâches complexes.
  </Accordion>

    La plupart des applications permettent d'enregistrer des prompts ou des personas. Créez des modèles pour les tâches courantes.
  </Accordion>

  <Accordion title="Surveillez l'utilisation">
    Consultez votre utilisation dans le tableau de bord LemonData pour suivre les coûts et gérer votre budget.
  </Accordion>
</AccordionGroup>
