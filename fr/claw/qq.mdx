Connectez QQ (l'application de messagerie la plus populaire en Chine) à votre instance Claw en utilisant l'API officielle QQ Open Platform Bot.

## Prérequis

- Un compte [QQ](https://im.qq.com/)
- Accès à [QQ Open Platform](https://q.qq.com/) (compte développeur)
- Une instance Claw ([créez-en une ici](https://lemondata.cc/dashboard/claw))

## Guide de configuration

<Steps>
  <Step title="Créer une application QQ Bot">
    1. Allez sur [QQ Open Platform](https://q.qq.com/)
    2. Connectez-vous et allez dans la section **Bot**
    3. Cliquez sur **Create Bot**
    4. Remplissez les informations du bot (nom, avatar, description)
    5. Soumettez pour examen
  </Step>
  <Step title="Obtenir les identifiants">
    1. Une fois approuvé, allez dans les **Development Settings** de votre bot
    2. Copiez l'**AppID**
    3. Copiez l'**AppSecret** (Client Secret)
  </Step>
  <Step title="Configurer le Sandbox (pour les tests)">
    1. Dans les paramètres du bot, allez dans **Sandbox Configuration**
    2. Ajoutez votre numéro QQ à la whitelist du sandbox
    3. Cela vous permet de tester avant la mise en ligne

    <Tip>
    Commencez par le mode sandbox pour vérifier que tout fonctionne avant de soumettre pour examen de production.
    </Tip>
  </Step>
  <Step title="Configurer dans le tableau de bord LemonData">
    1. Allez sur votre [Claw Dashboard](https://lemondata.cc/dashboard/claw)
    2. Cliquez sur l'**icône des paramètres** de votre instance
    3. Ouvrez l'onglet **IM Integrations**
    4. Développez la section **QQ**
    5. Collez votre **AppID** et votre **AppSecret**
    6. Cliquez sur **Save**
  </Step>
  <Step title="Commencer à discuter">
    1. Dans QQ, recherchez votre bot ou ajoutez-le comme ami
    2. Envoyez un message
    3. Le premier utilisateur est **automatiquement associé** en tant que propriétaire — aucun code n'est nécessaire !
    4. Commencez à discuter avec l'IA !
  </Step>
</Steps>

## Types de messages pris en charge

| Type | Description |
|------|-------------|
| **C2C (Privé)** | Messages directs au bot |
| **Mention @ de groupe** | Mentionnez @ le bot dans un groupe QQ |
| **Canal** | Messages dans les canaux QQ (频道) |

## Dépannage

<AccordionGroup>
  <Accordion title="Le bot ne répond pas">
    - Vérifiez que l'AppID et l'AppSecret sont corrects
    - Vérifiez que votre numéro QQ est dans la whitelist du sandbox (si vous testez)
    - Assurez-vous que l'application du bot est approuvée
  </Accordion>
  <Accordion title="Erreur 'invalid appid'">
    - Vérifiez l'AppID — il doit s'agir d'une chaîne numérique
    - Assurez-vous d'utiliser le bon AppSecret (pas le token)
  </Accordion>
</AccordionGroup>
