<Note>
**Auto-hébergé** signifie que vous exécutez OpenClaw directement sur votre propre VPS ou machine. Vous ne payez que l'utilisation des modèles AI via votre solde LemonData — aucun frais d'instance.
</Note>

## Installation rapide

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Non interactif
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

L'installateur vous demandera votre **LemonData API Key**. Obtenez-en une depuis votre [Dashboard](https://lemondata.cc/dashboard).

## Ce qui est installé

<Steps>
  <Step title="Node.js 22+">
    Installé automatiquement si absent (via NodeSource sur Linux, Homebrew sur macOS)
  </Step>
  <Step title="OpenClaw">
    Package OpenClaw officiel depuis npm, préconfiguré avec LemonData API
  </Step>
  <Step title="15 modèles AI">
    Claude, GPT-5, Gemini, DeepSeek et plus — prêts à l'emploi
  </Step>
  <Step title="Plugins IM">
    Plugins DingTalk, WeCom et QQ préinstallés
  </Step>
  <Step title="Service système">
    Démarrage automatique au boot (systemd sur Linux, launchd sur macOS)
  </Step>
</Steps>

## Commandes de gestion

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # Vérifier le statut
    journalctl --user -u openclaw -f    # Voir les logs
    systemctl --user restart openclaw   # Redémarrer
    systemctl --user stop openclaw      # Arrêter
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # Voir les logs
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # Redémarrer
    launchctl stop cc.lemondata.openclaw  # Arrêter
    ```
  </Tab>
</Tabs>

## Configuration

Emplacement du fichier de configuration : `~/.openclaw/openclaw.json`

Après modification, redémarrez le service pour appliquer les changements.

### Changer le modèle par défaut

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Préfixes de modèles disponibles :
- `lemondata-claude/` — Modèles Claude (anthropic-messages API)
- `lemondata-minimax/` — Modèles MiniMax (anthropic-messages API)
- `lemondata/` — Tous les autres modèles (openai-completions API)

## Mise à jour

```bash
npm update -g openclaw
# Puis redémarrer le service
systemctl --user restart openclaw  # Linux
# ou
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Désinstallation

```bash
# Arrêter et désactiver le service
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# Supprimer OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
