## Étape 1 : Obtenez votre clé API

<Steps>
  <Step title="Créez un compte">
    Inscrivez-vous sur [lemondata.cc](https://lemondata.cc). Connexion possible avec e-mail, Gmail ou GitHub.
  </Step>
  <Step title="Ajoutez des crédits">
    Accédez au dashboard et ajoutez des crédits à votre compte. Tarification pay-as-you-go sans minimum.
  </Step>
  <Step title="Créez une clé API">
    Allez dans **Dashboard → API Keys** et créez une nouvelle clé. Copiez-la en toute sécurité - elle ne s'affiche qu'une seule fois.
  </Step>
</Steps>

<Warning>
  Gardez votre clé API en sécurité. Ne l'exposez jamais dans le code côté client ou dans des dépôts publics.
</Warning>

## Étape 3 : Effectuez votre première requête

<CodeGroup>

```bash cURL
curl https://api.lemondata.cc/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "What is the capital of France?"}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is the capital of France?"}
    ]
)

print(response.choices[0].message.content)
# Output: The capital of France is Paris.
```
