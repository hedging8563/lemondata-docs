title: "Créer une traduction"
openapi: "POST /v1/audio/translations"
description: "Traduire de l'audio en texte anglais"
---

## Aperçu

Traduisez de l'audio depuis n'importe quelle langue supportée en texte anglais. Contrairement à la transcription, ce point de terminaison produit toujours du texte en anglais, quelle que soit la langue d'entrée.

## Corps de la requête

<ParamField body="file" type="file" required>
  Le fichier audio à traduire. Formats supportés : `flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`. La taille maximale du fichier est de 25 Mo.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  Le modèle à utiliser. Actuellement, seul `whisper-1` est supporté.
</ParamField>

<ParamField body="prompt" type="string">
  Texte optionnel pour guider le style du modèle ou continuer un segment précédent. Doit être en anglais.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  Le format de sortie. Options : `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number">
  La température d'échantillonnage, entre 0 et 1. Des valeurs plus élevées (comme 0.8) produiront une sortie plus aléatoire, tandis que des valeurs plus basses (comme 0.2) la rendront plus focalisée et déterministe.
</ParamField>

## Réponse

<ResponseField name="text" type="string">
  Le texte anglais traduit.
</ResponseField>

Pour le format `verbose_json`, la réponse contient également :

<ResponseField name="language" type="string">
  La langue détectée de l'audio d'entrée.
</ResponseField>

<ResponseField name="duration" type="number">
  La durée de l'audio d'entrée (en secondes).
</ResponseField>

<ResponseField name="segments" type="array">
  Segments de texte traduit avec horodatage.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/translations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -F "file=@german_audio.mp3" \
  -F "model=whisper-1"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

with open("german_audio.mp3", "rb") as audio_file:
    response = client.audio.translations.create(
        model="whisper-1",
        file=audio_file
    )

print(response.text)
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/translations');

$file = new CURLFile('german_audio.mp3', 'audio/mpeg', 'german_audio.mp3');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => [
        'file' => $file,
        'model' => 'whisper-1'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['text'];
```

## Traduction vs Transcription

| Caractéristique | Traduction | Transcription |
|---------|-------------|---------------|
| Langue de sortie | Toujours en anglais | Identique à l'entrée |
| Cas d'utilisation | Convertir de l'audio étranger en anglais | Conserver la langue d'origine |
| Paramètre de langue | N/A | Indice optionnel |

<Note>
  Le point de terminaison de traduction détecte automatiquement la langue source et la traduit en anglais. Le paramètre `language` utilisé dans les transcriptions sera ignoré.
</Note>
