---
title: "Réorganiser les documents"
openapi: "POST /v1/rerank"
description: "Réorganiser les documents en fonction de leur pertinence par rapport à une requête"
---

Réorganisez les documents à l'aide d'un modèle de similitude sémantique. Idéal pour optimiser les résultats de recherche et les applications RAG.

## Corps de la requête

<ParamField body="model" type="string" required>
  L'ID du modèle de rerank à utiliser (par exemple : `BAAI/bge-reranker-v2-m3`, `qwen3-rerank`).
</ParamField>

<ParamField body="query" type="string" required>
  La requête utilisée pour classer les documents.
</ParamField>

<ParamField body="documents" type="array" required>
  La liste des documents à réorganiser (chaînes de caractères).
</ParamField>

<ParamField body="top_n" type="integer">
  Le nombre de premiers résultats à retourner. Par défaut, tous les documents.
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  Indique s'il faut inclure le texte original du document dans la réponse.
</ParamField>

## Réponse

<ResponseField name="results" type="array">
  Une liste de documents triés avec leurs scores.

  Chaque résultat contient :
  - `index` (integer) : Index du document original
  - `relevance_score` (number) : Score de pertinence (0-1)
  - `document` (string) : Texte original (si `return_documents=true`)
</ResponseField>

<ResponseField name="model" type="string">
  Le modèle utilisé pour le rerank.
</ResponseField>

<ResponseField name="usage" type="object">
  Statistiques d'utilisation des tokens.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/rerank" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "BAAI/bge-reranker-v2-m3",
    "query": "What is machine learning?",
    "documents": [
      "Machine learning is a subset of AI",
      "The weather is nice today",
      "Deep learning uses neural networks"
    ],
    "top_n": 2,
    "return_documents": true
  }'
```
