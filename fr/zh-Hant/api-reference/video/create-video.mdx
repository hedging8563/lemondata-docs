---

## Aperçu

La génération de vidéo est asynchrone. Vous soumettez une requête et recevez un task ID, puis vous effectuez un polling pour obtenir le résultat.

## Corps de la requête

<ParamField body="model" type="string" default="kling-video">
  Modèle vidéo (par exemple : `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Description textuelle de la vidéo à générer.
</ParamField>

<ParamField body="image_url" type="string">
  Image de départ optionnelle pour la génération image-to-video.
</ParamField>

<ParamField body="duration" type="integer">
  Durée de la vidéo, en secondes (dépend du modèle).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Ratio d'aspect (par exemple : `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Résolution de la vidéo (par exemple : `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Nombre d'images par seconde (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Contenu à éviter dans la génération de la vidéo.
</ParamField>

<ParamField body="seed" type="integer">
  Graine aléatoire pour des générations reproductibles.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Échelle de guidage sans classificateur (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Intensité du mouvement (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL de l'image de la frame de début.
</ParamField>

<ParamField body="end_image" type="string">
  URL de l'image de la frame de fin.
</ParamField>

<ParamField body="user" type="string">
  Identifiant unique de l'utilisateur final.
</ParamField>

## Réponse

<ResponseField name="task_id" type="string">
  Identifiant de tâche unique utilisé pour le polling.
</ResponseField>

<ResponseField name="status" type="string">
  Statut initial : `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Modèle utilisé.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

## Image-to-Video

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```

## Modèles disponibles

| Modèle | Type | Description |
|-------|------|-------------|
| `kling-video` | T2V/I2V | Mouvements réalistes, plusieurs versions (v1.0-v2.6) |
| `veo3.1` | T2V | Dernier modèle de Google |
| `veo3.1-pro` | T2V | Qualité professionnelle |
| `minimax/video-01` | T2V | Sens du rythme élevé et stabilité |
| `wan2.6-i2v` | I2V | Image-to-Video |
