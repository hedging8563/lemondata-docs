## Aperçu

[Cursor](https://cursor.sh) est un éditeur de code propulsé par l'IA. Vous pouvez utiliser LemonData comme fournisseur d'API personnalisé pour accéder à plus de 300 modèles.

## Configuration

1. Ouvrez les paramètres de Cursor (`Cmd/Ctrl + ,`)
2. Naviguez vers **Models** dans la barre latérale gauche
3. Faites défiler jusqu'à la section **OpenAI API Key**
4. Entrez votre clé API LemonData
5. Activez **Override OpenAI Base URL**
6. Définissez l'URL de base sur :

```
https://api.lemondata.cc/v1
```

## Sélection de modèles

Une fois configuré, vous pouvez sélectionner n'importe quel modèle LemonData dans Cursor :

### Modèles de codage recommandés

| Modèle | Idéal pour |
|-------|----------|
| `claude-opus-4-5` | Codage complexe, conception d'architecture |
| `claude-sonnet-4-5` | Codage général, débogage |
| `gpt-4o` | Génération de code, explication |
| `deepseek-r1` | Tâches de raisonnement intensif |
| `gpt-4o-mini` | Complétion rapide, tâches simples |

## Fonctionnalités

### Chat

Utilisez le panneau de chat IA (`Cmd/Ctrl + L`) avec n'importe quel modèle LemonData :

- Explication de code
- Correction de bugs
- Suggestions de refactorisation
- Génération de documentation

### Composer

La fonctionnalité Composer (`Cmd/Ctrl + I`) peut être utilisée avec les modèles LemonData pour :

- Édition multi-fichiers
- Implémentation de fonctionnalités
- Migration de code

### Complétion Tab

Pour la complétion Tab, nous recommandons d'utiliser des modèles rapides :

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
