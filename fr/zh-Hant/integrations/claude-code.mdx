## Présentation

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) est l'outil CLI d'assistance à la programmation par IA officiel d'Anthropic. Vous pouvez le configurer pour utiliser LemonData comme fournisseur d'API.

## Configuration

### Variables d'environnement

Définissez les variables d'environnement suivantes :

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Ajoutez ces lignes à votre fichier de configuration shell (`~/.bashrc`, `~/.zshrc`, etc.) pour les rendre permanentes.

### Fichier de configuration

Alternativement, créez ou modifiez `~/.claude/settings.json` :

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Utilisation

Une fois la configuration terminée, vous pouvez utiliser Claude Code normalement :

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Modèles disponibles

LemonData prend en charge tous les modèles Claude :

| Modèle | Description |
|-------|-------------|
| `claude-opus-4-5` | Le modèle le plus puissant, idéal pour le raisonnement complexe |
| `claude-sonnet-4-5` | Équilibre des performances, excellent pour la programmation |
| `claude-haiku-4-5` | Le plus rapide et le plus rentable |

## Sélection du modèle

Utilisez le flag `--model` pour spécifier un modèle :

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Ou définissez une valeur par défaut dans votre configuration :

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Réflexion étendue

Pour les tâches de raisonnement complexes, utilisez des modèles dotés de capacités de réflexion étendue :

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Dépannage

<AccordionGroup>
  <Accordion title="Erreur de connexion">
    Vérifiez que l'URL de base est exactement `https://api.lemondata.cc` (le SDK Anthropic n'a pas besoin du suffixe `/v1`).
  </Accordion>

  <Accordion title="Clé API invalide">
    Assurez-vous que votre clé API LemonData commence par `sk-` et que votre solde est suffisant.
  </Accordion>

  <Accordion title="Modèle introuvable">
    Vérifiez que le nom du modèle est correct. Utilisez le format `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
