## Aperçu

LemonData fonctionne avec n'importe quelle application de chat compatible avec OpenAI. Ce guide couvre la configuration des applications de chat populaires pour bureau et mobile.

## Applications supportées

| Application | Plateforme | Fonctionnalités |
|-------------|----------|----------|
| LobeChat | Web, Bureau, Auto-hébergé | Multi-fournisseurs, Plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Plugins riches, multiples fournisseurs |
| Chatbox | Windows, macOS, Linux | Léger, rapide |
| BotGem | iOS, macOS | Expérience Apple native |
| TypingMind | Web, Bureau | Personnalités personnalisées, Plugins |
| OpenCat | iOS, macOS | Natif, support hors ligne |
| ChatWise | Windows, macOS | Collaboration d'équipe |

## Étapes de configuration

### Étape 1 : Obtenir votre API Key

1. Connectez-vous au [Console LemonData](https://lemondata.cc/dashboard)
2. Accédez à [API Keys](https://lemondata.cc/dashboard/api)
3. Créez et copiez votre API Key (format : `sk-...`)

### Étape 2 : Configurer votre application

<Tabs>
  <Tab title="LobeChat">
    **Pour LobeChat auto-hébergé :**

    Configurez les variables d'environnement :
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Pour LobeChat Cloud :**
    1. Ouvrez **Settings** → **AI Service Provider**
    2. Sélectionnez **OpenAI**
    3. Entrez votre API Key LemonData
    4. Configurez le point de terminaison personnalisé : `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Ouvrez **Settings** → **Model Provider**
    2. Cliquez sur **Add Provider**
    3. Sélectionnez **OpenAI API Compatible**
    4. Entrez :
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Ajoutez des modèles : `gpt-4o`, `claude-sonnet-4-5`, etc.
    6. Enregistrez et testez
  </Tab>
