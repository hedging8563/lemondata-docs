## Overview

LemonData works with any OpenAI-compatible chat application. This guide covers configuration for popular desktop and mobile chat apps.

## Supported Applications

| Application | Platform | Features |
|-------------|----------|----------|
| LobeChat | Web, Desktop, Self-hosted | Multi-provider, plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Rich plugins, multiple providers |
| Chatbox | Windows, macOS, Linux | Lightweight, fast |
| BotGem | iOS, macOS | Native Apple experience |
| TypingMind | Web, Desktop | Custom personas, plugins |
| OpenCat | iOS, macOS | Native, offline support |
| ChatWise | Windows, macOS | Team collaboration |

## Configuration Steps

### Step 1: Get Your API Key

1. Log into [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navigate to [API Keys](https://lemondata.cc/dashboard/api)
3. Create and copy your API key (format: `sk-...`)

### Step 2: Configure Your App

<Tabs>
  <Tab title="LobeChat">
    **For Self-hosted LobeChat:**

    Set environment variables:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **For LobeChat Cloud:**
    1. Open **Settings** → **AI Service Provider**
    2. Select **OpenAI**
    3. Enter your LemonData API key
    4. Set custom endpoint: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Open **Settings** → **Model Provider**
    2. Click **Add Provider**
    3. Select **OpenAI API Compatible**
    4. Enter:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Add models: `gpt-4o`, `claude-sonnet-4-5`, etc.
    6. Save and test
  </Tab>
  <Tab title="Chatbox">
    1. Open **Settings** → **AI Model**
    2. Select **OpenAI API Compatible**
    3. Enter:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Click **Verify** to test connection
  </Tab>
  <Tab title="BotGem">
    1. Open **Settings** → **Providers**
    2. Tap **Add Custom Provider**
    3. Enter:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Add your preferred models
    5. Save configuration
  </Tab>
  <Tab title="TypingMind">
    1. Click **Settings** (gear icon)
    2. Select **Custom Endpoint**
    3. Enter:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Configure models and save
  </Tab>
</Tabs>

### Step 3: Add Models

After configuring the provider, add the models you want to use:

| Category | Models |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Configuration Reference

| Field | Value |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API Type | OpenAI Compatible |

**JSON Configuration (for apps that support it):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection Error">
    - Verify Base URL is exactly `https://api.lemondata.cc/v1`
    - Check for trailing slashes (remove if present)
    - Verify internet connectivity and firewall settings
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Double-check your API key
    - Verify the key is active in [LemonData Dashboard](https://lemondata.cc/dashboard/api)
    - Ensure no extra spaces in the key
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - Check your account balance in the dashboard
    - Add credits to your account
  </Accordion>

  <Accordion title="404 Model Not Available">
    - Verify model name is spelled correctly
    - Check model availability at [lemondata.cc/en/models](https://lemondata.cc/en/models)
    - Try a different model
  </Accordion>

  <Accordion title="Slow Responses">
    - Try a faster model (e.g., `gpt-4o-mini` instead of `gpt-4o`)
    - Reduce `max_tokens` parameter if available
    - Check your network connection
  </Accordion>
</AccordionGroup>
