## Overview

Google Gemini CLI is a command-line tool for interacting with Gemini models. LemonData provides a compatible endpoint that allows you to use Gemini CLI with access to 300+ models.

## System Requirements

- **Node.js**: Version 20.0+
- **OS**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, or Debian 10+

## Installation

```bash
npm install -g @google/gemini-cli
```

Verify installation:

```bash
gemini --version
```

## Configuration

### Step 1: Get Your API Key

1. Log into [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navigate to [API Keys](https://lemondata.cc/dashboard/api)
3. Create and copy your API key (format: `sk-...`)

### Step 2: Set Environment Variables

**Temporary (current session):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**Permanent configuration:**

Add to your shell configuration file:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
Restart your terminal after configuration for changes to take effect.
</Warning>

## Basic Usage

Start Gemini CLI from your project directory:

```bash
cd your-project
gemini
```

On first run, you'll:
1. Choose a theme
2. Confirm safety notice
3. Trust the working directory

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection Error">
    - Verify `GOOGLE_GEMINI_BASE_URL` is set to `https://api.lemondata.cc`
    - Note: No `/v1` suffix for Gemini endpoint
    - Check network connectivity
  </Accordion>

  <Accordion title="Authentication Failed">
    - Verify `GEMINI_API_KEY` environment variable is set
    - Check that the key starts with `sk-`
    - Ensure the key is active in LemonData dashboard
  </Accordion>

  <Accordion title="Model Not Available">
    - Check Gemini models availability at [lemondata.cc/en/models](https://lemondata.cc/en/models)
    - Try a different Gemini model variant
  </Accordion>
</AccordionGroup>
