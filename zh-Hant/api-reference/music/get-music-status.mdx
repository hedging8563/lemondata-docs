---
title: "獲取音樂狀態"
api: "GET /v1/music/generations/{id}"
description: "獲取音樂生成任務的狀態與結果"
---

輪詢此端點以檢查音樂生成任務的狀態。

## 路徑參數

<ParamField path="id" type="string" required>
  音樂生成任務 ID。
</ParamField>

## 回應

<ResponseField name="id" type="string">
  任務 ID。
</ResponseField>

<ResponseField name="status" type="string">
  任務狀態：`pending`、`processing`、`completed` 或 `failed`。
</ResponseField>

<ResponseField name="progress" type="integer">
  進度百分比 (0-100)。
</ResponseField>

<ResponseField name="audio_url" type="string">
  音訊檔案下載連結（完成時）。
</ResponseField>

<ResponseField name="video_url" type="string">
  帶有視覺化效果的影片版本連結（完成時）。
</ResponseField>

<ResponseField name="title" type="string">
  生成的歌曲標題。
</ResponseField>

<ResponseField name="lyrics" type="string">
  生成或提供的歌詞。
</ResponseField>

<ResponseField name="error" type="string">
  錯誤訊息（失敗時）。
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1/music/generations/music_abc123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "music_abc123"

while True:
    response = requests.get(
        f"https://api.lemondata.cc/v1/music/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"}
    )
    result = response.json()

    if result["status"] == "completed":
        print(f"Audio URL: {result['audio_url']}")
        break
    elif result["status"] == "failed":
        print(f"Error: {result['error']}")
        break

    time.sleep(5)  # Poll every 5 seconds
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
    "time"
)

func main() {
    taskID := "music_abc123"

    for {
        req, _ := http.NewRequest("GET",
            fmt.Sprintf("https://api.lemondata.cc/v1/music/generations/%s", taskID), nil)
        req.Header.Set("Authorization", "Bearer sk-your-api-key")

        client := &http.Client{}
        resp, _ := client.Do(req)

        var result map[string]interface{}
        json.NewDecoder(resp.Body).Decode(&result)
        resp.Body.Close()

        if result["status"] == "completed" {
            fmt.Printf("Audio URL: %s\n", result["audio_url"])
            break
        } else if result["status"] == "failed" {
            fmt.Printf("Error: %s\n", result["error"])
            break
        }

        time.Sleep(5 * time.Second)
    }
}
```

```php PHP

    if ($result['status'] === 'completed') {
        echo "Audio URL: {$result['audio_url']}\n";
        break;
    } elseif ($result['status'] === 'failed') {
        echo "Error: {$result['error']}\n";
        break;
    }

    sleep(5);
}
```
