## 概覽

影片生成是異步的。您提交請求並收到一個任務 ID，然後輪詢結果。

## Request Body

<ParamField body="model" type="string" default="sora-2">
  影片模型 ID。請參閱 [Video Generation Guide](/guides/video-generation) 以獲取涵蓋 11 個供應商、超過 80 個模型的完整列表。
</ParamField>

<ParamField body="prompt" type="string" required>
  要生成的影片文字描述。
</ParamField>

<ParamField body="image_url" type="string">
  用於圖生影片 (I2V) 生成的起始圖片 URL。
</ParamField>

<ParamField body="image" type="string">
  帶有數據 URL 前綴的 Base64 編碼圖片（例如 `data:image/jpeg;base64,...`）。圖生影片 (I2V) 生成中 `image_url` 的替代方案。
</ParamField>

<ParamField body="duration" type="integer">
  影片時長（以秒為單位，取決於模型）。
</ParamField>

<ParamField body="aspect_ratio" type="string">
  長寬比（例如 `16:9`、`9:16`、`1:1`）。
</ParamField>

<ParamField body="resolution" type="string">
  影片解析度（例如 `1080p`、`720p`）。
</ParamField>

<ParamField body="fps" type="integer">
  每秒幀數 (1-120)。
</ParamField>

<ParamField body="negative_prompt" type="string">
  影片生成中要避免的內容。
</ParamField>

<ParamField body="seed" type="integer">
  用於可重複生成的隨機種子。
</ParamField>

<ParamField body="cfg_scale" type="number">
  無分類器指導比例 (0-20)。
</ParamField>

<ParamField body="motion_strength" type="number">
  動作強度 (0-1)。
</ParamField>

<ParamField body="start_image" type="string">
  起始幀圖片的 URL。
</ParamField>

<ParamField body="end_image" type="string">
  結束幀圖片的 URL。
</ParamField>

<ParamField body="user" type="string">
  終端用戶的唯一識別碼。
</ParamField>

## Response

<ResponseField name="task_id" type="string">
  用於輪詢的唯一任務識別碼。
</ResponseField>

<ResponseField name="status" type="string">
  初始狀態：`pending`。
</ResponseField>

<ResponseField name="model" type="string">
  使用的模型。
</ResponseField>

## 圖生影片

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```

## 關鍵幀控制

使用 `start_image` 和 `end_image` 來控制第一幀和最後一幀（由 Seedance 2.0 支援）：

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "seedance-2-0",
        "prompt": "Smooth transition from day to night",
        "start_image": "https://example.com/day.jpg",
        "end_image": "https://example.com/night.jpg",
        "duration": 5
    }
)
```
