Telegram 是**最簡單**的連接平台。您只需要從 BotFather 獲取一個 bot token，即可在 2 分鐘內開始與 AI 聊天。

## 前提條件

- 一個 [Telegram](https://telegram.org/) 帳號
- 一個 Claw 實例（[在此處建立](https://lemondata.cc/dashboard/claw)）

## 設置指南

<Steps>
  <Step title="使用 BotFather 建立機器人">
    1. 打開 Telegram 並搜尋 **@BotFather**
    2. 發送 `/newbot`
    3. 為您的機器人選擇一個**顯示名稱**（例如：「My AI Assistant」）
    4. 選擇一個以 `bot` 結尾的**使用者名稱**（例如：`my_ai_assistant_bot`）
    5. BotFather 會回覆您的 **Bot Token** — 它看起來像這樣：
    ```
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
    ```
    <Warning>
    請務必保密您的 bot token！任何擁有此 token 的人都可以控制您的機器人。
    </Warning>
  </Step>
  <Step title="在 LemonData 控制面板中配置">
    1. 前往您的 [Claw Dashboard](https://lemondata.cc/dashboard/claw)
    2. 點擊您實例上的**設定圖示**
    3. 打開 **IM Integrations** 分頁
    4. 展開 **Telegram** 區塊
    5. 貼上您的 **Bot Token**
    6. 點擊 **Save**
  </Step>
  <Step title="開始聊天">
    1. 打開 Telegram 並透過使用者名稱找到您的機器人
    2. 發送任何訊息（例如：「Hello!」）
    3. 機器人將回覆一個**配對碼 (pairing code)**
    4. 在您的 Claw WebUI 中輸入該代碼以完成配對
    5. 您已準備就緒 — 開始與 AI 聊天吧！
  </Step>
</Steps>

## 選用：自定義機器人個人資料

建立機器人後，您可以使用 BotFather 進行自定義：

| 指令 | 用途 |
|---------|---------|
| `/setdescription` | 設置機器人的簡介/描述 |
| `/setabouttext` | 設置「關於」區塊 |
| `/setuserpic` | 上傳大頭貼 |
| `/setcommands` | 新增指令建議 |

## 在群組中使用

<Warning>
**隱私模式預設開啟。** 在群組聊天中，機器人只能看到：
- 以 `/` 開頭的指令
- 直接 @提及機器人的訊息
- 回覆機器人訊息的內容

要讓機器人看到**所有**群組訊息，請透過 BotFather 關閉隱私模式：傳送 `/setprivacy` → 選擇你的機器人 → 選擇 **Disable**。注意：此變更需要**移除並重新加入**機器人到現有群組才能生效。
</Warning>

### 群組設定

在控制面板的 **IM Integrations** 分頁中配置群組行為：

| 設定 | 選項 | 描述 |
|---------|---------|-------------|
| **Group Policy** | Open / Allowlist / Disabled | 控制誰可以在群組中觸發機器人 |
| **Require @mention** | On / Off | 機器人僅在被 @提及時回應（建議） |
| **Group IDs** | 聊天 ID 列表 | 限制為特定群組（留空 = 所有群組） |
| **Group Allowed Users** | 使用者 ID 列表 | 僅這些使用者可以觸發機器人（allowlist 模式） |

<Tip>
**Group Policy** 控制*誰*可以在群組中向機器人傳送訊息，而 **Group IDs** 控制機器人在*哪些*群組中回應。兩者搭配使用可實現精細控制。
</Tip>

## 私訊策略 (DM Policy)

控制誰可以向您的機器人傳送私訊：

| 策略 | 行為 |
|--------|----------|
| **Pairing**（預設） | 新使用者會收到配對碼 — 在 WebUI 中輸入以完成授權 |
| **Allowlist** | 僅您指定的使用者 ID 可以向機器人傳送訊息 |
| **Open** | 任何人都可以向機器人傳送訊息（請謹慎使用 — 會消耗您的 AI 額度） |

## 疑難排解

<AccordionGroup>
  <Accordion title="機器人沒有回應">
    - 驗證 bot token 是否正確（沒有多餘的空格）
    - 檢查您的實例是否正在執行（控制面板中顯示綠色狀態）
    - 嘗試從控制面板重新啟動實例
  </Accordion>
  <Accordion title="「配對碼」沒有出現">
    - 確保您是向機器人傳送**私訊**，而非在群組中
    - 配對系統會在新使用者傳送第一則訊息時啟動
  </Accordion>
  <Accordion title="機器人回應緩慢">
    - 回應時間取決於所選的 AI 模型
    - 較大的模型（GPT-4、Claude）可能需要幾秒鐘
    - 檢查您的 LemonData 餘額 — 額度不足會導致錯誤
  </Accordion>
</AccordionGroup>
