<Note>
**自託管**代表您直接在自己的 VPS 或機器上執行 OpenClaw。您只需透過 LemonData 餘額支付 AI 模型使用費用 — 無實例費用。
</Note>

## 快速安裝

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash 非互動式
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

安裝程式會提示您輸入 **LemonData API Key**。請在您的[控制面板](https://lemondata.cc/dashboard)中取得。

## 安裝內容

<Steps>
  <Step title="Node.js 22+">
    如果缺少會自動安裝（Linux 透過 NodeSource，macOS 透過 Homebrew）
  </Step>
  <Step title="OpenClaw">
    來自 npm 的官方 OpenClaw 套件，已預先設定 LemonData API
  </Step>
  <Step title="15 個 AI 模型">
    Claude、GPT-5、Gemini、DeepSeek 等 — 開箱即用
  </Step>
  <Step title="IM 外掛">
    預裝釘釘、企業微信和 QQ 外掛
  </Step>
  <Step title="系統服務">
    開機自動啟動（Linux 使用 systemd，macOS 使用 launchd）
  </Step>
</Steps>

## 系統需求

| 需求 | 詳情 |
|------|------|
| **作業系統** | Linux（Ubuntu/Debian/CentOS/RHEL）或 macOS |
| **架構** | x64 或 arm64 |
| **記憶體** | 最低 1 GB，建議 2 GB |
| **API Key** | [在此取得](https://lemondata.cc) |

## 管理指令

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # 檢查狀態
    journalctl --user -u openclaw -f    # 檢視日誌
    systemctl --user restart openclaw   # 重新啟動
    systemctl --user stop openclaw      # 停止
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # 檢視日誌
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # 重新啟動
    launchctl stop cc.lemondata.openclaw  # 停止
    ```
  </Tab>
</Tabs>

## 設定

設定檔位置：`~/.openclaw/openclaw.json`

編輯後請重新啟動服務以套用變更。

### 變更預設模型

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

可用的模型前綴：
- `lemondata-claude/` — Claude 模型（anthropic-messages API）
- `lemondata-minimax/` — MiniMax 模型（anthropic-messages API）
- `lemondata/` — 所有其他模型（openai-completions API）

## 升級

```bash
npm update -g openclaw
# 然後重新啟動服務
systemctl --user restart openclaw  # Linux
# 或
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## 解除安裝

```bash
# 停止並停用服務
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# 移除 OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
