title: "WeCom"
description: "透過 webhook 回呼將您的 Claw 實例連接至 WeCom (企業微信)"
---

將 WeCom (企業微信 / WeChat Work) 與您的 Claw 實例整合，為您的企業工作空間引入 AI。

## 先決條件

- 一個 [WeCom](https://work.weixin.qq.com/) 管理員帳號
- 一個 Claw 實例 ([在此建立](https://lemondata.cc/dashboard/claw))

## 設定指南

<Steps>
  <Step title="建立 WeCom 應用程式">
    1. 登入 [WeCom 管理後台](https://work.weixin.qq.com/wework_admin/frame)
    2. 前往 **應用管理** → **建立應用**
    3. 輸入應用程式名稱、上傳標誌並設定可見範圍
    4. 點擊 **建立**
  </Step>
  <Step title="獲取憑證">
    1. 記錄您的 **Corp ID** — 可在 **我的企業** → **企業資訊** 底部找到
    2. 在您的應用程式設定中，複製 **Secret** (Corp Secret)
  </Step>
  <Step title="設定回呼 URL">
    1. 在您的應用程式設定中，前往 **接收訊息** → **設定 API 接收**
    2. 當您展開 WeCom 區塊時，您的回呼 URL 會顯示在 LemonData 控制面板中：
    ```
    https://claw-{your-subdomain}.lemondata.cc/webhook/wecom
    ```
    3. 設定 **Token** (隨機字串) 和 **EncodingAESKey** (43 個字元的隨機字串)
    4. 點擊 **儲存** — WeCom 將驗證回呼 URL

    <Tip>
    回呼 URL 顯示在 LemonData 控制面板中，並附有複製按鈕。在驗證 URL 之前，請確保您的 Claw 實例正在運行。
    </Tip>
  </Step>
  <Step title="在 LemonData 控制面板中設定">
    1. 前往您的 [Claw 控制面板](https://lemondata.cc/dashboard/claw)
    2. 點擊實例上的 **設定圖示**
    3. 開啟 **IM Integrations** 標籤頁
    4. 展開 **WeCom** 區塊
    5. 複製頂部顯示的 **Callback URL**
    6. 填寫：**Corp ID**、**Corp Secret**、**Token**、**EncodingAESKey**
    7. 點擊 **儲存**
  </Step>
  <Step title="開始對話">
    1. 在 WeCom 中，於 **工作台** 找到您的應用程式
    2. 向應用程式發送訊息
    3. 完成配對程序
    4. 開始與 AI 對話！
  </Step>
</Steps>

<Warning>
WeCom 管理後台與您的 Claw 設定中的 **Token** 和 **EncodingAESKey** 必須完全一致。若不匹配將導致訊息驗證失敗。
</Warning>

## 疑難排解

<AccordionGroup>
  <Accordion title="回呼 URL 驗證失敗">
    - 確保您的 Claw 實例正在運行 (綠色狀態)
    - 驗證 URL 格式：`https://claw-{subdomain}.lemondata.cc/webhook/wecom`
    - 在 WeCom 驗證 **之前**，檢查 Token 和 EncodingAESKey 是否已儲存在 Claw 設定中
  </Accordion>
  <Accordion title="機器人沒有回應">
    - 驗證 Corp ID 和 Corp Secret 是否正確
    - 檢查應用程式對您的使用者帳號是否可見 (可見範圍設定)
    - 確保 WeCom 和 Claw 之間的 Token 和 EncodingAESKey 一致
  </Accordion>
</AccordionGroup>
