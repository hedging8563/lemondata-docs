## 概覽

LemonData 可與任何相容 OpenAI 的聊天應用程式配合使用。本指南涵蓋了熱門桌面端和行動端聊天應用程式的設定。

## 支援的應用程式

| 應用程式 | 平台 | 功能 |
|-------------|----------|----------|
| LobeChat | 網頁、桌面端、自託管 | 多供應商、外掛程式、RAG |
| CherryStudio | Windows, macOS, Linux | 豐富的外掛程式、多個供應商 |
| Chatbox | Windows, macOS, Linux | 輕量、快速 |
| BotGem | iOS, macOS | 原生 Apple 體驗 |
| TypingMind | 網頁、桌面端 | 自定義人格、外掛程式 |
| OpenCat | iOS, macOS | 原生、離線支援 |
| ChatWise | Windows, macOS | 團隊協作 |

## 設定步驟

### 步驟 1：獲取您的 API Key

1. 登入 [LemonData 控制台](https://lemondata.cc/dashboard)
2. 導覽至 [API Keys](https://lemondata.cc/dashboard/api)
3. 建立並複製您的 API Key（格式：`sk-...`）

### 步驟 2：設定您的應用程式

<Tabs>
  <Tab title="LobeChat">
    **針對自託管 LobeChat：**

    設定環境變數：
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **針對 LobeChat Cloud：**
    1. 開啟 **Settings** → **AI Service Provider**
    2. 選擇 **OpenAI**
    3. 輸入您的 LemonData API Key
    4. 設定自定義端點：`https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. 開啟 **Settings** → **Model Provider**
    2. 點擊 **Add Provider**
    3. 選擇 **OpenAI API Compatible**
    4. 輸入：
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. 新增模型：`gpt-4o`, `claude-sonnet-4-5` 等
    6. 儲存並測試
  </Tab>
  <Tab title="Chatbox">
    1. 開啟 **Settings** → **AI Model**
    2. 選擇 **OpenAI API Compatible**
    3. 輸入：
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. 點擊 **Verify** 以測試連線
  </Tab>
  <Tab title="BotGem">
    1. 開啟 **Settings** → **Providers**
    2. 點擊 **Add Custom Provider**
    3. 輸入：
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. 新增您偏好的模型
    5. 儲存設定
  </Tab>
  <Tab title="TypingMind">
    1. 點擊 **Settings**（齒輪圖示）
    2. 選擇 **Custom Endpoint**
    3. 輸入：
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. 設定模型並儲存
  </Tab>
</Tabs>

### 步驟 3：新增模型

設定供應商後，新增您想要使用的模型：

| 類別 | 模型 |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## 設定參考

| 欄位 | 值 |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API 類型 | OpenAI Compatible |

**JSON 設定（適用於支援的應用程式）：**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## 疑難排解

<AccordionGroup>
  <Accordion title="連線錯誤">
    - 驗證 Base URL 是否準確為 `https://api.lemondata.cc/v1`
    - 檢查結尾是否有斜線（如有請移除）
    - 驗證網路連線和防火牆設定
  </Accordion>

  <Accordion title="401 Unauthorized">
    - 再次檢查您的 API Key
    - 在 [LemonData 控制台](https://lemondata.cc/dashboard/api) 中驗證 Key 是否處於啟用狀態
    - 確保 Key 中沒有多餘的空格
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - 在控制台中檢查您的帳戶餘額
    - 為您的帳戶儲值
  </Accordion>

  <Accordion title="404 Model Not Available">
    - 驗證模型名稱拼寫是否正確
    - 在 [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models) 檢查模型可用性
    - 嘗試不同的模型
  </Accordion>

  <Accordion title="回應緩慢">
    - 嘗試更快的模型（例如使用 `gpt-4o-mini` 代替 `gpt-4o`）
    - 如果可行，請減少 `max_tokens` 參數
    - 檢查您的網路連線
  </Accordion>
</AccordionGroup>
