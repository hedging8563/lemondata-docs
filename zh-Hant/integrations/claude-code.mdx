## 總覽

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) 是 Anthropic 官方推出的 AI 輔助編程 CLI 工具。您可以將其配置為使用 LemonData 作為 API 提供商。

## 配置

### 環境變數

設置以下環境變數：

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

將這些內容添加到您的 shell 設定檔（`~/.bashrc`、`~/.zshrc` 等）中以使其持久生效。

### 設定檔

或者，創建或編輯 `~/.claude/settings.json`：

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## 使用方法

配置完成後，即可正常使用 Claude Code：

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## 可用模型

LemonData 支援所有 Claude 模型：

| 模型 | 描述 |
|-------|-------------|
| `claude-opus-4-5` | 最強大的模型，適用於複雜推理 |
| `claude-sonnet-4-5` | 性能平衡，非常適合編程 |
| `claude-haiku-4-5` | 速度最快，具備成本效益 |

## 模型選擇

使用 `--model` 標記指定模型：

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

或在您的配置中設置預設值：

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## 擴展思考

對於複雜的推理任務，請使用具備擴展思考功能的模型：

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## 故障排除

<AccordionGroup>
  <Accordion title="連線錯誤">
    驗證基礎 URL 是否準確為 `https://api.lemondata.cc`（Anthropic SDK 不需要 `/v1` 後綴）。
  </Accordion>

  <Accordion title="無效的 API 金鑰">
    確保您的 LemonData API 金鑰以 `sk-` 開頭且餘額充足。
  </Accordion>

  <Accordion title="找不到模型">
    檢查模型名稱是否正確。請使用 `claude-sonnet-4-5` 格式。
  </Accordion>
</AccordionGroup>
