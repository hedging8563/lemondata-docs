## Visão Geral

A LemonData utiliza o modelo **pay-as-you-go (pagamento conforme o uso)**. Você paga apenas pelo que utilizar, sem assinaturas ou compromissos de gasto mínimo.

## Como funciona o faturamento

1. **Adicione créditos** à sua conta
2. **Use a API** - os custos são deduzidos por requisição
3. **Monitore o uso** no seu painel de controle
4. **Recarregue quando o saldo estiver baixo**

## Contagem de Tokens

Tokens são as unidades básicas de processamento de texto:

- ~4 caracteres = 1 token (Inglês)
- ~1-2 caracteres = 1 token (Chinês)
- 1 imagem = depende do tamanho e dos detalhes

### Estimando Tokens

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English
```
