## Visão Geral

Dify é uma plataforma de desenvolvimento de aplicações de LLM de código aberto. Ela oferece orquestração visual de Prompt, pipelines de RAG, frameworks de Agent e capacidades de LLMOps. O LemonData pode ser configurado como um provedor de modelos personalizado no Dify.

## Vantagens

- Acesso a mais de 300 modelos de IA através de uma única interface
- Alterne entre modelos sem alterar a lógica da aplicação
- Escolha o melhor modelo para cada tarefa para otimizar custos
- Faturamento unificado e rastreamento de uso

## Pré-requisitos

- Conta LemonData com acesso à API
- Instalação do Dify (versão em nuvem ou implantação on-premise)

## Passos de Configuração

### Passo 1: Obtenha sua API Key

1. Faça login no [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navegue até [API Keys](https://lemondata.cc/dashboard/api)
3. Crie e copie sua API Key (formato: `sk-...`)

### Passo 2: Adicione um provedor de modelos personalizado

<Steps>
  <Step title="Abrir configurações">
    No Dify, vá para **Settings** → **Model Provider**
  </Step>
  <Step title="Adicionar provedor compatível com OpenAI">
    Clique em **Add Model Provider** e selecione **OpenAI-API-compatible**
  </Step>
  <Step title="Configurar provedor">
    Insira as seguintes configurações:

    | Campo | Valor |
    |-------|-------|
    | Nome do provedor | LemonData |
    | API Key | `sk-your-lemondata-key` |
    | API Base URL | `https://api.lemondata.cc/v1` |
  </Step>
  <Step title="Adicionar modelos">
    Adicione os modelos que deseja usar:
    - `gpt-4o`
    - `gpt-4o-mini`
    - `claude-sonnet-4-5`
    - `claude-opus-4-5`
    - `gemini-2.5-flash`
    - `gemini-2.5-pro`
    - `deepseek-r1`
  </Step>
</Steps>

### Passo 3: Testar conexão

1. Selecione um modelo (por exemplo, `gpt-4o-mini`)
2. Envie uma mensagem de teste
3. Confirme que você recebeu uma resposta

## Uso em Aplicações

### Chatbot

1. Crie uma nova aplicação de Chatbot
2. Selecione LemonData como provedor de modelos
3. Selecione seu modelo preferido
4. Configure o System Prompt e os parâmetros

### Agente (Agent)

1. Crie uma aplicação de Agent
2. Selecione um modelo poderoso (GPT-4o, Claude)
3. Adicione ferramentas e bases de conhecimento
4. Configure o comportamento do Agent

### Fluxo de Trabalho (Workflow)

1. Crie um workflow
2. Adicione um nó de LLM
3. Selecione o modelo LemonData para cada nó
4. Conecte os nós e configure o fluxo de dados

## Modelos Disponíveis

| Categoria | Modelo |
|----------|--------|
| Chat | GPT-4o, GPT-4o-mini, Claude Sonnet/Opus, Gemini, DeepSeek |
| Embeddings | text-embedding-3-small, text-embedding-3-large |
| Visão (Vision) | GPT-4o (suporta imagens), Claude Sonnet (suporta imagens) |

## Configuração de RAG

Para aplicações de RAG, configure o modelo de embedding:

1. Vá para **Settings** → **Model Provider**
2. Adicione o modelo de embedding: `text-embedding-3-small`
3. Defina-o como o modelo de embedding padrão nas configurações da base de conhecimento

## Melhores Práticas

<AccordionGroup>
  <Accordion title="Comece com modelos econômicos">
    Use o GPT-4o-mini durante as fases de teste e desenvolvimento e mude para modelos mais poderosos em ambiente de produção.
  </Accordion>

  <Accordion title="Use roteamento de modelos">
    Configure modelos diferentes para tarefas diferentes — use modelos rápidos para consultas simples e modelos poderosos para raciocínio complexo.
  </Accordion>

  <Accordion title="Monitore o uso">
    Combine as funções de LLMOps do Dify com o console do LemonData para rastrear custos e desempenho.
  </Accordion>
</AccordionGroup>

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Erro de conexão">
    - Confirme se a API Base URL é exatamente `https://api.lemondata.cc/v1`
    - Verifique se há uma barra no final
    - Verifique a conexão de rede
  </Accordion>

  <Accordion title="401 Não autorizado (Unauthorized)">
    - Verifique novamente a API Key
    - Certifique-se de que a Key está ativa no console do LemonData
  </Accordion>

  <Accordion title="Modelo não encontrado">
    - Confirme se o nome do modelo corresponde exatamente
    - Verifique a disponibilidade do modelo em [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
  </Accordion>
</AccordionGroup>
