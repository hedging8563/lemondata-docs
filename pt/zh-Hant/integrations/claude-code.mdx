## Visão Geral

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) é a ferramenta de CLI oficial da Anthropic para programação assistida por AI. Você pode configurá-la para usar o LemonData como provedor de API.

## Configuração

### Variáveis de Ambiente

Defina as seguintes variáveis de ambiente:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Adicione estas linhas ao seu arquivo de perfil do shell (`~/.bashrc`, `~/.zshrc`, etc.) para torná-las persistentes.

### Arquivo de Configuração

Alternativamente, crie ou edite `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Uso

Após a configuração, você pode usar o Claude Code normalmente:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Modelos Disponíveis

O LemonData suporta todos os modelos Claude:

| Modelo | Descrição |
|-------|-------------|
| `claude-opus-4-5` | O modelo mais poderoso, ideal para raciocínio complexo |
| `claude-sonnet-4-5` | Equilíbrio de desempenho, excelente para programação |
| `claude-haiku-4-5` | O mais rápido e econômico |

## Seleção de Modelo

Use a flag `--model` para especificar um modelo:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Ou defina um valor padrão em sua configuração:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Pensamento Estendido

Para tarefas de raciocínio complexas, use modelos com recursos de pensamento estendido:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Erro de Conexão">
    Verifique se a URL base é exatamente `https://api.lemondata.cc` (o SDK da Anthropic não requer o sufixo `/v1`).
  </Accordion>

  <Accordion title="Chave de API Inválida">
    Certifique-se de que sua chave de API do LemonData comece com `sk-` e que você tenha saldo suficiente.
  </Accordion>

  <Accordion title="Modelo Não Encontrado">
    Verifique se o nome do modelo está correto. Use o formato `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
