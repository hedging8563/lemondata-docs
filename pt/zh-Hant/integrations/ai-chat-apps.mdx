## Visão Geral

A LemonData funciona com qualquer aplicativo de chat compatível com OpenAI. Este guia abrange a configuração de aplicativos de chat populares para desktop e dispositivos móveis.

## Aplicativos Suportados

| Aplicativo | Plataforma | Recursos |
|-------------|----------|----------|
| LobeChat | Web, Desktop, Auto-hospedado | Multi-provedor, Plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Plugins avançados, múltiplos provedores |
| Chatbox | Windows, macOS, Linux | Leve, rápido |
| BotGem | iOS, macOS | Experiência Apple nativa |
| TypingMind | Web, Desktop | Personas personalizadas, Plugins |
| OpenCat | iOS, macOS | Nativo, suporte offline |
| ChatWise | Windows, macOS | Colaboração em equipe |

## Passos de Configuração

### Passo 1: Obtenha sua API Key

1. Faça login no [Console da LemonData](https://lemondata.cc/dashboard)
2. Navegue até [API Keys](https://lemondata.cc/dashboard/api)
3. Crie e copie sua API Key (formato: `sk-...`)

### Passo 2: Configure seu aplicativo

<Tabs>
  <Tab title="LobeChat">
    **Para LobeChat auto-hospedado:**

    Configure as variáveis de ambiente:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Para LobeChat Cloud:**
    1. Abra **Settings** → **AI Service Provider**
    2. Selecione **OpenAI**
    3. Insira sua LemonData API Key
    4. Configure o endpoint personalizado: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Abra **Settings** → **Model Provider**
    2. Clique em **Add Provider**
    3. Selecione **OpenAI API Compatible**
    4. Insira:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Adicione modelos: `gpt-4o`, `claude-sonnet-4-5`, etc.
    6. Salve e teste
  </Tab>
  <Tab title="Chatbox">
    1. Abra **Settings** → **AI Model**
    2. Selecione **OpenAI API Compatible**
    3. Insira:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Clique em **Verify** para testar a conexão
  </Tab>
  <Tab title="BotGem">
    1. Abra **Settings** → **Providers**
    2. Clique em **Add Custom Provider**
    3. Insira:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Adicione seus modelos preferidos
    5. Salve as configurações
  </Tab>
  <Tab title="TypingMind">
    1. Clique em **Settings** (ícone de engrenagem)
    2. Selecione **Custom Endpoint**
    3. Insira:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Configure o modelo e salve
  </Tab>
</Tabs>

### Passo 3: Adicione modelos

Após configurar o provedor, adicione os modelos que deseja usar:

| Categoria | Modelo |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Referência de Configuração

| Campo | Valor |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| Tipo de API | OpenAI Compatible |

**Configuração JSON (para aplicativos compatíveis):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Erro de Conexão">
    - Verifique se a Base URL é exatamente `https://api.lemondata.cc/v1`
    - Verifique se há uma barra no final (remova se houver)
    - Verifique a conexão de rede e as configurações de firewall
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Verifique novamente sua API Key
    - Verifique se a Key está ativa no [Console da LemonData](https://lemondata.cc/dashboard/api)
    - Certifique-se de que não há espaços extras na Key
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - Verifique o saldo da sua conta no console
    - Recarregue sua conta
  </Accordion>

  <Accordion title="404 Model Not Available">
    - Verifique se o nome do modelo está escrito corretamente
    - Verifique a disponibilidade do modelo em [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - Tente um modelo diferente
  </Accordion>

  <Accordion title="Resposta Lenta">
    - Tente um modelo mais rápido (por exemplo, use `gpt-4o-mini` em vez de `gpt-4o`)
    - Se possível, reduza o parâmetro `max_tokens`
    - Verifique sua conexão de rede
  </Accordion>
</AccordionGroup>

## Dicas

<AccordionGroup>
  <Accordion title="Use o modelo adequado">
    Escolha o modelo com base na tarefa: use GPT-4o-mini para chats rápidos e Claude ou GPT-4o para tarefas complexas.
  </Accordion>

  <Accordion title="Salve prompts comuns">
    A maioria dos aplicativos suporta salvar prompts ou personas. Crie modelos para tarefas comuns.
  </Accordion>

  <Accordion title="Monitore o uso">
    Verifique seu uso no Console da LemonData para acompanhar os custos e gerenciar seu orçamento.
  </Accordion>
</AccordionGroup>
