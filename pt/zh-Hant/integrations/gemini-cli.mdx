## Visão Geral

O Google Gemini CLI é uma ferramenta de linha de comando para interagir com os modelos Gemini. O LemonData fornece um endpoint compatível que permite usar o Gemini CLI e acessar mais de 300 modelos.

## Requisitos do Sistema

- **Node.js**: Versão 20.0+
- **Sistema Operacional**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, ou Debian 10+

## Instalação

```bash
npm install -g @google/gemini-cli
```

Verifique a instalação:

```bash
gemini --version
```

## Configuração

### Passo 1: Obtenha sua API Key

1. Faça login no [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navegue até [API Keys](https://lemondata.cc/dashboard/api)
3. Crie e copie sua API Key (formato: `sk-...`)

### Passo 2: Configure as variáveis de ambiente

**Temporário (sessão atual):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**Configuração permanente:**

Adicione ao seu arquivo de configuração do shell:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
Reinicie o terminal após a configuração para que as alterações entrem em vigor.
</Warning>

<Note>
**Limitações conhecidas**: Em alguns casos, se houver uma sessão do Google em cache, o Gemini CLI pode não seguir a `GOOGLE_GEMINI_BASE_URL`. Se você encontrar problemas de conexão, tente limpar o cache da CLI ou iniciar uma nova sessão de terminal.
</Note>

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Erro de conexão">
    - Verifique se `GOOGLE_GEMINI_BASE_URL` está definido como `https://api.lemondata.cc`
    - Nota: O endpoint do Gemini não requer o sufixo `/v1`
    - Verifique a conexão de rede
  </Accordion>

  <Accordion title="Falha na autenticação">
    - Verifique se a variável de ambiente `GEMINI_API_KEY` está definida
    - Verifique se a Key começa com `sk-`
    - Certifique-se de que a Key está ativa no dashboard do LemonData
  </Accordion>

  <Accordion title="Modelo indisponível">
    - Verifique a disponibilidade dos modelos Gemini em [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - Tente diferentes variantes do modelo Gemini
  </Accordion>
</AccordionGroup>
