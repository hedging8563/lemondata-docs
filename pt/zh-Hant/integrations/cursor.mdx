## Visão Geral

[Cursor](https://cursor.sh) é um editor de código impulsionado por IA. Você pode usar o LemonData como um provedor de API personalizado para acessar mais de 300 modelos.

## Configuração

1. Abra as configurações do Cursor (`Cmd/Ctrl + ,`)
2. Navegue até **Models** na barra lateral esquerda
3. Role para baixo até a seção **OpenAI API Key**
4. Insira sua API key do LemonData
5. Ative **Override OpenAI Base URL**
6. Defina a Base URL para:

```
https://api.lemondata.cc/v1
```

## Seleção de Modelos

Após a configuração, você pode selecionar qualquer modelo do LemonData no Cursor:

### Modelos de codificação recomendados

| Modelo | Ideal para |
|-------|----------|
| `claude-opus-4-5` | Codificação complexa, design de arquitetura |
| `claude-sonnet-4-5` | Codificação geral, depuração |
| `gpt-4o` | Geração de código, explicação |
| `deepseek-r1` | Tarefas de raciocínio pesado |
| `gpt-4o-mini` | Preenchimento rápido, tarefas simples |

## Recursos

### Chat

Use o painel de chat de IA (`Cmd/Ctrl + L`) com qualquer modelo do LemonData:

- Explicação de código
- Correção de bugs
- Sugestões de refatoração
- Geração de documentação

### Composer

O recurso Composer (`Cmd/Ctrl + I`) pode ser usado com modelos do LemonData para:

- Edição de múltiplos arquivos
- Implementação de funcionalidades
- Migração de código

### Preenchimento com Tab

Para o preenchimento com Tab, recomendamos o uso de modelos rápidos:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`

## Resolução de Problemas

<AccordionGroup>
  <Accordion title="Modelos não aparecem">
    O Cursor armazena a lista de modelos em cache. Tente reiniciar o Cursor após configurar a API.
  </Accordion>

  <Accordion title="Resposta lenta">
    Verifique sua seleção de modelo. Para tarefas rápidas, use modelos mais velozes como o `gpt-4o-mini`.
  </Accordion>

  <Accordion title="Erro de autenticação">
    Certifique-se de que sua API key está correta e que você possui saldo suficiente.
  </Accordion>
</AccordionGroup>
