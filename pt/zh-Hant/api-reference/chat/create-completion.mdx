---
title: "Criar Chat Completion"
openapi: "POST /v1/chat/completions"
description: "Cria uma conclusão para as mensagens de chat"
---

## Corpo da Requisição (Request Body)

<ParamField body="model" type="string" required>
  O ID do modelo a ser usado. Consulte [Models](https://lemondata.cc/zh-TW/models) para ver as opções disponíveis.
</ParamField>

<ParamField body="messages" type="array" required>
  Uma lista de mensagens que compõem a conversa.

  Cada objeto de mensagem contém:
  - `role` (string): `system`, `user` ou `assistant`
  - `content` (string | array): O conteúdo da mensagem
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Temperatura de amostragem entre 0 e 2. Valores mais altos tornarão a saída mais aleatória.
</ParamField>

<ParamField body="max_tokens" type="integer">
  O número máximo de tokens a serem gerados.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Se for true, incrementos parciais de mensagens serão enviados como eventos SSE.
</ParamField>

<ParamField body="stream_options" type="object">
  Opções de streaming. Defina `include_usage: true` para receber o uso de tokens nos chunks do stream.
</ParamField>

<ParamField body="top_p" type="number" default="1">
  Parâmetro de amostragem de núcleo (Nucleus sampling). Recomendamos ajustar este parâmetro ou a temperature, mas não ambos.
</ParamField>

<ParamField body="frequency_penalty" type="number" default="0">
  Número entre -2.0 e 2.0. Valores positivos penalizam tokens repetidos.
</ParamField>

<ParamField body="presence_penalty" type="number" default="0">
  Número entre -2.0 e 2.0. Valores positivos penalizam tokens que já apareceram no texto.
</ParamField>

<ParamField body="stop" type="string | array">
  Até 4 sequências onde a API interromperá a geração de tokens.
</ParamField>

<ParamField body="tools" type="array">
  Uma lista de ferramentas que o modelo pode chamar (function calling).
</ParamField>

<ParamField body="tool_choice" type="string | object">
  Controla como o modelo utiliza as ferramentas. Opções: `auto`, `none`, `required` ou um objeto de ferramenta específico.
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  Se as chamadas de função paralelas estão habilitadas. Defina como false para chamar funções sequencialmente.
</ParamField>

<ParamField body="max_completion_tokens" type="integer">
  O número máximo de tokens para a conclusão. Uma alternativa ao `max_tokens`, recomendada para modelos mais novos como o1/o3.
</ParamField>

<ParamField body="reasoning_effort" type="string">
  Esforço de raciocínio para modelos o1/o3. Opções: `low`, `medium`, `high`.
</ParamField>

<ParamField body="seed" type="integer">
  Semente aleatória para amostragem determinística.
</ParamField>

<ParamField body="n" type="integer" default="1">
  O número de conclusões a serem geradas (1-128).
</ParamField>

<ParamField body="logprobs" type="boolean">
  Se deve retornar probabilidades logarítmicas (log probabilities).
</ParamField>

<ParamField body="top_logprobs" type="integer">
  O número de probabilidades logarítmicas mais prováveis a serem retornadas (0-20). Requer `logprobs: true`.
</ParamField>

<ParamField body="top_k" type="integer">
  Parâmetro de amostragem Top-K (aplicável a modelos Anthropic/Gemini).
</ParamField>

<ParamField body="response_format" type="object">
  Especificação do formato de resposta. Use `{"type": "json_object"}` para habilitar o modo JSON, ou `{"type": "json_schema", "json_schema": {...}}` para saída estruturada.
</ParamField>

<ParamField body="logit_bias" type="object">
  Modifica a probabilidade de tokens específicos aparecerem. Mapeia IDs de tokens (como strings) para valores de viés entre -100 e 100.
</ParamField>

<ParamField body="user" type="string">
  Um identificador exclusivo que representa seu usuário final, usado para monitoramento de abusos.
</ParamField>

<ParamField body="cache_control" type="object">
  Opções de controle de cache da LemonData.

  - `type` (string): Estratégia de cache - `default`, `no_cache`, `no_store`, `response_only`, `semantic_only`
  - `max_age` (integer): Tempo de vida do cache (TTL) em segundos (máximo 86400)
</ParamField>

## Resposta (Response)

<ResponseField name="id" type="string">
  Identificador exclusivo para a conclusão.
</ResponseField>

<ResponseField name="object" type="string">
  Sempre `chat.completion`.
</ResponseField>

<ResponseField name="created" type="integer">
  O timestamp Unix de quando a conclusão foi criada.
</ResponseField>

<ResponseField name="model" type="string">
  O modelo usado para a conclusão.
</ResponseField>

<ResponseField name="choices" type="array">
  Uma lista de opções de conclusão.

  Cada opção contém:
  - `index` (integer): O índice da opção
  - `message` (object): A mensagem gerada
  - `finish_reason` (string): O motivo pelo qual o modelo parou (`stop`, `length`, `tool_calls`)
</ResponseField>

<ResponseField name="usage" type="object">
  Estatísticas de uso de tokens.

  - `prompt_tokens` (integer): Número de tokens no prompt
  - `completion_tokens` (integer): Número de tokens na conclusão
  - `total_tokens` (integer): Número total de tokens usados
</ResponseField>

from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"}
    ],
    temperature=0.7,
    max_tokens=1000
)

print(response.choices[0].message.content)

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateChatCompletion(
        context.Background(),
        openai.ChatCompletionRequest{
            Model: "gpt-4o",
            Messages: []openai.ChatCompletionMessage{
                {Role: openai.ChatMessageRoleSystem, Content: "You are a helpful assistant."},
                {Role: openai.ChatMessageRoleUser, Content: "Hello!"},
            },
            Temperature: 0.7,
            MaxTokens:   1000,
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Choices[0].Message.Content)
}
```
