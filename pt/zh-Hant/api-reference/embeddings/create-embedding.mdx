---
title: "Criar embeddings"
openapi: "POST /v1/embeddings"
description: "Cria um vetor de embedding que representa o texto de entrada"
---

## Corpo da requisição

<ParamField body="model" type="string" required>
  O ID do modelo de embedding a ser usado (ex: `text-embedding-3-small`).
</ParamField>

<ParamField body="input" type="string | array" required>
  O texto de entrada para o qual gerar o embedding. Pode ser uma string ou um array de strings.
</ParamField>

<ParamField body="encoding_format" type="string" default="float">
  O formato do embedding: `float` ou `base64`.
</ParamField>

<ParamField body="dimensions" type="integer">
  O número de dimensões de saída (depende do modelo específico).
</ParamField>

<ParamField body="user" type="string">
  Um identificador exclusivo que representa seu usuário final, usado para monitoramento de abuso.
</ParamField>

## Modelos disponíveis

| Modelo | Dimensões | Descrição |
|-------|------------|-------------|
| `text-embedding-3-large` | 3072 | Melhor qualidade |
| `text-embedding-3-small` | 1536 | Equilibrado |
| `text-embedding-ada-002` | 1536 | Legado |

## Resposta

<ResponseField name="object" type="string">
  Fixado como `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Array de objetos de embedding.

  Cada objeto contém:
  - `object` (string): `embedding`
  - `index` (integer): O índice no array de entrada
  - `embedding` (array): Vetor de embedding
</ResponseField>

<ResponseField name="model" type="string">
  O modelo utilizado.
</ResponseField>

<ResponseField name="usage" type="object">
  Uso de tokens, incluindo `prompt_tokens` e `total_tokens`.
</ResponseField>

```bash cURL
    "model": "text-embedding-3-small",
    "input": "The quick brown fox jumps over the lazy dog"

```
