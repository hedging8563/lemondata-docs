---
title: "Criar Modelo 3D"
openapi: "POST /v1/3d/generations"
description: "Cria uma tarefa de geração de modelo 3D"
---

Gere modelos 3D a partir de texto ou imagens usando Tripo3D e outros provedores. Esta é uma API assíncrona.

## Corpo da Requisição (Request Body)

<ParamField body="model" type="string" default="tripo3d-v2.5">
  O modelo a ser usado (ex: `tripo3d-v2.5`, `tripo3d-v2`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Descrição em texto do modelo 3D a ser gerado.
</ParamField>

<ParamField body="image" type="string">
  Imagem codificada em Base64 para geração de imagem para 3D.
</ParamField>

<ParamField body="image_url" type="string">
  URL da imagem para geração de imagem para 3D.
</ParamField>

<ParamField body="format" type="string" default="glb">
  Formato de saída: `glb`, `fbx`, `obj` ou `usdz`.
</ParamField>

<ParamField body="quality" type="string" default="standard">
  Nível de qualidade: `draft`, `standard` ou `high`.
</ParamField>

<ParamField body="style" type="string">
  Predefinição de estilo do modelo.
</ParamField>

<ParamField body="seed" type="integer">
  Valor de semente (seed) para gerações reproduzíveis.
</ParamField>

<ParamField body="user" type="string">
  Identificador único para o usuário final.
</ParamField>

## Resposta (Response)

<ResponseField name="id" type="string">
  ID da tarefa usado para consultar o status (polling).
</ResponseField>

<ResponseField name="status" type="string">
  Status da tarefa: `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Timestamp Unix de quando a tarefa foi criada.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/3d/generations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tripo3d-v2.5",
    "prompt": "A detailed medieval castle with towers",
    "format": "glb",
    "quality": "high"
  }'
```
