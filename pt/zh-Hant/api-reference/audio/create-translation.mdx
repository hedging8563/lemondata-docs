## Visão Geral

Traduza áudio de qualquer idioma suportado para texto em inglês. Diferente da transcrição (`transcription`), este endpoint sempre gera texto em inglês, independentemente do idioma de entrada.

## Corpo da Requisição

<ParamField body="file" type="file" required>
  O arquivo de áudio a ser traduzido. Formatos suportados: `flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`. O tamanho máximo do arquivo é 25 MB.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  O modelo a ser utilizado. Atualmente, apenas `whisper-1` é suportado.
</ParamField>

<ParamField body="prompt" type="string">
  Texto opcional para guiar o estilo do modelo ou continuar um segmento anterior. Deve estar em inglês.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  O formato de saída. Opções: `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number">
  A temperatura de amostragem, entre 0 e 1. Valores mais altos (como 0.8) tornarão a saída mais aleatória, enquanto valores mais baixos (como 0.2) a tornarão mais focada e determinística.
</ParamField>

## Resposta

<ResponseField name="text" type="string">
  O texto traduzido em inglês.
</ResponseField>

Para o formato `verbose_json`, a resposta também inclui:

<ResponseField name="language" type="string">
  O idioma detectado do áudio de entrada.
</ResponseField>

<ResponseField name="duration" type="number">
  A duração do áudio de entrada (em segundos).
</ResponseField>

<ResponseField name="segments" type="array">
  Segmentos de texto traduzidos com carimbos de data/hora (timestamps).
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/translations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -F "file=@german_audio.mp3" \
  -F "model=whisper-1"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

with open("german_audio.mp3", "rb") as audio_file:
    response = client.audio.translations.create(
        model="whisper-1",
        file=audio_file
    )

print(response.text)
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/translations');

$file = new CURLFile('german_audio.mp3', 'audio/mpeg', 'german_audio.mp3');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => [
        'file' => $file,
        'model' => 'whisper-1'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['text'];
```

## Tradução vs Transcrição

| Característica | Tradução | Transcrição |
|---------|-------------|---------------|
| Idioma de saída | Sempre inglês | Igual à entrada |
| Caso de uso | Converter áudio em idioma estrangeiro para inglês | Preservar o idioma original |
| Parâmetro de idioma | Não aplicável | Dica opcional |

<Note>
  O endpoint de tradução detecta automaticamente o idioma de origem e o traduz para o inglês. O parâmetro `language` da transcrição será ignorado.
</Note>
