---
title: "Reordenar Documentos"
openapi: "POST /v1/rerank"
description: "Reordena documentos com base na relevância para uma consulta"
---

Use modelos de similaridade semântica para reordenar documentos. Ideal para otimizar resultados de busca e aplicações RAG.

## Corpo da Requisição

<ParamField body="model" type="string" required>
  O ID do modelo de rerank a ser utilizado (ex: `BAAI/bge-reranker-v2-m3`, `qwen3-rerank`).
</ParamField>

<ParamField body="query" type="string" required>
  A string de consulta usada para classificar os documentos.
</ParamField>

<ParamField body="documents" type="array" required>
  Uma lista de documentos (strings) para serem reordenados.
</ParamField>

<ParamField body="top_n" type="integer">
  O número de resultados principais a serem retornados. O padrão são todos os documentos.
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  Se deve incluir o texto original do documento na resposta.
</ParamField>

## Resposta

<ResponseField name="results" type="array">
  Uma lista ordenada de documentos com pontuações.

  Cada resultado contém:
  - `index` (integer): O índice original do documento
  - `relevance_score` (number): Pontuação de relevância (0-1)
  - `document` (string): O texto original (se `return_documents=true`)
</ResponseField>

<ResponseField name="model" type="string">
  O modelo utilizado para o rerank.
</ResponseField>

<ResponseField name="usage" type="object">
  Estatísticas de uso de tokens.
</ResponseField>

```bash cURL
curl -X POST "https://api.lemondata.cc/v1/rerank" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "BAAI/bge-reranker-v2-m3",
    "query": "What is machine learning?",
    "documents": [
      "Machine learning is a subset of AI",
      "The weather is nice today",
      "Deep learning uses neural networks"
    ],
    "top_n": 2,
    "return_documents": true
  }'
```
