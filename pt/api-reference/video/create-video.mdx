---

## Visão Geral

A geração de vídeo é assíncrona. Você envia uma solicitação e recebe um `task_id`, depois realiza o polling para obter o resultado.

## Corpo da Requisição

<ParamField body="model" type="string" default="kling-video">
  Modelo de vídeo (ex: `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Descrição em texto do vídeo a ser gerado.
</ParamField>

<ParamField body="image_url" type="string">
  Imagem inicial opcional para geração de imagem para vídeo (image-to-video).
</ParamField>

<ParamField body="duration" type="integer">
  Duração do vídeo em segundos (dependente do modelo).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Proporção de tela (ex: `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Resolução do vídeo (ex: `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Quadros por segundo (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  O que evitar na geração do vídeo.
</ParamField>

<ParamField body="seed" type="integer">
  Semente aleatória para geração reproduzível.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Escala de orientação livre de classificador (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Intensidade do movimento (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL da imagem do quadro inicial.
</ParamField>

<ParamField body="end_image" type="string">
  URL da imagem do quadro final.
</ParamField>

<ParamField body="user" type="string">
  Um identificador exclusivo para o usuário final.
</ParamField>

## Resposta

<ResponseField name="task_id" type="string">
  Identificador exclusivo da tarefa para polling.
</ResponseField>

<ResponseField name="status" type="string">
  Status inicial: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Modelo utilizado.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```
