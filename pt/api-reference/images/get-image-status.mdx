---
title: "Obter status da imagem"
openapi: "GET /v1/images/generations/{task_id}"
description: "Obtém o status e resultado de uma tarefa de geração de imagem assíncrona"
---

## Parâmetros de caminho

<ParamField path="task_id" type="string" required>
  O ID da tarefa retornado da requisição inicial de geração de imagem.
</ParamField>

## Resposta

<ResponseField name="created" type="integer">
  Timestamp Unix da criação.
</ResponseField>

<ResponseField name="task_id" type="string">
  O identificador da tarefa.
</ResponseField>

<ResponseField name="status" type="string">
  Status da tarefa: `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="data" type="array">
  Array de imagens geradas (preenchido quando `status` é `completed`).

  Cada objeto contém:
  - `url` (string): URL da imagem gerada
  - `revised_prompt` (string): O prompt usado para geração
</ResponseField>

<ResponseField name="error" type="string">
  Mensagem de erro (presente apenas quando `status` é `failed`).
</ResponseField>

```python Python
import requests

response = requests.get(
    "https://api.lemondata.cc/v1/images/generations/img_abc123def456",
    headers={"Authorization": "Bearer sk-your-api-key"}
)

data = response.json()
print(f"Status: {data['status']}")

if data["status"] == "completed":
    print(f"URL da imagem: {data['data'][0]['url']}")
```

## Melhores práticas de consulta

<Tip>
**Intervalo de consulta recomendado**: 3-5 segundos. A maioria das tarefas de geração de imagem é concluída em 30-120 segundos, dependendo do modelo.
</Tip>

```python
import requests
import time

def poll_image_task(task_id, api_key, max_wait=300, interval=3):
    """Consulta o resultado da geração de imagem com timeout."""
    url = f"https://api.lemondata.cc/v1/images/generations/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        if data["status"] == "completed":
            return data["data"][0]["url"]
        elif data["status"] == "failed":
            raise Exception(data.get("error", "Falha na geração"))

        time.sleep(interval)

    raise TimeoutError(f"Tarefa {task_id} não foi concluída em {max_wait}s")

# Uso
image_url = poll_image_task("img_abc123def456", "sk-your-api-key")
print(f"Imagem gerada: {image_url}")
```
