<Note>
**Auto-hospedado** significa que você executa o OpenClaw diretamente no seu próprio VPS ou máquina. Você paga apenas pelo uso de modelos AI através do seu saldo LemonData — sem taxas de instância.
</Note>

## Instalação rápida

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Não interativo
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

O instalador solicitará seu **LemonData API Key**. Obtenha um no seu [Dashboard](https://lemondata.cc/dashboard).

## O que é instalado

<Steps>
  <Step title="Node.js 22+">
    Instalado automaticamente se ausente (via NodeSource no Linux, Homebrew no macOS)
  </Step>
  <Step title="OpenClaw">
    Pacote oficial do OpenClaw via npm, pré-configurado com LemonData API
  </Step>
  <Step title="15 modelos AI">
    Claude, GPT-5, Gemini, DeepSeek e mais — prontos para usar
  </Step>
  <Step title="Plugins de IM">
    Plugins DingTalk, WeCom e QQ pré-instalados
  </Step>
  <Step title="Serviço do sistema">
    Inicia automaticamente na inicialização (systemd no Linux, launchd no macOS)
  </Step>
</Steps>

## Requisitos

| Requisito | Detalhes |

## Comandos de gerenciamento

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # Verificar status
    journalctl --user -u openclaw -f    # Ver logs
    systemctl --user restart openclaw   # Reiniciar
    systemctl --user stop openclaw      # Parar
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # Ver logs
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # Reiniciar
    launchctl stop cc.lemondata.openclaw  # Parar
    ```
  </Tab>
</Tabs>

## Configuração

Localização do arquivo de configuração: `~/.openclaw/openclaw.json`

Após editar, reinicie o serviço para aplicar as alterações.

### Alterar o modelo padrão

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Prefixos de modelos disponíveis:
- `lemondata-claude/` — Modelos Claude (anthropic-messages API)
- `lemondata-minimax/` — Modelos MiniMax (anthropic-messages API)
- `lemondata/` — Todos os outros modelos (openai-completions API)

## Atualização

```bash
npm update -g openclaw
# Depois reiniciar o serviço
systemctl --user restart openclaw  # Linux
# ou
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Desinstalação

```bash
# Parar e desativar o serviço
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# Remover OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```

## Auto-hospedado vs Gerenciado

| | Auto-hospedado | Gerenciado (Claw) |
|---|---|---|
| **Custo da instância** | Gratuito | $20 / 30 dias |
| **Uso de AI** | Pagamento por uso | Pagamento por uso |
| **Infraestrutura** | Seu servidor | Nós gerenciamos tudo |
| **Atualizações** | Manuais (`npm update`) | Automáticas |
| **Suporte** | Comunidade | Prioritário |
