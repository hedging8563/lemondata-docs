O Telegram é a plataforma mais **simples** de conectar. Você só precisa de um bot token do BotFather e estará conversando com a IA em menos de 2 minutos.

## Pré-requisitos

- Uma conta no [Telegram](https://telegram.org/)
- Uma instância Claw ([crie uma aqui](https://lemondata.cc/dashboard/claw))

## Guia de Configuração

<Steps>
  <Step title="Crie um Bot com o BotFather">
    1. Abra o Telegram e pesquise por **@BotFather**
    2. Envie `/newbot`
    3. Escolha um **nome de exibição** para o seu bot (ex: "Meu Assistente de IA")
    4. Escolha um **nome de usuário** terminando em `bot` (ex: `my_ai_assistant_bot`)
    5. O BotFather responderá com o seu **Bot Token** — ele se parece com isto:
    ```
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
    ```
    <Warning>
    Mantenha seu bot token em segredo! Qualquer pessoa com este token pode controlar seu bot.
    </Warning>
  </Step>
  <Step title="Configure no Painel da LemonData">
    1. Vá para o seu [Painel do Claw](https://lemondata.cc/dashboard/claw)
    2. Clique no **ícone de configurações** na sua instância
    3. Abra a aba **IM Integrations**
    4. Expanda a seção **Telegram**
    5. Cole o seu **Bot Token**
    6. Clique em **Save**
  </Step>
  <Step title="Comece a Conversar">
    1. Abra o Telegram e encontre seu bot pelo nome de usuário
    2. Envie qualquer mensagem (ex: "Olá!")
    3. O primeiro usuário é **pareado automaticamente** como o proprietário — nenhum código é necessário!
    4. Comece a conversar com a IA!
  </Step>
</Steps>

## Opcional: Personalizar Perfil do Bot

Após criar seu bot, você pode personalizá-lo com o BotFather:

| Comando | Finalidade |
|---------|---------|
| `/setdescription` | Definir a bio/descrição do bot |
| `/setabouttext` | Definir a seção "Sobre" |
| `/setuserpic` | Fazer upload de uma foto de perfil |
| `/setcommands` | Adicionar sugestões de comando |

## Uso em Grupos

<Warning>
**O Privacy Mode está ativado por padrão** para bots do Telegram. Em chats de grupo, o bot verá apenas mensagens que:
- Começam com `/` (comandos)
- Mencionam o bot diretamente com @
- São respostas às mensagens do bot

Para permitir que o bot veja **todas** as mensagens do grupo, desative o Privacy Mode via BotFather: envie `/setprivacy` → selecione seu bot → escolha **Disable**. Observação: você deve **remover e adicionar novamente** o bot aos grupos existentes para que essa alteração entre em vigor.
</Warning>

### Configurações de Grupo

Configure o comportamento do grupo na aba **IM Integrations** do seu Painel:

| Configuração | Opções | Descrição |
|---------|---------|-------------|
| **Group Policy** | Open (padrão) / Allowlist / Disabled | Controla quem pode acionar o bot em grupos |
| **Require @mention** | On / Off | O bot responde apenas quando mencionado com @ (recomendado) |
| **Group IDs** | Lista de IDs de chat | Restringir a grupos específicos (vazio = todos os grupos) |
| **Group Allowed Users** | Lista de IDs de usuário | Apenas estes usuários podem acionar o bot (modo allowlist) |

<Tip>
**Group Policy** controla *quem* pode enviar mensagens para o bot em grupos, enquanto **Group IDs** controla em *quais* grupos o bot responde. Use ambos juntos para um controle detalhado.
</Tip>

## Política de DM

Controle quem pode enviar mensagens diretas para o seu bot:

| Política | Comportamento |
|--------|----------|
| **Pairing** (padrão) | Novos usuários recebem um código de pareamento — insira-o na WebUI para autorizar |
| **Allowlist** | Apenas os IDs de usuário que você especificar podem enviar mensagens para o bot |
| **Open** | Qualquer pessoa pode enviar mensagens para o bot (use com cautela — consome seus créditos de IA) |

## Solução de Problemas

<AccordionGroup>
  <Accordion title="O bot não responde">
    - Verifique se o bot token está correto (sem espaços extras)
    - Verifique se sua instância está em execução (status verde no painel)
    - Tente reiniciar a instância pelo painel
  </Accordion>
  <Accordion title="'Pairing code' não aparece">
    - Certifique-se de que está enviando uma **mensagem direta** para o bot, não em um grupo
    - O sistema de pareamento é ativado na primeira mensagem de um novo usuário
  </Accordion>
  <Accordion title="O bot responde lentamente">
    - O tempo de resposta depende do modelo de IA selecionado
    - Modelos maiores (Claude Opus, GPT-5) podem levar alguns segundos
    - Verifique seu saldo na LemonData — créditos insuficientes causarão erros
  </Accordion>
</AccordionGroup>
