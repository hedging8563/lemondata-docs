## Visão Geral

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) é a ferramenta de CLI oficial da Anthropic para codificação assistida por IA. Você pode configurá-la para usar o LemonData como provedor de API.

## Configuração

### Variáveis de Ambiente

Defina as seguintes variáveis de ambiente:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Adicione estas ao seu perfil do shell (`~/.bashrc`, `~/.zshrc`, etc.) para persistência.

### Arquivo de Configuração

Alternativamente, crie ou edite `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Uso

Após a configuração, use o Claude Code normalmente:

```bash
# Iniciar uma sessão interativa
claude

# Executar um único comando
claude "Explain this code" < main.py

# Modo chat
claude chat
```

## Modelos Disponíveis

O LemonData suporta todos os modelos Claude:

| Modelo | Descrição |
|-------|-------------|
| `claude-opus-4-5` | Mais capaz, raciocínio complexo |
| `claude-sonnet-4-5` | Desempenho equilibrado, excelente para codificação |
| `claude-haiku-4-5` | Mais rápido, custo-benefício |

## Seleção de Modelo

Especifique um modelo com a flag `--model`:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Ou defina um padrão em sua configuração:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Extended Thinking

Para tarefas de raciocínio complexas, use modelos com extended thinking:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Erros de conexão">
    Verifique se a URL base é exatamente `https://api.lemondata.cc` (sem o sufixo `/v1` para o Anthropic SDK).
  </Accordion>

  <Accordion title="Chave de API inválida">
    Certifique-se de que sua chave de API do LemonData comece com `sk-` e tenha saldo suficiente.
  </Accordion>

  <Accordion title="Modelo não encontrado">
    Verifique se o nome do modelo está correto. Use o formato `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
