## Visão Geral

A LemonData funciona com qualquer aplicativo de chat compatível com OpenAI. Este guia aborda a configuração para aplicativos de chat populares de desktop e dispositivos móveis.

## Aplicativos Suportados

| Aplicativo | Plataforma | Recursos |
|-------------|----------|----------|
| LobeChat | Web, Desktop, Auto-hospedado | Multi-provedor, plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Plugins avançados, múltiplos provedores |
| Chatbox | Windows, macOS, Linux | Leve, rápido |
| BotGem | iOS, macOS | Experiência Apple nativa |
| TypingMind | Web, Desktop | Personas personalizadas, plugins |
| OpenCat | iOS, macOS | Nativo, suporte offline |
| ChatWise | Windows, macOS | Colaboração em equipe |

## Passos de Configuração

### Passo 1: Obtenha sua Chave de API

1. Faça login no [Painel da LemonData](https://lemondata.cc/dashboard)
2. Navegue até [Chaves de API](https://lemondata.cc/dashboard/api)
3. Crie e copie sua chave de API (formato: `sk-...`)

### Passo 2: Configure seu Aplicativo

<Tabs>
  <Tab title="LobeChat">
    **Para LobeChat auto-hospedado:**

    Defina as variáveis de ambiente:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Para LobeChat Cloud:**
    1. Abra **Settings** → **AI Service Provider**
    2. Selecione **OpenAI**
    3. Insira sua chave de API da LemonData
    4. Defina o endpoint personalizado: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Abra **Settings** → **Model Provider**
    2. Clique em **Add Provider**
    3. Selecione **OpenAI API Compatible**
    4. Insira:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Adicione modelos: `gpt-4o`, `claude-sonnet-4-5`, etc.
    6. Salve e teste
  </Tab>
  <Tab title="Chatbox">
    1. Abra **Settings** → **AI Model**
    2. Selecione **OpenAI API Compatible**
    3. Insira:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Clique em **Verify** para testar a conexão
  </Tab>
  <Tab title="BotGem">
    1. Abra **Settings** → **Providers**
    2. Toque em **Add Custom Provider**
    3. Insira:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Adicione seus modelos preferidos
    5. Salve a configuração
  </Tab>
  <Tab title="TypingMind">
    1. Clique em **Settings** (ícone de engrenagem)
    2. Selecione **Custom Endpoint**
    3. Insira:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Configure os modelos e salve
  </Tab>
</Tabs>

### Passo 3: Adicione Modelos

Após configurar o provedor, adicione os modelos que deseja usar:

| Categoria | Modelos |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Referência de Configuração

| Campo | Valor |
|-------|-------|
| URL Base | `https://api.lemondata.cc/v1` |
| Chave de API | `sk-your-lemondata-key` |
| Tipo de API | Compatível com OpenAI |

**Configuração JSON (para aplicativos que a suportam):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Erro de Conexão">
    - Verifique se a URL Base é exatamente `https://api.lemondata.cc/v1`
    - Verifique se há barras no final (remova se houver)
    - Verifique a conectividade com a internet e as configurações de firewall
  </Accordion>

  <Accordion title="401 Não Autorizado">
    - Verifique novamente sua chave de API
    - Verifique se a chave está ativa no [Painel da LemonData](https://lemondata.cc/dashboard/api)
    - Certifique-se de que não há espaços extras na chave
  </Accordion>

  <Accordion title="402 Saldo Insuficiente">
    - Verifique o saldo da sua conta no painel
    - Adicione créditos à sua conta
  </Accordion>

  <Accordion title="404 Modelo Não Disponível">
    - Verifique se o nome do modelo está escrito corretamente
    - Verifique a disponibilidade do modelo em [lemondata.cc/en/models](https://lemondata.cc/pt/models)
    - Tente um modelo diferente
  </Accordion>

  <Accordion title="Respostas Lentas">
    - Tente um modelo mais rápido (ex: `gpt-4o-mini` em vez de `gpt-4o`)
    - Reduza o parâmetro `max_tokens` se disponível
    - Verifique sua conexão de rede
  </Accordion>
</AccordionGroup>

## Dicas

<AccordionGroup>
  <Accordion title="Use modelos apropriados">
    Combine o modelo com sua tarefa - use GPT-4o-mini para chats rápidos, Claude ou GPT-4o para tarefas complexas.
  </Accordion>

  <Accordion title="Salve prompts usados com frequência">
    A maioria dos aplicativos suporta salvar prompts ou personas. Crie modelos para tarefas comuns.
  </Accordion>

  <Accordion title="Monitore o uso">
    Verifique seu uso no painel da LemonData para acompanhar os custos e gerenciar seu orçamento.
  </Accordion>
</AccordionGroup>
