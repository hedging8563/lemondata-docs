## Visão Geral

O [Cursor](https://cursor.sh) é um editor de código baseado em IA. Você pode usar o LemonData como um provedor de API personalizado para acessar mais de 300 modelos.

## Configuração

1. Abra as Configurações do Cursor (`Cmd/Ctrl + ,`)
2. Navegue até **Models** na barra lateral esquerda
3. Role para baixo até a seção **OpenAI API Key**
4. Insira sua chave de API do LemonData
5. Ative **Override OpenAI Base URL**
6. Defina a Base URL para:

```
https://api.lemondata.cc/v1
```

## Seleção de Modelos

Após a configuração, você pode selecionar qualquer modelo do LemonData no Cursor:

### Modelos Recomendados para Codificação

| Modelo | Melhor Para |
|-------|----------|
| `claude-opus-4-5` | Codificação complexa, arquitetura |
| `claude-sonnet-4-5` | Codificação geral, depuração |
| `gpt-4o` | Geração de código, explicações |
| `deepseek-r1` | Tarefas com uso intensivo de raciocínio |
| `gpt-4o-mini` | Preenchimentos rápidos, tarefas simples |

## Recursos

### Chat

Use o painel de chat de IA (`Cmd/Ctrl + L`) com qualquer modelo do LemonData:

- Explicações de código
- Correção de bugs
- Sugestões de refatoração
- Geração de documentação

### Composer

O recurso Composer (`Cmd/Ctrl + I`) funciona com modelos do LemonData para:

- Edições em múltiplos arquivos
- Implementação de recursos
- Migrações de código

### Tab Completion

Para preenchimentos com tab, recomendamos o uso de modelos rápidos:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Modelos não aparecem">
    O Cursor armazena listas de modelos em cache. Tente reiniciar o Cursor após configurar a API.
  </Accordion>

  <Accordion title="Respostas lentas">
    Verifique sua seleção de modelos. Use modelos mais rápidos como `gpt-4o-mini` para tarefas rápidas.
  </Accordion>

  <Accordion title="Erros de autenticação">
    Verifique se sua chave de API está correta e se possui saldo suficiente.
  </Accordion>
</AccordionGroup>
