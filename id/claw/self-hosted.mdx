<Note>
**Self-hosted** berarti Anda menjalankan OpenClaw langsung di VPS atau mesin Anda sendiri. Anda hanya membayar penggunaan model AI melalui saldo LemonData — tanpa biaya instans.
</Note>

## Instalasi Cepat

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Non-interactive
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

Installer akan meminta **LemonData API Key** Anda. Dapatkan dari [Dashboard](https://lemondata.cc/dashboard).

## Yang Diinstal

<Steps>
  <Step title="Node.js 22+">
    Diinstal otomatis jika belum ada (melalui NodeSource di Linux, Homebrew di macOS)
  </Step>
  <Step title="OpenClaw">
    Paket OpenClaw resmi dari npm, sudah dikonfigurasi dengan LemonData API
  </Step>
  <Step title="15 Model AI">
    Claude, GPT-5, Gemini, DeepSeek, dan lainnya — siap digunakan
  </Step>
  <Step title="Plugin IM">
    Plugin DingTalk, WeCom, dan QQ sudah terinstal
  </Step>
  <Step title="Layanan Sistem">
    Otomatis berjalan saat boot (systemd di Linux, launchd di macOS)
  </Step>
</Steps>

## Perintah Manajemen

    ```bash
    journalctl --user -u openclaw -f    # Lihat log
    ```

    ```bash
    launchctl stop cc.lemondata.openclaw  # Hentikan
    ```

## Konfigurasi

Lokasi file konfigurasi: `~/.openclaw/openclaw.json`

Setelah mengedit, mulai ulang layanan untuk menerapkan perubahan.

### Ubah Model Default

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Prefiks model yang tersedia:
- `lemondata-claude/` — Model Claude (anthropic-messages API)
- `lemondata-minimax/` — Model MiniMax (anthropic-messages API)
- `lemondata/` — Semua model lainnya (openai-completions API)

## Upgrade

```bash
npm update -g openclaw
# Lalu mulai ulang layanan
systemctl --user restart openclaw  # Linux
# atau
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Hapus Instalasi

```bash
# Hentikan dan nonaktifkan layanan
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# Hapus OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```

## Self-Hosted vs Terkelola

| | Self-Hosted | Terkelola (Claw) |
|---|---|---|
| **Biaya instans** | Gratis | $20 / 30 hari |
| **Penggunaan AI** | Bayar sesuai pemakaian | Bayar sesuai pemakaian |
| **Infrastruktur** | Server Anda | Kami kelola semuanya |
| **Pembaruan** | Manual (`npm update`) | Otomatis |
| **Dukungan** | Komunitas | Prioritas |
