## Langkah 1: Dapatkan API Key Anda

<Steps>
  <Step title="Buat akun">
    Daftar di [lemondata.cc](https://lemondata.cc). Masuk dengan email, Gmail, atau GitHub.
  </Step>
  <Step title="Tambah kredit">
    Buka dashboard dan tambahkan kredit ke akun Anda. Harga pay-as-you-go tanpa minimum.
  </Step>
  <Step title="Buat API key">
    Buka **Dashboard â†’ API Keys** dan buat kunci baru. Salin dengan aman - kunci ini hanya ditampilkan sekali.
  </Step>
</Steps>

<Warning>
  Jaga keamanan API key Anda. Jangan pernah membagikannya di kode sisi klien atau repositori publik.
</Warning>

## Langkah 3: Buat Permintaan Pertama Anda

<CodeGroup>

```bash cURL
curl https://api.lemondata.cc/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "What is the capital of France?"}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is the capital of France?"}
    ]
)

print(response.choices[0].message.content)
# Output: The capital of France is Paris.
```
