## Ringkasan

LemonData menggunakan **harga pay-as-you-go**. Anda hanya membayar untuk apa yang Anda gunakan, tanpa langganan atau komitmen minimum.

## Cara Kerja Penagihan

1. **Tambahkan kredit** ke akun Anda
2. **Gunakan API** - biaya dipotong per permintaan
3. **Pantau penggunaan** di dashboard Anda
4. **Isi ulang** saat saldo Anda rendah

### Penagihan Tugas Asinkron (Video/Musik/3D)

<Warning>
  **Penting**: Tugas pembuatan video, musik, dan 3D **ditagih saat pengiriman**, bukan saat selesai.
</Warning>

Untuk tugas pembuatan asinkron (video, musik, model 3D):

1. **Penagihan terjadi segera** saat tugas dikirim
2. Anda menerima `task_id` untuk memeriksa status
3. **Tidak ada pengembalian dana otomatis** jika tugas gagal

Ini karena penyedia upstream menagih kami saat tugas dikirim, terlepas dari hasilnya. Jika Anda mengalami kegagalan tugas, silakan hubungi support@lemondata.cc.

```python
# Contoh: Penagihan pembuatan video
response = client.post("/v1/video/generations", json={
    "model": "kling-v1",
    "prompt": "Matahari terbenam di atas laut"
})
# ⚠️ Anda ditagih SEKARANG, bukan saat video siap

task_id = response.json()["task_id"]
# Periksa status...
```

## Penghitungan Token

Token adalah unit dasar pemrosesan teks:

- ~4 karakter = 1 token (Inggris)
- ~1-2 karakter = 1 token (Mandarin)
- 1 gambar = bervariasi berdasarkan ukuran dan detail
