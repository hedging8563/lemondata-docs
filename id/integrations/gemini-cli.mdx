## Ringkasan

Google Gemini CLI adalah alat baris perintah (command-line tool) untuk berinteraksi dengan model Gemini. LemonData menyediakan endpoint yang kompatibel yang memungkinkan Anda menggunakan Gemini CLI dengan akses ke 300+ model.

## Persyaratan Sistem

- **Node.js**: Versi 20.0+
- **OS**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, atau Debian 10+

## Instalasi

```bash
npm install -g @google/gemini-cli
```

Verifikasi instalasi:

```bash
gemini --version
```

## Konfigurasi

### Langkah 1: Dapatkan API Key Anda

1. Masuk ke [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navigasi ke [API Keys](https://lemondata.cc/dashboard/api)
3. Buat dan salin API key Anda (format: `sk-...`)

### Langkah 2: Atur Environment Variables

**Sementara (sesi saat ini):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**Konfigurasi permanen:**

Tambahkan ke file konfigurasi shell Anda:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
Mulai ulang terminal Anda setelah konfigurasi agar perubahan dapat diterapkan.
</Warning>

<Note>
**Batasan yang diketahui**: Dalam beberapa kasus, Gemini CLI mungkin tidak mematuhi `GOOGLE_GEMINI_BASE_URL` jika terdapat sesi Google yang tersimpan di cache. Jika Anda mengalami masalah koneksi, coba hapus cache CLI atau mulai sesi terminal baru.
</Note>
