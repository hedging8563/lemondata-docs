## Ringkasan

LemonData bekerja dengan aplikasi chat apa pun yang kompatibel dengan OpenAI. Panduan ini mencakup konfigurasi untuk aplikasi chat desktop dan seluler yang populer.

## Aplikasi yang Didukung

| Aplikasi | Platform | Fitur |
|-------------|----------|----------|
| LobeChat | Web, Desktop, Self-hosted | Multi-provider, plugin, RAG |
| CherryStudio | Windows, macOS, Linux | Plugin yang kaya, beberapa provider |
| Chatbox | Windows, macOS, Linux | Ringan, cepat |
| BotGem | iOS, macOS | Pengalaman Apple native |
| TypingMind | Web, Desktop | Persona kustom, plugin |
| OpenCat | iOS, macOS | Native, dukungan offline |
| ChatWise | Windows, macOS | Kolaborasi tim |

## Langkah-langkah Konfigurasi

### Langkah 1: Dapatkan API Key Anda

1. Masuk ke [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Buka [API Keys](https://lemondata.cc/dashboard/api)
3. Buat dan salin API key Anda (format: `sk-...`)

### Langkah 2: Konfigurasikan Aplikasi Anda

<Tabs>
  <Tab title="LobeChat">
    **Untuk LobeChat Self-hosted:**

    Atur environment variables:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Untuk LobeChat Cloud:**
    1. Buka **Settings** → **AI Service Provider**
    2. Pilih **OpenAI**
    3. Masukkan LemonData API key Anda
    4. Atur endpoint kustom: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Buka **Settings** → **Model Provider**
    2. Klik **Add Provider**
    3. Pilih **OpenAI API Compatible**
    4. Masukkan:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Tambahkan model: `gpt-4o`, `claude-sonnet-4-5`, dll.
    6. Simpan dan uji
  </Tab>
  <Tab title="Chatbox">
    1. Buka **Settings** → **AI Model**
    2. Pilih **OpenAI API Compatible**
    3. Masukkan:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Klik **Verify** untuk menguji koneksi
  </Tab>
  <Tab title="BotGem">
    1. Buka **Settings** → **Providers**
    2. Ketuk **Add Custom Provider**
    3. Masukkan:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Tambahkan model pilihan Anda
    5. Simpan konfigurasi
  </Tab>
  <Tab title="TypingMind">
    1. Klik **Settings** (ikon gerigi)
    2. Pilih **Custom Endpoint**
    3. Masukkan:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Konfigurasikan model dan simpan
  </Tab>
</Tabs>

### Langkah 3: Tambahkan Model

Setelah mengonfigurasi provider, tambahkan model yang ingin Anda gunakan:

| Kategori | Model |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Referensi Konfigurasi

| Bidang | Nilai |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API Type | OpenAI Compatible |

**Konfigurasi JSON (untuk aplikasi yang mendukungnya):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Pemecahan Masalah

<AccordionGroup>
  <Accordion title="Kesalahan Koneksi">
    - Verifikasi Base URL tepat `https://api.lemondata.cc/v1`
    - Periksa garis miring di akhir (hapus jika ada)
    - Verifikasi konektivitas internet dan pengaturan firewall
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Periksa kembali API key Anda
    - Verifikasi bahwa key aktif di [LemonData Dashboard](https://lemondata.cc/dashboard/api)
    - Pastikan tidak ada spasi tambahan pada key
  </Accordion>

  <Accordion title="402 Saldo Tidak Mencukupi">
    - Periksa saldo akun Anda di dashboard
    - Tambahkan kredit ke akun Anda
  </Accordion>

  <Accordion title="404 Model Tidak Tersedia">
    - Verifikasi nama model dieja dengan benar
    - Periksa ketersediaan model di [lemondata.cc/en/models](https://lemondata.cc/id/models)
    - Coba model lain
  </Accordion>

  <Accordion title="Respons Lambat">
    - Coba model yang lebih cepat (misalnya, `gpt-4o-mini` alih-alih `gpt-4o`)
    - Kurangi parameter `max_tokens` jika tersedia
    - Periksa koneksi jaringan Anda
  </Accordion>
</AccordionGroup>

## Tips

<AccordionGroup>
  <Accordion title="Gunakan model yang sesuai">
    Sesuaikan model dengan tugas Anda - gunakan GPT-4o-mini untuk chat cepat, Claude atau GPT-4o untuk tugas yang kompleks.
  </Accordion>

  <Accordion title="Simpan prompt yang sering digunakan">
    Sebagian besar aplikasi mendukung penyimpanan prompt atau persona. Buat templat untuk tugas-tugas umum.
  </Accordion>

  <Accordion title="Pantau penggunaan">
    Periksa penggunaan Anda di dashboard LemonData untuk melacak biaya dan mengelola anggaran Anda.
  </Accordion>
</AccordionGroup>
