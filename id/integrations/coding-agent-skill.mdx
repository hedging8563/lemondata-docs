## Langkah 3: Mulai Menggunakan

Menggunakan skill ini semudah melakukan chat!

### Contoh 1: Menggunakan GPT-4

**Anda berkata:**
```
I want to use GPT-4 in my Python project
```

**Claude Code akan:**
1. Meminta API Key Anda
2. Mencari API terkait GPT-4
3. Menampilkan opsi API yang tersedia
4. Menghasilkan kode Python yang lengkap
5. Menjelaskan cara menggunakannya

**Anda mendapatkan:**
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

### Manajemen Biaya

- Setiap panggilan API menghabiskan kredit
- Atur batas penggunaan di [dashboard](https://lemondata.cc/dashboard) Anda
- Gunakan volume data kecil untuk pengujian

### Penanganan Kesalahan

Kode yang dihasilkan mencakup penanganan kesalahan dasar, tetapi Anda mungkin ingin:
- Menambahkan mekanisme percobaan ulang (retry)
- Mencatat log kesalahan
- Menyediakan pesan kesalahan yang ramah pengguna
