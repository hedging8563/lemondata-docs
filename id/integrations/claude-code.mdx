## Ringkasan

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) adalah alat CLI resmi Anthropic untuk pengodean berbantuan AI. Anda dapat mengonfigurasinya untuk menggunakan LemonData sebagai penyedia API.

## Konfigurasi

### Variabel Lingkungan

Atur variabel lingkungan berikut:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Tambahkan ini ke profil shell Anda (`~/.bashrc`, `~/.zshrc`, dll.) untuk persistensi.

### Berkas Konfigurasi

Atau, buat atau edit `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Penggunaan

Setelah konfigurasi, gunakan Claude Code secara normal:

```bash
# Mulai sesi interaktif
claude

# Jalankan perintah tunggal
claude "Explain this code" < main.py

# Mode chat
claude chat
```

## Model yang Tersedia

LemonData mendukung semua model Claude:

| Model | Deskripsi |
|-------|-------------|
| `claude-opus-4-5` | Paling mumpuni, penalaran kompleks |
| `claude-sonnet-4-5` | Performa seimbang, sangat baik untuk pengodean |
| `claude-haiku-4-5` | Tercepat, hemat biaya |

## Pemilihan Model

Tentukan model dengan flag `--model`:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Atau atur default dalam konfigurasi Anda:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Extended Thinking

Untuk tugas penalaran yang kompleks, gunakan model dengan extended thinking:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Pemecahan Masalah

<AccordionGroup>
  <Accordion title="Kesalahan koneksi">
    Verifikasi bahwa URL dasar tepat `https://api.lemondata.cc` (tanpa akhiran `/v1` untuk SDK Anthropic).
  </Accordion>

  <Accordion title="Kunci API tidak valid">
    Pastikan kunci API LemonData Anda dimulai dengan `sk-` dan memiliki saldo yang cukup.
  </Accordion>

  <Accordion title="Model tidak ditemukan">
    Periksa apakah nama model sudah benar. Gunakan format `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
