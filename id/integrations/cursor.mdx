## Ringkasan

[Cursor](https://cursor.sh) adalah editor kode bertenaga AI. Anda dapat menggunakan LemonData sebagai penyedia API kustom untuk mengakses 300+ model.

## Konfigurasi

1. Buka Pengaturan Cursor (`Cmd/Ctrl + ,`)
2. Navigasikan ke **Models** di bilah sisi kiri
3. Gulir ke bawah ke bagian **OpenAI API Key**
4. Masukkan kunci API LemonData Anda
5. Aktifkan **Override OpenAI Base URL**
6. Atur Base URL ke:

```
https://api.lemondata.cc/v1
```

## Pemilihan Model

Setelah konfigurasi, Anda dapat memilih model LemonData apa pun di Cursor:

### Model yang Direkomendasikan untuk Coding

| Model | Terbaik Untuk |
|-------|----------|
| `claude-opus-4-5` | Coding kompleks, arsitektur |
| `claude-sonnet-4-5` | Coding umum, debugging |
| `gpt-4o` | Pembuatan kode, penjelasan |
| `deepseek-r1` | Tugas berat penalaran |
| `gpt-4o-mini` | Penyelesaian cepat, tugas sederhana |

## Fitur

### Chat

Gunakan panel chat AI (`Cmd/Ctrl + L`) dengan model LemonData apa pun:

- Penjelasan kode
- Perbaikan bug
- Saran refactoring
- Pembuatan dokumentasi

### Composer

Fitur Composer (`Cmd/Ctrl + I`) berfungsi dengan model LemonData untuk:

- Pengeditan multi-file
- Implementasi fitur
- Migrasi kode

### Tab Completion

Untuk tab completion, kami merekomendasikan penggunaan model yang cepat:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
