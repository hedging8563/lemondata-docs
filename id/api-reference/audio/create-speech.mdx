---

## Otorisasi

<ParamField header="Authorization" type="string" required>
  Token Bearer untuk autentikasi API. Format: `Bearer sk-your-api-key`
</ParamField>

## Body Permintaan

<ParamField body="model" type="string" default="tts-1">
  Model TTS: `tts-1` (standar) atau `tts-1-hd` (kualitas lebih tinggi).
</ParamField>

<ParamField body="input" type="string" required>
  Teks untuk menghasilkan audio. Maksimum 4096 karakter.
</ParamField>

<ParamField body="voice" type="string" required>
  Suara yang digunakan: `alloy`, `ash`, `ballad`, `coral`, `echo`, `fable`, `nova`, `onyx`, `sage`, `shimmer`, `verse`.
</ParamField>

<ParamField body="response_format" type="string" default="mp3">
  Format audio: `mp3`, `opus`, `aac`, `flac`, `wav`, `pcm`.
</ParamField>

<ParamField body="speed" type="number" default="1.0">
  Kecepatan audio (0,25 hingga 4,0).
</ParamField>

## Respons

Mengembalikan file audio dalam format yang diminta.

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1-hd",
    "voice": "nova",
    "input": "Hello, welcome to LemonData!"
  }' \
  --output speech.mp3
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.audio.speech.create(
    model="tts-1-hd",
    voice="nova",
    input="Hello, welcome to LemonData!"
)

response.stream_to_file("speech.mp3")
```

```javascript JavaScript
import OpenAI from 'openai';
import fs from 'fs';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.audio.speech.create({
  model: 'tts-1-hd',
  voice: 'nova',
  input: 'Hello, welcome to LemonData!'
});

const buffer = Buffer.from(await response.arrayBuffer());
fs.writeFileSync('speech.mp3', buffer);
```
