---
title: "Buat Video"
openapi: "POST /v1/videos/generations"
description: "Membuat tugas pembuatan video"
---

## Otorisasi

<ParamField header="Authorization" type="string" required>
  Token Bearer untuk autentikasi API. Format: `Bearer sk-your-api-key`
</ParamField>

## Ringkasan

Pembuatan video bersifat asinkron. Anda mengirimkan permintaan dan menerima ID tugas, lalu melakukan polling untuk hasilnya.

## Body Permintaan

<ParamField body="model" type="string" default="kling-video">
  Model video (misalnya, `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Deskripsi teks dari video yang akan dibuat.
</ParamField>

<ParamField body="image_url" type="string">
  Gambar awal opsional untuk pembuatan image-to-video.
</ParamField>

<ParamField body="duration" type="integer">
  Durasi video dalam detik (tergantung model).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Rasio aspek (misalnya, `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Resolusi video (misalnya, `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Frame per detik (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Apa yang harus dihindari dalam pembuatan video.
</ParamField>

<ParamField body="seed" type="integer">
  Seed acak untuk pembuatan yang dapat direproduksi.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Skala classifier-free guidance (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Intensitas gerakan (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL gambar frame awal.
</ParamField>

<ParamField body="end_image" type="string">
  URL gambar frame akhir.
</ParamField>

<ParamField body="user" type="string">
  Pengidentifikasi unik untuk pengguna akhir.
</ParamField>

## Respons

<ResponseField name="task_id" type="string">
  Pengidentifikasi tugas unik untuk polling.
</ResponseField>

<ResponseField name="status" type="string">
  Status awal: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Model yang digunakan.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",

    }
)
```

## Model yang Tersedia

| Model | Tipe | Deskripsi |
|-------|------|-------------|
| `kling-video` | T2V/I2V | Gerakan realistis, beberapa versi (v1.0-v2.6) |
| `veo3.1` | T2V | Terbaru dari Google |
| `veo3.1-pro` | T2V | Kualitas profesional |
| `minimax/video-01` | T2V | Ritme dan stabilitas tinggi |
| `wan2.6-i2v` | I2V | Gambar ke video |
