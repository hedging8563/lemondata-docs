---
title: "Edit Gambar"
api: "POST /v1/images/edits"
description: "Mengedit gambar berdasarkan prompt dan gambar sumber"
---

## Ikhtisar

Membuat gambar yang diedit atau diperluas berdasarkan gambar asli dan prompt. Memerlukan content type `multipart/form-data`.

## Isi Permintaan

<ParamField body="image" type="file" required>
  Gambar yang akan diedit. Harus berupa file PNG yang valid, kurang dari 4MB, dan berbentuk persegi. Gambar harus memiliki transparansi di area yang ingin diedit.
</ParamField>

<ParamField body="prompt" type="string" required>
  Deskripsi teks dari edit yang diinginkan.
</ParamField>

<ParamField body="mask" type="file">
  Gambar tambahan yang area transparan sepenuhnya menunjukkan di mana gambar harus diedit. Harus berupa file PNG yang valid, kurang dari 4MB, dan memiliki dimensi yang sama dengan `image`.
</ParamField>

<ParamField body="model" type="string" default="dall-e-2">
  Model yang digunakan untuk edit gambar. Saat ini hanya `dall-e-2` yang didukung.
</ParamField>

<ParamField body="n" type="integer" default="1">
  Jumlah gambar yang akan dihasilkan. Harus antara 1 dan 10.
</ParamField>

<ParamField body="size" type="string">
  Ukuran gambar yang dihasilkan. Harus salah satu dari `256x256`, `512x512`, atau `1024x1024`.
</ParamField>

<ParamField body="response_format" type="string" default="url">
  Format pengembalian gambar yang dihasilkan. Harus `url` atau `b64_json`.
</ParamField>

<ParamField body="user" type="string">
  Pengidentifikasi unik yang mewakili pengguna akhir Anda untuk pemantauan penyalahgunaan.
</ParamField>

## Respons

<ResponseField name="created" type="integer">
  Timestamp Unix saat gambar dibuat.
</ResponseField>

<ResponseField name="data" type="array">
  Array gambar yang dihasilkan.

  Setiap objek berisi:
  - `url` (string): URL gambar yang diedit (jika response_format adalah `url`)
  - `b64_json` (string): Gambar yang dikodekan Base64 (jika response_format adalah `b64_json`)
</ResponseField>

```go Go
package main

import (
    "bytes"
    "fmt"
    "io"
    "mime/multipart"
    "net/http"
    "os"
)

func main() {
    body := &bytes.Buffer{}
    writer := multipart.NewWriter(body)

    image, _ := os.Open("sunlit_lounge.png")
    defer image.Close()
    part, _ := writer.CreateFormFile("image", "sunlit_lounge.png")
    io.Copy(part, image)

    mask, _ := os.Open("mask.png")
    defer mask.Close()
    maskPart, _ := writer.CreateFormFile("mask", "mask.png")
    io.Copy(maskPart, mask)

    writer.WriteField("prompt", "A sunlit indoor lounge area with a pool")
    writer.WriteField("n", "1")
    writer.WriteField("size", "1024x1024")
    writer.Close()

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/images/edits", body)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", writer.FormDataContentType())

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

## Catatan

<Note>
  Edit gambar hanya tersedia dengan DALL-E 2. Gambar harus memiliki area transparan yang menunjukkan di mana edit harus diterapkan, atau Anda dapat menyediakan gambar mask terpisah.
</Note>
