---
title: "Dapatkan Status Gambar"
api: "GET /v1/images/generations/{task_id}"
description: "Mendapatkan status dan hasil dari tugas pembuatan gambar asinkron"
---

## Parameter Path

<ParamField path="task_id" type="string" required>
  ID tugas yang dikembalikan dari permintaan pembuatan gambar awal.
</ParamField>

## Respons

<ResponseField name="created" type="integer">
  Timestamp Unix saat pembuatan.
</ResponseField>

<ResponseField name="task_id" type="string">
  Pengidentifikasi tugas.
</ResponseField>

<ResponseField name="status" type="string">
  Status tugas: `pending`, `processing`, `completed`, atau `failed`.
</ResponseField>

<ResponseField name="data" type="array">
  Array gambar yang dihasilkan (diisi ketika `status` adalah `completed`).

  Setiap objek berisi:
  - `url` (string): URL gambar yang dihasilkan
  - `revised_prompt` (string): Prompt yang digunakan untuk pembuatan
</ResponseField>

<ResponseField name="error" type="string">
  Pesan error (hanya ada ketika `status` adalah `failed`).
</ResponseField>

```python Python
import requests

response = requests.get(
    "https://api.lemondata.cc/v1/images/generations/img_abc123def456",
    headers={"Authorization": "Bearer sk-your-api-key"}
)

data = response.json()
print(f"Status: {data['status']}")

if data["status"] == "completed":
    print(f"Image URL: {data['data'][0]['url']}")
```

## Praktik Terbaik Polling

<Tip>
**Interval polling yang disarankan**: 3-5 detik. Sebagian besar tugas pembuatan gambar selesai dalam 30-120 detik tergantung pada model.
</Tip>

```python
import requests
import time

def poll_image_task(task_id, api_key, max_wait=300, interval=3):
    """Polling untuk hasil pembuatan gambar dengan timeout."""
    url = f"https://api.lemondata.cc/v1/images/generations/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        if data["status"] == "completed":
            return data["data"][0]["url"]
        elif data["status"] == "failed":
            raise Exception(data.get("error", "Pembuatan gagal"))

        time.sleep(interval)

    raise TimeoutError(f"Tugas {task_id} tidak selesai dalam {max_wait} detik")

# Penggunaan
image_url = poll_image_task("img_abc123def456", "sk-your-api-key")
print(f"Gambar yang dihasilkan: {image_url}")
```
