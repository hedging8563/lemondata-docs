---
title: "Buat Variasi Gambar"
openapi: "POST /v1/images/variations"
description: "Membuat variasi dari gambar yang diberikan"
---

## Otorisasi

<ParamField header="Authorization" type="string" required>
  Token Bearer untuk autentikasi API. Format: `Bearer sk-your-api-key`
</ParamField>

## Ikhtisar

Membuat variasi dari gambar yang diberikan. Memerlukan content type `multipart/form-data`.

## Isi Permintaan

<ParamField body="image" type="file" required>
  Gambar yang digunakan sebagai dasar untuk variasi. Harus berupa file PNG yang valid, kurang dari 4MB, dan berbentuk persegi.
</ParamField>

<ParamField body="model" type="string" default="dall-e-2">
  Model yang digunakan untuk variasi gambar. Saat ini hanya `dall-e-2` yang didukung.
</ParamField>

<ParamField body="n" type="integer" default="1">
  Jumlah gambar yang akan dihasilkan. Harus antara 1 dan 10.
</ParamField>

<ParamField body="size" type="string">
  Ukuran gambar yang dihasilkan. Harus salah satu dari `256x256`, `512x512`, atau `1024x1024`.
</ParamField>

<ParamField body="response_format" type="string" default="url">
  Format pengembalian gambar yang dihasilkan. Harus `url` atau `b64_json`.
</ParamField>

<ParamField body="user" type="string">
  Pengidentifikasi unik yang mewakili pengguna akhir Anda untuk pemantauan penyalahgunaan.
</ParamField>

## Respons

<ResponseField name="created" type="integer">
  Timestamp Unix saat gambar dibuat.
</ResponseField>

<ResponseField name="data" type="array">
  Array variasi gambar yang dihasilkan.

  Setiap objek berisi:
  - `url` (string): URL gambar variasi (jika response_format adalah `url`)
  - `b64_json` (string): Gambar yang dikodekan Base64 (jika response_format adalah `b64_json`)
</ResponseField>

```go Go
package main

import (
    "bytes"
    "fmt"
    "io"
    "mime/multipart"
    "net/http"
    "os"
)

func main() {
    body := &bytes.Buffer{}
    writer := multipart.NewWriter(body)

    file, _ := os.Open("cat.png")
    defer file.Close()
    part, _ := writer.CreateFormFile("image", "cat.png")
    io.Copy(part, file)

    writer.WriteField("n", "2")
    writer.WriteField("size", "1024x1024")
    writer.Close()

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/images/variations", body)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", writer.FormDataContentType())

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

}

## Catatan

<Note>
  Variasi gambar hanya tersedia dengan DALL-E 2. Model akan menghasilkan gambar baru yang mirip dalam gaya dan konten dengan gambar asli.
</Note>
