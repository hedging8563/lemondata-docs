---
title: "Dapatkan Model"
openapi: "GET /v1beta/models/{model}"
description: "Dapatkan metadata model menggunakan format Google Gemini API"
---

Mengembalikan metadata untuk model tertentu dalam format Google Gemini API.

## Parameter Path

<ParamField path="model" type="string" required>
  Nama model (misalnya, `gemini-2.5-pro`, `gemini-2.5-flash`). Alias juga diterima.
</ParamField>

## Autentikasi

Opsional. Mendukung metode autentikasi yang sama dengan endpoint Gemini lainnya:
- parameter query `?key=YOUR_API_KEY`
- header `x-goog-api-key: YOUR_API_KEY`
- header `Authorization: Bearer YOUR_API_KEY`

## Respons

<ResponseField name="name" type="string">
  Nama sumber daya model dalam format `models/{model}`.
</ResponseField>

<ResponseField name="displayName" type="string">
  Nama model yang dapat dibaca manusia.
</ResponseField>

<ResponseField name="inputTokenLimit" type="integer">
  Batas maksimum token input (jendela konteks).
</ResponseField>

<ResponseField name="outputTokenLimit" type="integer">
  Batas maksimum token output.
</ResponseField>

<ResponseField name="supportedGenerationMethods" type="array">
  Daftar metode generasi yang didukung (misalnya, `generateContent`, `countTokens`, `embedContent`).
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1beta/models/gemini-2.5-pro"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.lemondata.cc"}
)

model = genai.get_model("models/gemini-2.5-pro")
print(f"Name: {model.name}")
print(f"Input limit: {model.input_token_limit}")
print(f"Output limit: {model.output_token_limit}")
```
</RequestExample>
