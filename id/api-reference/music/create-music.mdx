---
title: "Buat Musik"
api: "POST /v1/music/generations"
description: "Membuat tugas pembuatan musik menggunakan Suno"
---

Hasilkan musik dan lirik menggunakan AI. Ini adalah API asinkron - Anda akan menerima ID tugas untuk melakukan polling penyelesaian.

## Body Permintaan

<ParamField body="model" type="string" default="suno_music">
  Model yang digunakan: `suno_music` untuk pembuatan musik, `suno_lyrics` hanya untuk lirik.
</ParamField>

<ParamField body="prompt" type="string" required>
  Deskripsi musik yang akan dihasilkan. Dapat mencakup gaya, suasana hati, instrumen, dll.
</ParamField>

<ParamField body="title" type="string">
  Judul untuk lagu yang dihasilkan.
</ParamField>

<ParamField body="tags" type="string">
  Tag gaya (misalnya, "pop, upbeat, electronic").
</ParamField>

<ParamField body="action" type="string">
  Tipe pembuatan: `MUSIC` (default) atau `LYRICS`.
</ParamField>

<ParamField body="mv" type="string">
  Versi model yang digunakan.
</ParamField>

<ParamField body="continue_clip_id" type="string">
  ID klip sebelumnya untuk dilanjutkan.
</ParamField>

<ParamField body="continue_at" type="number">
  Timestamp (dalam detik) untuk dilanjutkan.
</ParamField>

<ParamField body="user" type="string">
  Pengidentifikasi unik untuk pengguna akhir.
</ParamField>

## Respons

<ResponseField name="id" type="string">
  ID tugas untuk polling status.
</ResponseField>

<ResponseField name="status" type="string">
  Status tugas: `pending`, `processing`, `completed`, atau `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Unix timestamp pembuatan tugas.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/music/generations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "suno_music",
    "prompt": "An upbeat electronic dance track with heavy bass",
    "title": "Night Drive",
    "tags": "electronic, EDM, energetic"
  }'
```
