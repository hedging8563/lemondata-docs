---
title: "Rerank Dokumen"
openapi: "POST /v1/rerank"
description: "Mengurutkan ulang dokumen berdasarkan relevansi dengan kueri"
---

Gunakan model kemiripan semantik untuk mengurutkan ulang dokumen. Cocok untuk mengoptimalkan hasil pencarian dan aplikasi RAG.

## Request Body

<ParamField body="model" type="string" required>
  ID model rerank yang akan digunakan (contoh: `BAAI/bge-reranker-v2-m3`, `qwen3-rerank`).
</ParamField>

<ParamField body="query" type="string" required>
  Kueri yang digunakan untuk memberi peringkat pada dokumen.
</ParamField>

<ParamField body="documents" type="array" required>
  Daftar dokumen yang akan diurutkan ulang (string).
</ParamField>

<ParamField body="top_n" type="integer">
  Jumlah hasil teratas yang akan dikembalikan. Default adalah semua dokumen.
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  Apakah akan menyertakan teks dokumen asli dalam respons.
</ParamField>

## Respons

<ResponseField name="results" type="array">
  Daftar dokumen yang telah diurutkan dengan skor.

  Setiap hasil berisi:
  - `index` (integer): Indeks dokumen asli
  - `relevance_score` (number): Skor relevansi (0-1)
  - `document` (string): Teks asli (jika `return_documents=true`)
</ResponseField>

<ResponseField name="model" type="string">
  Model yang digunakan untuk rerank.
</ResponseField>

<ResponseField name="usage" type="object">
  Statistik penggunaan token.
</ResponseField>

curl -X POST "https://api.lemondata.cc/v1/rerank" \

  }'
