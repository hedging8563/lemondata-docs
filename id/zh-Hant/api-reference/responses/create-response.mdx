Responses API adalah API percakapan stateful terbaru dari OpenAI. LemonData menyediakan dukungan untuk format ini pada model yang kompatibel.

## Request Body

<ParamField body="model" type="string" required>
  ID model yang akan digunakan. Silakan merujuk ke [Models](https://lemondata.cc/zh-TW/models) untuk melihat opsi yang tersedia.
</ParamField>

<ParamField body="input" type="array" required>
  Daftar item input yang membentuk percakapan.

  Setiap item dapat berupa:
  - `message`: pesan percakapan yang berisi peran dan konten
  - `function_call`: permintaan panggilan fungsi
  - `function_call_output`: output dari panggilan fungsi
</ParamField>

<ParamField body="instructions" type="string">
  Instruksi sistem untuk model (setara dengan pesan sistem).
</ParamField>

<ParamField body="max_output_tokens" type="integer">
  Jumlah maksimum token yang akan dihasilkan.
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Suhu pengambilan sampel antara 0 hingga 2.
</ParamField>

<ParamField body="tools" type="array">
  Daftar alat yang mungkin dipanggil oleh model.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Jika true, mengembalikan aliran peristiwa (event stream).
</ParamField>

<ParamField body="previous_response_id" type="string">
  ID respons sebelumnya, digunakan untuk melanjutkan percakapan.
</ParamField>

<ParamField body="store" type="boolean" default="true">
  Apakah akan menyimpan respons untuk pengambilan nanti.
</ParamField>

<ParamField body="metadata" type="object">
  Metadata yang dilampirkan pada respons, digunakan untuk tujuan pelacakan.
</ParamField>

<ParamField body="text" type="object">
  Opsi konfigurasi pembuatan teks.
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  Apakah akan mengizinkan beberapa panggilan alat secara bersamaan.
</ParamField>

<ParamField body="top_p" type="number">
  Parameter nucleus sampling (0-1).
</ParamField>

<ParamField body="reasoning" type="object">
  Konfigurasi penalaran untuk model o1/o3.

  - `effort` (string): tingkat intensitas penalaran (`low`, `medium`, `high`)
</ParamField>

## Respons

<ResponseField name="id" type="string">
  Pengidentifikasi unik untuk respons.
</ResponseField>

<ResponseField name="object" type="string">
  Selalu bernilai `response`.
</ResponseField>

<ResponseField name="created_at" type="integer">
  Timestamp Unix saat respons dibuat.
</ResponseField>

<ResponseField name="output" type="array">
  Daftar item output yang dihasilkan oleh model.
</ResponseField>

<ResponseField name="usage" type="object">
  Statistik penggunaan token.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/responses" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "input": [
      {"type": "message", "role": "user", "content": "Hello!"}
    ],
    "max_output_tokens": 1000
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.responses.create(
    model="gpt-4o",
    input=[
        {"type": "message", "role": "user", "content": "Hello!"}
    ],
    max_output_tokens=1000
)

print(response.output)
```


```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model": "gpt-4o",
        "input": []map[string]interface{}{
            {"type": "message", "role": "user", "content": "Hello!"},
        },
        "max_output_tokens": 1000,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/responses", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result["output"])
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/responses');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'gpt-4o',
        'input' => [
            ['type' => 'message', 'role' => 'user', 'content' => 'Hello!']
        ],
        'max_output_tokens' => 1000
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
print_r($data['output']);
```
