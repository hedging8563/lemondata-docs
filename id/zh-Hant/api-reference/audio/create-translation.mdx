## Ikhtisar

Terjemahkan audio dari bahasa apa pun yang didukung ke dalam teks bahasa Inggris. Berbeda dengan transkripsi, endpoint ini selalu menghasilkan teks bahasa Inggris, apa pun bahasa inputnya.

## Request Body

<ParamField body="file" type="file" required>
  File audio yang akan diterjemahkan. Format yang didukung: `flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`. Ukuran file maksimum adalah 25 MB.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  Model yang akan digunakan. Saat ini hanya mendukung `whisper-1`.
</ParamField>

<ParamField body="prompt" type="string">
  Teks opsional untuk memandu gaya model atau melanjutkan segmen sebelumnya. Harus dalam bahasa Inggris.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  Format output. Opsi: `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number">
  Sampling temperature, antara 0 dan 1. Nilai yang lebih tinggi (seperti 0.8) akan menghasilkan output yang lebih acak, sedangkan nilai yang lebih rendah (seperti 0.2) akan membuat output lebih terfokus dan deterministik.
</ParamField>

## Respons

<ResponseField name="text" type="string">
  Teks bahasa Inggris yang telah diterjemahkan.
</ResponseField>

Untuk format `verbose_json`, respons juga mencakup:

<ResponseField name="language" type="string">
  Bahasa audio input yang terdeteksi.
</ResponseField>

<ResponseField name="duration" type="number">
  Durasi audio input (dalam detik).
</ResponseField>

<ResponseField name="segments" type="array">
  Segmen teks terjemahan dengan timestamp.
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"

with open("german_audio.mp3", "rb") as audio_file:
    response = client.audio.translations.create(
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/translations');

$file = new CURLFile('german_audio.mp3', 'audio/mpeg', 'german_audio.mp3');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => [
        'file' => $file,
        'model' => 'whisper-1'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['text'];
```

## Terjemahan vs Transkripsi

| Fitur | Terjemahan | Transkripsi |
|---------|-------------|---------------|
| Bahasa Output | Selalu bahasa Inggris | Sama dengan input |
| Kasus Penggunaan | Mengonversi audio bahasa asing ke bahasa Inggris | Mempertahankan bahasa asli |
| Parameter Bahasa | Tidak berlaku | Prompt opsional |

<Note>
  Endpoint terjemahan akan secara otomatis mendeteksi bahasa sumber dan menerjemahkannya ke dalam bahasa Inggris. Parameter `language` dalam transkripsi akan diabaikan.
</Note>
