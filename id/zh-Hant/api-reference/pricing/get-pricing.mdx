title: "Dapatkan Harga"
openapi: "GET /v1/pricing"
description: "Dapatkan informasi harga untuk model"
---

## Parameter Query

<ParamField query="model" type="string">
  ID model untuk mendapatkan harga. Jika tidak ditentukan, harga untuk semua model akan dikembalikan.
</ParamField>

## Respons

<ResponseField name="object" type="string">
  Selalu bernilai `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Array dari objek harga model.

  Setiap objek berisi:
  - `model` (string): Identifikasi model
  - `pricing` (object): Informasi harga
    - `input_per_1m` (string | null): Harga dalam USD per 1M input tokens
    - `output_per_1m` (string | null): Harga dalam USD per 1M output tokens
    - `per_request` (string | null): Harga dalam USD per permintaan (berlaku untuk model dengan harga tetap)
    - `is_lock_price` (boolean): Apakah harga tetap per permintaan
    - `currency` (string): Selalu bernilai `USD`
  - `provider` (string): Penyedia model
</ResponseField>

## Tipe Harga

| Tipe | Deskripsi | Contoh Model |
|------|-------------|----------------|
| **Berbasis Token** | Ditagih berdasarkan input/output tokens | GPT-4o, Claude, Gemini |
| **Per Permintaan** | Harga tetap per permintaan | DALL-E 3, Sora, Suno |
