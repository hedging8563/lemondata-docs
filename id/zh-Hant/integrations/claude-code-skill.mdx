## Langkah 3: Mulai Menggunakan

Menggunakan skill ini semudah mengobrol!

### Contoh 1: Menggunakan GPT-4

**Anda berkata:**
```
I want to use GPT-4 in my Python project
```

**Claude Code akan:**
1. Menanyakan API Key Anda
2. Mencari API yang terkait dengan GPT-4
3. Menampilkan opsi API yang tersedia
4. Menghasilkan kode Python yang lengkap
5. Menjelaskan cara menggunakannya

**Anda mendapatkan:**
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

### Keamanan API Key

  <Card title="âŒ Jangan pernah lakukan ini" icon="xmark" color="#ef4444">
    - Menaruh API Key langsung dalam kode frontend
    - Mengirimkan API Key ke GitHub

  <Card title="âœ… Praktik Terbaik" icon="check" color="#22c55e">

### Manajemen Biaya

- Setiap pemanggilan API akan mengonsumsi kuota
- Atur batasan penggunaan di [Dashboard](https://lemondata.cc/dashboard) Anda
- Gunakan volume data kecil saat melakukan pengujian

### Penanganan Kesalahan

Kode yang dihasilkan menyertakan penanganan kesalahan dasar, tetapi Anda mungkin juga perlu:
- Menambahkan mekanisme percobaan ulang (retry)
- Mencatat log kesalahan
- Menyediakan pesan kesalahan yang ramah pengguna
