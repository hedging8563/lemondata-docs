## Ikhtisar

OpenCode adalah asisten pengodean AI sumber terbuka yang berjalan di terminal. Ini mendukung beberapa penyedia LLM dan dapat dikonfigurasi untuk menggunakan API LemonData guna mengakses lebih dari 300 model.

## Instalasi

<Tabs>
  <Tab title="Skrip Instalasi">
    ```bash
    curl -fsSL https://opencode.ai/install | bash
    ```
  </Tab>
  <Tab title="Homebrew">
    ```bash
    brew install opencode-ai/tap/opencode
    ```

    Atau gunakan tap komunitas (diperbarui lebih sering):
    ```bash
    brew install anomalyco/tap/opencode
    ```
  </Tab>
  <Tab title="Go">
    ```bash
    go install github.com/opencode-ai/opencode@latest
    ```
  </Tab>
</Tabs>

Verifikasi instalasi:

```bash
opencode --version
```

## Konfigurasi

### Langkah 1: Atur variabel lingkungan

```bash
export OPENAI_API_KEY="sk-your-lemondata-key"
export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"
```

Untuk konfigurasi permanen, tambahkan ke `~/.bashrc` atau `~/.zshrc`:

```bash
echo 'export OPENAI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
echo 'export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"' >> ~/.zshrc
source ~/.zshrc
```

### Langkah 2: Konfigurasi OpenCode

Buat atau edit `~/.config/opencode/opencode.json` (global) atau `opencode.json` di direktori akar proyek Anda:

```json
{
  "provider": {
    "openai": {
      "options": {
        "apiKey": "sk-your-lemondata-key",
        "baseURL": "https://api.lemondata.cc/v1"
      }
    }
  },
  "model": "gpt-4o",
  "agent": {
    "coder": {
      "model": "gpt-4o"
    }
  }
}
```

## Penggunaan Dasar

**Mulai mode interaktif:**

```bash
opencode
```

**Jalankan dengan prompt:**

```bash
opencode "Explain this codebase"
```

**Tentukan model:**

```bash
opencode --model gpt-4o "Fix the bugs in main.py"
```

## Model yang Tersedia

| Model | Terbaik untuk |
|-------|----------|
| `gpt-4o` | Tugas kompleks, arsitektur kode |
| `gpt-4o-mini` | Perbaikan cepat, kueri sederhana |
| `claude-sonnet-4-5` | Peninjauan kode, penulisan dokumentasi |
| `claude-opus-4-5` | Penalaran kompleks |
| `gemini-2.5-flash` | Respons cepat |
| `deepseek-r1` | Desain algoritma |

## Perintah Umum

**Analisis kode:**

```bash
opencode "What does this function do?" < src/utils.ts
```

**Hasilkan kode:**

```bash
opencode "Create a REST API with Express"
```

**Tinjau perubahan:**

```bash
git diff | opencode "Review these changes"
```

**Perbaiki kesalahan:**

```bash
opencode "Fix the TypeScript errors in this project"
```

## Perintah Interaktif

| Perintah | Deskripsi |
|---------|-------------|
| `/help` | Tampilkan perintah yang tersedia |
| `/model <name>` | Beralih ke model yang berbeda |
| `/clear` | Bersihkan riwayat percakapan |
| `/exit` | Keluar dari OpenCode |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Kesalahan Koneksi">
    - Verifikasi bahwa `LOCAL_ENDPOINT` telah diatur ke `https://api.lemondata.cc/v1`
    - Periksa koneksi jaringan
    - Coba uji menggunakan `curl https://api.lemondata.cc/v1/models`
  </Accordion>

  <Accordion title="Autentikasi Gagal">
    - Verifikasi bahwa variabel lingkungan `OPENAI_API_KEY` telah diatur
    - Periksa apakah kunci dimulai dengan `sk-`
    - Pastikan kunci dalam keadaan aktif di dasbor LemonData
  </Accordion>

  <Accordion title="Model Tidak Ditemukan">
    - Periksa apakah nama model dalam konfigurasi cocok sepenuhnya
    - Verifikasi ketersediaan model di [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
  </Accordion>
</AccordionGroup>
