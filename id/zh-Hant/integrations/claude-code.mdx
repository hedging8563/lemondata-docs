## Ikhtisar

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) adalah alat CLI pemrograman berbantuan AI resmi dari Anthropic. Anda dapat mengonfigurasinya untuk menggunakan LemonData sebagai penyedia API.

## Konfigurasi

### Variabel Lingkungan

Atur variabel lingkungan berikut:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Tambahkan baris ini ke file profil shell Anda (`~/.bashrc`, `~/.zshrc`, dll.) agar perubahan bersifat permanen.

### File Konfigurasi

Atau, buat atau edit `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Cara Penggunaan

Setelah konfigurasi selesai, Anda dapat menggunakan Claude Code seperti biasa:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Model yang Tersedia

LemonData mendukung semua model Claude:

| Model | Deskripsi |
|-------|-------------|
| `claude-opus-4-5` | Model paling kuat, cocok untuk penalaran kompleks |
| `claude-sonnet-4-5` | Performa seimbang, sangat cocok untuk pemrograman |
| `claude-haiku-4-5` | Kecepatan tercepat, hemat biaya |

## Pemilihan Model

Gunakan flag `--model` untuk menentukan model:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Atau atur nilai default dalam konfigurasi Anda:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Pemikiran yang Diperluas

Untuk tugas penalaran yang kompleks, gunakan model dengan fitur pemikiran yang diperluas (extended thinking):

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
