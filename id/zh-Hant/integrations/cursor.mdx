## Ikhtisar

[Cursor](https://cursor.sh) adalah editor kode bertenaga AI. Anda dapat menggunakan LemonData sebagai penyedia API kustom untuk mengakses lebih dari 300 model.

## Konfigurasi

1. Buka pengaturan Cursor (`Cmd/Ctrl + ,`)
2. Navigasi ke **Models** di bilah sisi kiri
3. Gulir ke bawah ke bagian **OpenAI API Key**
4. Masukkan API key LemonData Anda
5. Aktifkan **Override OpenAI Base URL**
6. Atur Base URL ke:

```
https://api.lemondata.cc/v1
```

## Pemilihan Model

Setelah konfigurasi selesai, Anda dapat memilih model LemonData apa pun di Cursor:

### Model penulisan kode yang direkomendasikan

| Model | Terbaik untuk |
|-------|----------|
| `claude-opus-4-5` | Pengkodean kompleks, desain arsitektur |
| `claude-sonnet-4-5` | Pengkodean umum, debugging |
| `gpt-4o` | Generasi kode, penjelasan |
| `deepseek-r1` | Tugas penalaran berat |
| `gpt-4o-mini` | Penyelesaian cepat, tugas sederhana |

## Fitur

### Chat

Gunakan panel obrolan AI (`Cmd/Ctrl + L`) dengan model LemonData apa pun:

- Penjelasan kode
- Perbaikan Bug
- Saran refactoring
- Generasi dokumentasi

### Composer

Fitur Composer (`Cmd/Ctrl + I`) dapat digunakan dengan model LemonData untuk:

- Pengeditan multi-file
- Implementasi fitur
- Migrasi kode

### Penyelesaian Tab

Untuk penyelesaian Tab, kami merekomendasikan penggunaan model yang cepat:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
