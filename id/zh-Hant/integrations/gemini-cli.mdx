## Ikhtisar

Google Gemini CLI adalah alat baris perintah untuk berinteraksi dengan model Gemini. LemonData menyediakan endpoint yang kompatibel, memungkinkan Anda menggunakan Gemini CLI dan mengakses lebih dari 300 model.

## Persyaratan Sistem

- **Node.js**: Versi 20.0+
- **Sistem Operasi**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, atau Debian 10+

## Instalasi

```bash
npm install -g @google/gemini-cli
```

Verifikasi instalasi:

```bash
gemini --version
```

## Konfigurasi

### Langkah 1: Dapatkan API Key Anda

1. Login ke [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navigasi ke [API Keys](https://lemondata.cc/dashboard/api)
3. Buat dan salin API Key Anda (format: `sk-...`)

### Langkah 2: Atur Variabel Lingkungan

**Sementara (sesi saat ini):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**Konfigurasi Permanen:**

Tambahkan ke file konfigurasi shell Anda:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
Silakan mulai ulang terminal setelah konfigurasi selesai agar perubahan dapat diterapkan.
</Warning>

<Note>
**Batasan yang Diketahui**: Dalam beberapa kasus, Gemini CLI mungkin tidak mengikuti `GOOGLE_GEMINI_BASE_URL` jika terdapat sesi Google yang tersimpan di cache. Jika Anda mengalami masalah koneksi, coba hapus cache CLI atau mulai sesi terminal baru.
</Note>

## Penggunaan Dasar

Jalankan Gemini CLI dari direktori proyek Anda:

```bash
cd your-project
gemini
```

Saat pertama kali dijalankan, Anda perlu:
1. Pilih tema
2. Konfirmasi pemberitahuan keamanan
3. Percayai direktori kerja

```bash
# Uji koneksi

```

## Penyelesaian Masalah

<AccordionGroup>
  <Accordion title="Kesalahan Koneksi">
    - Verifikasi bahwa `GOOGLE_GEMINI_BASE_URL` telah diatur ke `https://api.lemondata.cc`
    - Catatan: Endpoint Gemini tidak memerlukan akhiran `/v1`
    - Periksa koneksi internet
  </Accordion>

  <Accordion title="Autentikasi Gagal">
    - Verifikasi bahwa variabel lingkungan `GEMINI_API_KEY` telah diatur
    - Periksa apakah Key dimulai dengan `sk-`
    - Pastikan Key tersebut aktif di dashboard LemonData
  </Accordion>

  <Accordion title="Model Tidak Tersedia">
    - Periksa ketersediaan model Gemini di [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - Coba varian model Gemini yang berbeda
  </Accordion>
</AccordionGroup>
