## Ikhtisar

LemonData **kompatibel dengan OpenAI**, yang berarti Anda hanya perlu mengubah base URL untuk menggunakan SDK OpenAI apa pun. Kami juga mendukung SDK Anthropic dan Google Gemini asli.

## SDK Resmi

## Konfigurasi

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

# 使用任何模型
response = client.chat.completions.create(
    model="gpt-4o",  # 或 claude-sonnet-4-5, gemini-2.5-flash 等
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### JavaScript / TypeScript

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }]
});
```
