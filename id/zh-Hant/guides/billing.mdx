## Ikhtisar

LemonData menggunakan model **pay-as-you-go**. Anda hanya membayar untuk apa yang Anda gunakan, tanpa langganan atau komitmen pengeluaran minimum.

## Cara Kerja Penagihan

1. **Isi ulang saldo** ke akun Anda
2. **Gunakan API** - biaya dipotong per permintaan
3. **Pantau penggunaan** di dasbor Anda
4. **Isi ulang saat saldo rendah**

## Penghitungan Token

Token adalah unit dasar pemrosesan teks:

- ~4 karakter = 1 token (Bahasa Inggris)
- ~1-2 karakter = 1 token (Bahasa Mandarin)
- 1 gambar = tergantung pada ukuran dan detail

### Estimasi Token

```python

    return len(text) / 4  # Approximate for English
```
