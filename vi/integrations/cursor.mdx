## Tổng quan

[Cursor](https://cursor.sh) là một trình soạn thảo mã nguồn được hỗ trợ bởi AI. Bạn có thể sử dụng LemonData như một nhà cung cấp API tùy chỉnh để truy cập hơn 300 mô hình.

## Cấu hình

1. Mở Cài đặt Cursor (`Cmd/Ctrl + ,`)
2. Đi đến mục **Models** ở thanh bên trái
3. Cuộn xuống phần **OpenAI API Key**
4. Nhập API key của LemonData
5. Bật tùy chọn **Override OpenAI Base URL**
6. Thiết lập Base URL thành:

```
https://api.lemondata.cc/v1
```

## Lựa chọn mô hình

Sau khi cấu hình, bạn có thể chọn bất kỳ mô hình LemonData nào trong Cursor:

### Các mô hình được đề xuất cho lập trình

| Mô hình | Tốt nhất cho |
|-------|----------|
| `claude-opus-4-5` | Lập trình phức tạp, kiến trúc |
| `claude-sonnet-4-5` | Lập trình tổng quát, gỡ lỗi |
| `gpt-4o` | Tạo mã, giải thích mã |
| `deepseek-r1` | Các tác vụ yêu cầu suy luận cao |
| `gpt-4o-mini` | Hoàn thành nhanh, các tác vụ đơn giản |

## Tính năng

### Chat

Sử dụng bảng chat AI (`Cmd/Ctrl + L`) với bất kỳ mô hình LemonData nào:

- Giải thích mã
- Sửa lỗi
- Gợi ý tái cấu trúc mã
- Tạo tài liệu

### Composer

Tính năng Composer (`Cmd/Ctrl + I`) hoạt động với các mô hình LemonData cho:

- Chỉnh sửa nhiều tệp
- Triển khai tính năng
- Di chuyển mã nguồn

### Tab Completion

Đối với tính năng tự động hoàn thành (tab completion), chúng tôi khuyên dùng các mô hình tốc độ nhanh:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
