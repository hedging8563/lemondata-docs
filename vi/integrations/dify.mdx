## Tổng quan

Dify là một nền tảng phát triển ứng dụng LLM mã nguồn mở. Nó cung cấp khả năng điều phối prompt trực quan, pipeline RAG, agent framework và các tính năng LLMOps. LemonData có thể được cấu hình như một nhà cung cấp mô hình tùy chỉnh trong Dify.

## Lợi ích

- Truy cập hơn 300 mô hình AI thông qua một giao diện duy nhất
- Chuyển đổi giữa các mô hình mà không cần thay đổi logic ứng dụng
- Tối ưu hóa chi phí bằng cách chọn mô hình tốt nhất cho từng tác vụ
- Thanh toán và theo dõi mức độ sử dụng tập trung

## Điều kiện tiên quyết

- Tài khoản LemonData có quyền truy cập API
- Cài đặt Dify (bản cloud hoặc tự lưu trữ)

## Các bước cấu hình

### Bước 1: Lấy API Key của bạn

1. Đăng nhập vào [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Đi tới [API Keys](https://lemondata.cc/dashboard/api)
3. Tạo và sao chép API key của bạn (định dạng: `sk-...`)

### Bước 2: Thêm nhà cung cấp mô hình tùy chỉnh

<Steps>
  <Step title="Mở Cài đặt">
    Trong Dify, đi tới **Settings** → **Model Provider**
  </Step>
  <Step title="Thêm nhà cung cấp tương thích OpenAI">
    Nhấp vào **Add Model Provider** và chọn **OpenAI-API-compatible**
  </Step>
  <Step title="Cấu hình nhà cung cấp">
    Nhập các cài đặt sau:

    | Trường | Giá trị |
    |-------|-------|
    | Provider Name | LemonData |
    | API Key | `sk-your-lemondata-key` |
    | API Base URL | `https://api.lemondata.cc/v1` |
  </Step>
  <Step title="Thêm mô hình">
    Thêm các mô hình bạn muốn sử dụng:
    - `gpt-4o`
    - `gpt-4o-mini`
    - `claude-sonnet-4-5`
    - `claude-opus-4-5`
    - `gemini-2.5-flash`
    - `gemini-2.5-pro`
    - `deepseek-r1`
  </Step>
</Steps>

### Bước 3: Kiểm tra kết nối

1. Chọn một mô hình (ví dụ: `gpt-4o-mini`)
2. Gửi một tin nhắn thử nghiệm
3. Xác nhận bạn nhận được phản hồi

## Sử dụng trong ứng dụng

### Chatbot

1. Tạo một ứng dụng Chatbot mới
2. Chọn LemonData làm nhà cung cấp mô hình
3. Chọn mô hình ưu tiên của bạn
4. Cấu hình system prompt và các tham số

### Agent

1. Tạo một ứng dụng Agent
2. Chọn một mô hình có năng lực (GPT-4o, Claude)
3. Thêm các công cụ và cơ sở tri thức
4. Cấu hình hành vi của agent

### Workflow

1. Tạo một Workflow
2. Thêm các node LLM
3. Chọn các mô hình LemonData cho từng node
4. Kết nối các node và cấu hình luồng dữ liệu

## Các mô hình hiện có

| Danh mục | Mô hình |
|----------|--------|
| Chat | GPT-4o, GPT-4o-mini, Claude Sonnet/Opus, Gemini, DeepSeek |
| Embeddings | text-embedding-3-small, text-embedding-3-large |
| Vision | GPT-4o (với hình ảnh), Claude Sonnet (với hình ảnh) |

## Cấu hình RAG

Đối với các ứng dụng RAG, hãy cấu hình embeddings:

1. Đi tới **Settings** → **Model Provider**
2. Thêm mô hình embedding: `text-embedding-3-small`
3. Đặt làm mô hình embedding mặc định trong cài đặt cơ sở tri thức
