## Tổng quan

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) là công cụ CLI chính thức của Anthropic dành cho việc lập trình với sự hỗ trợ của AI. Bạn có thể cấu hình công cụ này để sử dụng LemonData làm nhà cung cấp API.

## Cấu hình

### Biến môi trường

Thiết lập các biến môi trường sau:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Thêm các dòng này vào shell profile của bạn (`~/.bashrc`, `~/.zshrc`, v.v.) để duy trì cấu hình.

### Tệp cấu hình

Ngoài ra, bạn có thể tạo hoặc chỉnh sửa tệp `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Cách sử dụng

Sau khi cấu hình, hãy sử dụng Claude Code như bình thường:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Các mô hình hiện có

LemonData hỗ trợ tất cả các mô hình Claude:

| Mô hình | Mô tả |
|-------|-------------|
| `claude-opus-4-5` | Khả năng mạnh mẽ nhất, suy luận phức tạp |
| `claude-sonnet-4-5` | Hiệu suất cân bằng, tuyệt vời cho lập trình |
| `claude-haiku-4-5` | Nhanh nhất, tiết kiệm chi phí |

## Lựa chọn mô hình

Chỉ định một mô hình bằng cờ `--model`:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Hoặc thiết lập mặc định trong cấu hình của bạn:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Suy luận mở rộng

Đối với các tác vụ suy luận phức tạp, hãy sử dụng các mô hình có tính năng suy luận mở rộng:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
