## Tổng quan

LemonData hoạt động với bất kỳ ứng dụng chat nào tương thích với OpenAI. Hướng dẫn này bao gồm cách cấu hình cho các ứng dụng chat phổ biến trên máy tính và thiết bị di động.

## Các ứng dụng được hỗ trợ

| Ứng dụng | Nền tảng | Tính năng |
|-------------|----------|----------|
| LobeChat | Web, Máy tính, Tự lưu trữ (Self-hosted) | Đa nhà cung cấp, plugin, RAG |
| CherryStudio | Windows, macOS, Linux | Plugin phong phú, nhiều nhà cung cấp |
| Chatbox | Windows, macOS, Linux | Nhẹ, nhanh |
| BotGem | iOS, macOS | Trải nghiệm Apple nguyên bản |
| TypingMind | Web, Máy tính | Persona tùy chỉnh, plugin |
| OpenCat | iOS, macOS | Nguyên bản, hỗ trợ ngoại tuyến |
| ChatWise | Windows, macOS | Cộng tác nhóm |

## Các bước cấu hình

### Bước 1: Lấy API Key của bạn

1. Đăng nhập vào [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Đi tới [API Keys](https://lemondata.cc/dashboard/api)
3. Tạo và sao chép API key của bạn (định dạng: `sk-...`)

### Bước 2: Cấu hình ứng dụng của bạn

<Tabs>
  <Tab title="LobeChat">
    **Đối với LobeChat tự lưu trữ (Self-hosted):**

    Thiết lập các biến môi trường:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Đối với LobeChat Cloud:**
    1. Mở **Settings** → **AI Service Provider**
    2. Chọn **OpenAI**
    3. Nhập LemonData API key của bạn
    4. Thiết lập endpoint tùy chỉnh: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Mở **Settings** → **Model Provider**
    2. Nhấp vào **Add Provider**
    3. Chọn **OpenAI API Compatible**
    4. Nhập:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Thêm các model: `gpt-4o`, `claude-sonnet-4-5`, v.v.
    6. Lưu và kiểm tra
  </Tab>
  <Tab title="Chatbox">
    1. Mở **Settings** → **AI Model**
    2. Chọn **OpenAI API Compatible**
    3. Nhập:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Nhấp vào **Verify** để kiểm tra kết nối
  </Tab>
  <Tab title="BotGem">
    1. Mở **Settings** → **Providers**
    2. Nhấn vào **Add Custom Provider**
    3. Nhập:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Thêm các model ưa thích của bạn
    5. Lưu cấu hình
  </Tab>
  <Tab title="TypingMind">
    1. Nhấp vào **Settings** (biểu tượng bánh răng)
    2. Chọn **Custom Endpoint**
    3. Nhập:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Cấu hình các model và lưu
  </Tab>
</Tabs>

### Bước 3: Thêm các Model

Sau khi cấu hình nhà cung cấp, hãy thêm các model bạn muốn sử dụng:

| Danh mục | Model |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Tham chiếu cấu hình

| Trường | Giá trị |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API Type | OpenAI Compatible |

**Cấu hình JSON (dành cho các ứng dụng hỗ trợ):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Xử lý sự cố

<AccordionGroup>
  <Accordion title="Lỗi kết nối">
    - Xác minh Base URL chính xác là `https://api.lemondata.cc/v1`
    - Kiểm tra các dấu gạch chéo ở cuối (xóa nếu có)
    - Kiểm tra kết nối internet và cài đặt tường lửa
  </Accordion>

  <Accordion title="401 Unauthorized (Không được phép)">
    - Kiểm tra kỹ API key của bạn
    - Xác minh key đang hoạt động trong [LemonData Dashboard](https://lemondata.cc/dashboard/api)
    - Đảm bảo không có khoảng trắng thừa trong key
  </Accordion>

  <Accordion title="402 Insufficient Balance (Số dư không đủ)">
    - Kiểm tra số dư tài khoản trong dashboard
    - Nạp thêm tiền vào tài khoản của bạn
  </Accordion>

  <Accordion title="404 Model Not Available (Model không khả dụng)">
    - Xác minh tên model được viết chính xác
    - Kiểm tra tính khả dụng của model tại [lemondata.cc/en/models](https://lemondata.cc/vi/models)
    - Thử một model khác
  </Accordion>

  <Accordion title="Phản hồi chậm">
    - Thử một model nhanh hơn (ví dụ: `gpt-4o-mini` thay vì `gpt-4o`)
    - Giảm tham số `max_tokens` nếu có
    - Kiểm tra kết nối mạng của bạn
  </Accordion>
</AccordionGroup>

## Mẹo

<AccordionGroup>
  <Accordion title="Sử dụng các model phù hợp">
    Chọn model phù hợp với tác vụ của bạn - sử dụng GPT-4o-mini cho các cuộc trò chuyện nhanh, Claude hoặc GPT-4o cho các tác vụ phức tạp.
  </Accordion>

  <Accordion title="Lưu các prompt thường dùng">
    Hầu hết các ứng dụng đều hỗ trợ lưu prompt hoặc persona. Hãy tạo các mẫu cho những tác vụ phổ biến.
  </Accordion>

  <Accordion title="Theo dõi mức sử dụng">
    Kiểm tra mức sử dụng của bạn trong LemonData dashboard để theo dõi chi phí và quản lý ngân sách.
  </Accordion>
</AccordionGroup>
