<Note>
**Tự lưu trữ** nghĩa là bạn chạy OpenClaw trực tiếp trên VPS hoặc máy tính của mình. Bạn chỉ trả phí sử dụng mô hình AI thông qua số dư LemonData — không có phí phiên bản.
</Note>

## Cài đặt nhanh

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Non-interactive
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

Trình cài đặt sẽ yêu cầu bạn nhập **LemonData API Key**. Lấy một key từ [Bảng điều khiển](https://lemondata.cc/dashboard).

## Những gì được cài đặt

<Steps>
  <Step title="Node.js 22+">
    Tự động cài đặt nếu chưa có (qua NodeSource trên Linux, Homebrew trên macOS)
  </Step>
  <Step title="OpenClaw">
    Gói OpenClaw chính thức từ npm, được cấu hình sẵn với LemonData API
  </Step>
  <Step title="15 mô hình AI">
    Claude, GPT-5, Gemini, DeepSeek, và nhiều hơn nữa — sẵn sàng sử dụng
  </Step>
  <Step title="Plugin nhắn tin">
    Plugin DingTalk, WeCom và QQ được cài đặt sẵn
  </Step>
  <Step title="Dịch vụ hệ thống">
    Tự động khởi động cùng hệ thống (systemd trên Linux, launchd trên macOS)
  </Step>
</Steps>

## Yêu cầu

| Yêu cầu | Chi tiết |

## Lệnh quản lý

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # Kiểm tra trạng thái
    journalctl --user -u openclaw -f    # Xem nhật ký
    systemctl --user restart openclaw   # Khởi động lại
    ```

## Cấu hình

Vị trí tệp cấu hình: `~/.openclaw/openclaw.json`

Sau khi chỉnh sửa, khởi động lại dịch vụ để áp dụng thay đổi.

### Thay đổi mô hình mặc định

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Các tiền tố mô hình khả dụng:
- `lemondata-claude/` — Các mô hình Claude (anthropic-messages API)
- `lemondata-minimax/` — Các mô hình MiniMax (anthropic-messages API)
- `lemondata/` — Tất cả các mô hình khác (openai-completions API)

## Nâng cấp

```bash
npm update -g openclaw
# Sau đó khởi động lại dịch vụ
systemctl --user restart openclaw  # Linux
# hoặc
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Gỡ cài đặt

```bash
# Dừng và vô hiệu hóa dịch vụ
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# Xóa OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
