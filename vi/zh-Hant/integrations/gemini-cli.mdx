## Tổng quan

Google Gemini CLI là một công cụ dòng lệnh dùng để tương tác với các mô hình Gemini. LemonData cung cấp một endpoint tương thích, cho phép bạn sử dụng Gemini CLI và truy cập hơn 300 mô hình.

## Yêu cầu hệ thống

- **Node.js**: Phiên bản 20.0+
- **Hệ điều hành**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, hoặc Debian 10+

## Cài đặt

```bash
npm install -g @google/gemini-cli
```

Xác minh cài đặt:

```bash
gemini --version
```

## Cấu hình

### Bước 1: Lấy API Key của bạn

1. Đăng nhập vào [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Điều hướng đến [API Keys](https://lemondata.cc/dashboard/api)
3. Tạo và sao chép API Key của bạn (định dạng: `sk-...`)

### Bước 2: Thiết lập biến môi trường

**Tạm thời (phiên làm việc hiện tại):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**Cấu hình vĩnh viễn:**

Thêm vào tệp cấu hình shell của bạn:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
Vui lòng khởi động lại terminal sau khi cấu hình để các thay đổi có hiệu lực.
</Warning>

<Note>
**Hạn chế đã biết**: Trong một số trường hợp, Gemini CLI có thể không tuân theo `GOOGLE_GEMINI_BASE_URL` nếu tồn tại phiên làm việc Google được lưu trong bộ nhớ cache. Nếu bạn gặp sự cố kết nối, hãy thử xóa bộ nhớ cache CLI hoặc bắt đầu một phiên làm việc terminal mới.
</Note>

## Cách sử dụng cơ bản

Khởi chạy Gemini CLI từ thư mục dự án của bạn:

```bash
cd your-project
gemini
```

Trong lần chạy đầu tiên, bạn cần:
1. Chọn chủ đề (theme)
2. Xác nhận thông báo an toàn
3. Tin cậy thư mục làm việc

## Các mô hình khả dụng

| Mô hình | Mô tả |
|-------|-------------|
| `gemini-2.5-pro` | Mô hình Gemini mạnh mẽ nhất |
| `gemini-2.5-flash` | Nhanh và hiệu quả, phù hợp cho hầu hết các tác vụ |
| `gemini-2.0-flash` | Phiên bản thế hệ trước, ổn định |

## Xác minh cấu hình

```bash
# Kiểm tra biến môi trường
echo $GEMINI_API_KEY
echo $GOOGLE_GEMINI_BASE_URL

# Kiểm tra kết nối
gemini
```
