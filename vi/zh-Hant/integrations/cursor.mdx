## Tổng quan

[Cursor](https://cursor.sh) là một trình soạn thảo mã nguồn được hỗ trợ bởi AI. Bạn có thể sử dụng LemonData như một nhà cung cấp API tùy chỉnh để truy cập hơn 300 mô hình.

## Cấu hình

1. Mở cài đặt Cursor (`Cmd/Ctrl + ,`)
2. Điều hướng đến mục **Models** ở thanh bên trái
3. Cuộn xuống phần **OpenAI API Key**
4. Nhập API key LemonData của bạn
5. Kích hoạt **Override OpenAI Base URL**
6. Thiết lập Base URL thành:

```
https://api.lemondata.cc/v1
```

## Lựa chọn mô hình

Sau khi cấu hình hoàn tất, bạn có thể chọn bất kỳ mô hình LemonData nào trong Cursor:

### Các mô hình lập trình được đề xuất

| Mô hình | Phù hợp nhất |
|-------|----------|
| `claude-opus-4-5` | Lập trình phức tạp, thiết kế kiến trúc |
| `claude-sonnet-4-5` | Lập trình tổng quát, gỡ lỗi |
| `gpt-4o` | Tạo mã, giải thích mã |
| `deepseek-r1` | Các tác vụ suy luận phức tạp |
| `gpt-4o-mini` | Hoàn thiện nhanh, tác vụ đơn giản |

## Tính năng

### Chat

Sử dụng bảng điều khiển hội thoại AI (`Cmd/Ctrl + L`) với bất kỳ mô hình LemonData nào:

- Giải thích mã
- Sửa lỗi (Bug)
- Đề xuất tái cấu trúc (Refactoring)
- Tạo tài liệu

### Composer

Tính năng Composer (`Cmd/Ctrl + I`) có thể được sử dụng với các mô hình LemonData để:

- Chỉnh sửa đa tệp
- Triển khai tính năng
- Di chuyển mã (Migration)

### Hoàn thiện bằng phím Tab

Đối với tính năng hoàn thiện bằng phím Tab, chúng tôi khuyên dùng các mô hình nhanh:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
