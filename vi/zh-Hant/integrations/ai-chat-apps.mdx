## Tổng quan

LemonData có thể hoạt động với bất kỳ ứng dụng chat nào tương thích với OpenAI. Hướng dẫn này bao gồm cách cấu hình cho các ứng dụng chat phổ biến trên máy tính và thiết bị di động.

## Các ứng dụng được hỗ trợ

| Ứng dụng | Nền tảng | Tính năng |
|-------------|----------|----------|
| LobeChat | Web, Desktop, Self-hosted | Đa nhà cung cấp, Plugin, RAG |
| CherryStudio | Windows, macOS, Linux | Plugin phong phú, nhiều nhà cung cấp |
| Chatbox | Windows, macOS, Linux | Nhẹ, nhanh |
| BotGem | iOS, macOS | Trải nghiệm Apple nguyên bản |
| TypingMind | Web, Desktop | Nhân vật tùy chỉnh, Plugin |
| OpenCat | iOS, macOS | Nguyên bản, hỗ trợ ngoại tuyến |
| ChatWise | Windows, macOS | Cộng tác nhóm |

## Các bước thiết lập

### Bước 1: Lấy API Key của bạn

1. Đăng nhập vào [LemonData Console](https://lemondata.cc/dashboard)
2. Điều hướng đến [API Keys](https://lemondata.cc/dashboard/api)
3. Tạo và sao chép API Key của bạn (định dạng: `sk-...`)

### Bước 2: Cấu hình ứng dụng của bạn

<Tabs>
  <Tab title="LobeChat">
    **Đối với LobeChat Self-hosted:**

    Thiết lập biến môi trường:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Đối với LobeChat Cloud:**
    1. Mở **Settings** → **AI Service Provider**
    2. Chọn **OpenAI**
    3. Nhập LemonData API Key của bạn
    4. Thiết lập endpoint tùy chỉnh: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Mở **Settings** → **Model Provider**
    2. Nhấp vào **Add Provider**
    3. Chọn **OpenAI API Compatible**
    4. Nhập:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Thêm mô hình: `gpt-4o`, `claude-sonnet-4-5`, v.v.
    6. Lưu và kiểm tra
  </Tab>
  <Tab title="Chatbox">
    1. Mở **Settings** → **AI Model**
    2. Chọn **OpenAI API Compatible**
    3. Nhập:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Nhấp vào **Verify** để kiểm tra kết nối
  </Tab>
  <Tab title="BotGem">
    1. Mở **Settings** → **Providers**
    2. Nhấp vào **Add Custom Provider**
    3. Nhập:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Thêm các mô hình bạn ưu tiên
    5. Lưu cài đặt
  </Tab>
  <Tab title="TypingMind">
    1. Nhấp vào **Settings** (biểu tượng bánh răng)
    2. Chọn **Custom Endpoint**
    3. Nhập:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Cấu hình mô hình và lưu
  </Tab>
</Tabs>

### Bước 3: Thêm mô hình

Sau khi thiết lập nhà cung cấp, hãy thêm các mô hình bạn muốn sử dụng:

| Danh mục | Mô hình |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Tham khảo cấu hình

| Trường | Giá trị |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| Loại API | OpenAI Compatible |

**Cấu hình JSON (áp dụng cho các ứng dụng được hỗ trợ):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Khắc phục sự cố

<AccordionGroup>
  <Accordion title="Lỗi kết nối">
    - Xác minh xem Base URL có chính xác là `https://api.lemondata.cc/v1` hay không
    - Kiểm tra xem có dấu gạch chéo ở cuối không (nếu có hãy xóa đi)
    - Xác minh kết nối mạng và cài đặt tường lửa
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Kiểm tra lại API Key của bạn
    - Xác minh xem Key có đang ở trạng thái hoạt động trong [LemonData Console](https://lemondata.cc/dashboard/api) hay không
    - Đảm bảo không có khoảng trắng thừa trong Key
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - Kiểm tra số dư tài khoản của bạn trong Console
    - Nạp tiền vào tài khoản của bạn
  </Accordion>

  <Accordion title="404 Model Not Available">
    - Xác minh xem tên mô hình có được viết đúng chính tả không
    - Kiểm tra tính khả dụng của mô hình tại [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - Thử một mô hình khác
  </Accordion>

  <Accordion title="Phản hồi chậm">
    - Thử các mô hình nhanh hơn (ví dụ: sử dụng `gpt-4o-mini` thay vì `gpt-4o`)
    - Nếu có thể, hãy giảm tham số `max_tokens`
    - Kiểm tra kết nối mạng của bạn
  </Accordion>
</AccordionGroup>
