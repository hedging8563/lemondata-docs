## Tổng quan

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) là công cụ CLI hỗ trợ lập trình bằng AI chính thức từ Anthropic. Bạn có thể cấu hình công cụ này để sử dụng LemonData làm nhà cung cấp API.

## Cấu hình

### Biến môi trường

Thiết lập các biến môi trường sau:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Thêm các nội dung này vào tệp cấu hình shell của bạn (`~/.bashrc`, `~/.zshrc`, v.v.) để các thay đổi có hiệu lực vĩnh viễn.

### Tệp cấu hình

Hoặc, tạo hoặc chỉnh sửa `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Cách sử dụng

Sau khi cấu hình hoàn tất, bạn có thể sử dụng Claude Code như bình thường:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Các mô hình hỗ trợ

LemonData hỗ trợ tất cả các mô hình Claude:

| Mô hình | Mô tả |
|-------|-------------|
| `claude-opus-4-5` | Mô hình mạnh mẽ nhất, phù hợp cho các tác vụ suy luận phức tạp |
| `claude-sonnet-4-5` | Cân bằng về hiệu suất, rất phù hợp để lập trình |
| `claude-haiku-4-5` | Tốc độ nhanh nhất, tối ưu về chi phí |

## Lựa chọn mô hình

Sử dụng cờ `--model` để chỉ định mô hình:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Hoặc thiết lập giá trị mặc định trong cấu hình của bạn:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Suy luận mở rộng (Extended Thinking)

Đối với các tác vụ suy luận phức tạp, hãy sử dụng các mô hình có tính năng suy luận mở rộng:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
