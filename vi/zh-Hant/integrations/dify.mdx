## Tổng quan

Dify là một nền tảng phát triển ứng dụng LLM mã nguồn mở. Nó cung cấp khả năng điều phối Prompt trực quan, quy trình RAG, khung Agent và các khả năng LLMOps. LemonData có thể được cấu hình như một nhà cung cấp mô hình tùy chỉnh trong Dify.

## Ưu điểm

- Truy cập hơn 300+ mô hình AI thông qua một giao diện duy nhất
- Chuyển đổi mô hình mà không cần thay đổi logic ứng dụng
- Chọn mô hình tốt nhất cho mỗi tác vụ để tối ưu hóa chi phí
- Thống nhất việc thanh toán và theo dõi mức độ sử dụng

## Điều kiện tiên quyết

- Tài khoản LemonData có quyền truy cập API
- Cài đặt Dify (phiên bản Cloud hoặc tự triển khai)

## Các bước cấu hình

### Bước 1: Lấy API Key của bạn

1. Đăng nhập vào [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Điều hướng đến [API Keys](https://lemondata.cc/dashboard/api)
3. Tạo và sao chép API Key của bạn (định dạng: `sk-...`)

### Bước 2: Thêm nhà cung cấp mô hình tùy chỉnh

<Steps>
  <Step title="Mở cài đặt">
    Trong Dify, đi tới **Settings** → **Model Provider**
  </Step>
  <Step title="Thêm nhà cung cấp tương thích OpenAI">
    Nhấp vào **Add Model Provider** và chọn **OpenAI-API-compatible**
  </Step>
  <Step title="Cấu hình nhà cung cấp">
    Nhập các cài đặt sau:

    | Trường | Giá trị |
    |-------|-------|
    | Tên nhà cung cấp | LemonData |
    | API Key | `sk-your-lemondata-key` |
    | API Base URL | `https://api.lemondata.cc/v1` |
  </Step>
  <Step title="Thêm mô hình">
    Thêm các mô hình bạn muốn sử dụng:
    - `gpt-4o`
    - `gpt-4o-mini`
    - `claude-sonnet-4-5`
    - `claude-opus-4-5`
    - `gemini-2.5-flash`
    - `gemini-2.5-pro`
    - `deepseek-r1`
  </Step>
</Steps>

### Bước 3: Kiểm tra kết nối

1. Chọn một mô hình (ví dụ: `gpt-4o-mini`)
2. Gửi một tin nhắn thử nghiệm
3. Xác nhận rằng bạn đã nhận được phản hồi

## Sử dụng trong ứng dụng

### Chatbot

1. Tạo một ứng dụng Chatbot mới
2. Chọn LemonData làm nhà cung cấp mô hình
3. Chọn mô hình ưa thích của bạn
4. Cấu hình System Prompt và các tham số

### Agent

1. Tạo một ứng dụng Agent
2. Chọn một mô hình mạnh mẽ (GPT-4o, Claude)
3. Thêm công cụ và cơ sở kiến thức
4. Cấu hình hành vi của Agent

### Workflow

1. Tạo một workflow
2. Thêm nút LLM
3. Chọn mô hình LemonData cho mỗi nút
4. Kết nối các nút và cấu hình luồng dữ liệu

## Các mô hình có sẵn

| Danh mục | Mô hình |
|----------|--------|
| Trò chuyện (Chat) | GPT-4o, GPT-4o-mini, Claude Sonnet/Opus, Gemini, DeepSeek |
| Nhúng (Embeddings) | text-embedding-3-small, text-embedding-3-large |
| Thị giác (Vision) | GPT-4o (hỗ trợ hình ảnh), Claude Sonnet (hỗ trợ hình ảnh) |

## Cấu hình RAG

Đối với các ứng dụng RAG, vui lòng cấu hình mô hình nhúng:

1. Đi tới **Settings** → **Model Provider**
2. Thêm mô hình nhúng: `text-embedding-3-small`
3. Thiết lập nó làm mô hình nhúng mặc định trong cài đặt cơ sở kiến thức
