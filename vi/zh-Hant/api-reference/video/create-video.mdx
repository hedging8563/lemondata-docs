## Tổng quan

Quá trình tạo video là không đồng bộ. Bạn gửi một yêu cầu và nhận được một `task_id`, sau đó thực hiện polling để lấy kết quả.

## Thân yêu cầu (Request Body)

<ParamField body="model" type="string" default="kling-video">
  Mô hình video (ví dụ: `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Mô tả văn bản cho video cần tạo.
</ParamField>

<ParamField body="image_url" type="string">
  Hình ảnh bắt đầu tùy chọn cho quá trình tạo video từ hình ảnh (image-to-video).
</ParamField>

<ParamField body="duration" type="integer">
  Thời lượng video, tính bằng giây (tùy thuộc vào mô hình).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Tỷ lệ khung hình (ví dụ: `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Độ phân giải video (ví dụ: `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Số khung hình trên giây (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Nội dung cần tránh trong quá trình tạo video.
</ParamField>

<ParamField body="seed" type="integer">
  Seed ngẫu nhiên để có thể tái tạo kết quả.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Thang đo hướng dẫn không phân loại (Classifier-Free Guidance scale) (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Cường độ chuyển động (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL của hình ảnh khung hình bắt đầu.
</ParamField>

<ParamField body="end_image" type="string">
  URL của hình ảnh khung hình kết thúc.
</ParamField>

<ParamField body="user" type="string">
  Mã định danh duy nhất của người dùng cuối.
</ParamField>

## Phản hồi (Response)

<ResponseField name="task_id" type="string">
  Mã định danh tác vụ duy nhất được sử dụng để polling.
</ResponseField>

<ResponseField name="status" type="string">
  Trạng thái ban đầu: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Mô hình được sử dụng.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

## Tạo video từ hình ảnh (Image-to-Video)

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```

## Các mô hình hiện có

| Mô hình | Loại | Mô tả |
|-------|------|-------------|
| `kling-video` | T2V/I2V | Chuyển động thực tế, nhiều phiên bản (v1.0-v2.6) |
| `veo3.1` | T2V | Mô hình mới nhất của Google |
| `veo3.1-pro` | T2V | Chất lượng chuyên nghiệp |
| `minimax/video-01` | T2V | Nhịp điệu cao và độ ổn định |
| `wan2.6-i2v` | I2V | Tạo video từ hình ảnh (Image-to-Video) |
