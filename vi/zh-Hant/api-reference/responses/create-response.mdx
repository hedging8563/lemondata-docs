---

Responses API là API hội thoại có trạng thái mới hơn của OpenAI. LemonData cung cấp hỗ trợ định dạng này cho các mô hình tương thích.

## Thân yêu cầu

<ParamField body="model" type="string" required>
  ID của mô hình sẽ sử dụng. Vui lòng tham khảo [Models](https://lemondata.cc/zh-TW/models) để xem các tùy chọn có sẵn.
</ParamField>

<ParamField body="input" type="array" required>
  Danh sách các mục đầu vào tạo nên cuộc hội thoại.

  Mỗi mục có thể là:
  - `message`: tin nhắn hội thoại bao gồm vai trò và nội dung
  - `function_call`: yêu cầu gọi hàm
  - `function_call_output`: đầu ra của việc gọi hàm
</ParamField>

<ParamField body="instructions" type="string">
  Các chỉ dẫn hệ thống cho mô hình (tương đương với tin nhắn hệ thống).
</ParamField>

<ParamField body="max_output_tokens" type="integer">
  Số lượng token tối đa được tạo ra.
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Nhiệt độ lấy mẫu trong khoảng từ 0 đến 2.
</ParamField>

<ParamField body="tools" type="array">
  Danh sách các công cụ mà mô hình có thể gọi.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Nếu là true, sẽ trả về một luồng sự kiện (event stream).
</ParamField>

<ParamField body="previous_response_id" type="string">
  ID của phản hồi trước đó, được sử dụng để tiếp tục cuộc hội thoại.
</ParamField>

<ParamField body="store" type="boolean" default="true">
  Có lưu trữ phản hồi để truy xuất sau này hay không.
</ParamField>

<ParamField body="metadata" type="object">
  Metadata được đính kèm vào phản hồi, dùng cho mục đích theo dõi.
</ParamField>

<ParamField body="text" type="object">
  Các tùy chọn cấu hình tạo văn bản.
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  Có cho phép thực hiện nhiều cuộc gọi công cụ đồng thời hay không.
</ParamField>

<ParamField body="top_p" type="number">
  Tham số nucleus sampling (0-1).
</ParamField>

<ParamField body="reasoning" type="object">
  Cấu hình suy luận cho các mô hình o1/o3.

  - `effort` (string): Mức độ nỗ lực
