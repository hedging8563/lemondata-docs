---
title: "Tạo nội dung"
openapi: "POST /v1beta/models/{model}:generateContent"
description: "Tạo nội dung bằng định dạng Google Gemini API"
---

LemonData hỗ trợ định dạng Google Gemini API gốc cho các mô hình Gemini. Điều này cho phép khả năng tương thích trực tiếp với Google AI SDK.

## Tham số đường dẫn

<ParamField path="model" type="string" required>
  Tên mô hình (ví dụ: `gemini-2.5-pro`, `gemini-2.5-flash`).
</ParamField>

## Tham số truy vấn

<ParamField query="key" type="string">
  API key (phương pháp thay thế cho xác thực header).
</ParamField>

## Xác thực

Endpoint Gemini hỗ trợ nhiều phương thức xác thực:
- Tham số truy vấn `?key=YOUR_API_KEY`
- Header `x-goog-api-key: YOUR_API_KEY`
- Header `Authorization: Bearer YOUR_API_KEY`

## Body yêu cầu

<ParamField body="contents" type="array" required>
  Nội dung hội thoại.

  Mỗi đối tượng nội dung bao gồm:
  - `role` (string): `user` hoặc `model`
  - `parts` (array): Các phần nội dung (văn bản hoặc dữ liệu inline)
</ParamField>

<ParamField body="systemInstruction" type="object">
  Chỉ dẫn hệ thống cho mô hình.
</ParamField>

<ParamField body="generationConfig" type="object">
  Cấu hình tạo nội dung:
  - `temperature` (number): Nhiệt độ lấy mẫu
  - `topP` (number): Xác suất lấy mẫu hạt nhân (nucleus sampling)
  - `topK` (integer): Lấy mẫu Top-K
  - `maxOutputTokens` (integer): Số lượng token đầu ra tối đa
  - `stopSequences` (array): Các chuỗi dừng
</ParamField>

<ParamField body="safetySettings" type="array">
  Cài đặt bộ lọc an toàn.
</ParamField>

## Phản hồi

<ResponseField name="candidates" type="array">
  Các nội dung ứng viên được tạo ra.
</ResponseField>

<ResponseField name="usageMetadata" type="object">
  Thông tin sử dụng token.
</ResponseField>

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.lemondata.cc"}
)

model = genai.GenerativeModel("gemini-2.5-pro")
response = model.generate_content("Hello, Gemini!")

print(response.text)
```


```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "contents": []map[string]interface{}{
            {
                "parts": []map[string]string{
                    {"text": "Hello, Gemini!"},
                },
            },
        },
        "generationConfig": map[string]interface{}{
            "temperature":    0.7,
            "maxOutputTokens": 1024,
        },
    }

    jsonData, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST",
        "https://api.lemondata.cc/v1beta/models/gemini-2.5-pro:generateContent?key=sk-your-api-key",
        bytes.NewBuffer(jsonData))
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
