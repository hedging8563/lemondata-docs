---
title: "Lấy thông tin Model"
api: "GET /v1/models/{model}"
description: "Truy xuất một phiên bản model"
---

## Tham số đường dẫn

<ParamField path="model" type="string" required>
  ID của model cần truy xuất (ví dụ: `gpt-4o`, `claude-sonnet-4-5`).
</ParamField>

## Phản hồi

<ResponseField name="id" type="string">
  Định danh của model.
</ResponseField>

<ResponseField name="object" type="string">
  Luôn là `model`.
</ResponseField>

<ResponseField name="created" type="integer">
  Dấu thời gian tạo (timestamp).
</ResponseField>

<ResponseField name="owned_by" type="string">
  Nhà cung cấp model.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1/models/gpt-4o" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

model = client.models.retrieve("gpt-4o")
print(f"Model: {model.id}, Provider: {model.owned_by}")
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const model = await client.models.retrieve('gpt-4o');
console.log(`Model: ${model.id}, Provider: ${model.owned_by}`);
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    model, err := client.GetModel(context.Background(), "gpt-4o")
    if err != nil {
        panic(err)
    }
    fmt.Printf("Model: %s, Provider: %s\n", model.ID, model.OwnedBy)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/models/gpt-4o');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$model = json_decode($response, true);
echo "Model: {$model['id']}, Provider: {$model['owned_by']}\n";
```

## Xử lý lỗi

Nếu model không tồn tại, bạn sẽ nhận được lỗi 404:

```json
{
  "error": {
    "message": "Model 'invalid-model' not found",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```
