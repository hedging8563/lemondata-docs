## Tổng quan

LemonData sử dụng **mô hình giá pay-as-you-go**. Bạn chỉ trả tiền cho những gì bạn sử dụng, không có phí đăng ký định kỳ hoặc cam kết tối thiểu.

## Cách thức thanh toán hoạt động

1. **Nạp credit** vào tài khoản của bạn
2. **Sử dụng API** - chi phí được khấu trừ trên mỗi yêu cầu
3. **Theo dõi mức sử dụng** trong dashboard của bạn
4. **Nạp thêm tiền** khi số dư của bạn ở mức thấp

### Thanh toán tác vụ bất đồng bộ (Video/Âm nhạc/3D)

<Warning>
  **Quan trọng**: Các tác vụ tạo video, âm nhạc và 3D được **tính phí khi gửi**, không phải khi hoàn thành.
</Warning>

Đối với các tác vụ tạo bất đồng bộ (video, âm nhạc, mô hình 3D):

1. **Thanh toán xảy ra ngay lập tức** khi gửi tác vụ
2. Bạn nhận được `task_id` để truy vấn trạng thái
3. **Không hoàn tiền tự động** nếu tác vụ thất bại

Điều này là do các nhà cung cấp upstream tính phí chúng tôi khi tác vụ được gửi, bất kể kết quả. Nếu bạn gặp lỗi tác vụ, vui lòng liên hệ support@lemondata.cc.

```python
# Ví dụ: Thanh toán tạo video
response = client.post("/v1/video/generations", json={
    "model": "kling-v1",
    "prompt": "Hoàng hôn trên biển"
})
# ⚠️ Bạn bị tính phí NGAY BÂY GIỜ, không phải khi video sẵn sàng

task_id = response.json()["task_id"]
# Truy vấn trạng thái...
```

## Cách tính Token

Token là các đơn vị cơ bản của quá trình xử lý văn bản:

- ~4 ký tự = 1 token (Tiếng Anh)
- ~1-2 ký tự = 1 token (Tiếng Trung)
- 1 hình ảnh = thay đổi tùy theo kích thước và độ chi tiết

### Ước tính Token

```python
# Ước tính sơ bộ
def estimate_tokens(text):
    return len(text) / 4  # Xấp xỉ cho tiếng Anh

# Số lượng thực tế (cho các mô hình OpenAI)
import tiktoken
encoder = tiktoken.encoding_for_model("gpt-4o")
tokens = encoder.encode("Your text here")
print(f"Token count: {len(tokens)}")
```

```json
  }
}
```

## Cảnh báo số dư thấp

Cấu hình cảnh báo khi số dư của bạn giảm xuống:

1. Đi tới **Dashboard → Settings → Notifications**
2. Thiết lập số tiền ngưỡng của bạn
3. Nhận thông báo qua email

## Nạp Credit

### Phương thức thanh toán

- Alipay (支付宝)
- WeChat Pay (微信支付)
- Stripe (Visa, Mastercard)

### Các bước thực hiện

1. Đăng nhập vào [Dashboard](https://lemondata.cc/dashboard)
2. Nhấp vào **Add Credits**
3. Chọn số tiền và phương thức thanh toán
4. Hoàn tất thanh toán

Credit được cộng ngay lập tức sau khi xác nhận thanh toán.

## Giới hạn API Key

Bạn có thể thiết lập giới hạn chi tiêu cho từng API key riêng lẻ:

1. Đi tới **Dashboard → API Keys**
2. Nhấp vào một key để chỉnh sửa
3. Thiết lập **Usage Limit**

Khi đạt đến giới hạn, các yêu cầu với key đó sẽ trả về lỗi `402 Payment Required`.

## Hóa đơn

Đối với tài khoản doanh nghiệp, hóa đơn luôn có sẵn:

1. Đi tới **Dashboard → Billing**
2. Xem lịch sử giao dịch
3. Tải xuống hóa đơn dưới dạng PDF
