## Tổng quan

LemonData hỗ trợ **ba định dạng API gốc** với một API key duy nhất. Hãy chọn định dạng phù hợp nhất với nhu cầu sử dụng của bạn - không cần thay đổi cấu hình.

## Tại sao nên dùng Đa Định Dạng?

| Lợi ích | Mô tả |
|---------|-------------|
| **Không cần chuyển đổi SDK** | Sử dụng bất kỳ mô hình nào với SDK ưa thích của bạn |
| **Tính năng gốc** | Truy cập các khả năng đặc thù của từng định dạng |
| **Di chuyển dễ dàng** | Chuyển đổi từ các API chính thức chỉ với việc thay đổi base URL |
| **Thanh toán tập trung** | Một tài khoản, một API key, tất cả định dạng |

## So sánh Định dạng

| Tính năng | OpenAI | Anthropic | Gemini |
|---------|--------|-----------|--------|
| **Endpoint** | `/v1/chat/completions` | `/v1/messages` | `/v1beta/models/:model:generateContent` |
| **Auth Header** | `Authorization: Bearer` | `x-api-key` | `Authorization: Bearer` |
| **System Prompt** | Trong mảng messages | Trường `system` riêng biệt | Trong `systemInstruction` |
| **Extended Thinking** | ❌ | ✅ | ❌ |
| **Streaming** | ✅ SSE | ✅ SSE | ✅ SSE |
| **Tool Calling** | ✅ | ✅ | ✅ |
| **Vision** | ✅ | ✅ | ✅ |

## Định dạng OpenAI

client = OpenAI(
    api_key="sk-your-lemondata-key",
    base_url="https://api.lemondata.cc/v1"
)
