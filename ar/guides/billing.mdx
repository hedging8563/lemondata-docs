## نظرة عامة

تستخدم LemonData **نظام تسعير الدفع حسب الاستخدام**. أنت تدفع فقط مقابل ما تستخدمه، دون اشتراكات أو التزامات حد أدنى.

## كيف تعمل الفواتير

1. **أضف رصيداً** إلى حسابك
2. **استخدم الـ API** - يتم خصم التكاليف لكل طلب
3. **راقب الاستخدام** في لوحة التحكم الخاصة بك
4. **أعد الشحن** عندما ينخفض رصيدك

### فوترة المهام غير المتزامنة (فيديو/موسيقى/3D)

<Warning>
  **مهم**: يتم **فوترة مهام إنشاء الفيديو والموسيقى و3D عند الإرسال**، وليس عند الاكتمال.
</Warning>

بالنسبة لمهام الإنشاء غير المتزامنة (فيديو، موسيقى، نماذج 3D):

1. **تتم الفوترة فوراً** عند إرسال المهمة
2. تتلقى `task_id` للاستعلام عن الحالة
3. **لا يوجد استرداد تلقائي** إذا فشلت المهمة

هذا لأن مزودي الخدمة الأصليين يفرضون علينا رسوماً عند إرسال المهمة، بغض النظر عن النتيجة. إذا واجهت فشلاً في المهام، يرجى التواصل مع support@lemondata.cc.

```python
# مثال: فوترة إنشاء الفيديو
response = client.post("/v1/video/generations", json={
    "model": "kling-v1",
    "prompt": "غروب الشمس فوق المحيط"
})
# ⚠️ يتم فوترتك الآن، وليس عندما يكون الفيديو جاهزاً

task_id = response.json()["task_id"]
# الاستعلام عن الحالة...
```

## حساب الـ Tokens

الـ Tokens هي الوحدات الأساسية لمعالجة النصوص:

- ~4 أحرف = 1 token (الإنجليزية)
- ~1-2 حرف = 1 token (الصينية)
- صورة واحدة = تختلف حسب الحجم والتفاصيل

### تقدير الـ Tokens

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English

# Actual count (for OpenAI models)
import tiktoken
encoder = tiktoken.encoding_for_model("gpt-4o")
tokens = encoder.encode("Your text here")
print(f"Token count: {len(tokens)}")
```

## تتبع الاستخدام

### لوحة التحكم

راقب استخدامك في [لوحة التحكم](https://lemondata.cc/dashboard):

- الرصيد في الوقت الفعلي
- سجل الاستخدام حسب النموذج
- تفاصيل التكاليف
- استخدام مفتاح الـ API

### استجابة الـ API

تتضمن كل استجابة معلومات الاستخدام:

```json
{
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 100,
    "total_tokens": 150
  }
}
```

## تنبيهات انخفاض الرصيد

قم بتكوين التنبيهات عندما ينخفض رصيدك:

1. انتقل إلى **لوحة التحكم ← الإعدادات ← الإشعارات**
2. حدد مبلغ الحد الأدنى الخاص بك
3. تلقي إشعارات عبر البريد الإلكتروني

## إضافة رصيد

### طرق الدفع

- Alipay (支付宝)
- WeChat Pay (微信支付)
- Stripe (Visa, Mastercard)

### الخطوات

1. قم بتسجيل الدخول إلى [لوحة التحكم](https://lemondata.cc/dashboard)
2. انقر على **إضافة رصيد**
3. اختر المبلغ وطريقة الدفع
4. أكمل عملية الدفع

يتم إضافة الرصيد فوراً بعد تأكيد الدفع.

## حدود مفتاح الـ API

يمكنك تعيين حدود للإنفاق على مفاتيح الـ API الفردية:

1. انتقل إلى **لوحة التحكم ← مفاتيح الـ API**
2. انقر على مفتاح للتعديل
3. حدد **حد الاستخدام**

عند الوصول إلى الحد الأقصى، ستُرجع الطلبات باستخدام هذا المفتاح خطأ `402 Payment Required`.

## الفواتير

بالنسبة لحسابات الشركات، تتوفر الفواتير:

1. انتقل إلى **لوحة التحكم ← الفواتير**
2. عرض سجل المعاملات
3. تحميل الفواتير بصيغة PDF
