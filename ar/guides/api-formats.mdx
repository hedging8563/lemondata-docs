## نظرة عامة

يدعم LemonData **ثلاثة تنسيقات API أصلية** بمفتاح API واحد. اختر التنسيق الذي يناسب حالة الاستخدام الخاصة بك بشكل أفضل - لا حاجة لتغييرات في الإعدادات.

  <Card title="تنسيق OpenAI" icon="plug">

    التنسيق القياسي، التوافق الأوسع
  </Card>
  <Card title="تنسيق Anthropic" icon="message">
    `/v1/messages`
    التفكير الموسع، ميزات Claude الأصلية
  </Card>
  <Card title="تنسيق Gemini" icon="sparkles">
    `/v1beta/models/:model:generateContent`
    التكامل مع منظومة Google
  </Card>
</CardGroup>

## لماذا التنسيقات المتعددة؟

| الميزة | الوصف |
|---------|-------------|
| **لا حاجة لتبديل SDK** | استخدم أي نموذج مع SDK المفضل لديك |
| **الميزات الأصلية** | الوصول إلى الإمكانيات الخاصة بكل تنسيق |
| **هجرة سهلة** | التبديل من واجهات برمجة التطبيقات الرسمية بمجرد تغيير base URL |
| **فوترة موحدة** | حساب واحد، مفتاح API واحد، جميع التنسيقات |

## مقارنة التنسيقات

| الميزة | OpenAI | Anthropic | Gemini |
|---------|--------|-----------|--------|
| **نقطة النهاية (Endpoint)** | `/v1/chat/completions` | `/v1/messages` | `/v1beta/models/:model:generateContent` |
| **رأس المصادقة (Auth Header)** | `Authorization: Bearer` | `x-api-key` | `Authorization: Bearer` |
| **موجه النظام (System Prompt)** | في مصفوفة messages | حقل `system` منفصل | في `systemInstruction` |
| **التفكير الموسع (Extended Thinking)** | ❌ | ✅ | ❌ |
| **البث (Streaming)** | ✅ SSE | ✅ SSE | ✅ SSE |
| **استدعاء الأدوات (Tool Calling)** | ✅ | ✅ | ✅ |
| **الرؤية (Vision)** | ✅ | ✅ | ✅ |
