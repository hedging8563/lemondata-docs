---
title: "Get Music Status"
api: "GET /v1/music/generations/{id}"
description: "Retrieves the status and result of a music generation task"
---

Poll this endpoint to check the status of your music generation task.

## Path Parameters

<ParamField path="id" type="string" required>
  The music generation task ID.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Task ID.
</ResponseField>

<ResponseField name="status" type="string">
  Task status: `pending`, `processing`, `completed`, or `failed`.
</ResponseField>

<ResponseField name="progress" type="integer">
  Progress percentage (0-100).
</ResponseField>

<ResponseField name="audio_url" type="string">
  URL to download the audio file (when completed).
</ResponseField>

<ResponseField name="video_url" type="string">
  URL to the video version with visualizer (when completed).
</ResponseField>

<ResponseField name="title" type="string">
  Generated song title.
</ResponseField>

<ResponseField name="lyrics" type="string">
  Generated or provided lyrics.
</ResponseField>

<ResponseField name="error" type="string">
  Error message (when failed).
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1/music/generations/music_abc123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "music_abc123"

while True:
    response = requests.get(
        f"https://api.lemondata.cc/v1/music/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"}
    )
    result = response.json()

    if result["status"] == "completed":
        print(f"Audio URL: {result['audio_url']}")
        break
    elif result["status"] == "failed":
        print(f"Error: {result['error']}")
        break

    time.sleep(5)  # Poll every 5 seconds
```

```javascript JavaScript
const taskId = 'music_abc123';

async function pollForMusic() {
  while (true) {
    const response = await fetch(
      `https://api.lemondata.cc/v1/music/generations/${taskId}`,
      { headers: { 'Authorization': 'Bearer sk-your-api-key' } }
    );

    const result = await response.json();

    if (result.status === 'completed') {
      console.log(`Audio URL: ${result.audio_url}`);
      return result.audio_url;
    } else if (result.status === 'failed') {
      throw new Error(result.error);
    }

    await new Promise(r => setTimeout(r, 5000));
  }
}
```
