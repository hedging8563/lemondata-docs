## نظرة عامة

Google Gemini CLI هي أداة سطر أوامر للتفاعل مع نماذج Gemini. توفر LemonData نقطة نهاية متوافقة تتيح لك استخدام Gemini CLI مع الوصول إلى أكثر من 300 نموذج.

## متطلبات النظام

- **Node.js**: الإصدار 20.0+
- **نظام التشغيل**: Windows 10/11، macOS 10.15+، Ubuntu 20.04+، أو Debian 10+

## التثبيت

```bash
npm install -g @google/gemini-cli
```

التحقق من التثبيت:

```bash
gemini --version
```

## التكوين

### الخطوة 1: احصل على API Key الخاص بك

1. سجل الدخول إلى [لوحة تحكم LemonData](https://lemondata.cc/dashboard)
2. انتقل إلى [مفاتيح API](https://lemondata.cc/dashboard/api)
3. أنشئ وانسخ API key الخاص بك (التنسيق: `sk-...`)

### الخطوة 2: تعيين متغيرات البيئة

**مؤقت (الجلسة الحالية):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**تكوين دائم:**

أضف إلى ملف تكوين shell الخاص بك:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
أعد تشغيل terminal بعد التكوين لتفعيل التغييرات.
</Warning>

<Note>
**قيد معروف**: في بعض الحالات، قد لا يحترم Gemini CLI متغير `GOOGLE_GEMINI_BASE_URL` إذا كانت هناك جلسة Google مخزنة مؤقتاً. إذا واجهت مشكلات في الاتصال، جرب مسح ذاكرة التخزين المؤقت لـ CLI أو بدء جلسة terminal جديدة.
</Note>

## الاستخدام الأساسي

ابدأ Gemini CLI من دليل مشروعك:

```bash
cd your-project
gemini
```

في التشغيل الأول، ستقوم بـ:
1. اختيار سمة (theme)
2. تأكيد إشعار السلامة
3. الوثوق بدليل العمل

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
  <Accordion title="خطأ في الاتصال">
    - تحقق من أن `GOOGLE_GEMINI_BASE_URL` مضبوط على `https://api.lemondata.cc`
    - ملاحظة: لا توجد لاحقة `/v1` لنقطة نهاية Gemini
    - تحقق من اتصال الشبكة
  </Accordion>

  <Accordion title="فشل المصادقة">
    - تحقق من تعيين متغير البيئة `GEMINI_API_KEY`
    - تأكد من أن المفتاح يبدأ بـ `sk-`
    - تأكد من أن المفتاح نشط في لوحة تحكم LemonData
  </Accordion>

  <Accordion title="النموذج غير متاح">
    - تحقق من توفر نماذج Gemini على [lemondata.cc/en/models](https://lemondata.cc/en/models)
    - جرب متغير نموذج Gemini مختلف
  </Accordion>
