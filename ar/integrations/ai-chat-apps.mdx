## نظرة عامة

يعمل LemonData مع أي تطبيق دردشة متوافق مع OpenAI. يغطي هذا الدليل خطوات التكوين لتطبيقات الدردشة الشهيرة على أجهزة الكمبيوتر والهواتف المحمولة.

## التطبيقات المدعومة

| التطبيق | المنصة | الميزات |
|-------------|----------|----------|
| LobeChat | الويب، سطح المكتب، استضافة ذاتية | متعدد المزودين، إضافات، RAG |
| CherryStudio | ويندوز، ماك، لينكس | إضافات غنية، مزودون متعددون |
| Chatbox | ويندوز، ماك، لينكس | خفيف الوزن، سريع |
| BotGem | iOS، macOS | تجربة Apple أصلية |
| TypingMind | الويب، سطح المكتب | شخصيات مخصصة، إضافات |
| OpenCat | iOS، macOS | أصلي، دعم العمل بدون اتصال |
| ChatWise | ويندوز، ماك | تعاون الفريق |

## خطوات التكوين

### الخطوة 1: الحصول على API Key الخاص بك

1. قم بتسجيل الدخول إلى [LemonData Dashboard](https://lemondata.cc/dashboard)
2. انتقل إلى [API Keys](https://lemondata.cc/dashboard/api)
3. قم بإنشاء ونسخ API key الخاص بك (بصيغة: `sk-...`)

### الخطوة 2: تكوين تطبيقك

<Tabs>
  <Tab title="LobeChat">
    **بالنسبة لـ LobeChat المستضاف ذاتياً:**

    قم بتعيين متغيرات البيئة:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **بالنسبة لـ LobeChat Cloud:**
    1. افتح **Settings** ← **AI Service Provider**
    2. اختر **OpenAI**
    3. أدخل LemonData API key الخاص بك
    4. قم بتعيين endpoint مخصص: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. افتح **Settings** ← **Model Provider**
    2. انقر فوق **Add Provider**
    3. اختر **OpenAI API Compatible**
    4. أدخل:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. أضف النماذج: `gpt-4o`، `claude-sonnet-4-5`، إلخ.
    6. احفظ واختبر
  </Tab>
  <Tab title="Chatbox">
    1. افتح **Settings** ← **AI Model**
    2. اختر **OpenAI API Compatible**
    3. أدخل:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. انقر فوق **Verify** لاختبار الاتصال
  </Tab>
  <Tab title="BotGem">
    1. افتح **Settings** ← **Providers**
    2. اضغط على **Add Custom Provider**
    3. أدخل:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. أضف نماذجك المفضلة
    5. احفظ التكوين
  </Tab>
  <Tab title="TypingMind">
    1. انقر فوق **Settings** (أيقونة الترس)
    2. اختر **Custom Endpoint**
    3. أدخل:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. قم بتكوين النماذج واحفظ
  </Tab>
</Tabs>

### الخطوة 3: إضافة النماذج

بعد تكوين المزود، أضف النماذج التي تريد استخدامها:

| الفئة | النماذج |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## مرجع التكوين

| الحقل | القيمة |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| نوع API | متوافق مع OpenAI |

**تكوين JSON (للتطبيقات التي تدعم ذلك):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
  <Accordion title="خطأ في الاتصال">
    - تحقق من أن Base URL هو بالضبط `https://api.lemondata.cc/v1`
    - تحقق من وجود شرطات مائلة في النهاية (قم بإزالتها إذا وجدت)
    - تحقق من الاتصال بالإنترنت وإعدادات جدار الحماية
  </Accordion>

  <Accordion title="401 غير مصرح به">
    - تحقق مرة أخرى من API key الخاص بك
    - تحقق من أن المفتاح نشط في [LemonData Dashboard](https://lemondata.cc/dashboard/api)
    - تأكد من عدم وجود مسافات إضافية في المفتاح
  </Accordion>

  <Accordion title="402 رصيد غير كافٍ">
    - تحقق من رصيد حسابك في لوحة التحكم
    - أضف رصيداً إلى حسابك
  </Accordion>

  <Accordion title="404 النموذج غير متوفر">
    - تحقق من كتابة اسم النموذج بشكل صحيح
    - تحقق من توفر النموذج في [lemondata.cc/en/models](https://lemondata.cc/ar/models)
    - جرب نموذجاً مختلفاً
  </Accordion>

  <Accordion title="استجابات بطيئة">
    - جرب نموذجاً أسرع (على سبيل المثال، `gpt-4o-mini` بدلاً من `gpt-4o`)
    - قم بتقليل معلمة `max_tokens` إذا كانت متاحة
    - تحقق من اتصال الشبكة الخاص بك
  </Accordion>
</AccordionGroup>
