## نظرة عامة

[Cursor](https://cursor.sh) هو محرر أكواد مدعوم بالذكاء الاصطناعي. يمكنك استخدام LemonData كمزود API مخصص للوصول إلى أكثر من 300 نموذج.

## الإعداد

1. افتح إعدادات Cursor (`Cmd/Ctrl + ,`)
2. انتقل إلى **Models** في الشريط الجانبي الأيسر
3. قم بالتمرير لأسفل إلى قسم **OpenAI API Key**
4. أدخل مفتاح API الخاص بـ LemonData
5. قم بتفعيل **Override OpenAI Base URL**
6. قم بتعيين Base URL إلى:

```
https://api.lemondata.cc/v1
```

## اختيار النموذج

بعد الإعداد، يمكنك اختيار أي نموذج من LemonData في Cursor:

### النماذج الموصى بها للبرمجة

| النموذج | الأفضل لـ |
|-------|----------|
| `claude-opus-4-5` | البرمجة المعقدة، الهندسة المعمارية |
| `claude-sonnet-4-5` | البرمجة العامة، تصحيح الأخطاء |
| `gpt-4o` | توليد الأكواد، الشروحات |
| `deepseek-r1` | المهام التي تتطلب تفكيراً منطقياً مكثفاً |
| `gpt-4o-mini` | الإكمال السريع، المهام البسيطة |

## الميزات

### الدردشة (Chat)

استخدم لوحة دردشة الذكاء الاصطناعي (`Cmd/Ctrl + L`) مع أي نموذج من LemonData:

- شروحات الأكواد
- إصلاح الأخطاء
- اقتراحات إعادة هيكلة الكود (Refactoring)
- توليد التوثيق

### الملحن (Composer)

تعمل ميزة Composer (`Cmd/Ctrl + I`) مع نماذج LemonData من أجل:

- تعديلات على ملفات متعددة
- تنفيذ الميزات
- ترحيل الأكواد (Code migrations)

### الإكمال التلقائي (Tab Completion)

بالنسبة للإكمال التلقائي، نوصي باستخدام النماذج السريعة:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
