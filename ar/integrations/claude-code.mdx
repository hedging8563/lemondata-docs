## نظرة عامة

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) هي أداة CLI الرسمية من Anthropic للبرمجة بمساعدة الذكاء الاصطناعي. يمكنك تهيئتها لاستخدام LemonData كمزود API.

## التهيئة

### متغيرات البيئة

قم بتعيين متغيرات البيئة التالية:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

أضف هذه المتغيرات إلى ملف تعريف shell الخاص بك (`~/.bashrc` أو `~/.zshrc` وما إلى ذلك) لضمان استمرارها.

### ملف التهيئة

بدلاً من ذلك، قم بإنشاء أو تعديل الملف `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## الاستخدام

بعد التهيئة، استخدم Claude Code بشكل طبيعي:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## النماذج المتاحة

يدعم LemonData جميع نماذج Claude:

| النموذج | الوصف |
|-------|-------------|
| `claude-opus-4-5` | الأكثر قدرة، تفكير معقد |
| `claude-sonnet-4-5` | أداء متوازن، ممتاز للبرمجة |
| `claude-haiku-4-5` | الأسرع، وفعال من حيث التكلفة |

## اختيار النموذج

حدد نموذجاً باستخدام علامة `--model`:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

أو قم بتعيين نموذج افتراضي في التهيئة الخاصة بك:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## التفكير الممتد (Extended Thinking)

لمهام التفكير المعقدة، استخدم النماذج التي تدعم التفكير الممتد:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
