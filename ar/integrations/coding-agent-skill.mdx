<Note>
تُعلم هذه المهارة وكيل البرمجة الخاص بك كيفية دمج أي من **أكثر من 300 واجهة برمجة تطبيقات (AI APIs)** من LemonData بمجرد وصف ما تحتاجه. وهي تعمل مع **Claude Code و Cursor و GitHub Copilot و Windsurf** وأي أداة تدعم ملفات المهارات/القواعد (skill/rules).

واجهة برمجة التطبيقات (API) تتبع نهج **[Agent-First](/guides/agent-first-api)** — حتى لو خمن الوكيل اسم النموذج بشكل خاطئ، فإن استجابة الخطأ تخبره بالضبط بكيفية إصلاحه.
</Note>

## كيفية العمل

1. **تصف ما تحتاجه** — "أريد استخدام GPT-4 في مشروع Python الخاص بي"
2. **يقوم الوكيل باستدعاء API** — قد يخمن اسم نموذج أو نقطة نهاية
3. **في حال حدوث أي خطأ** — تعيد API تلميحات منظمة (`did_you_mean` ، `suggestions` ، `retry_after`)
4. **يقوم الوكيل بالتصحيح الذاتي** — يقرأ التلميحات ويعيد المحاولة بالمعلمات الصحيحة
5. **تحصل على كود يعمل** — عادةً من المحاولة الأولى أو الثانية

## التثبيت

<Tabs>
  <Tab title="npx (موصى به)">
    يتم التثبيت لجميع وكلاء البرمجة المكتشفين تلقائياً:

    ```bash
    npx add-skill hedging8563/lemondata-api-skill -y
    ```

  </Tab>


    **خاص بالمشروع** (مشترك مع الفريق):
    ```bash
    git clone https://github.com/hedging8563/lemondata-api-skill.git .claude/skills/lemondata-api-integration
    ```
  </Tab>
