<Note>
**التثبيت الذاتي** يعني أنك تقوم بتشغيل OpenClaw مباشرة على خادم VPS الخاص بك أو جهازك. أنت تدفع فقط مقابل استخدام نماذج الذكاء الاصطناعي من رصيد LemonData الخاص بك — بدون رسوم للمثيل.
</Note>

## التثبيت السريع

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Non-interactive
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

سيطلب منك المثبّت **مفتاح LemonData API Key** الخاص بك. احصل على واحد من [لوحة التحكم](https://lemondata.cc/dashboard).

## ما الذي يتم تثبيته

<Steps>
  <Step title="Node.js 22+">
    يتم تثبيته تلقائيًا إذا لم يكن موجودًا (عبر NodeSource على Linux، Homebrew على macOS)
  </Step>
  <Step title="OpenClaw">
    حزمة OpenClaw الرسمية من npm، مُعدّة مسبقًا مع LemonData API
  </Step>
  <Step title="15 نموذج ذكاء اصطناعي">
    Claude، GPT-5، Gemini، DeepSeek، والمزيد — جاهزة للاستخدام
  </Step>
  <Step title="إضافات المراسلة الفورية">
    إضافات DingTalk و WeCom و QQ مثبتة مسبقًا
  </Step>
  <Step title="خدمة النظام">
    تبدأ تلقائيًا عند التشغيل (systemd على Linux، launchd على macOS)
  </Step>
</Steps>

## أوامر الإدارة

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # التحقق من الحالة
    journalctl --user -u openclaw -f    # عرض السجلات
    systemctl --user restart openclaw   # إعادة التشغيل
    systemctl --user stop openclaw      # الإيقاف
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # عرض السجلات
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # إعادة التشغيل
    launchctl stop cc.lemondata.openclaw  # الإيقاف
    ```
  </Tab>
</Tabs>

## الإعدادات

موقع ملف الإعدادات: `~/.openclaw/openclaw.json`

بعد التعديل، أعد تشغيل الخدمة لتطبيق التغييرات.

### تغيير النموذج الافتراضي

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

بادئات النماذج المتاحة:
- `lemondata-claude/` — نماذج Claude (anthropic-messages API)
- `lemondata-minimax/` — نماذج MiniMax (anthropic-messages API)
- `lemondata/` — جميع النماذج الأخرى (openai-completions API)

## الترقية

```bash
npm update -g openclaw
# ثم أعد تشغيل الخدمة
systemctl --user restart openclaw  # Linux
# أو
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## إلغاء التثبيت

```bash
# إيقاف وتعطيل الخدمة
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# إزالة OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
