## نظرة عامة

OpenCode هو مساعد برمجة مفتوح المصدر يعمل بالذكاء الاصطناعي (AI) في Terminal. يدعم العديد من مزودي LLM ويمكن تكوينه لاستخدام LemonData API للوصول إلى أكثر من 300 نموذج.

## التثبيت

<Tabs>
  <Tab title="سكربت التثبيت">
    ```bash
    curl -fsSL https://opencode.ai/install | bash
    ```
  </Tab>
  <Tab title="Homebrew">
    ```bash
    brew install opencode-ai/tap/opencode
    ```

    أو استخدم tap المجتمع (يتم تحديثه بشكل متكرر):
    ```bash
    brew install anomalyco/tap/opencode
    ```
  </Tab>
  <Tab title="Go">
    ```bash
    go install github.com/opencode-ai/opencode@latest
    ```
  </Tab>
</Tabs>

التحقق من التثبيت:

```bash
opencode --version
```

## التكوين

### الخطوة 1: إعداد متغيرات البيئة

```bash
export OPENAI_API_KEY="sk-your-lemondata-key"
export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"
```

للتكوين الدائم، أضفها إلى `~/.bashrc` أو `~/.zshrc`:

```bash
echo 'export OPENAI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
echo 'export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"' >> ~/.zshrc
source ~/.zshrc
```

### الخطوة 2: تكوين OpenCode

قم بإنشاء أو تحرير `~/.config/opencode/opencode.json` (عالمي) أو `opencode.json` في المجلد الرئيسي لمشروعك:

```json
{
  "provider": {
    "openai": {
      "options": {
        "apiKey": "sk-your-lemondata-key",
        "baseURL": "https://api.lemondata.cc/v1"
      }
    }
  },
  "model": "gpt-4o",
  "agent": {
    "coder": {
      "model": "gpt-4o"
    }
  }
}
```

## الاستخدام الأساسي

**بدء الوضع التفاعلي:**

```bash
opencode
```

**التشغيل باستخدام موجه (Prompt):**

```bash
opencode "Explain this codebase"
```

**تحديد النموذج:**

```bash
opencode --model gpt-4o "Fix the bugs in main.py"
```
