## نظرة عامة

[Cursor](https://cursor.sh) هو محرر أكواد مدعوم بالذكاء الاصطناعي (AI). يمكنك استخدام LemonData كمزود API مخصص للوصول إلى أكثر من 300 نموذج.

## الإعداد

1. افتح إعدادات Cursor (`Cmd/Ctrl + ,`)
2. انتقل إلى **Models** في الشريط الجانبي الأيسر
3. قم بالتمرير لأسفل إلى قسم **OpenAI API Key**
4. أدخل LemonData API key الخاص بك
5. قم بتمكين **Override OpenAI Base URL**
6. قم بتعيين Base URL إلى:

```
https://api.lemondata.cc/v1
```

## اختيار النماذج

بمجرد الانتهاء من الإعداد، يمكنك اختيار أي نموذج من نماذج LemonData في Cursor:

### نماذج البرمجة الموصى بها

| النموذج | الأفضل لـ |
|-------|----------|
| `claude-opus-4-5` | البرمجة المعقدة، تصميم البنية التحتية |
| `claude-sonnet-4-5` | البرمجة العامة، تصحيح الأخطاء (Debugging) |
| `gpt-4o` | توليد الأكواد، الشرح |
| `deepseek-r1` | مهام الاستدلال الثقيلة |
| `gpt-4o-mini` | الإكمال السريع، المهام البسيطة |

## الميزات

### Chat

استخدم لوحة الدردشة المدعومة بالذكاء الاصطناعي (`Cmd/Ctrl + L`) مع أي نموذج من نماذج LemonData:

- شرح الأكواد
- إصلاح الأخطاء (Bugs)
- اقتراحات إعادة الهيكلة (Refactoring)
- توليد التوثيق

### Composer

يمكن استخدام ميزة Composer (`Cmd/Ctrl + I`) مع نماذج LemonData من أجل:

- تحرير ملفات متعددة
- تنفيذ الميزات
- ترحيل الأكواد (Code Migration)

### إكمال Tab

بالنسبة لإكمال Tab، نوصي باستخدام نماذج سريعة:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
