## نظرة عامة

واجهة سطر أوامر Google Gemini (Gemini CLI) هي أداة سطر أوامر للتفاعل مع نماذج Gemini. توفر LemonData نقطة نهاية (endpoint) متوافقة تتيح لك استخدام Gemini CLI والوصول إلى أكثر من 300 نموذج.

## متطلبات النظام

- **Node.js**: الإصدار 20.0+
- **نظام التشغيل**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, أو Debian 10+

## التثبيت

```bash
npm install -g @google/gemini-cli
```

التحقق من التثبيت:

```bash
gemini --version
```

## التهيئة

### الخطوة 1: الحصول على API Key الخاص بك

1. قم بتسجيل الدخول إلى [LemonData Dashboard](https://lemondata.cc/dashboard)
2. انتقل إلى [API Keys](https://lemondata.cc/dashboard/api)
3. قم بإنشاء ونسخ API Key الخاص بك (بصيغة: `sk-...`)

### الخطوة 2: إعداد متغيرات البيئة

**مؤقت (للجلسة الحالية):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**تهيئة دائمة:**

أضف إلى ملف تهيئة shell الخاص بك:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
يرجى إعادة تشغيل الجهاز الطرفي (terminal) بعد اكتمال التهيئة لتفعيل التغييرات.
</Warning>

<Note>
**قيود معروفة**: في بعض الحالات، قد لا تتبع Gemini CLI قيمة `GOOGLE_GEMINI_BASE_URL` إذا كانت هناك جلسة Google مخزنة مؤقتاً. إذا واجهت مشاكل في الاتصال، فحاول مسح ذاكرة التخزين المؤقت لـ CLI أو بدء جلسة terminal جديدة.
</Note>

## الاستخدام الأساسي

قم بتشغيل Gemini CLI من دليل المشروع الخاص بك:

```bash
cd your-project
gemini
```

عند التشغيل لأول مرة، ستحتاج إلى:
1. اختيار سمة (theme)
2. تأكيد إشعار السلامة
3. الوثوق بدليل العمل

## النماذج المتاحة

| النموذج | الوصف |
|-------|-------------|
| `gemini-2.5-pro` | أقوى نماذج Gemini |
| `gemini-2.5-flash` | سريع وفعال، مناسب لمعظم المهام |
| `gemini-2.0-flash` | الإصدار السابق، مستقر |
