## نظرة عامة

يعمل LemonData مع أي تطبيق دردشة متوافق مع OpenAI. يغطي هذا الدليل إعداد تطبيقات الدردشة الشائعة لسطح المكتب والأجهزة المحمولة.

## التطبيقات المدعومة

| التطبيق | المنصة | الميزات |
|-------------|----------|----------|
| LobeChat | الويب، سطح المكتب، الاستضافة الذاتية | متعدد المزودين، إضافات، RAG |
| CherryStudio | Windows, macOS, Linux | إضافات غنية، مزودون متعددون |
| Chatbox | Windows, macOS, Linux | خفيف الوزن، سريع |
| BotGem | iOS, macOS | تجربة Apple أصلية |
| TypingMind | الويب، سطح المكتب | شخصيات مخصصة، إضافات |
| OpenCat | iOS, macOS | أصلي، دعم بدون اتصال |
| ChatWise | Windows, macOS | تعاون الفريق |

## خطوات الإعداد

### الخطوة 1: الحصول على API Key الخاص بك

1. قم بتسجيل الدخول إلى [لوحة تحكم LemonData](https://lemondata.cc/dashboard)
2. انتقل إلى [API Keys](https://lemondata.cc/dashboard/api)
3. قم بإنشاء ونسخ API Key الخاص بك (التنسيق: `sk-...`)

### الخطوة 2: إعداد تطبيقك

<Tabs>
  <Tab title="LobeChat">
    **بالنسبة لـ LobeChat المستضاف ذاتياً:**

    إعداد متغيرات البيئة:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **بالنسبة لـ LobeChat Cloud:**
    1. افتح **Settings** ← **AI Service Provider**
    2. اختر **OpenAI**
    3. أدخل LemonData API Key الخاص بك
    4. قم بإعداد نقطة النهاية المخصصة: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. افتح **Settings** ← **Model Provider**
    2. انقر على **Add Provider**
    3. اختر **OpenAI API Compatible**
    4. أدخل:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. أضف النماذج: `gpt-4o`, `claude-sonnet-4-5` وغيرها
    6. احفظ واختبر
  </Tab>
  <Tab title="Chatbox">
    1. افتح **Settings** ← **AI Model**
    2. اختر **OpenAI API Compatible**
    3. أدخل:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. انقر على **Verify** لاختبار الاتصال
  </Tab>
  <Tab title="BotGem">
    1. افتح **Settings** ← **Providers**
    2. انقر على **Add Custom Provider**
    3. أدخل:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. أضف نماذجك المفضلة
    5. احفظ الإعدادات
  </Tab>
  <Tab title="TypingMind">
    1. انقر على **Settings** (أيقونة الترس)
    2. اختر **Custom Endpoint**
    3. أدخل:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. قم بإعداد النماذج واحفظ
  </Tab>
</Tabs>

### الخطوة 3: إضافة النماذج

بعد إعداد المزود، أضف النماذج التي ترغب في استخدامها:

| الفئة | النموذج |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## مرجع الإعدادات

| الحقل | القيمة |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| نوع API | OpenAI Compatible |

**تكوين JSON (للتطبيقات المدعومة):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
  <Accordion title="الخطأ في الاتصال">
    - تحقق من أن Base URL هو بالضبط `https://api.lemondata.cc/v1`
    - تحقق مما إذا كان هناك شرطة مائلة في النهاية (قم بإزالتها إذا وجدت)
    - تحقق من اتصال الشبكة وإعدادات جدار الحماية
  </Accordion>

  <Accordion title="401 Unauthorized">
    - تحقق مرة أخرى من API Key الخاص بك
    - تحقق من أن Key في حالة نشطة في [لوحة تحكم LemonData](https://lemondata.cc/dashboard/api)
    - تأكد من عدم وجود مسافات زائدة في Key
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - تحقق من رصيد حسابك في لوحة التحكم
    - قم بشحن رصيد حسابك
  </Accordion>

  <Accordion title="404 Model Not Available">
    - تحقق من صحة إملاء اسم النموذج
    - تحقق من توفر النموذج في [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - جرب نموذجاً مختلفاً
  </Accordion>

  <Accordion title="استجابة بطيئة">
    - جرب نموذجاً أسرع (على سبيل المثال، استخدم `gpt-4o-mini` بدلاً من `gpt-4o`)
    - إذا كان ذلك ممكناً، قم بتقليل معلمة `max_tokens`
    - تحقق من اتصال الشبكة الخاص بك
  </Accordion>
</AccordionGroup>
