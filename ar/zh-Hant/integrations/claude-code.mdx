## نظرة عامة

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) هي أداة CLI رسمية للمساعدة في البرمجة بالذكاء الاصطناعي من Anthropic. يمكنك تهيئتها لاستخدام LemonData كمزود API.

## التهيئة

### متغيرات البيئة

قم بتعيين متغيرات البيئة التالية:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

أضف هذه الأسطر إلى ملف تعريف shell الخاص بك (`~/.bashrc` أو `~/.zshrc` وما إلى ذلك) لجعلها دائمة.

### ملف التهيئة

بدلاً من ذلك، قم بإنشاء أو تحرير `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## طريقة الاستخدام

بمجرد الانتهاء من التهيئة، يمكنك استخدام Claude Code بشكل طبيعي:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## النماذج المتاحة

يدعم LemonData جميع نماذج Claude:

| النموذج | الوصف |
|-------|-------------|
| `claude-opus-4-5` | أقوى نموذج، مناسب للاستنتاج المعقد |
| `claude-sonnet-4-5` | أداء متوازن، مثالي جداً للبرمجة |
| `claude-haiku-4-5` | الأسرع والأكثر فعالية من حيث التكلفة |

## اختيار النموذج

استخدم علامة `--model` لتحديد النموذج:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

أو قم بتعيين القيمة الافتراضية في التهيئة الخاصة بك:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## التفكير الموسع

للمهام الاستنتاجية المعقدة، استخدم النماذج التي تدعم ميزة التفكير الموسع:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
