## نظرة عامة

عملية توليد الفيديو غير متزامنة. تقوم بإرسال الطلب وتتلقى معرف مهمة `task_id` ثم تقوم بالاستعلام (polling) عن النتيجة.

## جسم الطلب (Request Body)

<ParamField body="model" type="string" default="kling-video">
  نموذج الفيديو (على سبيل المثال: `kling-video`، `veo3.1`، `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  الوصف النصي للفيديو المراد توليده.
</ParamField>

<ParamField body="image_url" type="string">
  صورة البداية الاختيارية لتوليد فيديو من صورة (image-to-video).
</ParamField>

<ParamField body="duration" type="integer">
  مدة الفيديو بالثواني (تعتمد على النموذج).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  نسبة العرض إلى الارتفاع (على سبيل المثال: `16:9`، `9:16`، `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  دقة الفيديو (على سبيل المثال: `1080p`، `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  عدد الإطارات في الثانية (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  المحتوى الذي يجب تجنبه في توليد الفيديو.
</ParamField>

<ParamField body="seed" type="integer">
  بذرة عشوائية (seed) لعمليات التوليد القابلة للتكرار.
</ParamField>

<ParamField body="cfg_scale" type="number">
  مقياس التوجيه بدون مصنف (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  قوة الحركة (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  رابط URL لصورة الإطار الأول.
</ParamField>

<ParamField body="end_image" type="string">
  رابط URL لصورة الإطار الأخير.
</ParamField>

<ParamField body="user" type="string">
  المعرف الفريد للمستخدم النهائي.
</ParamField>

## الاستجابة

<ResponseField name="task_id" type="string">
  معرف المهمة الفريد المستخدم للاستعلام (polling).
</ResponseField>

<ResponseField name="status" type="string">
  الحالة الأولية: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  النموذج المستخدم.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

## تحويل الصورة إلى فيديو (Image to Video)

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```

## النماذج المتاحة

| النموذج | النوع | الوصف |
|-------|------|-------------|
| `kling-video` | T2V/I2V | حركة واقعية، إصدارات متعددة (v1.0-v2.6) |
| `veo3.1` | T2V | أحدث نموذج من Google |
| `veo3.1-pro` | T2V | جودة احترافية |
| `minimax/video-01` | T2V | إيقاع عالٍ واستقرار |
| `wan2.6-i2v` | I2V | تحويل الصورة إلى فيديو |
