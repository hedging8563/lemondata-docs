---
title: "إنشاء الموسيقى"
openapi: "POST /v1/music/generations"
description: "إنشاء مهمة توليد موسيقى باستخدام Suno"
---

استخدم الذكاء الاصطناعي لتوليد الموسيقى والكلمات. هذا API غير متزامن - ستتلقى معرف مهمة (ID) لاستخدامه في الاستعلام عن حالة الإكمال.

## معلمات الطلب

<ParamField body="model" type="string" default="suno_music">
  النموذج المراد استخدامه: `suno_music` لتوليد الموسيقى، و `suno_lyrics` للكلمات فقط.
</ParamField>

<ParamField body="prompt" type="string" required>
  وصف الموسيقى المراد توليدها. يمكن أن يتضمن النمط، والمزاج، والآلات الموسيقية، وما إلى ذلك.
</ParamField>

<ParamField body="title" type="string">
  عنوان الأغنية المولدة.
</ParamField>

<ParamField body="tags" type="string">
  علامات النمط (مثال: "pop, upbeat, electronic").
</ParamField>

<ParamField body="action" type="string">
  نوع التوليد: `MUSIC` (افتراضي) أو `LYRICS`.
</ParamField>

<ParamField body="mv" type="string">
  إصدار النموذج المراد استخدامه.
</ParamField>

<ParamField body="continue_clip_id" type="string">
  معرف المقطع السابق للمتابعة منه.
</ParamField>

<ParamField body="continue_at" type="number">
  الطابع الزمني للمتابعة (بالثواني).
</ParamField>

<ParamField body="user" type="string">
  المعرف الفريد للمستخدم النهائي.
</ParamField>

## الاستجابة

<ResponseField name="id" type="string">
  معرف المهمة المستخدم للاستعلام عن الحالة.
</ResponseField>

<ResponseField name="status" type="string">
  حالة المهمة: `pending` أو `processing` أو `completed` أو `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  طابع Unix الزمني لإنشاء المهمة.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "suno_music",
        "prompt": "An upbeat electronic dance track with heavy bass",
        "title":  "Night Drive",
        "tags":   "electronic, EDM, energetic",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/music/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```

```php PHP

        'title' => 'Night Drive',
        'tags' => 'electronic, EDM, energetic'
    ])
]);

curl_close($ch);

echo "Task ID: " . $data['id'];
```
