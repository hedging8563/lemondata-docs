## نظرة عامة

ترجمة الصوت من أي لغة مدعومة إلى نص باللغة الإنجليزية. على عكس النسخ النصي (transcription)، يقوم هذا المسار دائمًا بإخراج نص باللغة الإنجليزية بغض النظر عن لغة الإدخال.

## جسم الطلب

<ParamField body="file" type="file" required>
  ملف الصوت المراد ترجمته. التنسيقات المدعومة: `flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`. الحد الأقصى لحجم الملف هو 25 ميجابايت.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  النموذج المراد استخدامه. حاليًا يتم دعم `whisper-1` فقط.
</ParamField>

<ParamField body="prompt" type="string">
  نص اختياري لتوجيه أسلوب النموذج أو مواصلة مقطع سابق. يجب أن يكون باللغة الإنجليزية.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  تنسيق الإخراج. الخيارات: `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number">
  درجة حرارة العينات، بين 0 و 1. القيم الأعلى (مثل 0.8) ستؤدي إلى مخرجات أكثر عشوائية، بينما القيم الأقل (مثل 0.2) ستجعل المخرجات أكثر تركيزًا وحتمية.
</ParamField>

## الاستجابة

<ResponseField name="text" type="string">
  النص الإنجليزي المترجم.
</ResponseField>

بالنسبة لتنسيق `verbose_json`، تتضمن الاستجابة أيضًا:

<ResponseField name="language" type="string">
  لغة الصوت المدخل المكتشفة.
</ResponseField>

<ResponseField name="duration" type="number">
  مدة الصوت المدخل (بالثواني).
</ResponseField>

<ResponseField name="segments" type="array">
  مقاطع النص المترجم مع الطوابع الزمنية.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/translations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -F "file=@german_audio.mp3" \
  -F "model=whisper-1"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

with open("german_audio.mp3", "rb") as audio_file:
    response = client.audio.translations.create(
        model="whisper-1",
        file=audio_file
    )

print(response.text)
```

```javascript JavaScript
import OpenAI from 'openai';
import fs from 'fs';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.audio.translations.create({
  model: 'whisper-1',
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/translations');

$file = new CURLFile('german_audio.mp3', 'audio/mpeg', 'german_audio.mp3');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => [
        'file' => $file,
        'model' => 'whisper-1'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['text'];
```

## الترجمة مقابل النسخ النصي

| الميزة | الترجمة | النسخ النصي |
|---------|-------------|---------------|
| لغة الإخراج | دائمًا الإنجليزية | نفس لغة الإدخال |
| حالة الاستخدام | تحويل الصوت الأجنبي إلى الإنجليزية | الحفاظ على اللغة الأصلية |
| معلمة اللغة | غير قابل للتطبيق | تلميح اختياري |

<Note>
  يقوم مسار الترجمة تلقائيًا باكتشاف اللغة المصدر وترجمتها إلى الإنجليزية. سيتم تجاهل معلمة `language` المستخدمة في النسخ النصي.
</Note>
