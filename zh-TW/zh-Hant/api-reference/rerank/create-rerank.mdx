---
title: "重排文件"
openapi: "POST /v1/rerank"
description: "根據與查詢的相關性對文件進行重排"
---

使用語義相似度模型對文件進行重排。適用於優化搜尋結果和 RAG 應用程式。

## 請求主體

<ParamField body="model" type="string" required>
  要使用的重排模型 ID（例如：`BAAI/bge-reranker-v2-m3`、`qwen3-rerank`）。
</ParamField>

<ParamField body="query" type="string" required>
  用於對文件進行排名的查詢語句。
</ParamField>

<ParamField body="documents" type="array" required>
  要進行重排的文件列表（字串）。
</ParamField>

<ParamField body="top_n" type="integer">
  要返回的前幾個結果數量。預設為所有文件。
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  是否在回應中包含原始文件文本。
</ParamField>

## 回應

<ResponseField name="results" type="array">
  帶有評分的已排序文件列表。

  每個結果包含：
  - index (integer)：原始文件索引
  - relevance_score (number)：相關性評分 (0-1)
  - document (string)：原始文本（若 return_documents=true）
</ResponseField>

<ResponseField name="model" type="string">
  用於重排的模型。
</ResponseField>

<ResponseField name="usage" type="object">
  Token 使用量統計。
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/rerank" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "BAAI/bge-reranker-v2-m3",
    "query": "What is machine learning?",
    "documents": [
      "Machine learning is a subset of AI",
      "The weather is nice today",
      "Deep learning uses neural networks"
    ],
    "top_n": 2,
    "return_documents": true
  }'
```
