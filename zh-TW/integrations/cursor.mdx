## 概覽

[Cursor](https://cursor.sh) 是一款 AI 驅動的程式碼編輯器。您可以將 LemonData 作為自定義 API 提供商，以存取 300 多種模型。

## 配置

1. 打開 Cursor 設定 (`Cmd/Ctrl + ,`)
2. 導覽至左側側邊欄的 **Models**
3. 向下捲動至 **OpenAI API Key** 區塊
4. 輸入您的 LemonData API key
5. 啟用 **Override OpenAI Base URL**
6. 將 Base URL 設定為：

```
https://api.lemondata.cc/v1
```

## 模型選擇

配置完成後，您可以在 Cursor 中選擇任何 LemonData 模型：

### 推薦的程式碼編寫模型

| 模型 | 最適合 |
|-------|----------|
| `claude-opus-4-5` | 複雜編碼、架構設計 |
| `claude-sonnet-4-5` | 一般編碼、除錯 |
| `gpt-4o` | 程式碼生成、解釋 |
| `deepseek-r1` | 重推理任務 |
| `gpt-4o-mini` | 快速補全、簡單任務 |

## 功能

### Chat

配合任何 LemonData 模型使用 AI 對話面板 (`Cmd/Ctrl + L`)：

- 程式碼解釋
- 修復 Bug
- 重構建議
- 生成文件

### Composer

Composer 功能 (`Cmd/Ctrl + I`) 可配合 LemonData 模型用於：

- 多檔案編輯
- 功能實現
- 程式碼遷移

### Tab 補全

對於 Tab 補全，我們建議使用快速模型：

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
