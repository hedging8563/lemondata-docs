## Resumen

LemonData funciona con cualquier aplicación de chat compatible con OpenAI. Esta guía cubre la configuración de aplicaciones de chat populares para escritorio y dispositivos móviles.

## Aplicaciones compatibles

| Aplicación | Plataforma | Características |
|-------------|----------|----------|
| LobeChat | Web, escritorio, auto-alojado | Multi-proveedor, plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Plugins variados, múltiples proveedores |
| Chatbox | Windows, macOS, Linux | Ligero, rápido |
| BotGem | iOS, macOS | Experiencia nativa de Apple |
| TypingMind | Web, escritorio | Personalidades personalizadas, plugins |
| OpenCat | iOS, macOS | Nativo, soporte offline |
| ChatWise | Windows, macOS | Colaboración en equipo |

## Pasos de configuración

### Paso 1: Obtén tu API Key

1. Inicia sesión en el [Panel de control de LemonData](https://lemondata.cc/dashboard)
2. Navega a [API Keys](https://lemondata.cc/dashboard/api)
3. Crea y copia tu API Key (formato: `sk-...`)

### Paso 2: Configura tu aplicación

<Tabs>
  <Tab title="LobeChat">
    **Para LobeChat auto-alojado:**

    Configura las variables de entorno:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Para LobeChat Cloud:**
    1. Abre **Settings** → **AI Service Provider**
    2. Selecciona **OpenAI**
    3. Ingresa tu API Key de LemonData
    4. Configura el endpoint personalizado: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Abre **Settings** → **Model Provider**
    2. Haz clic en **Add Provider**
    3. Selecciona **OpenAI API Compatible**
    4. Ingresa:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Agrega modelos: `gpt-4o`, `claude-sonnet-4-5` etc.
    6. Guarda y prueba
  </Tab>
  <Tab title="Chatbox">
    1. Abre **Settings** → **AI Model**
    2. Selecciona **OpenAI API Compatible**
    3. Ingresa:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Haz clic en **Verify** para probar la conexión
  </Tab>
  <Tab title="BotGem">
    1. Abre **Settings** → **Providers**
    2. Haz clic en **Add Custom Provider**
    3. Ingresa:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Agrega tus modelos preferidos
    5. Guarda la configuración
  </Tab>
  <Tab title="TypingMind">
    1. Haz clic en **Settings** (icono de engranaje)
    2. Selecciona **Custom Endpoint**
    3. Ingresa:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Configura los modelos y guarda
  </Tab>
</Tabs>

### Paso 3: Agrega modelos

Después de configurar el proveedor, agrega los modelos que deseas usar:

| Categoría | Modelo |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Referencia de configuración

| Campo | Valor |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| Tipo de API | OpenAI Compatible |

**Configuración JSON (para aplicaciones compatibles):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Solución de problemas

<AccordionGroup>
  <Accordion title="Error de conexión">
    - Verifica que la Base URL sea exactamente `https://api.lemondata.cc/v1`
    - Comprueba si hay una barra diagonal al final (elimínala si la hay)
    - Verifica la conexión a internet y la configuración del firewall
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Vuelve a comprobar tu API Key
    - Verifica que la Key esté activa en el [Panel de control de LemonData](https://lemondata.cc/dashboard/api)
    - Asegúrate de que no haya espacios adicionales en la Key
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - Consulta el saldo de tu cuenta en el panel de control
    - Recarga tu cuenta
  </Accordion>

  <Accordion title="404 Model Not Available">
    - Verifica que el nombre del modelo esté escrito correctamente
    - Consulta la disponibilidad del modelo en [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - Prueba con un modelo diferente
  </Accordion>

  <Accordion title="Respuesta lenta">
    - Prueba con un modelo más rápido (por ejemplo, usa `gpt-4o-mini` en lugar de `gpt-4o`)
    - Si es posible, reduce el parámetro `max_tokens`
    - Comprueba tu conexión a internet
  </Accordion>
</AccordionGroup>

## Consejos

<AccordionGroup>
  <Accordion title="Usa el modelo adecuado">
    Elige el modelo según la tarea: usa GPT-4o-mini para chats rápidos y Claude o GPT-4o para tareas complejas.
  </Accordion>

  <Accordion title="Guarda prompts frecuentes">
    La mayoría de las aplicaciones permiten guardar prompts o personalidades. Crea plantillas para tareas comunes.
  </Accordion>

  <Accordion title="Monitorea el uso">
    Revisa tu uso en el panel de control de LemonData para realizar un seguimiento de los costos y gestionar tu presupuesto.
  </Accordion>
</AccordionGroup>
