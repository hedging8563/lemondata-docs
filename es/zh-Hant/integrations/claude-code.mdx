## Resumen

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) es la herramienta CLI de programación asistida por AI oficial de Anthropic. Puede configurarla para utilizar LemonData como proveedor de API.

## Configuración

### Variables de entorno

Establezca las siguientes variables de entorno:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Agregue estas líneas a su archivo de configuración de shell (`~/.bashrc`, `~/.zshrc`, etc.) para que los cambios sean persistentes.

### Archivo de configuración

Alternativamente, cree o edite `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Uso

Una vez configurado, puede utilizar Claude Code con normalidad:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Modelos disponibles

LemonData es compatible con todos los modelos de Claude:

| Modelo | Descripción |
|-------|-------------|
| `claude-opus-4-5` | El modelo más potente, ideal para razonamiento complejo |
| `claude-sonnet-4-5` | Rendimiento equilibrado, excelente para programación |
| `claude-haiku-4-5` | El más rápido y rentable |

## Selección de modelo

Utilice el flag `--model` para especificar un modelo:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

O establezca un valor predeterminado en su configuración:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Pensamiento extendido

Para tareas de razonamiento complejo, utilice modelos con capacidades de pensamiento extendido:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Solución de problemas

<AccordionGroup>
  <Accordion title="Error de conexión">
    Verifique que la URL base sea exactamente `https://api.lemondata.cc` (el SDK de Anthropic no requiere el sufijo `/v1`).
  </Accordion>

  <Accordion title="Clave API inválida">
    Asegúrese de que su clave API de LemonData comience con `sk-` y tenga saldo suficiente.
  </Accordion>

  <Accordion title="Modelo no encontrado">
    Compruebe que el nombre del modelo sea correcto. Utilice el formato `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
