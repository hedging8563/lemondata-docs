## Resumen

[Cursor](https://cursor.sh) es un editor de código impulsado por AI. Puedes usar LemonData como un proveedor de API personalizado para acceder a más de 300 modelos.

## Configuración

1. Abre la configuración de Cursor (`Cmd/Ctrl + ,`)
2. Navega a **Models** en la barra lateral izquierda
3. Desplázate hacia abajo hasta la sección **OpenAI API Key**
4. Introduce tu LemonData API key
5. Habilita **Override OpenAI Base URL**
6. Establece la Base URL como:

```
https://api.lemondata.cc/v1
```

## Selección de modelos

Una vez configurado, puedes seleccionar cualquier modelo de LemonData en Cursor:

### Modelos de codificación recomendados

| Modelo | Ideal para |
|-------|----------|
| `claude-opus-4-5` | Codificación compleja, diseño de arquitectura |
| `claude-sonnet-4-5` | Codificación general, depuración |
| `gpt-4o` | Generación de código, explicación |
| `deepseek-r1` | Tareas de razonamiento pesado |
| `gpt-4o-mini` | Autocompletado rápido, tareas simples |

## Funciones

### Chat

Usa el panel de chat de AI (`Cmd/Ctrl + L`) con cualquier modelo de LemonData:

- Explicación de código
- Corrección de errores
- Sugerencias de refactorización
- Generación de documentación

### Composer

La función Composer (`Cmd/Ctrl + I`) se puede usar con los modelos de LemonData para:

- Edición de múltiples archivos
- Implementación de funciones
- Migración de código

### Autocompletado de Tab

Para el autocompletado de Tab, recomendamos usar modelos rápidos:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
