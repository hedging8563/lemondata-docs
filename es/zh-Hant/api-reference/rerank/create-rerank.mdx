---
title: "Reordenar documentos"
openapi: "POST /v1/rerank"
description: "Reordena documentos según su relevancia con la consulta"
---

Utiliza modelos de similitud semántica para reordenar documentos. Ideal para optimizar resultados de búsqueda y aplicaciones RAG.

## Cuerpo de la solicitud

<ParamField body="model" type="string" required>
  ID del modelo de rerank a utilizar (por ejemplo: `BAAI/bge-reranker-v2-m3`, `qwen3-rerank`).
</ParamField>

<ParamField body="query" type="string" required>
  La consulta utilizada para clasificar los documentos.
</ParamField>

<ParamField body="documents" type="array" required>
  Lista de documentos a reordenar (cadenas de texto).
</ParamField>

<ParamField body="top_n" type="integer">
  Número de resultados principales a devolver. Por defecto son todos los documentos.
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  Indica si se debe incluir el texto original del documento en la respuesta.
</ParamField>

## Respuesta

<ResponseField name="results" type="array">
  Lista de documentos ordenados con sus puntuaciones.

  Cada resultado contiene:
  - `index` (integer): Índice del documento original
  - `relevance_score` (number): Puntuación de relevancia (0-1)
  - `document` (string): Texto original (si `return_documents=true`)
</ResponseField>

<ResponseField name="model" type="string">
  El modelo utilizado para el rerank.
</ResponseField>

<ResponseField name="usage" type="object">
  Estadísticas de uso de tokens.
</ResponseField>
