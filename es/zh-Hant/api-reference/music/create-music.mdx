---
title: "Crear música"
openapi: "POST /v1/music/generations"
description: "Crear una tarea de generación de música con Suno"
---

Genera música y letras usando IA. Esta es una API asíncrona: recibirá un ID de tarea para consultar el estado de finalización.

## Parámetros de la solicitud

<ParamField body="model" type="string" default="suno_music">
  El modelo a utilizar: `suno_music` para la generación de música, `suno_lyrics` solo para letras.
</ParamField>

<ParamField body="prompt" type="string" required>
  Descripción de la música a generar. Puede incluir estilo, estado de ánimo, instrumentos, etc.
</ParamField>

<ParamField body="title" type="string">
  Título de la canción generada.
</ParamField>

<ParamField body="tags" type="string">
  Etiquetas de estilo (por ejemplo: "pop, upbeat, electronic").
</ParamField>

<ParamField body="action" type="string">
  Tipo de generación: `MUSIC` (predeterminado) o `LYRICS`.
</ParamField>

<ParamField body="mv" type="string">
  Versión del modelo a utilizar.
</ParamField>

<ParamField body="continue_clip_id" type="string">
  ID del clip anterior para continuar.
</ParamField>

<ParamField body="continue_at" type="number">
  Marca de tiempo para continuar (en segundos).
</ParamField>

<ParamField body="user" type="string">
  Identificador único del usuario final.
</ParamField>

## Respuesta

<ResponseField name="id" type="string">
  ID de la tarea utilizado para consultar el estado.
</ResponseField>

<ResponseField name="status" type="string">
  Estado de la tarea: `pending`, `processing`, `completed` o `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Marca de tiempo Unix de la creación de la tarea.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "suno_music",
        "prompt": "An upbeat electronic dance track with heavy bass",
        "title":  "Night Drive",
        "tags":   "electronic, EDM, energetic",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/music/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```
