---
title: "Crear modelo 3D"
openapi: "POST /v1/3d/generations"
description: "Crea una tarea de generación de modelos 3D"
---

Genera modelos 3D a partir de texto o imágenes utilizando Tripo3D y otros proveedores. Esta es una API asíncrona.

## Request Body

<ParamField body="model" type="string" default="tripo3d-v2.5">
  Modelo a utilizar (por ejemplo, `tripo3d-v2.5`, `tripo3d-v2`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Descripción de texto del modelo 3D a generar.
</ParamField>

<ParamField body="image" type="string">
  Imagen codificada en Base64 para la generación de imagen a 3D.
</ParamField>

<ParamField body="image_url" type="string">
  URL de la imagen para la generación de imagen a 3D.
</ParamField>

<ParamField body="format" type="string" default="glb">
  Formato de salida: `glb`, `fbx`, `obj` o `usdz`.
</ParamField>

<ParamField body="quality" type="string" default="standard">
  Nivel de calidad: `draft`, `standard` o `high`.
</ParamField>

<ParamField body="style" type="string">
  Preajuste de estilo para el modelo.
</ParamField>

<ParamField body="seed" type="integer">
  Semilla para una generación reproducible.
</ParamField>

<ParamField body="user" type="string">
  Un identificador único para el usuario final.
</ParamField>

## Response

<ResponseField name="id" type="string">
  ID de la tarea para consultar el estado.
</ResponseField>

<ResponseField name="status" type="string">
  Estado de la tarea: `pending`, `processing`, `completed` o `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Marca de tiempo Unix de la creación de la tarea.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":   "tripo3d-v2.5",
        "prompt":  "A detailed medieval castle with towers",
        "format":  "glb",
        "quality": "high",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/3d/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```
