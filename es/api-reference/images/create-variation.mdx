---
title: "Crear variacion de imagen"
openapi: "POST /v1/images/variations"
description: "Crea una variacion de una imagen dada"
---

## Autorización

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticación de API. Formato: `Bearer sk-your-api-key`
</ParamField>

## Descripcion general

Crea una variacion de una imagen dada. Requiere el tipo de contenido `multipart/form-data`.

## Cuerpo de la solicitud

<ParamField body="image" type="file" required>
  La imagen a usar como base para la variacion. Debe ser un archivo PNG valido, menor de 4MB y cuadrado.
</ParamField>

<ParamField body="model" type="string" default="dall-e-2">
  El modelo a usar para las variaciones de imagen. Actualmente solo se admite `dall-e-2`.
</ParamField>

<ParamField body="n" type="integer" default="1">
  El numero de imagenes a generar. Debe estar entre 1 y 10.
</ParamField>

<ParamField body="size" type="string">
  El tamano de las imagenes generadas. Debe ser uno de `256x256`, `512x512` o `1024x1024`.
</ParamField>

<ParamField body="response_format" type="string" default="url">
  El formato en que se devuelven las imagenes generadas. Debe ser `url` o `b64_json`.
</ParamField>

<ParamField body="user" type="string">
  Un identificador unico que representa a su usuario final para monitoreo de abuso.
</ParamField>

## Respuesta

<ResponseField name="created" type="integer">
  Marca de tiempo Unix de cuando se crearon las imagenes.
</ResponseField>

<ResponseField name="data" type="array">
  Array de variaciones de imagen generadas.

  Cada objeto contiene:
  - `url` (string): URL de la imagen de variacion (si response_format es `url`)
  - `b64_json` (string): Imagen codificada en Base64 (si response_format es `b64_json`)
</ResponseField>

```go Go
package main

import (
    "bytes"
    "fmt"
    "io"
    "mime/multipart"
    "net/http"
    "os"
)

func main() {
    body := &bytes.Buffer{}
    writer := multipart.NewWriter(body)

    file, _ := os.Open("cat.png")
    defer file.Close()
    part, _ := writer.CreateFormFile("image", "cat.png")
    io.Copy(part, file)

    writer.WriteField("n", "2")
    writer.WriteField("size", "1024x1024")
    writer.Close()

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/images/variations", body)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", writer.FormDataContentType())

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

$data = json_decode($response, true);

}

## Notas

<Note>
  Las variaciones de imagen solo estan disponibles con DALL-E 2. El modelo generara nuevas imagenes que son similares en estilo y contenido a la imagen original.
</Note>
