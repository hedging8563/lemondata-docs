---
title: "Crear voz"
openapi: "POST /v1/audio/speech"
description: "Genera audio a partir del texto de entrada"
---

## Autorización

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticación de API. Formato: `Bearer sk-your-api-key`
</ParamField>

## Cuerpo de la solicitud

<ParamField body="model" type="string" default="tts-1">
  Modelo TTS: `tts-1` (estándar) o `tts-1-hd` (mayor calidad).
</ParamField>

<ParamField body="input" type="string" required>
  El texto para el cual se generará el audio. Máximo 4096 caracteres.
</ParamField>

<ParamField body="voice" type="string" required>
  Voz a utilizar: `alloy`, `ash`, `ballad`, `coral`, `echo`, `fable`, `nova`, `onyx`, `sage`, `shimmer`, `verse`.
</ParamField>

<ParamField body="response_format" type="string" default="mp3">
  Formato de audio: `mp3`, `opus`, `aac`, `flac`, `wav`, `pcm`.
</ParamField>

<ParamField body="speed" type="number" default="1.0">
  Velocidad del audio (0.25 a 4.0).
</ParamField>

## Respuesta

Devuelve el archivo de audio en el formato solicitado.

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1-hd",
    "voice": "nova",
    "input": "Hello, welcome to LemonData!"
  }' \
  --output speech.mp3
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.audio.speech.create(
    model="tts-1-hd",
    voice="nova",
    input="Hello, welcome to LemonData!"

response.stream_to_file("speech.mp3")
```

```javascript JavaScript
import OpenAI from 'openai';
import fs from 'fs';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.audio.speech.create({
  model: 'tts-1-hd',
  voice: 'nova',
  input: 'Hello, welcome to LemonData!'
});

const buffer = Buffer.from(await response.arrayBuffer());
fs.writeFileSync('speech.mp3', buffer);
```

```go Go
package main

import (
    "context"
    "io"
    "os"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateSpeech(
        context.Background(),
        openai.CreateSpeechRequest{
            Model: openai.TTSModel1HD,
            Voice: openai.VoiceNova,
            Input: "Hello, welcome to LemonData!",
        },
    )
    if err != nil {
        panic(err)
    }
    defer resp.Close()

    file, _ := os.Create("speech.mp3")
    defer file.Close()
    io.Copy(file, resp)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/speech');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'tts-1-hd',
        'voice' => 'nova',
        'input' => 'Hello, welcome to LemonData!'
    ])
]);

$audio = curl_exec($ch);
curl_close($ch);

file_put_contents('speech.mp3', $audio);
```

## Muestras de voz

| Voz | Descripción |
|-------|-------------|
| `alloy` | Neutral, equilibrada |
| `ash` | Calma, pausada |
| `ballad` | Melódica, expresiva |
| `coral` | Cálida, acogedora |
| `echo` | Cálida, conversacional |
| `fable` | Expresiva, narrativa |
| `nova` | Amigable, clara |
| `onyx` | Profunda, autoritaria |
| `sage` | Sabia, reflexiva |
| `shimmer` | Suave, delicada |
| `verse` | Dinámica, versátil |
