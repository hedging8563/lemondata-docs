## Par치metros de Consulta

<ParamField query="model" type="string">
  ID del modelo para el cual obtener los precios. Si no se especifica, devuelve los precios de todos los modelos.
</ParamField>

## Respuesta

<ResponseField name="object" type="string">
  Siempre `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Array de objetos de precios de modelos.

  Cada objeto contiene:
  - `model` (string): Identificador del modelo
  - `pricing` (object): Informaci칩n de precios
    - `input_per_1m` (string | null): USD por cada 1M de tokens de entrada
    - `output_per_1m` (string | null): USD por cada 1M de tokens de salida
    - `per_request` (string | null): USD por solicitud (para modelos de precio fijo)
    - `is_lock_price` (boolean): Indica si el precio es fijo por solicitud
    - `currency` (string): Siempre `USD`
  - `provider` (string): Proveedor del modelo
</ResponseField>
