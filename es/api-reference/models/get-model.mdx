---
title: "Obtener Modelo"
api: "GET /v1/models/{model}"
description: "Recupera una instancia de modelo"
---

## Parámetros de Ruta

<ParamField path="model" type="string" required>
  El ID del modelo a recuperar (ej., `gpt-4o`, `claude-sonnet-4-5`).
</ParamField>

## Respuesta

<ResponseField name="id" type="string">
  Identificador del modelo.
</ResponseField>

<ResponseField name="object" type="string">
  Siempre `model`.
</ResponseField>

<ResponseField name="created" type="integer">
  Marca de tiempo de creación.
</ResponseField>

<ResponseField name="owned_by" type="string">
  Proveedor del modelo.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1/models/gpt-4o" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

model = client.models.retrieve("gpt-4o")
print(f"Modelo: {model.id}, Proveedor: {model.owned_by}")
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const model = await client.models.retrieve('gpt-4o');
console.log(`Modelo: ${model.id}, Proveedor: ${model.owned_by}`);
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    model, err := client.GetModel(context.Background(), "gpt-4o")
    if err != nil {
        panic(err)
    }
    fmt.Printf("Modelo: %s, Proveedor: %s\n", model.ID, model.OwnedBy)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/models/gpt-4o');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$model = json_decode($response, true);
echo "Modelo: {$model['id']}, Proveedor: {$model['owned_by']}\n";
```

## Manejo de Errores

Si el modelo no existe, recibirás un error 404:

```json
{
  "error": {
    "message": "Model 'invalid-model' not found",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```
