<Note>
**Autoalojado** significa que ejecutas OpenClaw directamente en tu propio VPS o máquina. Solo pagas por el uso de modelos AI a través de tu saldo de LemonData — sin costos de instancia.
</Note>

## Instalación rápida

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash No interactivo
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

El instalador te pedirá tu **LemonData API Key**. Obtén una desde tu [Dashboard](https://lemondata.cc/dashboard).

## Qué se instala

<Steps>
  <Step title="Node.js 22+">
    Se instala automáticamente si no está presente (vía NodeSource en Linux, Homebrew en macOS)
  </Step>
  <Step title="OpenClaw">
    Paquete oficial de OpenClaw desde npm, preconfigurado con LemonData API
  </Step>
  <Step title="15 modelos AI">
    Claude, GPT-5, Gemini, DeepSeek y más — listos para usar
  </Step>
  <Step title="Plugins de IM">
    Plugins de DingTalk, WeCom y QQ preinstalados
  </Step>
  <Step title="Servicio del sistema">
    Se inicia automáticamente al arrancar (systemd en Linux, launchd en macOS)
  </Step>
</Steps>

## Requisitos

| Requisito | Detalles |
|-----------|----------|
| **Sistema operativo** | Linux (Ubuntu/Debian/CentOS/RHEL) o macOS |
| **Arquitectura** | x64 o arm64 |
| **RAM** | 1 GB mínimo, 2 GB recomendado |
| **API Key** | [Obtener aquí](https://lemondata.cc) |

## Comandos de gestión

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # Verificar estado
    journalctl --user -u openclaw -f    # Ver logs
    systemctl --user restart openclaw   # Reiniciar
    systemctl --user stop openclaw      # Detener
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # Ver logs
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # Reiniciar
    launchctl stop cc.lemondata.openclaw  # Detener
    ```
  </Tab>
</Tabs>

## Configuración

Ubicación del archivo de configuración: `~/.openclaw/openclaw.json`

Después de editar, reinicia el servicio para aplicar los cambios.

### Cambiar el modelo predeterminado

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Prefijos de modelos disponibles:
- `lemondata-claude/` — Modelos Claude (anthropic-messages API)
- `lemondata-minimax/` — Modelos MiniMax (anthropic-messages API)
- `lemondata/` — Todos los demás modelos (openai-completions API)

## Actualización

```bash
npm update -g openclaw
# Luego reiniciar el servicio
systemctl --user restart openclaw  # Linux
# o
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Desinstalación

```bash
# Detener y deshabilitar el servicio
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# Eliminar OpenClaw
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
