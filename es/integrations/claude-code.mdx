## Descripción general

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) es la herramienta de CLI oficial de Anthropic para la codificación asistida por IA. Puede configurarla para usar LemonData como proveedor de API.

## Configuración

### Variables de entorno

Establezca las siguientes variables de entorno:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Añada estas a su perfil de shell (`~/.bashrc`, `~/.zshrc`, etc.) para que sean persistentes.

### Archivo de configuración

Alternativamente, cree o edite `~/.claude/settings.json`:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Uso

Después de la configuración, use Claude Code normalmente:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Modelos disponibles

LemonData es compatible con todos los modelos de Claude:

| Modelo | Descripción |
|-------|-------------|
| `claude-opus-4-5` | El más capaz, razonamiento complejo |
| `claude-sonnet-4-5` | Rendimiento equilibrado, excelente para codificación |
| `claude-haiku-4-5` | El más rápido, rentable |

## Selección de modelo

Especifique un modelo con el flag `--model`:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

O establezca un valor predeterminado en su configuración:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Pensamiento extendido

Para tareas de razonamiento complejo, use modelos con pensamiento extendido (extended thinking):

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Solución de problemas

<AccordionGroup>
  <Accordion title="Errores de conexión">
    Verifique que la URL base sea exactamente `https://api.lemondata.cc` (sin el sufijo `/v1` para el SDK de Anthropic).
  </Accordion>

  <Accordion title="Clave API no válida">
    Asegúrese de que su clave API de LemonData comience con `sk-` y tenga saldo suficiente.
  </Accordion>

  <Accordion title="Modelo no encontrado">
    Compruebe que el nombre del modelo sea correcto. Use el formato `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
