## Visión general

LemonData funciona con cualquier aplicación de chat compatible con OpenAI. Esta guía cubre la configuración para aplicaciones de chat populares de escritorio y móviles.

## Aplicaciones compatibles

| Aplicación | Plataforma | Características |
|-------------|----------|----------|
| LobeChat | Web, Escritorio, Auto-alojado | Multi-proveedor, plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Plugins enriquecidos, múltiples proveedores |
| Chatbox | Windows, macOS, Linux | Ligero, rápido |
| BotGem | iOS, macOS | Experiencia nativa de Apple |
| TypingMind | Web, Escritorio | Personas personalizadas, plugins |
| OpenCat | iOS, macOS | Nativo, soporte offline |
| ChatWise | Windows, macOS | Colaboración en equipo |

## Pasos de configuración

### Paso 1: Obtenga su API Key

1. Inicie sesión en el [Dashboard de LemonData](https://lemondata.cc/dashboard)
2. Navegue a [API Keys](https://lemondata.cc/dashboard/api)
3. Cree y copie su API key (formato: `sk-...`)

### Paso 2: Configure su aplicación

<Tabs>
  <Tab title="LobeChat">
    **Para LobeChat auto-alojado:**

    Establezca las variables de entorno:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Para LobeChat Cloud:**
    1. Abra **Settings** → **AI Service Provider**
    2. Seleccione **OpenAI**
    3. Ingrese su API key de LemonData
    4. Establezca el endpoint personalizado: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Abra **Settings** → **Model Provider**
    2. Haga clic en **Add Provider**
    3. Seleccione **OpenAI API Compatible**
    4. Ingrese:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Agregue modelos: `gpt-4o`, `claude-sonnet-4-5`, etc.
    6. Guarde y pruebe
  </Tab>
  <Tab title="Chatbox">
    1. Abra **Settings** → **AI Model**
    2. Seleccione **OpenAI API Compatible**
    3. Ingrese:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Haga clic en **Verify** para probar la conexión
  </Tab>
  <Tab title="BotGem">
    1. Abra **Settings** → **Providers**
    2. Toque en **Add Custom Provider**
    3. Ingrese:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Agregue sus modelos preferidos
    5. Guarde la configuración
  </Tab>
  <Tab title="TypingMind">
    1. Haga clic en **Settings** (icono de engranaje)
    2. Seleccione **Custom Endpoint**
    3. Ingrese:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Configure los modelos y guarde
  </Tab>
</Tabs>

### Paso 3: Agregue modelos

Después de configurar el proveedor, agregue los modelos que desea utilizar:

| Categoría | Modelos |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Referencia de configuración

| Campo | Valor |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| Tipo de API | OpenAI Compatible |

**Configuración JSON (para aplicaciones que lo soportan):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Solución de problemas

<AccordionGroup>
  <Accordion title="Error de conexión">
    - Verifique que la Base URL sea exactamente `https://api.lemondata.cc/v1`
    - Busque barras diagonales al final (elimínelas si están presentes)
    - Verifique la conectividad a internet y la configuración del firewall
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Verifique nuevamente su API key
    - Verifique que la clave esté activa en el [Dashboard de LemonData](https://lemondata.cc/dashboard/api)
    - Asegúrese de que no haya espacios adicionales en la clave
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - Verifique el saldo de su cuenta en el dashboard
    - Agregue créditos a su cuenta
  </Accordion>

  <Accordion title="404 Model Not Available">
    - Verifique que el nombre del modelo esté escrito correctamente
    - Verifique la disponibilidad del modelo en [lemondata.cc/en/models](https://lemondata.cc/es/models)
    - Pruebe con un modelo diferente
  </Accordion>

  <Accordion title="Respuestas lentas">
    - Pruebe un modelo más rápido (por ejemplo, `gpt-4o-mini` en lugar de `gpt-4o`)
    - Reduzca el parámetro `max_tokens` si está disponible
    - Verifique su conexión de red
  </Accordion>
</AccordionGroup>

## Consejos

<AccordionGroup>
  <Accordion title="Use los modelos adecuados">
    Adapte el modelo a su tarea: use GPT-4o-mini para chats rápidos, Claude o GPT-4o para tareas complejas.
  </Accordion>

  <Accordion title="Guarde los prompts utilizados con frecuencia">
    La mayoría de las aplicaciones permiten guardar prompts o personas. Cree plantillas para tareas comunes.
  </Accordion>

  <Accordion title="Monitoree el uso">
    Consulte su uso en el dashboard de LemonData para realizar un seguimiento de los costos y gestionar su presupuesto.
  </Accordion>
</AccordionGroup>
