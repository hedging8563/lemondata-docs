## Resumen

Dify es una plataforma de desarrollo de aplicaciones de LLM de código abierto. Proporciona orquestación visual de prompts, pipelines de RAG, frameworks de agentes y capacidades de LLMOps. LemonData puede configurarse como un proveedor de modelos personalizado en Dify.

## Beneficios

- Accede a más de 300 modelos de IA a través de una sola interfaz
- Cambia entre modelos sin modificar la lógica de la aplicación
- Optimiza costos seleccionando el mejor modelo para cada tarea
- Facturación unificada y seguimiento de uso

## Requisitos previos

- Cuenta de LemonData con acceso a la API
- Instalación de Dify (en la nube o auto-hospedado)

## Pasos de configuración

### Paso 1: Obtén tu API Key

1. Inicia sesión en el [LemonData Dashboard](https://lemondata.cc/dashboard)
2. Navega a [API Keys](https://lemondata.cc/dashboard/api)
3. Crea y copia tu API key (formato: `sk-...`)

### Paso 2: Agrega el proveedor de modelos personalizado

<Steps>
  <Step title="Abre la configuración">
    En Dify, ve a **Settings** → **Model Provider**
  </Step>
  <Step title="Agrega un proveedor compatible con OpenAI">
    Haz clic en **Add Model Provider** y selecciona **OpenAI-API-compatible**
  </Step>
  <Step title="Configura el proveedor">
    Ingresa la siguiente configuración:

    | Campo | Valor |
    |-------|-------|
    | Provider Name | LemonData |
    | API Key | `sk-your-lemondata-key` |
    | API Base URL | `https://api.lemondata.cc/v1` |
  </Step>
  <Step title="Agrega modelos">
    Agrega los modelos que desees utilizar:
    - `gpt-4o`
    - `gpt-4o-mini`
    - `claude-sonnet-4-5`
    - `claude-opus-4-5`
    - `gemini-2.5-flash`
    - `gemini-2.5-pro`
    - `deepseek-r1`
  </Step>
</Steps>

### Paso 3: Prueba la conexión

1. Selecciona un modelo (por ejemplo, `gpt-4o-mini`)
2. Envía un mensaje de prueba
3. Verifica que recibas una respuesta

## Uso en aplicaciones

### Chatbot

1. Crea una nueva aplicación de Chatbot
2. Selecciona LemonData como el proveedor de modelos
3. Elige tu modelo preferido
4. Configura el prompt de sistema y los parámetros

### Agente

1. Crea una aplicación de Agente
2. Selecciona un modelo capaz (GPT-4o, Claude)
3. Agrega herramientas y bases de conocimiento
4. Configura el comportamiento del agente

### Workflow

1. Crea un Workflow
2. Agrega nodos de LLM
3. Selecciona modelos de LemonData para cada nodo
4. Conecta los nodos y configura el flujo de datos

## Modelos disponibles

| Categoría | Modelos |
|----------|--------|
| Chat | GPT-4o, GPT-4o-mini, Claude Sonnet/Opus, Gemini, DeepSeek |
| Embeddings | text-embedding-3-small, text-embedding-3-large |
| Visión | GPT-4o (con imágenes), Claude Sonnet (con imágenes) |

## Configuración de RAG

Para aplicaciones de RAG, configura los embeddings:

1. Ve a **Settings** → **Model Provider**
2. Agrega el modelo de embedding: `text-embedding-3-small`
3. Establécelo como el modelo de embedding predeterminado en la configuración de la base de conocimiento

## Mejores prácticas

<AccordionGroup>
  <Accordion title="Comienza con modelos económicos">
    Usa GPT-4o-mini para pruebas y desarrollo, y cambia a modelos más potentes para producción.
  </Accordion>

  <Accordion title="Usa enrutamiento de modelos">
    Configura diferentes modelos para diferentes tareas: modelos rápidos para consultas simples y modelos potentes para razonamiento complejo.
  </Accordion>

  <Accordion title="Monitorea el uso">
    Utiliza las funciones de LLMOps de Dify junto con el dashboard de LemonData para realizar un seguimiento de los costos y el rendimiento.
  </Accordion>
</AccordionGroup>

## Solución de problemas

<AccordionGroup>
  <Accordion title="Error de conexión">
    - Verifica que la API Base URL sea exactamente `https://api.lemondata.cc/v1`
    - Comprueba si hay barras diagonales al final
    - Verifica la conectividad de red
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Verifica nuevamente la API key
    - Asegúrate de que la clave esté activa en el dashboard de LemonData
  </Accordion>

  <Accordion title="Modelo no encontrado">
    - Verifica que el nombre del modelo coincida exactamente
    - Consulta la disponibilidad de los modelos en [lemondata.cc/en/models](https://lemondata.cc/es/models)
  </Accordion>
</AccordionGroup>
