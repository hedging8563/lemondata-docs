---
title: "Create Speech"
openapi: "POST /v1/audio/speech"
description: "Generates audio from the input text"
---

## Request Body

<ParamField body="model" type="string" default="tts-1">
  TTS model: `tts-1` (standard) or `tts-1-hd` (higher quality).
</ParamField>

<ParamField body="input" type="string" required>
  The text to generate audio for. Maximum 4096 characters.
</ParamField>

<ParamField body="voice" type="string" required>
  Voice to use: `alloy`, `ash`, `ballad`, `coral`, `echo`, `fable`, `nova`, `onyx`, `sage`, `shimmer`, `verse`.
</ParamField>

<ParamField body="response_format" type="string" default="mp3">
  Audio format: `mp3`, `opus`, `aac`, `flac`, `wav`, `pcm`.
</ParamField>

<ParamField body="speed" type="number" default="1.0">
  Speed of audio (0.25 to 4.0).
</ParamField>

## Response

Returns the audio file in the requested format.

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1-hd",
    "voice": "nova",
    "input": "Hello, welcome to LemonData!"
  }' \
  --output speech.mp3
```


```go Go
package main

import (
    "context"
    "io"
    "os"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateSpeech(
        context.Background(),
        openai.CreateSpeechRequest{
            Model: openai.TTSModel1HD,
            Voice: openai.VoiceNova,
            Input: "Hello, welcome to LemonData!",
        },
    )
    if err != nil {
        panic(err)
    }
    defer resp.Close()

    file, _ := os.Create("speech.mp3")
    defer file.Close()
    io.Copy(file, resp)
}
```
