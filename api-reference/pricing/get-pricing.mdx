---
title: "Get Pricing"
api: "GET /v1/pricing"
description: "Get pricing information for models"
---

## Query Parameters

<ParamField query="model" type="string">
  Model ID to get pricing for. If not specified, returns pricing for all models.
</ParamField>

## Response

<ResponseField name="object" type="string">
  Always `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Array of model pricing objects.

  Each object contains:
  - `model` (string): Model identifier
  - `pricing` (object): Pricing information
    - `input_per_1m` (string | null): USD per 1M input tokens
    - `output_per_1m` (string | null): USD per 1M output tokens
    - `per_request` (string | null): USD per request (for fixed-price models)
    - `is_lock_price` (boolean): Whether pricing is fixed per request
    - `currency` (string): Always `USD`
  - `provider` (string): Model provider
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1/pricing?model=gpt-4o" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
response = requests.get(
    "https://api.lemondata.cc/v1/pricing",
    headers={"Authorization": "Bearer sk-your-api-key"},
    params={"model": "gpt-4o"}
)

print(response.json())
```

## Pricing Types

| Type | Description | Example Models |
|------|-------------|----------------|
| **Token-based** | Charged per input/output tokens | GPT-4o, Claude, Gemini |
| **Per-request** | Fixed price per request | DALL-E 3, Sora, Suno |
