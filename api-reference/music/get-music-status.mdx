---
title: "Get Music Status"
api: "GET /v1/music/generations/{id}"
description: "Retrieves the status and result of a music generation task"
---

Poll this endpoint to check the status of your music generation task.

## Path Parameters

<ParamField path="id" type="string" required>
  The music generation task ID.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Task ID.
</ResponseField>

<ResponseField name="status" type="string">
  Task status: `pending`, `processing`, `completed`, or `failed`.
</ResponseField>

<ResponseField name="progress" type="integer">
  Progress percentage (0-100).
</ResponseField>

<ResponseField name="audio" type="object">
  Audio result (when completed).

  - `url` (string): URL to download the audio file
  - `duration` (number): Duration in seconds
  - `format` (string): Audio format (mp3, wav, etc.)
</ResponseField>

<ResponseField name="clips" type="array">
  Generated music clips (Suno returns multiple variations).
</ResponseField>

<ResponseField name="error" type="object">
  Error details (when failed).
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1/music/generations/music_abc123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "music_abc123"

while True:
    response = requests.get(
        f"https://api.lemondata.cc/v1/music/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"}
    )
    result = response.json()

    if result["status"] == "completed":
        print(f"Audio URL: {result['audio']['url']}")
        break
    elif result["status"] == "failed":
        print(f"Error: {result['error']['message']}")
        break

    time.sleep(5)  # Poll every 5 seconds
```
</RequestExample>
