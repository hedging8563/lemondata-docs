---
title: "WhatsApp"
description: "Verbinden Sie Ihre Claw-Instanz über QR-Code-Pairing mit WhatsApp"
---

Verbinden Sie WhatsApp mit Ihrer Claw-Instanz, indem Sie einen QR-Code scannen – genau wie bei WhatsApp Web.

## Voraussetzungen

- Ein Telefon mit installiertem [WhatsApp](https://www.whatsapp.com/) und einem aktiven Konto
- Eine Claw-Instanz ([hier erstellen](https://lemondata.cc/dashboard/claw))

<Warning>
WhatsApp verwendet Ihre **echte Telefonnummer**. Über den Bot gesendete Nachrichten erscheinen so, als kämen sie von Ihrem WhatsApp-Konto. Verwenden Sie eine dedizierte Nummer, wenn Sie eine Trennung wünschen.
</Warning>

<Warning>
**VoIP-Nummern werden nicht unterstützt.** WhatsApp erfordert eine echte SIM-basierte Telefonnummer. Virtuelle Nummern (Google Voice, Skype usw.) funktionieren nicht für die QR-Code-Kopplung.
</Warning>

## Einrichtungsanleitung

<Steps>
  <Step title="WhatsApp-Konfiguration speichern">
    1. Gehen Sie zu Ihrem [Claw Dashboard](https://lemondata.cc/dashboard/claw)
    2. Klicken Sie auf das **Einstellungen-Symbol** Ihrer Instanz
    3. Öffnen Sie den Tab **IM Integrations**
    4. Erweitern Sie den Bereich **WhatsApp**

    Klicken Sie zuerst auf **Speichern** — Sie müssen die Konfiguration speichern, bevor Sie den QR-Code scannen
  </Step>
  <Step title="QR-Code scannen">
    1. Klicken Sie auf die Schaltfläche **Scan QR Code**
    2. Ein QR-Code wird angezeigt (gültig für ca. 60 Sekunden)
    3. Öffnen Sie auf Ihrem Telefon WhatsApp → **Einstellungen** → **Verknüpfte Geräte** → **Gerät hinzufügen**
    4. Scannen Sie den auf dem Bildschirm angezeigten QR-Code
    5. Warten Sie, bis die Verbindung hergestellt ist
  </Step>
  <Step title="Zugriffsrichtlinie konfigurieren">
    WhatsApp verwendet die **allowlist**-Richtlinie – nur von Ihnen angegebene Telefonnummern können mit dem KI-Bot interagieren.

    Fügen Sie erlaubte Telefonnummern im LemonData-Dashboard hinzu (internationales Format, z. B. `+1234567890`).

    <Tip>
    Im Gegensatz zu anderen Plattformen unterstützt WhatsApp den Pairing-Mechanismus nicht. Sie müssen erlaubte Kontakte explizit hinzufügen.
    </Tip>
  </Step>
  <Step title="Chatten starten">
    Senden Sie eine Nachricht von einem erlaubten Kontakt an die verbundene WhatsApp-Nummer – die KI wird antworten!
  </Step>
</Steps>

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="QR-Code abgelaufen">
    Klicken Sie erneut auf **Scan QR Code**, um einen neuen Code zu generieren. QR-Codes laufen nach ca. 60 Sekunden ab.
  </Accordion>
  <Accordion title="WhatsApp-Verbindung getrennt">
    WhatsApp kann verknüpfte Geräte nach Inaktivität trennen. Scannen Sie den QR-Code erneut, um die Verbindung wiederherzustellen.
  </Accordion>
  <Accordion title="Nachrichten erhalten keine KI-Antworten">
    - Überprüfen Sie, ob die Nummer des Absenders in Ihrer allowlist steht (falls die allowlist-Richtlinie verwendet wird)
    - Überprüfen Sie, ob Ihre Instanz läuft
    - Überprüfen Sie Ihr LemonData-Guthaben
  </Accordion>
</AccordionGroup>
