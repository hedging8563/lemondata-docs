## Schnellstart

<Steps>
  <Step title="Instanz erstellen">
    Gehen Sie zu Ihrem [LemonData Dashboard](https://lemondata.cc/dashboard/claw) und klicken Sie auf **Instanz erstellen**. Wählen Sie eine Subdomain – Ihre Instanz wird unter `claw-{subdomain}.lemondata.cc` erreichbar sein.
  </Step>
  <Step title="KI-Modelle konfigurieren">
    Ihre Instanz ist mit beliebten Modellen vorkonfiguriert. Sie können in den Instanzeinstellungen anpassen, welche Modelle verfügbar sind.
  </Step>
  <Step title="Chat-Plattform verbinden">
    Wählen Sie eine oder mehrere Messaging-Plattformen aus und folgen Sie der Einrichtungsanleitung. Die Konfiguration der meisten Plattformen dauert weniger als 5 Minuten.
  </Step>
  <Step title="Chatten starten">
    Senden Sie eine Nachricht an Ihren Bot – er wird sich automatisch mit Ihnen koppeln und mit KI antworten!
  </Step>
</Steps>

## So funktioniert das Auto-Pairing

Die meisten Plattformen (Telegram, Discord, Slack, Feishu, DingTalk, QQ) verwenden einen **Auto-Pairing**-Mechanismus:

1. Senden Sie Ihrem Bot eine Nachricht auf einer beliebigen verbundenen Plattform
2. Der **erste Benutzer** pro Kanal wird automatisch als **Besitzer** gekoppelt – kein Code erforderlich!
3. Weitere Benutzer erhalten einen **Pairing-Code**, der in der Claw WebUI unter `claw-{subdomain}.lemondata.cc` eingegeben werden muss

<Note>
**WhatsApp** verwendet QR-Code-Scanning anstelle von Pairing. **WeCom** nutzt Webhook-Callbacks und erfordert kein Pairing.
</Note>

<Warning>
Standardmäßig können nur gekoppelte Benutzer mit Ihrem Bot interagieren. Sie können dies in den DM Policy-Einstellungen ändern, aber seien Sie vorsichtig mit der „Open“-Policy, da sie es jedem ermöglicht, Ihren Bot (und Ihr KI-Guthaben) zu nutzen.
</Warning>
