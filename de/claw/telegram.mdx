Telegram ist die **einfachste** Plattform für eine Verbindung. Sie benötigen lediglich einen Bot-Token von BotFather und können in weniger als 2 Minuten mit der KI chatten.

## Voraussetzungen

- Ein [Telegram](https://telegram.org/)-Konto
- Eine Claw-Instanz ([hier erstellen](https://lemondata.cc/dashboard/claw))

## Einrichtungsanleitung

<Steps>
  <Step title="Bot mit BotFather erstellen">
    1. Öffnen Sie Telegram und suchen Sie nach **@BotFather**
    2. Senden Sie `/newbot`
    3. Wählen Sie einen **Anzeigenamen** für Ihren Bot (z. B. „Mein KI-Assistent“)
    4. Wählen Sie einen **Benutzernamen**, der auf `bot` endet (z. B. `my_ai_assistant_bot`)
    5. BotFather antwortet mit Ihrem **Bot-Token** — dieser sieht so aus:
    ```
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
    ```
    <Warning>
    Halten Sie Ihren Bot-Token geheim! Jeder mit diesem Token kann Ihren Bot steuern.
    </Warning>
  </Step>
  <Step title="Im LemonData Dashboard konfigurieren">
    1. Gehen Sie zu Ihrem [Claw Dashboard](https://lemondata.cc/dashboard/claw)
    2. Klicken Sie auf das **Einstellungen-Symbol** Ihrer Instanz
    3. Öffnen Sie den Tab **IM Integrations**
    4. Erweitern Sie den Bereich **Telegram**
    5. Fügen Sie Ihren **Bot-Token** ein
    6. Klicken Sie auf **Save**
  </Step>
  <Step title="Chatten starten">
    1. Öffnen Sie Telegram und suchen Sie Ihren Bot über seinen Benutzernamen
    2. Senden Sie eine beliebige Nachricht (z. B. „Hallo!“)
    3. Der Bot antwortet mit einem **Pairing-Code**
    4. Geben Sie den Code in Ihrer Claw WebUI ein, um das Pairing abzuschließen
    5. Alles bereit — fangen Sie an, mit der KI zu chatten!
  </Step>
</Steps>

## Optional: Bot-Profil anpassen

Nachdem Sie Ihren Bot erstellt haben, können Sie ihn mit BotFather anpassen:

| Befehl | Zweck |
|---------|---------|
| `/setdescription` | Bio/Beschreibung des Bots festlegen |
| `/setabouttext` | Den „Über“-Bereich festlegen |
| `/setuserpic` | Ein Profilbild hochladen |
| `/setcommands` | Befehlsvorschläge hinzufügen |

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Bot antwortet nicht">
    - Überprüfen Sie, ob der Bot-Token korrekt ist (keine zusätzlichen Leerzeichen)
    - Prüfen Sie, ob Ihre Instanz läuft (grüner Status im Dashboard)
    - Versuchen Sie, die Instanz über das Dashboard neu zu starten
  </Accordion>
  <Accordion title="„Pairing-Code“ erscheint nicht">
    - Stellen Sie sicher, dass Sie eine **Direktnachricht** an den Bot senden und nicht in einer Gruppe
    - Das Pairing-System wird bei der ersten Nachricht eines neuen Benutzers aktiviert
  </Accordion>
  <Accordion title="Bot antwortet langsam">
    - Die Antwortzeit hängt vom ausgewählten KI-Modell ab
    - Größere Modelle (GPT-4, Claude) können einige Sekunden dauern
    - Überprüfen Sie Ihr LemonData-Guthaben — unzureichendes Guthaben führt zu Fehlern
  </Accordion>
</AccordionGroup>
