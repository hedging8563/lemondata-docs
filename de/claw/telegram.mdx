Telegram ist die **einfachste** Plattform für eine Verbindung. Sie benötigen lediglich einen Bot-Token von BotFather und können in weniger als 2 Minuten mit der KI chatten.

## Voraussetzungen

- Ein [Telegram](https://telegram.org/)-Konto
- Eine Claw-Instanz ([hier erstellen](https://lemondata.cc/dashboard/claw))

## Einrichtungsanleitung

<Steps>
  <Step title="Bot mit BotFather erstellen">
    1. Öffnen Sie Telegram und suchen Sie nach **@BotFather**
    2. Senden Sie `/newbot`
    3. Wählen Sie einen **Anzeigenamen** für Ihren Bot (z. B. „Mein KI-Assistent“)
    4. Wählen Sie einen **Benutzernamen**, der auf `bot` endet (z. B. `my_ai_assistant_bot`)
    5. BotFather antwortet mit Ihrem **Bot Token** — dieser sieht so aus:
    ```
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
    ```
    <Warning>
    Halten Sie Ihren Bot-Token geheim! Jeder, der diesen Token besitzt, kann Ihren Bot steuern.
    </Warning>
  </Step>
  <Step title="Im LemonData Dashboard konfigurieren">
    1. Gehen Sie zu Ihrem [Claw Dashboard](https://lemondata.cc/dashboard/claw)
    2. Klicken Sie auf das **Einstellungen-Symbol** Ihrer Instanz
    3. Öffnen Sie den Tab **IM Integrations**
    4. Erweitern Sie den Bereich **Telegram**
    5. Fügen Sie Ihren **Bot Token** ein
    6. Klicken Sie auf **Save**
  </Step>
  <Step title="Chatten starten">
    1. Öffnen Sie Telegram und suchen Sie Ihren Bot über seinen Benutzernamen
    2. Senden Sie eine beliebige Nachricht (z. B. „Hallo!“)
    3. Der erste Benutzer wird **automatisch als Besitzer gekoppelt** — kein Code erforderlich!
    4. Beginnen Sie mit der KI zu chatten!
  </Step>
</Steps>

## Optional: Bot-Profil anpassen

Nachdem Sie Ihren Bot erstellt haben, können Sie ihn mit BotFather anpassen:

| Befehl | Zweck |
|---------|---------|
| `/setdescription` | Bio/Beschreibung des Bots festlegen |
| `/setabouttext` | „Über uns“-Bereich festlegen |
| `/setuserpic` | Profilbild hochladen |
| `/setcommands` | Befehlsvorschläge hinzufügen |

## Nutzung in Gruppen

<Warning>
**Privacy Mode ist standardmäßig aktiviert** für Telegram-Bots. In Gruppen-Chats sieht der Bot nur Nachrichten, die:
- Mit `/` beginnen (Befehle)
- Den Bot direkt per @mention erwähnen
- Antworten auf die Nachrichten des Bots sind

Damit der Bot **alle** Gruppennachrichten sehen kann, deaktivieren Sie den Privacy Mode über BotFather: Senden Sie `/setprivacy` → wählen Sie Ihren Bot aus → wählen Sie **Disable**. Hinweis: Sie müssen den Bot aus bestehenden Gruppen **entfernen und erneut hinzufügen**, damit diese Änderung wirksam wird.
</Warning>

### Gruppeneinstellungen

Konfigurieren Sie das Gruppenverhalten im Tab **IM Integrations** Ihres Dashboards:

| Einstellung | Optionen | Beschreibung |
|---------|---------|-------------|
| **Group Policy** | Open (Standard) / Allowlist / Disabled | Steuert, wer den Bot in Gruppen auslösen kann |
| **Require @mention** | On / Off | Bot antwortet nur, wenn er per @mention erwähnt wird (empfohlen) |
| **Group IDs** | Liste der Chat-IDs | Auf bestimmte Gruppen beschränken (leer = alle Gruppen) |
| **Group Allowed Users** | Liste der Benutzer-IDs | Nur diese Benutzer können den Bot auslösen (Allowlist-Modus) |

<Tip>
**Group Policy** steuert, *wer* dem Bot in Gruppen Nachrichten senden kann, während **Group IDs** steuert, in *welchen* Gruppen der Bot antwortet. Verwenden Sie beides zusammen für eine präzise Steuerung.
</Tip>

## DM-Richtlinie

Steuern Sie, wer Ihrem Bot Direktnachrichten senden kann:

| Richtlinie | Verhalten |
|--------|----------|
| **Pairing** (Standard) | Neue Benutzer erhalten einen Kopplungscode — geben Sie diesen in der WebUI ein, um sie zu autorisieren |
| **Allowlist** | Nur von Ihnen angegebene Benutzer-IDs können dem Bot Nachrichten senden |
| **Open** | Jeder kann dem Bot Nachrichten senden (mit Vorsicht verwenden — verbraucht Ihre KI-Credits) |

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Bot antwortet nicht">
    - Überprüfen Sie, ob der Bot-Token korrekt ist (keine zusätzlichen Leerzeichen)
    - Prüfen Sie, ob Ihre Instanz läuft (grüner Status im Dashboard)
    - Versuchen Sie, die Instanz über das Dashboard neu zu starten
  </Accordion>
  <Accordion title="„Pairing code“ erscheint nicht">
    - Stellen Sie sicher, dass Sie eine **Direktnachricht** an den Bot senden und nicht in einer Gruppe schreiben
    - Das Kopplungssystem wird bei der ersten Nachricht eines neuen Benutzers aktiviert
  </Accordion>
  <Accordion title="Bot antwortet langsam">
    - Die Antwortzeit hängt vom ausgewählten KI-Modell ab
    - Größere Modelle (Claude Opus, GPT-5) können einige Sekunden dauern
    - Überprüfen Sie Ihr LemonData-Guthaben — unzureichende Credits führen zu Fehlern
  </Accordion>
</AccordionGroup>
