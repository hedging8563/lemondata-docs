<Note>
**Selbst gehostet** bedeutet, dass Sie OpenClaw direkt auf Ihrem eigenen VPS oder Rechner ausführen. Sie zahlen nur für die Nutzung von AI-Modellen über Ihr LemonData-Guthaben — keine Instanzgebühren.
</Note>

## Schnellinstallation

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Nicht-interaktiv
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

Das Installationsprogramm fragt Sie nach Ihrem **LemonData API Key**. Holen Sie sich einen über Ihr [Dashboard](https://lemondata.cc/dashboard).

## Was wird installiert

<Steps>
  <Step title="Node.js 22+">
    Wird automatisch installiert, falls nicht vorhanden (über NodeSource unter Linux, Homebrew unter macOS)
  </Step>
  <Step title="OpenClaw">
    Offizielles OpenClaw-Paket von npm, vorkonfiguriert mit LemonData API
  </Step>
  <Step title="15 AI-Modelle">
    Claude, GPT-5, Gemini, DeepSeek und mehr — sofort einsatzbereit
  </Step>
  <Step title="IM-Plugins">
    DingTalk-, WeCom- und QQ-Plugins vorinstalliert
  </Step>
  <Step title="Systemdienst">
    Startet automatisch beim Hochfahren (systemd unter Linux, launchd unter macOS)
  </Step>
</Steps>

## Voraussetzungen

| Voraussetzung | Details |

## Verwaltungsbefehle

<Tabs>
  <Tab title="Linux">
    ```bash
    systemctl --user status openclaw    # Status prüfen
    journalctl --user -u openclaw -f    # Logs anzeigen
    systemctl --user restart openclaw   # Neustarten
    systemctl --user stop openclaw      # Stoppen
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    tail -f ~/.openclaw/openclaw.log    # Logs anzeigen
    launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # Neustarten
    launchctl stop cc.lemondata.openclaw  # Stoppen
    ```
  </Tab>
</Tabs>

## Konfiguration

Speicherort der Konfigurationsdatei: `~/.openclaw/openclaw.json`

Nach der Bearbeitung starten Sie den Dienst neu, um die Änderungen zu übernehmen.

### Standardmodell ändern

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Verfügbare Modellpräfixe:
- `lemondata-claude/` — Claude-Modelle (anthropic-messages API)
- `lemondata-minimax/` — MiniMax-Modelle (anthropic-messages API)
- `lemondata/` — Alle anderen Modelle (openai-completions API)

## Aktualisierung

```bash
npm update -g openclaw
# Dann den Dienst neustarten
systemctl --user restart openclaw  # Linux
# oder
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Deinstallation

```bash
# Dienst stoppen und deaktivieren
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# OpenClaw entfernen
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
