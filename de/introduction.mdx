## Was ist LemonData?

LemonData ist eine KI-API-Aggregationsplattform, die einen einheitlichen Zugriff auf **über 300 KI-Modelle** von führenden Anbietern wie OpenAI, Anthropic, Google, DeepSeek und weiteren bietet.

## Unterstützte Funktionen

| Funktion | Beschreibung | Beispielmodelle |
|------------|-------------|----------------|
| **Chat** | Texterstellung und Konversation | GPT-4o, Claude 3.5, Gemini 2.0 |
| **Vision** | Bildverständnis und -analyse | GPT-4o, Claude 3.5 Sonnet, Gemini |
| **Bilderzeugung** | Bilder aus Text erstellen | DALL-E 3, Midjourney, Flux, Ideogram |
| **Videoerzeugung** | Videos aus Text/Bildern erstellen | Sora, Runway Gen-3, Kling |
| **Audio** | Text-to-Speech und Transkription | Whisper, TTS-1, MiniMax |
| **Embeddings** | Textvektorisierung | text-embedding-3-small |
| **Musik** | KI-Musikerzeugung | Suno |
| **3D-Erzeugung** | 3D-Modelle erstellen | Tripo3D |

## Schnelles Beispiel

<CodeGroup>

```bash cURL
curl https://api.lemondata.cc/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"

    client := openai.NewClientWithConfig(config)

    resp, _ := client.CreateChatCompletion(
        context.Background(),
        openai.ChatCompletionRequest{
            Model: "gpt-4o",
            Messages: []openai.ChatCompletionMessage{
                {Role: openai.ChatMessageRoleUser, Content: "Hello!"},
            },
        },
    )
    fmt.Println(resp.Choices[0].Message.Content)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/chat/completions');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'gpt-4o',
        'messages' => [
            ['role' => 'user', 'content' => 'Hello!']
        ]
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['choices'][0]['message']['content'];
```
