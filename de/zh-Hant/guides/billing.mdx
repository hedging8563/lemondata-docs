## Übersicht

LemonData nutzt **Pay-as-you-go**. Sie zahlen nur für das, was Sie nutzen, ohne Abonnements oder Mindestumsatzverpflichtungen.

## So funktioniert die Abrechnung

1. **Guthaben aufladen** auf Ihr Konto
2. **API nutzen** - Gebühren werden pro Anfrage abgezogen
3. **Nutzung überwachen** in Ihrem Dashboard
4. **Guthaben bei Bedarf aufladen**

## Token-Zählung

Token sind die Basiseinheiten der Textverarbeitung:

- ~4 Zeichen = 1 Token (Englisch)
- ~1-2 Zeichen = 1 Token (Chinesisch)
- 1 Bild = Abhängig von Größe und Details

### Token-Schätzung

```python
# Grobe Schätzung
def estimate_tokens(text):
    return len(text) / 4  # Ungefähr für Englisch
```
