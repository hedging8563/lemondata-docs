## Übersicht

Dieser Endpunkt bietet native Kompatibilität mit der Anthropic Messages API. Er ist für Claude-Modelle mit Funktionen wie Extended Thinking geeignet.

<Note>
Base URL für das Anthropic SDK: `https://api.lemondata.cc` (ohne `/v1` Suffix)
</Note>

## Request-Header

<ParamField header="x-api-key" type="string" required>
  Ihr LemonData API-Key. Eine Alternative zum Bearer token.
</ParamField>

<ParamField header="anthropic-version" type="string" required>
  Anthropic API-Version. Bitte verwenden Sie `2023-06-01`.
</ParamField>

## Request-Body

<ParamField body="model" type="string" required>
  Claude Modell-ID (z. B. `claude-sonnet-4-5`).
</ParamField>

<ParamField body="messages" type="array" required>
  Ein Array von Nachrichtobjekten, die `role` und `content` enthalten.
</ParamField>

<ParamField body="max_tokens" type="integer" required>
  Die maximale Anzahl der zu generierenden token.
</ParamField>

<ParamField body="system" type="string">
  System-Prompt (getrennt vom `messages` Array).
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Sampling-Temperatur (0-1).
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Stream-Antworten aktivieren.
</ParamField>

<ParamField body="thinking" type="object">
  Konfiguration für Extended Thinking (Claude Opus 4.5).

  - `type` (string): Auf `"enabled"` setzen, um es zu aktivieren
  - `budget_tokens` (integer): token-Budget für das Thinking
</ParamField>

<ParamField body="tools" type="array">
  Verfügbare Tools für das Modell.
</ParamField>

<ParamField body="tool_choice" type="object">
  Wie das Modell Tools verwenden soll. Optionen: `auto`, `any`, `tool` (spezifisches Tool).
</ParamField>

<ParamField body="top_p" type="number">
  Parameter für Nucleus-Sampling. Verwenden Sie entweder `temperature` oder `top_p`, nicht beides gleichzeitig.
</ParamField>

<ParamField body="top_k" type="integer">
  Stichprobenziehung nur aus den Top-K-Optionen für jedes token.
</ParamField>

<ParamField body="stop_sequences" type="array">
  Benutzerdefinierte Stopp-Sequenzen, die das Modell veranlassen, die Generierung zu beenden.
</ParamField>

<ParamField body="metadata" type="object">
  Metadaten, die der Anfrage beigefügt werden.
</ParamField>
