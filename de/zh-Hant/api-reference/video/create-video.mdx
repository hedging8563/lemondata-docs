## Übersicht

Die Video-Generierung erfolgt asynchron. Sie senden eine Anfrage und erhalten eine task ID, um anschließend das Ergebnis abzufragen (Polling).

## Request Body

<ParamField body="model" type="string" default="kling-video">
  Video-Modell (z. B.: `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Textbeschreibung des zu generierenden Videos.
</ParamField>

<ParamField body="image_url" type="string">
  Optionale Startgrafik für die Image-to-Video-Generierung.
</ParamField>

<ParamField body="duration" type="integer">
  Videodauer in Sekunden (abhängig vom Modell).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Seitenverhältnis (z. B.: `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Videoauflösung (z. B.: `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Bilder pro Sekunde (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Inhalte, die bei der Video-Generierung vermieden werden sollen.
</ParamField>

<ParamField body="seed" type="integer">
  Zufallswert (Seed) für reproduzierbare Ergebnisse.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Classifier-Free Guidance Scale (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Bewegungsintensität (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL des Startbildes.
</ParamField>

<ParamField body="end_image" type="string">
  URL des Endbildes.
</ParamField>

<ParamField body="user" type="string">
  Eindeutige Kennung des Endbenutzers.
</ParamField>

## Antwort

<ResponseField name="task_id" type="string">
  Eindeutige Task-ID für das Polling.
</ResponseField>

<ResponseField name="status" type="string">
  Initialer Status: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Verwendetes Modell.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

## Image-to-Video

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```
