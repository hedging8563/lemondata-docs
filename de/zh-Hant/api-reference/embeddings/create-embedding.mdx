---
title: "Embeddings erstellen"
openapi: "POST /v1/embeddings"
description: "Erstellt einen Embedding-Vektor, der den Eingabetext repräsentiert"
---

## Anfrage-Body

<ParamField body="model" type="string" required>
  Die ID des zu verwendenden Embedding-Modells (z. B. `text-embedding-3-small`).
</ParamField>

<ParamField body="input" type="string | array" required>
  Der Eingabetext, der eingebettet werden soll. Kann ein String oder ein Array von Strings sein.
</ParamField>

<ParamField body="encoding_format" type="string" default="float">
  Das Format des Embeddings: `float` oder `base64`.
</ParamField>

<ParamField body="dimensions" type="integer">
  Die Anzahl der Ausgabedimensionen (abhängig vom jeweiligen Modell).
</ParamField>

<ParamField body="user" type="string">
  Eine eindeutige Kennung, die Ihren Endbenutzer repräsentiert und zur Missbrauchsüberwachung verwendet wird.
</ParamField>

## Verfügbare Modelle

| Modell | Dimensionen | Beschreibung |
|-------|------------|-------------|
| `text-embedding-3-large` | 3072 | Beste Qualität |
| `text-embedding-3-small` | 1536 | Ausgewogen |
| `text-embedding-ada-002` | 1536 | Legacy |

## Antwort

<ResponseField name="object" type="string">
  Festgelegt auf `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Ein Array von Embedding-Objekten.

  Jedes Objekt enthält:
  - `object` (string): `embedding`
  - `index` (integer): Index im Eingabe-Array
  - `embedding` (array): Embedding-Vektor
</ResponseField>

<ResponseField name="model" type="string">
  Das verwendete Modell.
</ResponseField>

<ResponseField name="usage" type="object">
  Token-Nutzung, einschließlich `prompt_tokens` und `total_tokens`.
</ResponseField>
