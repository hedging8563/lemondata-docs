---

Die Responses API ist die neuere zustandsorientierte Chat-API von OpenAI. LemonData bietet Unterstützung für dieses Format für kompatible Modelle.

## Anfrage-Body

<ParamField body="model" type="string" required>
  Die ID des zu verwendenden Modells. Siehe [Models](https://lemondata.cc/zh-TW/models) für verfügbare Optionen.
</ParamField>

<ParamField body="input" type="array" required>
  Eine Liste von Eingabeelementen, die die Konversation bilden.

  Jedes Element kann sein:
  - `message`: Eine Chat-Nachricht, die Rolle und Inhalt enthält
  - `function_call`: Eine Funktionsaufrufanfrage
  - `function_call_output`: Die Ausgabe eines Funktionsaufrufs
</ParamField>

<ParamField body="instructions" type="string">
  Systemanweisungen für das Modell (entspricht einer Systemnachricht).
</ParamField>

<ParamField body="max_output_tokens" type="integer">
  Die maximale Anzahl der zu generierenden Token.
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Die Sampling-Temperatur zwischen 0 und 2.
</ParamField>

<ParamField body="tools" type="array">
  Eine Liste von Tools, die das Modell aufrufen kann.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Wenn true, wird ein Event-Stream zurückgegeben.
</ParamField>

<ParamField body="previous_response_id" type="string">
  Die ID der vorherigen Antwort, um die Konversation fortzusetzen.
</ParamField>

<ParamField body="store" type="boolean" default="true">
  Gibt an, ob die Antwort für einen späteren Abruf gespeichert werden soll.
</ParamField>

<ParamField body="metadata" type="object">
  Metadaten, die der Antwort zu Tracking-Zwecken beigefügt werden.
</ParamField>

<ParamField body="text" type="object">
  Konfigurationsoptionen für die Texterstellung.
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  Gibt an, ob mehrere Tool-Aufrufe gleichzeitig zulässig sind.
</ParamField>

<ParamField body="top_p" type="number">
  Parameter für Nucleus-Sampling (0-1).
</ParamField>

<ParamField body="reasoning" type="object">
  Konfiguration für das Reasoning bei o1/o3 Modellen.

  - `effort` (string): Stufe der Reasoning-Intensität (`low`, `medium`, `high`)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Die eindeutige Kennung der Antwort.
</ResponseField>

<ResponseField name="object" type="string">
  Festgelegt auf `response`.
</ResponseField>

<ResponseField name="created_at" type="integer">
  Der Unix-Zeitstempel der Erstellung der Antwort.
</ResponseField>

<ResponseField name="output" type="array">
  Eine Liste der vom Modell generierten Ausgabeelemente.
</ResponseField>

<ResponseField name="usage" type="object">
  Statistiken zur Token-Nutzung.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/responses" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "input": [
      {"type": "message", "role": "user", "content": "Hello!"}
    ],
    "max_output_tokens": 1000
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.responses.create(
    model="gpt-4o",
    input=[
        {"type": "message", "role": "user", "content": "Hello!"}
    ],
    max_output_tokens=1000
)

print(response.output)
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model": "gpt-4o",
        "input": []map[string]interface{}{
            {"type": "message", "role": "user", "content": "Hello!"},
        },
        "max_output_tokens": 1000,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/responses", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result["output"])
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/responses');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'gpt-4o',
        'input' => [
            ['type' => 'message', 'role' => 'user', 'content' => 'Hello!']
        ],
        'max_output_tokens' => 1000
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
print_r($data['output']);
```
