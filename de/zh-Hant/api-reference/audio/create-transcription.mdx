---
title: "Transkription erstellen"
openapi: "POST /v1/audio/transcriptions"
description: "Transkribiert Audio in die Eingabesprache"
---

## Request Body

<ParamField body="file" type="file" required>
  Die zu transkribierende Audiodatei. Unterstützte Formate: flac, mp3, mp4, mpeg, mpga, m4a, ogg, wav, webm.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  Das zu verwendende Modell. Derzeit wird nur `whisper-1` unterstützt.
</ParamField>

<ParamField body="language" type="string">
  Die Sprache des Audios im ISO-639-1-Format (z. B.: `en`, `zh`, `ja`).
</ParamField>

<ParamField body="prompt" type="string">
  Optionaler Text, um den Stil des Modells zu steuern oder ein vorheriges Segment fortzusetzen.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  Das Format der Ausgabe: `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number" default="0">
  Die Sampling-Temperatur (0 bis 1).
</ParamField>

<ParamField body="timestamp_granularities" type="array">
  Die Granularität der Zeitstempel: `word` und/oder `segment`. Erfordert `verbose_json`.
</ParamField>

## Antwort

<ResponseField name="text" type="string">
  Der transkribierte Text.
</ResponseField>

Für `verbose_json`:

<ResponseField name="task" type="string">
  Festgelegt auf `transcribe`.
</ResponseField>

<ResponseField name="language" type="string">
  Die erkannte Sprache.
</ResponseField>

<ResponseField name="duration" type="number">
  Die Dauer des Audios (Sekunden).
</ResponseField>

<ResponseField name="segments" type="array">
  Transkribierte Segmente inklusive Zeitstempel.
</ResponseField>

<ResponseField name="words" type="array">
  Zeitstempel auf Wortebene (falls angefordert).
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/transcriptions" \
  -H "Authorization: Bearer sk-your-api-key" \
  -F file="@audio.mp3" \
  -F model="whisper-1" \
  -F language="en"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

with open("audio.mp3", "rb") as audio_file:
    response = client.audio.transcriptions.create(
        model="whisper-1",
        file=audio_file,
        language="en"
    )

print(response.text)
```


```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranscription(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "audio.mp3",
            Language: "en",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/transcriptions');

$file = new CURLFile('audio.mp3', 'audio/mpeg', 'audio.mp3');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => [
        'file' => $file,
        'model' => 'whisper-1',
        'language' => 'en'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['text'];
```

## Übersetzung

Um Audio ins Englische zu übersetzen, verwenden Sie den Übersetzungs-Endpunkt:

```python
response = client.audio.translations.create(
    model="whisper-1",
    file=audio_file
)
```
