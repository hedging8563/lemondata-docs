## Übersicht

Übersetzen Sie Audio aus jeder unterstützten Sprache in englischen Text. Im Gegensatz zur Transkription gibt dieser Endpunkt unabhängig von der Eingabesprache immer englischen Text aus.

## Request-Body

<ParamField body="file" type="file" required>
  Die zu übersetzende Audiodatei. Unterstützte Formate: `flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`. Die maximale Dateigröße beträgt 25 MB.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  Das zu verwendende Modell. Derzeit wird nur `whisper-1` unterstützt.
</ParamField>

<ParamField body="prompt" type="string">
  Optionaler Text, um den Stil des Modells zu steuern oder ein vorheriges Segment fortzusetzen. Sollte in Englisch verfasst sein.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  Das Format der Ausgabe. Optionen: `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number">
  Die Sampling-Temperatur zwischen 0 und 1. Höhere Werte (wie 0,8) führen zu einer zufälligeren Ausgabe, während niedrigere Werte (wie 0,2) die Ausgabe fokussierter und deterministischer machen.
</ParamField>

## Antwort

<ResponseField name="text" type="string">
  Der übersetzte englische Text.
</ResponseField>

Für das Format `verbose_json` enthält die Antwort zusätzlich:

<ResponseField name="language" type="string">
  Die erkannte Sprache des Eingabe-Audios.
</ResponseField>

<ResponseField name="duration" type="number">
  Die Dauer des Eingabe-Audios (in Sekunden).
</ResponseField>

<ResponseField name="segments" type="array">
  Übersetzte Textsegmente mit Zeitstempeln.
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"

with open("german_audio.mp3", "rb") as audio_file:
    response = client.audio.translations.create(
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP

$ch = curl_init('https://api.lemondata.cc/v1/audio/translations');

    ]

curl_close($ch);
```

## Übersetzung vs. Transkription

| Eigenschaft | Übersetzung | Transkription |
|---------|-------------|---------------|
| Ausgabesprache | Immer Englisch | Gleich wie Eingabe |
| Anwendungsfall | Fremdsprachiges Audio in Englisch umwandeln | Originalsprache beibehalten |
| Sprachparameter | Nicht anwendbar | Optionaler Hinweis |

<Note>
  Der Übersetzungs-Endpunkt erkennt automatisch die Quellsprache und übersetzt sie ins Englische. Der Parameter `language` aus der Transkription wird ignoriert.
</Note>
