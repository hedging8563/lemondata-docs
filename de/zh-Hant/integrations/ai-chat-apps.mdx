## Übersicht

LemonData funktioniert mit jeder OpenAI-kompatiblen Chat-Anwendung. Dieser Leitfaden beschreibt die Konfiguration für beliebte Desktop- und Mobil-Chat-Anwendungen.

## Unterstützte Anwendungen

| Anwendung | Plattform | Funktionen |
|-------------|----------|----------|
| LobeChat | Web, Desktop, Self-hosted | Multi-Provider, Plugins, RAG |
| CherryStudio | Windows, macOS, Linux | Umfangreiche Plugins, mehrere Provider |
| Chatbox | Windows, macOS, Linux | Leichtgewichtig, schnell |
| BotGem | iOS, macOS | Native Apple-Erfahrung |
| TypingMind | Web, Desktop | Benutzerdefinierte Personas, Plugins |
| OpenCat | iOS, macOS | Nativ, Offline-Unterstützung |
| ChatWise | Windows, macOS | Team-Kollaboration |

## Konfigurationsschritte

### Schritt 1: Erhalten Sie Ihren API Key

1. Melden Sie sich im [LemonData Dashboard](https://lemondata.cc/dashboard) an
2. Navigieren Sie zu [API Keys](https://lemondata.cc/dashboard/api)
3. Erstellen und kopieren Sie Ihren API Key (Format: `sk-...`)

### Schritt 2: Konfigurieren Sie Ihre Anwendung

<Tabs>
  <Tab title="LobeChat">
    **Für selbstgehostetes LobeChat:**

    Umgebungsvariablen festlegen:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **Für LobeChat Cloud:**
    1. Öffnen Sie **Settings** → **AI Service Provider**
    2. Wählen Sie **OpenAI**
    3. Geben Sie Ihren LemonData API Key ein
    4. Konfigurieren Sie den benutzerdefinierten Endpunkt: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. Öffnen Sie **Settings** → **Model Provider**
    2. Klicken Sie auf **Add Provider**
    3. Wählen Sie **OpenAI API Compatible**
    4. Eingeben:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Modelle hinzufügen: `gpt-4o`, `claude-sonnet-4-5` usw.
    6. Speichern und testen
  </Tab>
  <Tab title="Chatbox">
    1. Öffnen Sie **Settings** → **AI Model**
    2. Wählen Sie **OpenAI API Compatible**
    3. Eingeben:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Klicken Sie auf **Verify**, um die Verbindung zu testen
  </Tab>
  <Tab title="BotGem">
    1. Öffnen Sie **Settings** → **Providers**
    2. Klicken Sie auf **Add Custom Provider**
    3. Eingeben:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Fügen Sie Ihre bevorzugten Modelle hinzu
    5. Einstellungen speichern
  </Tab>
  <Tab title="TypingMind">
    1. Klicken Sie auf **Settings** (Zahnrad-Symbol)
    2. Wählen Sie **Custom Endpoint**
    3. Eingeben:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Modelle konfigurieren und speichern
  </Tab>
</Tabs>

### Schritt 3: Modelle hinzufügen

Nachdem Sie den Provider konfiguriert haben, fügen Sie die Modelle hinzu, die Sie verwenden möchten:

| Kategorie | Modell |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Konfigurationsreferenz

| Feld | Wert |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API-Typ | OpenAI Compatible |

**JSON-Konfiguration (für unterstützte Anwendungen):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Verbindungsfehler">
    - Überprüfen Sie, ob die Base URL exakt `https://api.lemondata.cc/v1` lautet
    - Prüfen Sie auf einen abschließenden Schrägstrich (falls vorhanden, entfernen)
    - Überprüfen Sie die Netzwerkverbindung und Firewall-Einstellungen
  </Accordion>

  <Accordion title="401 Unauthorized">
    - Überprüfen Sie Ihren API Key erneut
    - Überprüfen Sie im [LemonData Dashboard](https://lemondata.cc/dashboard/api), ob der Key aktiv ist
    - Stellen Sie sicher, dass der Key keine zusätzlichen Leerzeichen enthält
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - Überprüfen Sie Ihr Kontoguthaben im Dashboard
    - Laden Sie Ihr Konto auf
  </Accordion>

  <Accordion title="404 Model Not Available">
    - Überprüfen Sie die Schreibweise des Modellnamens
    - Überprüfen Sie die Modellverfügbarkeit unter [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
    - Versuchen Sie ein anderes Modell
  </Accordion>

  <Accordion title="Langsame Antwortzeit">
    - Versuchen Sie ein schnelleres Modell (z. B. `gpt-4o-mini` anstelle von `gpt-4o`)
    - Reduzieren Sie den Parameter `max_tokens`, falls möglich
    - Überprüfen Sie Ihre Netzwerkverbindung
  </Accordion>
</AccordionGroup>

## Tipps

<AccordionGroup>
  <Accordion title="Verwenden Sie das richtige Modell">
    Wählen Sie das Modell basierend auf der Aufgabe: GPT-4o-mini für schnelle Chats, Claude oder GPT-4o für komplexe Aufgaben.
  </Accordion>

  <Accordion title="Häufig genutzte Prompts speichern">
    Die meisten Anwendungen unterstützen das Speichern von Prompts oder Personas. Erstellen Sie Vorlagen für häufige Aufgaben.
  </Accordion>

  <Accordion title="Nutzung überwachen">
    Überprüfen Sie Ihre Nutzung im LemonData Dashboard, um Kosten zu verfolgen und Ihr Budget zu verwalten.
  </Accordion>
</AccordionGroup>
