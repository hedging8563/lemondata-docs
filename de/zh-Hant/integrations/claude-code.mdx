## Übersicht

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) ist das offizielle KI-gestützte Programmierung-CLI-Tool von Anthropic. Sie können es so konfigurieren, dass LemonData als API-Provider verwendet wird.

## Konfiguration

### Umgebungsvariablen

Setzen Sie die folgenden Umgebungsvariablen:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Fügen Sie diese zu Ihrer Shell-Konfigurationsdatei hinzu (`~/.bashrc`, `~/.zshrc` usw.), um sie dauerhaft zu machen.

### Konfigurationsdatei

Alternativ können Sie die Datei `~/.claude/settings.json` erstellen oder bearbeiten:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Verwendung

Nach der Konfiguration können Sie Claude Code wie gewohnt verwenden:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Verfügbare Modelle

LemonData unterstützt alle Claude-Modelle:

| Modell | Beschreibung |
|-------|-------------|
| `claude-opus-4-5` | Das leistungsstärkste Modell für komplexe logische Schlussfolgerungen |
| `claude-sonnet-4-5` | Ausgewogene Leistung, ideal für die Programmierung |
| `claude-haiku-4-5` | Schnellstes Modell mit hoher Kosteneffizienz |

## Modellauswahl

Verwenden Sie das Flag `--model`, um ein Modell anzugeben:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Oder legen Sie einen Standardwert in Ihrer Konfiguration fest:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Erweitertes Denken

Verwenden Sie für komplexe Reasoning-Aufgaben Modelle mit der Funktion für erweitertes Denken:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Verbindungsfehler">
    Überprüfen Sie, ob die Basis-URL exakt `https://api.lemondata.cc` lautet (das Anthropic SDK benötigt kein `/v1`-Suffix).
  </Accordion>

  <Accordion title="Ungültiger API-Key">
    Stellen Sie sicher, dass Ihr LemonData API-Key mit `sk-` beginnt und über ausreichendes Guthaben verfügt.
  </Accordion>

  <Accordion title="Modell nicht gefunden">
    Überprüfen Sie, ob der Modellname korrekt ist. Verwenden Sie das Format `claude-sonnet-4-5`.
  </Accordion>
</AccordionGroup>
