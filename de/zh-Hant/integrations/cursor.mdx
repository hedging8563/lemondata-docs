## Übersicht

[Cursor](https://cursor.sh) ist ein KI-gestützter Code-Editor. Sie können LemonData als benutzerdefinierten API-Anbieter verwenden, um auf über 300 Modelle zuzugreifen.

## Konfiguration

1. Öffnen Sie die Cursor-Einstellungen (`Cmd/Ctrl + ,`)
2. Navigieren Sie zu **Models** in der linken Seitenleiste
3. Scrollen Sie nach unten zum Bereich **OpenAI API Key**
4. Geben Sie Ihren LemonData API key ein
5. Aktivieren Sie **Override OpenAI Base URL**
6. Setzen Sie die Base URL auf:

```
https://api.lemondata.cc/v1
```

## Modellauswahl

Nach der Konfiguration können Sie jedes LemonData-Modell in Cursor auswählen:

### Empfohlene Modelle für die Programmierung

| Modell | Bestens geeignet für |
|-------|----------|
| `claude-opus-4-5` | Komplexe Programmierung, Architekturdesign |
| `claude-sonnet-4-5` | Allgemeine Programmierung, Debugging |
| `gpt-4o` | Code-Generierung, Erklärungen |
| `deepseek-r1` | Aufgaben mit hoher Reasoning-Anforderung |
| `gpt-4o-mini` | Schnelle Vervollständigung, einfache Aufgaben |

## Funktionen

### Chat

Verwenden Sie das KI-Chat-Panel (`Cmd/Ctrl + L`) mit jedem LemonData-Modell:

- Code-Erklärungen
- Bug-Fixing
- Refactoring-Vorschläge
- Dokumentationserstellung

### Composer

Die Composer-Funktion (`Cmd/Ctrl + I`) kann mit LemonData-Modellen verwendet werden für:

- Bearbeitung mehrerer Dateien
- Feature-Implementierung
- Code-Migration

### Tab-Vervollständigung

Für die Tab-Vervollständigung empfehlen wir die Verwendung schneller Modelle:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Modelle werden nicht angezeigt">
    Cursor speichert die Modellliste im Cache. Bitte versuchen Sie, Cursor nach der API-Konfiguration neu zu starten.
  </Accordion>

  <Accordion title="Langsame Antwort">
    Überprüfen Sie Ihre Modellauswahl. Verwenden Sie für schnelle Aufgaben ein schnelleres Modell wie `gpt-4o-mini`.
  </Accordion>

  <Accordion title="Authentifizierungsfehler">
    Bitte stellen Sie sicher, dass Ihr API key korrekt ist und Ihr Guthaben ausreicht.
  </Accordion>
</AccordionGroup>
