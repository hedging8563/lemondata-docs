## Übersicht

Dify ist eine Open-Source-Entwicklungsplattform für LLM-Anwendungen. Sie bietet visuelle Prompt-Orchestrierung, RAG-Pipelines, Agent-Frameworks und LLMOps-Funktionen. LemonData kann als benutzerdefinierter Modellanbieter in Dify konfiguriert werden.

## Vorteile

- Zugriff auf über 300 KI-Modelle über eine einzige Schnittstelle
- Modellwechsel ohne Änderung der Anwendungslogik
- Wahl des besten Modells für jede Aufgabe zur Kostenoptimierung
- Einheitliche Abrechnung und Nutzungsverfolgung

## Voraussetzungen

- LemonData-Konto mit API-Zugriffsberechtigung
- Dify-Installation (Cloud-Version oder Self-Hosted)

## Konfigurationsschritte

### Schritt 1: Erhalten Sie Ihren API-Key

1. Melden Sie sich im [LemonData Dashboard](https://lemondata.cc/dashboard) an
2. Navigieren Sie zu [API Keys](https://lemondata.cc/dashboard/api)
3. Erstellen und kopieren Sie Ihren API-Key (Format: `sk-...`)

### Schritt 2: Hinzufügen eines benutzerdefinierten Modellanbieters

<Steps>
  <Step title="Einstellungen öffnen">
    Gehen Sie in Dify zu **Settings** → **Model Provider**
  </Step>
  <Step title="OpenAI-kompatiblen Anbieter hinzufügen">
    Klicken Sie auf **Add Model Provider** und wählen Sie **OpenAI-API-compatible**
  </Step>
  <Step title="Anbieter konfigurieren">
    Geben Sie die folgenden Einstellungen ein:

    | Feld | Wert |
    |-------|-------|
    | Anbietername | LemonData |
    | API Key | `sk-your-lemondata-key` |
    | API Base URL | `https://api.lemondata.cc/v1` |
  </Step>
  <Step title="Modelle hinzufügen">
    Fügen Sie die Modelle hinzu, die Sie verwenden möchten:
    - `gpt-4o`
    - `gpt-4o-mini`
    - `claude-sonnet-4-5`
    - `claude-opus-4-5`
    - `gemini-2.5-flash`
    - `gemini-2.5-pro`
    - `deepseek-r1`
  </Step>
</Steps>

### Schritt 3: Verbindung testen

1. Wählen Sie ein Modell aus (z. B. `gpt-4o-mini`)
2. Senden Sie eine Testnachricht
3. Bestätigen Sie, dass Sie eine Antwort erhalten

## Verwendung in Anwendungen

### Chatbot

1. Erstellen Sie eine neue Chatbot-Anwendung
2. Wählen Sie LemonData als Modellanbieter
3. Wählen Sie Ihr bevorzugtes Modell
4. Konfigurieren Sie den System Prompt und die Parameter

### Agent

1. Erstellen Sie eine Agent-Anwendung
2. Wählen Sie ein leistungsstarkes Modell (GPT-4o, Claude)
3. Fügen Sie Tools und Wissensdatenbanken hinzu
4. Konfigurieren Sie das Verhalten des Agents

### Workflow

1. Erstellen Sie einen Workflow
2. Fügen Sie einen LLM-Knoten hinzu
3. Wählen Sie für jeden Knoten ein LemonData-Modell
4. Verbinden Sie die Knoten und konfigurieren Sie den Datenfluss

## Verfügbare Modelle

| Kategorie | Modell |
|----------|--------|
| Chat | GPT-4o, GPT-4o-mini, Claude Sonnet/Opus, Gemini, DeepSeek |
| Embeddings | text-embedding-3-small, text-embedding-3-large |
| Vision | GPT-4o (unterstützt Bilder), Claude Sonnet (unterstützt Bilder) |

## RAG-Konfiguration

Konfigurieren Sie für RAG-Anwendungen das Embedding-Modell:

1. Gehen Sie zu **Settings** → **Model Provider**
2. Fügen Sie das Embedding-Modell hinzu: `text-embedding-3-small`
3. Legen Sie es in den Einstellungen der Wissensdatenbank als Standard-Embedding-Modell fest

## Best Practices

<AccordionGroup>
  <Accordion title="Beginnen Sie mit kostengünstigen Modellen">
    Verwenden Sie GPT-4o-mini während der Test- und Entwicklungsphase und wechseln Sie in der Produktionsumgebung zu leistungsstärkeren Modellen.
  </Accordion>

  <Accordion title="Modell-Routing verwenden">
    Konfigurieren Sie verschiedene Modelle für unterschiedliche Aufgaben — schnelle Modelle für einfache Abfragen, leistungsstarke Modelle für komplexe Schlussfolgerungen.
  </Accordion>

  <Accordion title="Nutzung überwachen">
    Kombinieren Sie die LLMOps-Funktionen von Dify mit der LemonData-Konsole, um Kosten und Leistung zu verfolgen.
  </Accordion>
</AccordionGroup>

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Verbindungsfehler">
    - Stellen Sie sicher, dass die API Base URL exakt `https://api.lemondata.cc/v1` lautet
    - Überprüfen Sie, ob am Ende ein Schrägstrich steht
    - Überprüfen Sie die Netzwerkverbindung
  </Accordion>

  <Accordion title="401 Nicht autorisiert (Unauthorized)">
    - Überprüfen Sie Ihren API-Key erneut
    - Stellen Sie sicher, dass der Key in der LemonData-Konsole aktiviert ist
  </Accordion>

  <Accordion title="Modell nicht gefunden">
    - Bestätigen Sie, dass der Modellname exakt übereinstimmt
    - Überprüfen Sie die Modellverfügbarkeit unter [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models)
  </Accordion>
</AccordionGroup>
