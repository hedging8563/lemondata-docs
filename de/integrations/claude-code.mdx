## Übersicht

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) ist das offizielle CLI-Tool von Anthropic für KI-gestützte Programmierung. Sie können es so konfigurieren, dass LemonData als API-Anbieter verwendet wird.

## Konfiguration

### Umgebungsvariablen

Setzen Sie die folgenden Umgebungsvariablen:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Fügen Sie diese für eine dauerhafte Speicherung zu Ihrem Shell-Profil hinzu (`~/.bashrc`, `~/.zshrc`, etc.).

### Konfigurationsdatei

Alternativ können Sie die Datei `~/.claude/settings.json` erstellen oder bearbeiten:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Verwendung

Verwenden Sie Claude Code nach der Konfiguration ganz normal:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Verfügbare Modelle

LemonData unterstützt alle Claude-Modelle:

| Modell | Beschreibung |
|-------|-------------|
| `claude-opus-4-5` | Leistungsfähigstes Modell, komplexes logisches Denken |
| `claude-sonnet-4-5` | Ausgewogene Leistung, hervorragend für die Programmierung |
| `claude-haiku-4-5` | Schnellstes Modell, kosteneffizient |

## Modellauswahl

Geben Sie ein Modell mit dem `--model`-Flag an:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Oder legen Sie einen Standardwert in Ihrer Konfiguration fest:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Extended Thinking

Verwenden Sie für komplexe Denkaufgaben Modelle mit Extended Thinking:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
