## Übersicht

[Cursor](https://cursor.sh) ist ein KI-gestützter Code-Editor. Sie können LemonData als benutzerdefinierten API-Anbieter verwenden, um auf über 300 Modelle zuzugreifen.

## Konfiguration

1. Öffnen Sie die Cursor-Einstellungen (`Cmd/Ctrl + ,`)
2. Navigieren Sie zu **Models** in der linken Seitenleiste
3. Scrollen Sie nach unten zum Abschnitt **OpenAI API Key**
4. Geben Sie Ihren LemonData API-Key ein
5. Aktivieren Sie **Override OpenAI Base URL**
6. Setzen Sie die Base URL auf:

```
https://api.lemondata.cc/v1
```

## Modellauswahl

Nach der Konfiguration können Sie jedes LemonData-Modell in Cursor auswählen:

### Empfohlene Modelle für das Coding

| Modell | Bestens geeignet für |
|-------|----------|
| `claude-opus-4-5` | Komplexes Coding, Architektur |
| `claude-sonnet-4-5` | Allgemeines Coding, Debugging |
| `gpt-4o` | Code-Generierung, Erklärungen |
| `deepseek-r1` | Aufgaben mit hohem logischem Anspruch |
| `gpt-4o-mini` | Schnelle Vervollständigungen, einfache Aufgaben |

## Funktionen

### Chat

Verwenden Sie das KI-Chat-Panel (`Cmd/Ctrl + L`) mit jedem LemonData-Modell:

- Code-Erklärungen
- Fehlerbehebung
- Refactoring-Vorschläge
- Dokumentationserstellung

### Composer

Die Composer-Funktion (`Cmd/Ctrl + I`) funktioniert mit LemonData-Modellen für:

- Bearbeitungen über mehrere Dateien hinweg
- Feature-Implementierung
- Code-Migrationen

### Tab-Vervollständigung

Für Tab-Vervollständigungen empfehlen wir die Verwendung schneller Modelle:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`

## Fehlerbehebung

  </Accordion>

  <Accordion title="Langsame Antworten">

  </Accordion>

  <Accordion title="Authentifizierungsfehler">
    Überprüfen Sie, ob Ihr API-Key korrekt ist und über ausreichendes Guthaben verfügt.
  </Accordion>
</AccordionGroup>
