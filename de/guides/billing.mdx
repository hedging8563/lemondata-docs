## Übersicht

LemonData nutzt ein **Pay-as-you-go-Preismodell**. Sie zahlen nur für das, was Sie tatsächlich nutzen, ohne Abonnements oder Mindestverpflichtungen.

## So funktioniert die Abrechnung

1. **Guthaben hinzufügen** zu Ihrem Konto
2. **Die API nutzen** – Kosten werden pro Anfrage abgezogen
3. **Nutzung überwachen** in Ihrem Dashboard
4. **Guthaben aufladen**, wenn Ihr Saldo niedrig ist

### Abrechnung asynchroner Aufgaben (Video/Musik/3D)

<Warning>
  **Wichtig**: Video-, Musik- und 3D-Generierungsaufgaben werden **bei der Übermittlung abgerechnet**, nicht bei Fertigstellung.
</Warning>

Für asynchrone Generierungsaufgaben (Video, Musik, 3D-Modelle):

1. **Die Abrechnung erfolgt sofort** bei Aufgabenübermittlung
2. Sie erhalten eine `task_id` zur Statusabfrage
3. **Keine automatische Rückerstattung** bei Aufgabenfehlern

Dies liegt daran, dass Upstream-Anbieter uns bei der Aufgabenübermittlung belasten, unabhängig vom Ergebnis. Bei Aufgabenfehlern wenden Sie sich bitte an support@lemondata.cc.

```python
# Beispiel: Abrechnung der Videogenerierung
response = client.post("/v1/video/generations", json={
    "model": "kling-v1",
    "prompt": "Ein Sonnenuntergang über dem Meer"
})
# ⚠️ Die Abrechnung erfolgt JETZT, nicht wenn das Video fertig ist

task_id = response.json()["task_id"]
# Status abfragen...
```

- ~4 Zeichen = 1 Token (Englisch)
- ~1-2 Zeichen = 1 Token (Chinesisch)
