title: "Sprache erstellen"
openapi: "POST /v1/audio/speech"
description: "Generiert Audio aus dem Eingabetext"
---

## Request Body

<ParamField body="model" type="string" default="tts-1">
  TTS-Modell: `tts-1` (Standard) oder `tts-1-hd` (höhere Qualität).
</ParamField>

<ParamField body="input" type="string" required>
  Der Text, für den Audio generiert werden soll. Maximal 4096 Zeichen.
</ParamField>

<ParamField body="voice" type="string" required>
  Zu verwendende Stimme: `alloy`, `ash`, `ballad`, `coral`, `echo`, `fable`, `nova`, `onyx`, `sage`, `shimmer`, `verse`.
</ParamField>

<ParamField body="response_format" type="string" default="mp3">
  Audioformat: `mp3`, `opus`, `aac`, `flac`, `wav`, `pcm`.
</ParamField>

<ParamField body="speed" type="number" default="1.0">
  Audiogeschwindigkeit (0.25 bis 4.0).
</ParamField>

## Antwort

Gibt die Audiodatei im angeforderten Format zurück.

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1-hd",
    "voice": "nova",
    "input": "Hello, welcome to LemonData!"
  }' \
  --output speech.mp3
```

```javascript JavaScript
import OpenAI from 'openai';
import fs from 'fs';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.audio.speech.create({
```

```go Go
package main

import (
    "context"
    "io"
    "os"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateSpeech(
        context.Background(),
        openai.CreateSpeechRequest{
            Model: openai.TTSModel1HD,
            Voice: openai.VoiceNova,
            Input: "Hello, welcome to LemonData!",
        },
    )
    if err != nil {
        panic(err)
    }
    defer resp.Close()

    file, _ := os.Create("speech.mp3")
    defer file.Close()
    io.Copy(file, resp)
}
```


## Sprachbeispiele

| Stimme | Beschreibung |
|-------|-------------|
| `alloy` | Neutral, ausgewogen |
| `ash` | Ruhig, besonnen |
| `ballad` | Melodisch, ausdrucksstark |
| `coral` | Warm, einladend |
| `echo` | Warm, gesprächig |
| `fable` | Ausdrucksstark, erzählend |
| `nova` | Freundlich, klar |
| `onyx` | Tief, autoritär |
| `sage` | Weise, nachdenklich |
| `shimmer` | Weich, sanft |
| `verse` | Dynamisch, vielseitig |
