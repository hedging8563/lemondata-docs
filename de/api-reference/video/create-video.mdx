---
title: "Video erstellen"
openapi: "POST /v1/videos/generations"
description: "Erstellt eine Video-Generierungsaufgabe"
---

## Übersicht

Die Videogenerierung erfolgt asynchron. Sie senden eine Anfrage und erhalten eine Task-ID, um anschließend das Ergebnis abzufragen.

## Request Body

<ParamField body="model" type="string" default="sora-2">
  Video-Modell-ID. Eine vollständige Liste der über 80 Modelle von 11 Anbietern finden Sie im [Video Generation Guide](/guides/video-generation).
</ParamField>

<ParamField body="prompt" type="string" required>
  Textbeschreibung des zu generierenden Videos.
</ParamField>

<ParamField body="image_url" type="string">
  URL des Startbildes für die Image-to-Video (I2V) Generierung.
</ParamField>

<ParamField body="image" type="string">
  Base64-kodiertes Bild mit Data-URL-Präfix (z. B. `data:image/jpeg;base64,...`). Alternative zu `image_url` für die I2V-Generierung.
</ParamField>

<ParamField body="duration" type="integer">
  Videodauer in Sekunden (modellabhängig).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Seitenverhältnis (z. B. `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Videoauflösung (z. B. `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Bilder pro Sekunde (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Was bei der Videogenerierung vermieden werden soll.
</ParamField>

<ParamField body="seed" type="integer">
  Zufälliger Seed für reproduzierbare Generierung.
</ParamField>

<ParamField body="cfg_scale" type="number">
  Classifier-free Guidance Scale (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Bewegungsintensität (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  URL des Startbildes.
</ParamField>

<ParamField body="end_image" type="string">
  URL des Endbildes.
</ParamField>

<ParamField body="user" type="string">
  Eine eindeutige Kennung für den Endbenutzer.
</ParamField>
