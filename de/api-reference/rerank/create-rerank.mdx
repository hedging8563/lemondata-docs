---
title: "Dokumente neu ordnen (Rerank)"
api: "POST /v1/rerank"
description: "Ordnet Dokumente nach Relevanz für eine Suchanfrage neu"
---

Ordnen Sie Dokumente mithilfe von semantischen Ähnlichkeitsmodellen neu. Nützlich zur Verbesserung von Suchergebnissen und RAG-Anwendungen.

## Request Body

<ParamField body="model" type="string" required>
  ID des zu verwendenden Reranker-Modells (z. B. `BAAI/bge-reranker-v2-m3`, `qwen3-rerank`).
</ParamField>

<ParamField body="query" type="string" required>
  Die Suchanfrage, gegen die die Dokumente bewertet werden sollen.
</ParamField>

<ParamField body="documents" type="array" required>
  Liste der neu zu ordnenden Dokumente (Strings).
</ParamField>

<ParamField body="top_n" type="integer">
  Anzahl der zurückzugebenden Top-Ergebnisse. Standardmäßig werden alle Dokumente zurückgegeben.
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  Gibt an, ob der ursprüngliche Dokumenttext in der Antwort enthalten sein soll.
</ParamField>

## Response

<ResponseField name="results" type="array">
  Rangliste der Dokumente mit Scores.

  Jedes Ergebnis enthält:
  - `index` (integer): Ursprünglicher Dokument-Index
  - `relevance_score` (number): Relevanz-Score (0-1)
  - `document` (string): Ursprünglicher Text (wenn `return_documents=true`)
</ResponseField>

<ResponseField name="model" type="string">
  Das für das Reranking verwendete Modell.
</ResponseField>

<ResponseField name="usage" type="object">
  Statistiken zur Token-Nutzung.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/rerank" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "BAAI/bge-reranker-v2-m3",
    "query": "What is machine learning?",
    "documents": [
      "Machine learning is a subset of AI",
      "The weather is nice today",
      "Deep learning uses neural networks"
    ],
    "top_n": 2,
    "return_documents": true
  }'
```
