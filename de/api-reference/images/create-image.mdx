---
title: "Bild erstellen"
openapi: "POST /v1/images/generations"
description: "Erstellt ein Bild basierend auf einem Prompt"
---

## Autorisierung

<ParamField header="Authorization" type="string" required>
  Bearer-Token für die API-Authentifizierung. Format: `Bearer sk-your-api-key`
</ParamField>

## Anfragekörper

<ParamField body="model" type="string" default="dall-e-3">
  Zu verwendendes Modell (z.B. `dall-e-3`, `flux-pro`, `midjourney`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Textbeschreibung des gewünschten Bildes.
</ParamField>

<ParamField body="n" type="integer" default="1">
  Anzahl der zu generierenden Bilder (1-4, modellabhängig).
</ParamField>

<ParamField body="size" type="string" default="1024x1024">
  Bildgröße. Optionen variieren je nach Modell:
  - DALL-E 3: `1024x1024`, `1792x1024`, `1024x1792`
  - Andere Modelle: `512x512`, `1024x1024`, etc.
</ParamField>

<ParamField body="quality" type="string" default="standard">
  Bildqualität (`standard` oder `hd`). Nur DALL-E 3.
</ParamField>

<ParamField body="response_format" type="string" default="url">
  Antwortformat: `url` oder `b64_json`.
</ParamField>

<ParamField body="style" type="string" default="vivid">
  Stil für DALL-E 3: `vivid` oder `natural`.
</ParamField>

<ParamField body="user" type="string">
  Eine eindeutige Kennung für den Endbenutzer.
</ParamField>

## Antwort

### Synchrone Antwort (DALL-E, Flux Schnell, etc.)

<ResponseField name="created" type="integer">
  Unix-Zeitstempel der Erstellung.
</ResponseField>

<ResponseField name="data" type="array">
  Array der generierten Bilder.

  Jedes Objekt enthält:
  - `url` (string): URL des generierten Bildes
  - `b64_json` (string): Base64-kodiertes Bild (falls angefordert)
  - `revised_prompt` (string): Der verwendete Prompt (DALL-E 3)
</ResponseField>

### Asynchrone Antwort (Midjourney, Flux Pro, Ideogram, etc.)

Einige Modelle benötigen längere Verarbeitungszeit und geben eine asynchrone Antwort zurück:

<ResponseField name="created" type="integer">
  Unix-Zeitstempel der Erstellung.
</ResponseField>

<ResponseField name="task_id" type="string">
  Eindeutige Aufgabenkennung für Abfragen.
</ResponseField>

<ResponseField name="status" type="string">
  Anfangsstatus: `pending`.
</ResponseField>

<ResponseField name="poll_url" type="string">
  Relative URL für Statusabfragen (z.B. `/v1/images/generations/{task_id}`).
</ResponseField>

<ResponseField name="data" type="array">
  Array mit Platzhalterdaten. `url` ist leer bis zur Fertigstellung.
</ResponseField>

<Note>
Wenn Sie `status: "pending"` erhalten, verwenden Sie den [Bildstatus abrufen](/de/api-reference/images/get-image-status) Endpunkt, um das Ergebnis abzufragen.
</Note>

## Verfügbare Modelle

| Modell | Typ | Funktionen |
|--------|-----|------------|
| `dall-e-3` | Sync | Beste Qualität, Prompt-Verbesserung |
| `dall-e-2` | Sync | Schneller, günstiger |
| `flux-pro` | **Async** | Fotorealistisch, hohe Qualität |
| `flux-schnell` | Sync | Sehr schnell |
| `midjourney` | **Async** | Künstlerischer Stil |
| `ideogram-v3` | **Async** | Beste Textdarstellung |
| `stable-diffusion-3` | Sync | Open Source, anpassbar |

<Tip>
**Asynchrone Modelle** geben `status: "pending"` zurück und erfordern Polling. Siehe [Bildstatus abrufen](/de/api-reference/images/get-image-status) für das Abrufen von Ergebnissen.
</Tip>

## Umgang mit asynchronen Antworten

Für asynchrone Modelle prüfen Sie, ob die Antwort `status: "pending"` enthält:

```python
import requests
import time

def generate_image(prompt, model="midjourney"):
    # Bildanfrage erstellen
    response = requests.post(
        "https://api.lemondata.cc/v1/images/generations",
        headers={"Authorization": "Bearer sk-your-api-key"},
        json={"model": model, "prompt": prompt}
    )
    data = response.json()

    # Prüfen ob asynchron
    if data.get("status") == "pending":
        task_id = data["task_id"]
        print(f"Async task started: {task_id}")

        # Auf Ergebnis warten
        while True:
            status_resp = requests.get(
                f"https://api.lemondata.cc/v1/images/generations/{task_id}",
                headers={"Authorization": "Bearer sk-your-api-key"}
            )
            status_data = status_resp.json()

            if status_data["status"] == "completed":
                return status_data["data"][0]["url"]
            elif status_data["status"] == "failed":
                raise Exception(status_data.get("error", "Generation failed"))

            time.sleep(3)
    else:
        # Synchrone Antwort
        return data["data"][0]["url"]

# Verwendung
url = generate_image("a beautiful sunset over mountains", model="midjourney")
print(f"Generated image: {url}")
```
