---
title: "Embedding erstellen"
openapi: "POST /v1/embeddings"
description: "Erstellt einen Embedding-Vektor, der den Eingabetext repräsentiert"
---

## Autorisierung

<ParamField header="Authorization" type="string" required>
  Bearer-Token für die API-Authentifizierung. Format: `Bearer sk-your-api-key`
</ParamField>

## Request Body

<ParamField body="model" type="string" required>
  ID des zu verwendenden Embedding-Modells (z. B. `text-embedding-3-small`).
</ParamField>

<ParamField body="input" type="string | array" required>
  Eingabetext für das Embedding. Kann ein String oder ein Array von Strings sein.
</ParamField>

<ParamField body="encoding_format" type="string" default="float">
  Format für die Embeddings: `float` oder `base64`.
</ParamField>

<ParamField body="dimensions" type="integer">
  Anzahl der Dimensionen für die Ausgabe (modellspezifisch).
</ParamField>

<ParamField body="user" type="string">
  Eine eindeutige Kennung, die Ihren Endbenutzer für die Missbrauchsüberwachung repräsentiert.
</ParamField>

## Verfügbare Modelle

| Modell | Dimensionen | Beschreibung |
|-------|------------|-------------|
| `text-embedding-3-large` | 3072 | Beste Qualität |
| `text-embedding-3-small` | 1536 | Ausgewogen |
| `text-embedding-ada-002` | 1536 | Legacy |

## Response

<ResponseField name="object" type="string">
  Immer `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Array von Embedding-Objekten.

  Jedes Objekt enthält:
  - `object` (string): `embedding`
  - `index` (integer): Index im Eingabe-Array
  - `embedding` (array): Der Embedding-Vektor
</ResponseField>

<ResponseField name="model" type="string">
  Verwendetes Modell.
</ResponseField>

<ResponseField name="usage" type="object">
  Token-Nutzung mit `prompt_tokens` und `total_tokens`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/embeddings" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "text-embedding-3-small",
    "input": "The quick brown fox jumps over the lazy dog"
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
