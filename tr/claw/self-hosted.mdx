## Hızlı Kurulum

<CodeGroup>
```bash Linux / macOS
curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```

```bash Non-interactive
LEMONDATA_API_KEY=sk-xxx curl -fsSL https://raw.githubusercontent.com/hedging8563/lemondata-openclaw/main/install.sh | bash
```
</CodeGroup>

Yükleyici sizden **LemonData API Key** isteyecektir. [Kontrol Paneli](https://lemondata.cc/dashboard)'nden bir tane edinin.

<Steps>

  </Step>
</Steps>

## Yapılandırma

Yapılandırma dosyası konumu: `~/.openclaw/openclaw.json`

Düzenledikten sonra değişiklikleri uygulamak için hizmeti yeniden başlatın.

### Varsayılan Modeli Değiştirme

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "lemondata-claude/claude-opus-4-6"
      }
    }
  }
}
```

Kullanılabilir model önekleri:
- `lemondata-claude/` — Claude modelleri (anthropic-messages API)
- `lemondata-minimax/` — MiniMax modelleri (anthropic-messages API)
- `lemondata/` — Diğer tüm modeller (openai-completions API)

## Yükseltme

```bash
npm update -g openclaw
# Ardından hizmeti yeniden başlatın
systemctl --user restart openclaw  # Linux
# veya
launchctl stop cc.lemondata.openclaw && launchctl start cc.lemondata.openclaw  # macOS
```

## Kaldırma

```bash
# Hizmeti durdur ve devre dışı bırak
systemctl --user stop openclaw && systemctl --user disable openclaw  # Linux
launchctl unload ~/Library/LaunchAgents/cc.lemondata.openclaw.plist  # macOS

# OpenClaw'u kaldır
npm uninstall -g openclaw
rm -rf ~/.openclaw
```
