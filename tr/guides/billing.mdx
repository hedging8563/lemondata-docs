## Genel Bakış

LemonData **kullandıkça öde fiyatlandırması** kullanır. Abonelik veya minimum taahhüt olmadan yalnızca kullandığınız kadar ödersiniz.

## Faturalandırma Nasıl Çalışır

1. **Hesabınıza kredi ekleyin**
2. **API'yi kullanın** - maliyetler istek başına düşülür
3. **Panelinizden kullanımı izleyin**
4. **Bakiyeniz azaldığında yükleme yapın**

## Token Sayımı

Tokenlar, metin işlemenin temel birimleridir:

- ~4 karakter = 1 token (İngilizce)
- ~1-2 karakter = 1 token (Çince)
- 1 görüntü = boyut ve detaya göre değişir

### Token Tahmini

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English

# Actual count (for OpenAI models)
import tiktoken
encoder = tiktoken.encoding_for_model("gpt-4o")
tokens = encoder.encode("Your text here")
print(f"Token count: {len(tokens)}")
```

## Kullanım Takibi

### Panel

Kullanımınızı [Panel](https://lemondata.cc/dashboard) üzerinden izleyin:

- Gerçek zamanlı bakiye
- Modele göre kullanım geçmişi
- Maliyet dökümü
- API anahtarı kullanımı

### API Yanıtı

Her yanıt kullanım bilgilerini içerir:

```json
{
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 100,
    "total_tokens": 150
  }
}
```

## Düşük Bakiye Uyarıları

Bakiyeniz düştüğünde uyarıları yapılandırın:

1. **Panel → Ayarlar → Bildirimler** bölümüne gidin
2. Eşik tutarınızı belirleyin
3. E-posta bildirimleri alın

## Kredi Ekleme

### Ödeme Yöntemleri

- Alipay (支付宝)
- WeChat Pay (微信支付)
- Stripe (Visa, Mastercard)

### Adımlar

1. [Panel](https://lemondata.cc/dashboard) hesabınıza giriş yapın
2. **Kredi Ekle** butonuna tıklayın
3. Tutarı ve ödeme yöntemini seçin
4. Ödemeyi tamamlayın

Krediler, ödeme onayından sonra anında eklenir.

## API Anahtarı Limitleri

Bireysel API anahtarları üzerinde harcama limitleri belirleyebilirsiniz:

1. **Panel → API Anahtarları** bölümüne gidin
2. Düzenlemek için bir anahtara tıklayın
3. **Kullanım Limiti** belirleyin

Limite ulaşıldığında, o anahtarla yapılan istekler `402 Payment Required` hatası döndürecektir.

## Faturalar

Kurumsal hesaplar için faturalar mevcuttur:

1. **Panel → Faturalandırma** bölümüne gidin
2. İşlem geçmişini görüntüleyin
3. Faturaları PDF olarak indirin
