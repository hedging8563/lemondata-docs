## Genel Bakış

[Claude Code](https://docs.anthropic.com/en/docs/claude-code), Anthropic'in yapay zeka destekli kodlama için resmi CLI aracıdır. LemonData'yı API sağlayıcısı olarak kullanacak şekilde yapılandırabilirsiniz.

## Yapılandırma

### Ortam Değişkenleri

Aşağıdaki ortam değişkenlerini ayarlayın:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Kalıcılık sağlamak için bunları kabuk profilinize (`~/.bashrc`, `~/.zshrc`, vb.) ekleyin.

### Yapılandırma Dosyası

Alternatif olarak, `~/.claude/settings.json` dosyasını oluşturun veya düzenleyin:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Kullanım

Yapılandırmadan sonra Claude Code'u normal şekilde kullanın:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Mevcut Modeller

LemonData tüm Claude modellerini destekler:

| Model | Açıklama |
|-------|-------------|
| `claude-opus-4-5` | En yetenekli, karmaşık muhakeme |
| `claude-sonnet-4-5` | Dengeli performans, kodlama için mükemmel |
| `claude-haiku-4-5` | En hızlı, maliyet etkin |

## Model Seçimi

`--model` bayrağı ile bir model belirtin:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Veya yapılandırmanızda bir varsayılan ayarlayın:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Genişletilmiş Düşünme

Karmaşık muhakeme görevleri için genişletilmiş düşünme özelliğine sahip modelleri kullanın:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
