## Genel Bakış

LemonData, herhangi bir OpenAI uyumlu sohbet uygulamasıyla çalışır. Bu kılavuz, popüler masaüstü ve mobil sohbet uygulamaları için yapılandırmayı kapsar.

## Desteklenen Uygulamalar

| Uygulama | Platform | Özellikler |
|-------------|----------|----------|
| LobeChat | Web, Masaüstü, Self-hosted | Çoklu sağlayıcı, eklentiler, RAG |
| CherryStudio | Windows, macOS, Linux | Zengin eklentiler, çoklu sağlayıcılar |
| Chatbox | Windows, macOS, Linux | Hafif, hızlı |
| BotGem | iOS, macOS | Yerel Apple deneyimi |
| TypingMind | Web, Masaüstü | Özel personalar, eklentiler |
| OpenCat | iOS, macOS | Yerel, çevrimdışı destek |
| ChatWise | Windows, macOS | Ekip iş birliği |

## Yapılandırma Adımları

### Adım 1: API Anahtarınızı Alın

1. [LemonData Dashboard](https://lemondata.cc/dashboard) paneline giriş yapın
2. [API Keys](https://lemondata.cc/dashboard/api) bölümüne gidin
3. API anahtarınızı oluşturun ve kopyalayın (format: `sk-...`)

### Adım 2: Uygulamanızı Yapılandırın

<Tabs>
  <Tab title="LobeChat">
    **Kendi sunucunuzda barındırılan LobeChat için:**

    Ortam değişkenlerini ayarlayın:
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **LobeChat Cloud için:**
    1. **Settings** → **AI Service Provider** kısmını açın
    2. **OpenAI** seçeneğini belirleyin
    3. LemonData API anahtarınızı girin
    4. Özel uç noktayı (endpoint) ayarlayın: `https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. **Settings** → **Model Provider** kısmını açın
    2. **Add Provider** butonuna tıklayın
    3. **OpenAI API Compatible** seçeneğini belirleyin
    4. Şunları girin:
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. Modelleri ekleyin: `gpt-4o`, `claude-sonnet-4-5`, vb.
    6. Kaydedin ve test edin
  </Tab>
  <Tab title="Chatbox">
    1. **Settings** → **AI Model** kısmını açın
    2. **OpenAI API Compatible** seçeneğini belirleyin
    3. Şunları girin:
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. Bağlantıyı test etmek için **Verify** butonuna tıklayın
  </Tab>
  <Tab title="BotGem">
    1. **Settings** → **Providers** kısmını açın
    2. **Add Custom Provider** seçeneğine dokunun
    3. Şunları girin:
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. Tercih ettiğiniz modelleri ekleyin
    5. Yapılandırmayı kaydedin
  </Tab>
  <Tab title="TypingMind">
    1. **Settings** (dişli simgesi) kısmına tıklayın
    2. **Custom Endpoint** seçeneğini belirleyin
    3. Şunları girin:
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. Modelleri yapılandırın ve kaydedin
  </Tab>
</Tabs>

### Adım 3: Modelleri Ekleyin

Sağlayıcıyı yapılandırdıktan sonra kullanmak istediğiniz modelleri ekleyin:

| Kategori | Modeller |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## Yapılandırma Referansı

| Alan | Değer |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API Type | OpenAI Compatible |

**JSON Yapılandırması (destekleyen uygulamalar için):**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## Sorun Giderme

<AccordionGroup>
  <Accordion title="Bağlantı Hatası">
    - Base URL'in tam olarak `https://api.lemondata.cc/v1` olduğunu doğrulayın
    - Sondaki eğik çizgileri kontrol edin (varsa kaldırın)
    - İnternet bağlantısını ve güvenlik duvarı ayarlarını doğrulayın
  </Accordion>

  <Accordion title="401 Yetkisiz (Unauthorized)">
    - API anahtarınızı tekrar kontrol edin
    - Anahtarın [LemonData Dashboard](https://lemondata.cc/dashboard/api) üzerinde aktif olduğunu doğrulayın
    - Anahtarda fazladan boşluk olmadığından emin olun
  </Accordion>

  <Accordion title="402 Yetersiz Bakiye (Insufficient Balance)">
    - Kontrol panelinden hesap bakiyenizi kontrol edin
    - Hesabınıza kredi ekleyin
  </Accordion>

  <Accordion title="404 Model Mevcut Değil (Model Not Available)">
    - Model adının doğru yazıldığını doğrulayın
    - [lemondata.cc/en/models](https://lemondata.cc/tr/models) adresinden modelin kullanılabilirliğini kontrol edin
    - Farklı bir model deneyin
  </Accordion>

  <Accordion title="Yavaş Yanıtlar">
    - Daha hızlı bir model deneyin (örneğin, `gpt-4o` yerine `gpt-4o-mini`)
    - Varsa `max_tokens` parametresini azaltın
    - Ağ bağlantınızı kontrol edin
  </Accordion>
</AccordionGroup>
