## Genel Bakış

[Cursor](https://cursor.sh) yapay zeka destekli bir kod düzenleyicidir. 300'den fazla modele erişmek için LemonData'yı özel bir API sağlayıcısı olarak kullanabilirsiniz.

## Yapılandırma

1. Cursor Ayarlarını açın (`Cmd/Ctrl + ,`)
2. Sol kenar çubuğundaki **Models** bölümüne gidin
3. Aşağıdaki **OpenAI API Key** bölümüne kaydırın
4. LemonData API anahtarınızı girin
5. **Override OpenAI Base URL** seçeneğini etkinleştirin
6. Base URL değerini şu şekilde ayarlayın:

```
https://api.lemondata.cc/v1
```

## Model Seçimi

Yapılandırmadan sonra, Cursor içinde herhangi bir LemonData modelini seçebilirsiniz:

### Kodlama İçin Önerilen Modeller

| Model | En İyi Kullanım Alanı |
|-------|----------|
| `claude-opus-4-5` | Karmaşık kodlama, mimari |
| `claude-sonnet-4-5` | Genel kodlama, hata ayıklama |
| `gpt-4o` | Kod oluşturma, açıklamalar |
| `deepseek-r1` | Yoğun muhakeme gerektiren görevler |
| `gpt-4o-mini` | Hızlı tamamlamalar, basit görevler |

## Özellikler

### Chat

Herhangi bir LemonData modeli ile yapay zeka sohbet panelini (`Cmd/Ctrl + L`) kullanın:

- Kod açıklamaları
- Hata düzeltme
- Yeniden yapılandırma (refactoring) önerileri
- Dokümantasyon oluşturma

### Composer

Composer özelliği (`Cmd/Ctrl + I`), şu işlemler için LemonData modelleriyle çalışır:

- Çoklu dosya düzenlemeleri
- Özellik uygulama
- Kod taşıma (migration)

### Tab Completion

Sekme tamamlamaları için hızlı modeller kullanmanızı öneririz:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
