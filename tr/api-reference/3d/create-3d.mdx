title: "3D Model Oluştur"
openapi: "POST /v1/3d/generations"
description: "Bir 3D model oluşturma görevi oluşturur"
---

## Yetkilendirme

<ParamField header="Authorization" type="string" required>
  API kimlik doğrulaması için Bearer token. Format: `Bearer sk-your-api-key`
</ParamField>

Tripo3D ve diğer sağlayıcıları kullanarak metin veya görsellerden 3D modeller oluşturun. Bu asenkron bir API'dir.

## İstek Gövdesi

<ParamField body="model" type="string" default="tripo3d-v2.5">
  Kullanılacak model (örneğin, `tripo3d-v2.5`, `tripo3d-v2`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Oluşturulacak 3D modelin metin açıklaması.
</ParamField>

<ParamField body="image" type="string">
  Görselden 3D'ye üretim için Base64 kodlu görsel.
</ParamField>

<ParamField body="image_url" type="string">
  Görselden 3D'ye üretim için görselin URL'si.
</ParamField>

<ParamField body="format" type="string" default="glb">
  Çıktı formatı: `glb`, `fbx`, `obj` veya `usdz`.
</ParamField>

<ParamField body="quality" type="string" default="standard">
  Kalite seviyesi: `draft`, `standard` veya `high`.
</ParamField>

<ParamField body="style" type="string">
  Model için stil ön ayarı.
</ParamField>

<ParamField body="seed" type="integer">
  Tekrarlanabilir üretim için seed değeri.
</ParamField>

<ParamField body="user" type="string">
  Son kullanıcı için benzersiz bir tanımlayıcı.
</ParamField>

## Yanıt

<ResponseField name="id" type="string">
  Durum sorgulama için görev kimliği (Task ID).
</ResponseField>

<ResponseField name="status" type="string">
  Görev durumu: `pending`, `processing`, `completed` veya `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Görevin oluşturulma zamanına ait Unix zaman damgası.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":   "tripo3d-v2.5",
        "prompt":  "A detailed medieval castle with towers",
        "format":  "glb",
        "quality": "high",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/3d/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```
