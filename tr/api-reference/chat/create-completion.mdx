title: "Sohbet Tamamlama Oluştur"

---

## Yetkilendirme

<ParamField header="Authorization" type="string" required>
  API kimlik doğrulaması için Bearer token. Format: `Bearer sk-your-api-key`
</ParamField>

## İstek Gövdesi

<ParamField body="model" type="string" required>
  Kullanılacak modelin kimliği. Mevcut seçenekler için [Models](https://lemondata.cc/tr/models) sayfasına bakın.
</ParamField>

<ParamField body="messages" type="array" required>
  Konuşmayı oluşturan mesajların listesi.

  Her mesaj nesnesi şunları içerir:
  - `role` (string): `system`, `user` veya `assistant`
  - `content` (string | array): Mesaj içeriği
</ParamField>

<ParamField body="temperature" type="number" default="1">
  0 ile 2 arasında örnekleme sıcaklığı. Daha yüksek değerler çıktıyı daha rastgele hale getirir.
</ParamField>

<ParamField body="max_tokens" type="integer">
  Oluşturulacak maksimum token sayısı.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Eğer true ise, kısmi mesaj farkları SSE olayları olarak gönderilir.
</ParamField>

<ParamField body="stream_options" type="object">
  Akış seçenekleri. Akış parçalarında token kullanımını almak için `include_usage: true` olarak ayarlayın.
</ParamField>

<ParamField body="top_p" type="number" default="1">
  Nucleus örnekleme parametresi. Bunu veya temperature değerini değiştirmenizi öneririz, her ikisini birden değil.
</ParamField>

<ParamField body="frequency_penalty" type="number" default="0">
  -2.0 ile 2.0 arasında bir sayı. Pozitif değerler tekrarlanan token'ları cezalandırır.
</ParamField>

<ParamField body="presence_penalty" type="number" default="0">
  -2.0 ile 2.0 arasında bir sayı. Pozitif değerler metinde zaten bulunan token'ları cezalandırır.
</ParamField>

<ParamField body="stop" type="string | array">
  API'nin token oluşturmayı durduracağı 4 adede kadar dizi.
</ParamField>

<ParamField body="tools" type="array">
  Modelin çağırabileceği araçların listesi (fonksiyon çağırma).
</ParamField>

<ParamField body="tool_choice" type="string | object">
  Modelin araçları nasıl kullanacağını kontrol eder. Seçenekler: `auto`, `none`, `required` veya belirli bir araç nesnesi.
</ParamField>
