---
title: "Müzik Oluştur"
api: "POST /v1/music/generations"
description: "Suno kullanarak bir müzik oluşturma görevi oluşturur"
---

Yapay zeka kullanarak müzik ve şarkı sözü oluşturun. Bu asenkron bir API'dir - tamamlanma durumunu sorgulamak için bir görev kimliği (task ID) alacaksınız.

## İstek Gövdesi

<ParamField body="model" type="string" default="suno_music">
  Kullanılacak model: müzik oluşturma için `suno_music`, yalnızca şarkı sözleri için `suno_lyrics`.
</ParamField>

<ParamField body="prompt" type="string" required>
  Oluşturulacak müziğin açıklaması. Tarz, ruh hali, enstrümanlar vb. içerebilir.
</ParamField>

<ParamField body="title" type="string">
  Oluşturulan şarkı için başlık.
</ParamField>

<ParamField body="tags" type="string">
  Tarz etiketleri (örneğin, "pop, upbeat, electronic").
</ParamField>

<ParamField body="action" type="string">
  Oluşturma türü: `MUSIC` (varsayılan) veya `LYRICS`.
</ParamField>

<ParamField body="mv" type="string">
  Kullanılacak model sürümü.
</ParamField>

<ParamField body="continue_clip_id" type="string">
  Devam edilecek önceki klibin kimliği (ID).
</ParamField>

<ParamField body="continue_at" type="number">
  Devam edilecek zaman damgası (saniye cinsinden).
</ParamField>

<ParamField body="user" type="string">
  Son kullanıcı için benzersiz bir tanımlayıcı.
</ParamField>

## Yanıt

<ResponseField name="id" type="string">
  Durum sorgulama için görev kimliği (task ID).
</ResponseField>

<ResponseField name="status" type="string">
  Görev durumu: `pending`, `processing`, `completed` veya `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Görevin oluşturulma zamanına ait Unix zaman damgası.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "suno_music",
        "prompt": "An upbeat electronic dance track with heavy bass",
        "title":  "Night Drive",
        "tags":   "electronic, EDM, energetic",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/music/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/music/generations');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'suno_music',
        'prompt' => 'An upbeat electronic dance track with heavy bass',
        'title' => 'Night Drive',
        'tags' => 'electronic, EDM, energetic'
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo "Task ID: " . $data['id'];
```
