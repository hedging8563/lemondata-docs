---

## Yetkilendirme

<ParamField header="Authorization" type="string" required>
  API kimlik doğrulaması için Bearer token. Format: `Bearer sk-your-api-key`
</ParamField>

## Yol Parametreleri

<ParamField path="task_id" type="string" required>
  İlk görsel oluşturma isteğinden döndürülen görev ID'si.
</ParamField>

## Yanıt

<ResponseField name="created" type="integer">
  Oluşturma Unix zaman damgası.
</ResponseField>

<ResponseField name="task_id" type="string">
  Görev tanımlayıcısı.
</ResponseField>

<ResponseField name="status" type="string">
  Görev durumu: `pending`, `processing`, `completed` veya `failed`.
</ResponseField>

<ResponseField name="data" type="array">
  Oluşturulan görseller dizisi (`status` `completed` olduğunda doldurulur).

  Her nesne şunları içerir:
  - `url` (string): Oluşturulan görselin URL'si
  - `revised_prompt` (string): Oluşturma için kullanılan prompt
</ResponseField>

<ResponseField name="error" type="string">
  Hata mesajı (sadece `status` `failed` olduğunda bulunur).
</ResponseField>

## Sorgulama En İyi Uygulamaları

<Tip>
**Önerilen sorgulama aralığı**: 3-5 saniye. Çoğu görsel oluşturma görevi modele bağlı olarak 30-120 saniye içinde tamamlanır.
</Tip>

```python
import requests
import time

def poll_image_task(task_id, api_key, max_wait=300, interval=3):
    """Zaman aşımı ile görsel oluşturma sonucunu sorgula."""
    url = f"https://api.lemondata.cc/v1/images/generations/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        if data["status"] == "completed":
            return data["data"][0]["url"]
        elif data["status"] == "failed":
            raise Exception(data.get("error", "Generation failed"))

        time.sleep(interval)

    raise TimeoutError(f"Task {task_id} did not complete within {max_wait}s")

# Kullanım
image_url = poll_image_task("img_abc123def456", "sk-your-api-key")
print(f"Generated image: {image_url}")
```
