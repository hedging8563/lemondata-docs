## Yetkilendirme

<ParamField header="Authorization" type="string" required>
  API kimlik doğrulaması için Bearer token. Format: `Bearer sk-your-api-key`
</ParamField>

## İstek Gövdesi

<ParamField body="model" type="string" default="tts-1">
  TTS modeli: `tts-1` (standart) veya `tts-1-hd` (daha yüksek kalite).
</ParamField>

<ParamField body="input" type="string" required>
  Ses oluşturulacak metin. Maksimum 4096 karakter.
</ParamField>

<ParamField body="voice" type="string" required>
  Kullanılacak ses: `alloy`, `ash`, `ballad`, `coral`, `echo`, `fable`, `nova`, `onyx`, `sage`, `shimmer`, `verse`.
</ParamField>

<ParamField body="response_format" type="string" default="mp3">
  Ses formatı: `mp3`, `opus`, `aac`, `flac`, `wav`, `pcm`.
</ParamField>

<ParamField body="speed" type="number" default="1.0">
  Ses hızı (0.25 ile 4.0 arası).
</ParamField>

## Yanıt

Ses dosyasını istenen formatta döndürür.

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1-hd",
    "voice": "nova",
    "input": "Hello, welcome to LemonData!"
  }' \
  --output speech.mp3
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.audio.speech.create(
    model="tts-1-hd",
    voice="nova",
    input="Hello, welcome to LemonData!"
)

response.stream_to_file("speech.mp3")
```

```javascript JavaScript
import OpenAI from 'openai';
import fs from 'fs';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.lemondata.cc/v1'
});

const response = await client.audio.speech.create({
  model: 'tts-1-hd',
  voice: 'nova',
  input: 'Hello, welcome to LemonData!'
});

const buffer = Buffer.from(await response.arrayBuffer());
fs.writeFileSync('speech.mp3', buffer);
```

```go Go
package main

import (
    "context"
    "io"
    "os"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateSpeech(
        context.Background(),
        openai.CreateSpeechRequest{
            Model: openai.TTSModel1HD,
            Voice: openai.VoiceNova,
            Input: "Hello, welcome to LemonData!",
        },
    )
    if err != nil {
        panic(err)
    }
    defer resp.Close()

    file, _ := os.Create("speech.mp3")
    defer file.Close()
    io.Copy(file, resp)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/speech');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'tts-1-hd',
        'voice' => 'nova',
        'input' => 'Hello, welcome to LemonData!'
    ])
]);

$audio = curl_exec($ch);
curl_close($ch);

file_put_contents('speech.mp3', $audio);
```

## Ses Örnekleri

| Ses | Açıklama |
|-------|-------------|
| `alloy` | Nötr, dengeli |
| `ash` | Sakin, ölçülü |
| `ballad` | Melodik, etkileyici |
| `coral` | Sıcak, davetkar |
| `echo` | Sıcak, konuşkan |
| `fable` | Etkileyici, anlatısal |
| `nova` | Arkadaş canlısı, net |
| `onyx` | Derin, otoriter |
| `sage` | Bilge, düşünceli |
| `shimmer` | Yumuşak, nazik |
| `verse` | Dinamik, çok yönlü |
