---
title: "Embedding Oluştur"
api: "POST /v1/embeddings"
description: "Girdi metnini temsil eden bir embedding vektörü oluşturur"
---

## İstek Gövdesi

<ParamField body="model" type="string" required>
  Kullanılacak embedding modelinin ID'si (örneğin, `text-embedding-3-small`).
</ParamField>

<ParamField body="input" type="string | array" required>
  Embed edilecek girdi metni. Bir string veya string dizisi (array) olabilir.
</ParamField>

<ParamField body="encoding_format" type="string" default="float">
  Embedding'ler için format: `float` veya `base64`.
</ParamField>

<ParamField body="dimensions" type="integer">
  Çıktı için boyut sayısı (modele özgü).
</ParamField>

<ParamField body="user" type="string">
  Kötüye kullanım izleme için son kullanıcınızı temsil eden benzersiz bir tanımlayıcı.
</ParamField>

## Mevcut Modeller

| Model | Boyutlar | Açıklama |
|-------|------------|-------------|
| `text-embedding-3-large` | 3072 | En iyi kalite |
| `text-embedding-3-small` | 1536 | Dengeli |
| `text-embedding-ada-002` | 1536 | Eski sürüm |

## Yanıt

<ResponseField name="object" type="string">
  Her zaman `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Embedding nesneleri dizisi.

  Her nesne şunları içerir:
  - `object` (string): `embedding`
  - `index` (integer): Girdi dizisindeki dizin
  - `embedding` (array): Embedding vektörü
</ResponseField>

<ResponseField name="model" type="string">
  Kullanılan model.
</ResponseField>

<ResponseField name="usage" type="object">
  `prompt_tokens` ve `total_tokens` ile token kullanımı.
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.embeddings.create(
    model="text-embedding-3-small",
    input="The quick brown fox jumps over the lazy dog"
)

embedding = response.data[0].embedding
print(f"Embedding dimension: {len(embedding)}")
print(f"First 5 values: {embedding[:5]}")
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateEmbeddings(
        context.Background(),
        openai.EmbeddingRequest{
            Model: openai.SmallEmbedding3,
            Input: []string{"The quick brown fox jumps over the lazy dog"},
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Printf("Embedding dimension: %d\n", len(resp.Data[0].Embedding))
}
```

## Toplu Embedding'ler

```python
# Birden fazla metni aynı anda embed edin
response = client.embeddings.create(
    model="text-embedding-3-small",
    input=[
        "First document text",
        "Second document text",
        "Third document text"
    ]
)

for i, data in enumerate(response.data):
    print(f"Document {i}: {len(data.embedding)} dimensions")
```
