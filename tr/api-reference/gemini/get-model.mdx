---
title: "Modeli Getir"
openapi: "GET /v1beta/models/{model}"
description: "Google Gemini API formatını kullanarak model meta verilerini getirin"
---

Belirli bir model için Google Gemini API formatında meta verileri döndürür.

## Yol Parametreleri

<ParamField path="model" type="string" required>
  Model adı (örneğin, `gemini-2.5-pro`, `gemini-2.5-flash`). Takma adlar da kabul edilir.
</ParamField>

## Kimlik Doğrulama

İsteğe bağlıdır. Diğer Gemini uç noktaları ile aynı kimlik doğrulama yöntemlerini destekler:
- `?key=YOUR_API_KEY` sorgu parametresi
- `x-goog-api-key: YOUR_API_KEY` başlığı
- `Authorization: Bearer YOUR_API_KEY` başlığı

## Yanıt

<ResponseField name="name" type="string">
  `models/{model}` formatındaki model kaynak adı.
</ResponseField>

<ResponseField name="displayName" type="string">
  İnsan tarafından okunabilir model adı.
</ResponseField>

<ResponseField name="inputTokenLimit" type="integer">
  Maksimum giriş token sınırı (bağlam penceresi).
</ResponseField>

<ResponseField name="outputTokenLimit" type="integer">
  Maksimum çıkış token sınırı.
</ResponseField>

<ResponseField name="supportedGenerationMethods" type="array">
  Desteklenen oluşturma yöntemlerinin listesi (örneğin, `generateContent`, `countTokens`, `embedContent`).
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.lemondata.cc/v1beta/models/gemini-2.5-pro"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.lemondata.cc"}
)

model = genai.get_model("models/gemini-2.5-pro")
print(f"Name: {model.name}")
print(f"Input limit: {model.input_token_limit}")
print(f"Output limit: {model.output_token_limit}")
```
</RequestExample>
