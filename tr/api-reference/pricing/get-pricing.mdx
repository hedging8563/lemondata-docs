## Sorgu Parametreleri

<ParamField query="model" type="string">
  Fiyatlandırması alınacak model ID'si. Belirtilmezse tüm modeller için fiyatlandırmayı döndürür.
</ParamField>

## Yanıt

<ResponseField name="object" type="string">
  Her zaman `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Model fiyatlandırma nesneleri dizisi.

  Her nesne şunları içerir:
  - `model` (string): Model tanımlayıcı
  - `pricing` (object): Fiyatlandırma bilgisi
    - `input_per_1m` (string | null): 1M giriş token başına USD
    - `output_per_1m` (string | null): 1M çıkış token başına USD
    - `per_request` (string | null): İstek başına USD (sabit fiyatlı modeller için)
    - `is_lock_price` (boolean): Fiyatlandırmanın istek başına sabit olup olmadığı
    - `currency` (string): Her zaman `USD`
  - `provider` (string): Model sağlayıcısı
</ResponseField>

```go Go
package main

import (

)

func main() {
```

## Fiyatlandırma Türleri

| Tür | Açıklama | Örnek Modeller |
|------|-------------|----------------|
| **Token tabanlı** | Giriş/çıkış token'ları başına ücretlendirilir | GPT-4o, Claude, Gemini |
| **İstek başına** | İstek başına sabit fiyat | DALL-E 3, Sora, Suno |
