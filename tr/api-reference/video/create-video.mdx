## Genel Bakış

Video oluşturma işlemi asenkrondur. Bir istek gönderirsiniz ve bir görev kimliği (`task_id`) alırsınız, ardından sonuç için sorgulama (polling) yaparsınız.

## İstek Gövdesi (Request Body)

<ParamField body="model" type="string" default="kling-video">
  Video modeli (örneğin, `kling-video`, `veo3.1`, `wan2.6-i2v`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Oluşturulacak videonun metin açıklaması.
</ParamField>

<ParamField body="image_url" type="string">
  Görselden videoya (image-to-video) oluşturma için isteğe bağlı başlangıç görseli.
</ParamField>

<ParamField body="duration" type="integer">
  Saniye cinsinden video süresi (modele bağlıdır).
</ParamField>

<ParamField body="aspect_ratio" type="string">
  En boy oranı (örneğin, `16:9`, `9:16`, `1:1`).
</ParamField>

<ParamField body="resolution" type="string">
  Video çözünürlüğü (örneğin, `1080p`, `720p`).
</ParamField>

<ParamField body="fps" type="integer">
  Saniye başına kare sayısı (1-120).
</ParamField>

<ParamField body="negative_prompt" type="string">
  Video oluşturma sırasında nelerin önleneceği.
</ParamField>

<ParamField body="seed" type="integer">
  Tekrarlanabilir oluşturma için rastgele tohum (seed).
</ParamField>

<ParamField body="cfg_scale" type="number">
  Classifier-free guidance ölçeği (0-20).
</ParamField>

<ParamField body="motion_strength" type="number">
  Hareket yoğunluğu (0-1).
</ParamField>

<ParamField body="start_image" type="string">
  Başlangıç karesi görselinin URL'si.
</ParamField>

<ParamField body="end_image" type="string">
  Bitiş karesi görselinin URL'si.
</ParamField>

<ParamField body="user" type="string">
  Son kullanıcı için benzersiz bir tanımlayıcı.
</ParamField>

## Yanıt (Response)

<ResponseField name="task_id" type="string">
  Sorgulama için benzersiz görev tanımlayıcı.
</ResponseField>

<ResponseField name="status" type="string">
  Başlangıç durumu: `pending`.
</ResponseField>

<ResponseField name="model" type="string">
  Kullanılan model.
</ResponseField>


```python Python
import requests

response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json"
    },
    json={
        "model": "kling-video",
        "prompt": "A cat walking through a garden, cinematic lighting",
        "duration": 5
    }
)

data = response.json()
task_id = data["task_id"]
print(f"Task ID: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

## Mevcut Modeller

| Model | Tür | Açıklama |
|-------|------|-------------|
| `kling-video` | T2V/I2V | Gerçekçi hareket, birden fazla sürüm (v1.0-v2.6) |
| `veo3.1` | T2V | Google'ın en yenisi |
| `veo3.1-pro` | T2V | Profesyonel kalite |
| `minimax/video-01` | T2V | Yüksek ritim ve kararlılık |
| `wan2.6-i2v` | I2V | Görselden videoya |
