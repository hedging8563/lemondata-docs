## Genel Bakış

Google Gemini CLI, Gemini modelleriyle etkileşim kurmak için kullanılan bir komut satırı aracıdır. LemonData, Gemini CLI'yı kullanmanıza ve 300'den fazla modele erişmenize olanak tanıyan uyumlu bir uç nokta sağlar.

## Sistem Gereksinimleri

- **Node.js**: Versiyon 20.0+
- **İşletim Sistemi**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+ veya Debian 10+

## Kurulum

```bash
npm install -g @google/gemini-cli
```

Kurulumu doğrulayın:

```bash
gemini --version
```

## Yapılandırma

### Adım 1: API Key'inizi Alın

1. [LemonData Dashboard](https://lemondata.cc/dashboard) adresine giriş yapın
2. [API Keys](https://lemondata.cc/dashboard/api) bölümüne gidin
3. API Key'inizi oluşturun ve kopyalayın (biçim: `sk-...`)

### Adım 2: Ortam Değişkenlerini Ayarlayın

**Geçici (mevcut oturum):**

```bash
export GEMINI_API_KEY="sk-your-lemondata-key"
export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"
```

**Kalıcı yapılandırma:**

Shell yapılandırma dosyanıza ekleyin:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.lemondata.cc"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-lemondata-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.lemondata.cc"
    ```
  </Tab>
</Tabs>

<Warning>
Değişikliklerin geçerli olması için yapılandırmadan sonra terminali yeniden başlatın.
</Warning>

<Note>
**Bilinen Sınırlamalar**: Bazı durumlarda, önbelleğe alınmış bir Google oturumu varsa Gemini CLI `GOOGLE_GEMINI_BASE_URL` değişkenini dikkate almayabilir. Eğer bağlantı sorunları yaşıyorsanız, CLI önbelleğini temizlemeyi veya yeni bir terminal oturumu başlatmayı deneyin.
</Note>

## Temel Kullanım

Proje dizininizden Gemini CLI'yı başlatın:

```bash
cd your-project
gemini
```

İlk kez çalıştırdığınızda şunları yapmanız gerekir:
1. Bir tema seçin
2. Güvenlik bildirimini onaylayın
3. Çalışma dizinine güvenin

```bash

gemini
```

## Sorun Giderme

<AccordionGroup>
  <Accordion title="Bağlantı Hatası">
    - `GOOGLE_GEMINI_BASE_URL` değişkeninin `https://api.lemondata.cc` olarak ayarlandığını doğrulayın
    - Not: Gemini uç noktaları `/v1` son ekine ihtiyaç duymaz
    - İnternet bağlantısını kontrol edin
  </Accordion>

  <Accordion title="Kimlik Doğrulama Başarısız">
    - `GEMINI_API_KEY` ortam değişkeninin ayarlandığını doğrulayın

  </Accordion>

  <Accordion title="Model Mevcut Değil">
    - [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models) adresinden Gemini modellerinin kullanılabilirliğini kontrol edin
