## Genel Bakış

Dify, açık kaynaklı bir LLM uygulama geliştirme platformudur. Görsel Prompt düzenleme, RAG işlem hatları (pipelines), Agent çerçeveleri ve LLMOps yetenekleri sunar. LemonData, Dify içinde özel bir model sağlayıcısı olarak yapılandırılabilir.

## Avantajlar

- Tek bir arayüz üzerinden 300'den fazla AI modeline erişim
- Uygulama mantığını değiştirmeden modeller arası geçiş yapma
- Maliyetleri optimize etmek için her görev için en iyi modeli seçme
- Birleşik faturalandırma ve kullanım takibi

## Ön Koşullar

- API erişimine sahip bir LemonData hesabı
- Dify kurulumu (Bulut sürümü veya self-hosted)

## Yapılandırma Adımları

### Adım 1: API Anahtarınızı Alın

1. [LemonData Dashboard](https://lemondata.cc/dashboard) adresinde oturum açın
2. [API Keys](https://lemondata.cc/dashboard/api) bölümüne gidin
3. API anahtarınızı oluşturun ve kopyalayın (format: `sk-...`)

### Adım 2: Özel Model Sağlayıcısı Ekleyin

<Steps>
  <Step title="Ayarları Açın">
    Dify içinde **Settings** → **Model Provider** bölümüne gidin
  </Step>
  <Step title="OpenAI Uyumlu Sağlayıcı Ekleyin">
    **Add Model Provider** düğmesine tıklayın ve **OpenAI-API-compatible** seçeneğini seçin
  </Step>
  <Step title="Sağlayıcıyı Yapılandırın">
    Aşağıdaki ayarları girin:

    | Alan | Değer |
    |-------|-------|
    | Sağlayıcı Adı | LemonData |
    | API Key | `sk-your-lemondata-key` |
    | API Base URL | `https://api.lemondata.cc/v1` |
  </Step>
  <Step title="Modelleri Ekleyin">
    Kullanmak istediğiniz modelleri ekleyin:
    - `gpt-4o`
    - `gpt-4o-mini`
    - `claude-sonnet-4-5`
    - `claude-opus-4-5`
    - `gemini-2.5-flash`
    - `gemini-2.5-pro`
    - `deepseek-r1`
  </Step>
</Steps>

### Adım 3: Bağlantıyı Test Edin

1. Bir model seçin (örneğin `gpt-4o-mini`)
2. Bir test mesajı gönderin
3. Yanıt aldığınızı doğrulayın

## Uygulamalarda Kullanım

### Sohbet Robotu (Chatbot)

1. Yeni bir Chatbot uygulaması oluşturun
2. Model sağlayıcısı olarak LemonData'yı seçin
3. Tercih ettiğiniz modeli seçin
4. Sistem istemini (System Prompt) ve parametreleri yapılandırın

### Aracı (Agent)

1. Bir Agent uygulaması oluşturun
2. Güçlü bir model seçin (GPT-4o, Claude)
3. Araçlar ve bilgi tabanı ekleyin
4. Agent davranışını yapılandırın

### İş Akışı (Workflow)

1. Bir iş akışı oluşturun
2. Bir LLM düğümü (node) ekleyin
3. Her düğüm için bir LemonData modeli seçin
4. Düğümleri bağlayın ve veri akışını yapılandırın

## Kullanılabilir Modeller

| Kategori | Modeller |
|----------|--------|
| Sohbet (Chat) | GPT-4o, GPT-4o-mini, Claude Sonnet/Opus, Gemini, DeepSeek |
| Yerleştirmeler (Embeddings) | text-embedding-3-small, text-embedding-3-large |
| Görsel (Vision) | GPT-4o (resim destekli), Claude Sonnet (resim destekli) |

## RAG Yapılandırması

RAG uygulamaları için yerleştirme (embedding) modellerini yapılandırın:

1. **Settings** → **Model Provider** bölümüne gidin
2. Yerleştirme modelini ekleyin: `text-embedding-3-small`
3. Bilgi tabanı ayarlarında bunu varsayılan yerleştirme modeli olarak ayarlayın
