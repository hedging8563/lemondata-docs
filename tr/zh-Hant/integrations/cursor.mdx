## Genel Bakış

[Cursor](https://cursor.sh), AI destekli bir kod düzenleyicisidir. 300'den fazla modele erişmek için LemonData'yı özel bir API sağlayıcısı olarak kullanabilirsiniz.

## Yapılandırma

1. Cursor ayarlarını açın (`Cmd/Ctrl + ,`)
2. Sol kenar çubuğundaki **Models** bölümüne gidin
3. **OpenAI API Key** bölümüne aşağı kaydırın
4. LemonData API key'inizi girin
5. **Override OpenAI Base URL** seçeneğini etkinleştirin
6. Base URL'yi şu şekilde ayarlayın:

```
https://api.lemondata.cc/v1
```

## Model Seçimi

Yapılandırma tamamlandıktan sonra Cursor içinde herhangi bir LemonData modelini seçebilirsiniz:

### Önerilen Kodlama Modelleri

| Model | En Uygun Kullanım |
|-------|----------|
| `claude-opus-4-5` | Karmaşık kodlama, mimari tasarım |
| `claude-sonnet-4-5` | Genel kodlama, hata ayıklama |
| `gpt-4o` | Kod oluşturma, açıklama |
| `deepseek-r1` | Ağır akıl yürütme görevleri |
| `gpt-4o-mini` | Hızlı tamamlama, basit görevler |

## Özellikler

### Chat

Herhangi bir LemonData modeli ile AI sohbet panelini (`Cmd/Ctrl + L`) kullanın:

- Kod açıklaması
- Hata düzeltme
- Yeniden yapılandırma (refactoring) önerileri
- Dokümantasyon oluşturma

### Composer

Composer özelliği (`Cmd/Ctrl + I`), LemonData modelleri ile şunlar için kullanılabilir:

- Çoklu dosya düzenleme
- Özellik uygulama
- Kod taşıma (migration)

### Tab Tamamlama

Tab tamamlama için hızlı modelleri kullanmanızı öneririz:

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
