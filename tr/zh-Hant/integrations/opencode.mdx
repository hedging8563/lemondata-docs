## Genel Bakış

OpenCode, terminalde çalışan açık kaynaklı bir yapay zeka (AI) kodlama asistanıdır. Birden fazla LLM sağlayıcısını destekler ve 300'den fazla modele erişmek için LemonData API'sını kullanacak şekilde yapılandırılabilir.

## Kurulum

<Tabs>
  <Tab title="Kurulum Betiği">

    curl -fsSL https://opencode.ai/install | bash
    ```
  </Tab>
  <Tab title="Homebrew">

    ```

    Veya topluluk tap'ini kullanın (daha sık güncellenir):
    ```bash

    go install github.com/opencode-ai/opencode@latest
    ```
  </Tab>
</Tabs>

Kurulumu doğrulayın:

```bash
opencode --version
```

## Yapılandırma

### Adım 1: Ortam Değişkenlerini Ayarlayın

```bash
export OPENAI_API_KEY="sk-your-lemondata-key"
export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"
```

Kalıcı yapılandırma için bunları `~/.bashrc` veya `~/.zshrc` dosyanıza ekleyin:

```bash
echo 'export OPENAI_API_KEY="sk-your-lemondata-key"' >> ~/.zshrc
echo 'export LOCAL_ENDPOINT="https://api.lemondata.cc/v1"' >> ~/.zshrc
source ~/.zshrc
```

### Adım 2: OpenCode'u Yapılandırın

Projenizin kök dizininde `~/.config/opencode/opencode.json` (global) veya `opencode.json` dosyasını oluşturun veya düzenleyin:

```json
{
  "provider": {
    "openai": {
      "options": {
        "apiKey": "sk-your-lemondata-key",
        "baseURL": "https://api.lemondata.cc/v1"
      }
    }
  },
  "model": "gpt-4o",
  "agent": {
    "coder": {
      "model": "gpt-4o"
    }
  }
}
```

## Temel Kullanım

**Etkileşimli modu başlatın:**

```bash
opencode
```

**Bir istem (prompt) ile çalıştırın:**

```bash
opencode "Explain this codebase"
```

**Modeli belirtin:**

```bash
opencode --model gpt-4o "Fix the bugs in main.py"
```

## Kullanılabilir Modeller

| Model | En İyi Kullanım Alanı |
|-------|----------|
| `gpt-4o` | Karmaşık görevler, kod mimarisi |
| `gpt-4o-mini` | Hızlı düzeltmeler, basit sorgular |
| `claude-sonnet-4-5` | Kod incelemesi, dokümantasyon yazımı |
| `claude-opus-4-5` | Karmaşık akıl yürütme |
| `gemini-2.5-flash` | Hızlı yanıt |
| `deepseek-r1` | Algoritma tasarımı |

## Yaygın Komutlar

**Kodu analiz edin:**

```bash
opencode "What does this function do?" < src/utils.ts
```

**Kod oluşturun:**

```bash
opencode "Create a REST API with Express"
```

**Değişiklikleri inceleyin:**

```bash
git diff | opencode "Review these changes"
```

**Hataları düzeltin:**

```bash
opencode "Fix the TypeScript errors in this project"
```

## Etkileşimli Komutlar

| Komut | Açıklama |
|---------|-------------|
| `/help` | Kullanılabilir komutları gösterir |
| `/model <name>` | Farklı bir modele geçer |
| `/clear` | Sohbet geçmişini temizler |
| `/exit` | OpenCode'dan çıkar |

## Sorun Giderme

<AccordionGroup>
  <Accordion title="Bağlantı Hatası">
    - `LOCAL_ENDPOINT` değişkeninin `https://api.lemondata.cc/v1` olarak ayarlandığını doğrulayın
    - İnternet bağlantısını kontrol edin
    - Test etmek için `curl https://api.lemondata.cc/v1/models` komutunu deneyin
  </Accordion>

  <Accordion title="Kimlik Doğrulama Başarısız">
    - `OPENAI_API_KEY` ortam değişkeninin ayarlandığını doğrulayın
    - Anahtarın `sk-` ile başladığını kontrol edin
    - Anahtarın LemonData kontrol panelinde etkin olduğundan emin olun
  </Accordion>

  <Accordion title="Model Bulunamadı">
    - Yapılandırmadaki model adının tam olarak eşleştiğini kontrol edin
    - [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models) adresinden modelin kullanılabilirliğini doğrulayın
  </Accordion>
</AccordionGroup>
