## Genel Bakış

[Claude Code](https://docs.anthropic.com/en/docs/claude-code), Anthropic tarafından resmi olarak yayınlanan AI destekli bir programlama CLI aracıdır. LemonData'yı bir API sağlayıcısı olarak kullanacak şekilde yapılandırabilirsiniz.

## Yapılandırma

### Ortam Değişkenleri

Aşağıdaki ortam değişkenlerini ayarlayın:

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

Bunları kalıcı hale getirmek için shell profilinize (`~/.bashrc`, `~/.zshrc` vb.) ekleyin.

### Yapılandırma Dosyası

Alternatif olarak, `~/.claude/settings.json` dosyasını oluşturun veya düzenleyin:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## Kullanım

Yapılandırma tamamlandıktan sonra Claude Code'u normal şekilde kullanabilirsiniz:

```bash
# Start an interactive session
claude

# Run a single command
claude "Explain this code" < main.py

# Chat mode
claude chat
```

## Mevcut Modeller

LemonData tüm Claude modellerini destekler:

| Model | Açıklama |
|-------|-------------|
| `claude-opus-4-5` | Karmaşık akıl yürütme için en güçlü model |
| `claude-sonnet-4-5` | Dengeli performans, programlama için ideal |
| `claude-haiku-4-5` | En hızlı ve maliyet etkin |

## Model Seçimi

Bir model belirtmek için `--model` bayrağını kullanın:

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

Veya yapılandırmanızda varsayılan bir değer ayarlayın:

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## Genişletilmiş Düşünme (Extended Thinking)

Karmaşık akıl yürütme görevleri için genişletilmiş düşünme özelliklerine sahip bir model kullanın:

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```
