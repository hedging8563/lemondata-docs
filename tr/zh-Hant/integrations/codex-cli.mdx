## Genel Bakış

OpenAI Codex, terminalde kod okuyabilen, değiştirebilen ve yürütebilen hafif bir kodlama aracısı (coding agent) olarak hizmet veren açık kaynaklı bir komut satırı aracıdır (CLI). GPT modelleri üzerine inşa edilmiştir ve kod üretimi için optimize edilmiştir.

## Sistem Gereksinimleri

- **İşletim Sistemi**: macOS, Linux (resmi destek), WSL üzerinden Windows
- **Node.js**: Sürüm 18+
- **npm**: Sürüm 10.x.x veya daha yüksek

## Kurulum

```bash
sudo npm install -g @openai/codex@latest
```

Kurulumu doğrulayın:

```bash
codex --version
```

## Yapılandırma

### Adım 1: API Key Ayarlama

**Geçici (mevcut oturum):**

```bash
export OPENAI_API_KEY="sk-your-lemondata-key"
```

**Kalıcı yapılandırma:**

Şu dosyalara ekleyin: `~/.bashrc`, `~/.zshrc` veya `~/.bash_profile`:

```bash
export OPENAI_API_KEY="sk-your-lemondata-key"
```

Ardından yeniden yükleyin:

```bash
source ~/.zshrc  # veya source ~/.bashrc
```

### Adım 2: config.toml Yapılandırması

`~/.codex/config.toml` dosyasını düzenleyin:

```toml
model = "gpt-4o"
model_provider = "lemondata"

[model_providers.lemondata]
name = "LemonData"
base_url = "https://api.lemondata.cc/v1"
env_key = "OPENAI_API_KEY"
wire_api = "chat"
```

<Note>
Eğer yapılandırma dosyası mevcut değilse, dosyayı oluşturmak için bir kez `codex` komutunu çalıştırın ve ardından düzenleme yapın.
</Note>

## Temel Kullanım

**Etkileşimli modu başlatın:**

```bash
codex
```

**Doğrudan komut:**

```bash
codex "Fix the bug in main.py line 42"
```

**Model belirtin:**

```bash
codex -m gpt-4o "Build a REST API server"
```

## Mevcut Modeller

| Model | Kullanım Senaryoları |
|-------|----------|
| `gpt-4o` | Karmaşık kodlama görevleri, mimari tasarım |
| `gpt-4o-mini` | Hızlı düzeltmeler, basit görevler |
| `claude-sonnet-4-5` | Kod incelemesi, dokümantasyon yazımı |
| `deepseek-r1` | Algoritma tasarımı, akıl yürütme |

## Etkileşimli Komutlar

| Komut | Açıklama |
|---------|-------------|
| `/help` | Yardımı göster |
| `/exit` veya `Ctrl+C` | Çıkış |
| `/clear` | Sohbeti temizle |
| `/config` | Yapılandırmayı görüntüle |
| `/model <name>` | Modeli değiştir |
| `/tokens` | Token kullanımını görüntüle |

## Yapılandırmayı Doğrulama

```bash
# Çevre değişkenlerini kontrol edin
echo $OPENAI_API_KEY

# API bağlantısını test edin
codex "Hello, Codex!"

# Yapılandırmayı görüntüle
cat ~/.codex/config.toml
```

## Yaygın Kullanım Durumları

**Kod incelemesi:**

```bash
git diff | codex "Review these code changes"
```

**Commit mesajı oluşturma:**

```bash
git diff --staged | codex "Generate a commit message for these changes"
```

**Hataları düzeltme:**

```bash
codex "Fix the TypeScript errors in src/components/"
```

**Kodu açıklama:**

```bash
cat main.py | codex "Explain what this code does"
```
