## Genel Bakış

LemonData, adil kullanım ve platform kararlılığını sağlamak için hız sınırları uygular. Sınırlar hesap seviyesine göre değişiklik gösterir.

## Hız Sınırı Seviyeleri

| Seviye | İstek/Dakika | Açıklama |
|------|-------------|-------------|
| **User** | 1,000 | Tüm hesaplar için varsayılan seviye |
| **Partner** | 3,000 | Entegrasyon ortakları için |
| **VIP** | 10,000 | Yüksek hacimli kullanıcılar |

<Note>
  Hız sınırları değişikliğe tabidir. Özel sınırlar için lütfen support@lemondata.cc ile iletişime geçin.
</Note>

## Hız Sınırı Başlıkları

Her API yanıtı hız sınırı bilgilerini içerir:

```
X-RateLimit-Limit: 60          # Dakika başına sınırınız
X-RateLimit-Remaining: 55      # Kalan istek sayısı
X-RateLimit-Reset: 1234567890  # Sınırın sıfırlanacağı Unix zaman damgası
```

## Hız Sınırı Aşıldı

Sınırı aştığınızda bir `429` yanıtı alırsınız:

```json
{
  "error": {
    "message": "Rate limit exceeded. Please slow down.",
    "type": "rate_limit_exceeded"
  }
}
```

Ek başlık ile birlikte:
```
Retry-After: 60  # Yeniden denemeden önce beklenecek saniye
```

## Hız Sınırlarını Yönetme

### Üstel Geri Çekilme (Exponential Backoff)

Otomatik yeniden denemeler için üstel geri çekilme uygulayın:

```python
import time
from openai import OpenAI, RateLimitError

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

def make_request_with_backoff(messages, max_retries=5):
    for attempt in range(max_retries):
        try:
            return client.chat.completions.create(
                model="gpt-4o",
                messages=messages
            )
        except RateLimitError as e:
            if attempt == max_retries - 1:
                raise

            wait_time = 2 ** attempt  # 1, 2, 4, 8, 16 saniye
            print(f"Hız sınırı aşıldı. {wait_time}s bekleniyor...")
            time.sleep(wait_time)
```

### İstek Kuyruğa Alma

Yüksek hacimli uygulamalar için istek kuyruğa alma uygulayın:

```python
import asyncio
from collections import deque

class RateLimitedClient:
    def __init__(self, requests_per_minute=60):
        self.rpm = requests_per_minute
        self.interval = 60 / requests_per_minute
        self.last_request = 0

    async def request(self, messages):
        # Hız sınırına uymak için gerekirse bekle
        now = asyncio.get_event_loop().time()
        wait_time = max(0, self.last_request + self.interval - now)
        if wait_time > 0:
            await asyncio.sleep(wait_time)

        self.last_request = asyncio.get_event_loop().time()
        return await self.client.chat.completions.create(
            model="gpt-4o",
            messages=messages
        )
```

### Toplu İşleme (Batch Processing)

Büyük ölçekli işlemler için gecikmeli toplu işleme kullanın:

```python
def process_batch(items, batch_size=50, delay=1):
    results = []
    for i in range(0, len(items), batch_size):
        batch = items[i:i + batch_size]
        for item in batch:
            result = client.chat.completions.create(
                model="gpt-4o",
                messages=[{"role": "user", "content": item}]
            )
            results.append(result)
        time.sleep(delay)  # Gruplar arasında duraklat
    return results
```
