## Genel Bakış

Bu uç nokta, yerel Anthropic Messages API uyumluluğu sağlar. Genişletilmiş düşünme (extended thinking) gibi özelliklere sahip Claude modelleri için uygundur.

<Note>
Anthropic SDK için Base URL: `https://api.lemondata.cc` (`/v1` son eki olmadan)
</Note>

## İstek Başlıkları

<ParamField header="x-api-key" type="string" required>
  LemonData API anahtarınız. Bearer token alternatifi.
</ParamField>

<ParamField header="anthropic-version" type="string" required>
  Anthropic API sürümü. Lütfen `2023-06-01` kullanın.
</ParamField>

## İstek Gövdesi

<ParamField body="model" type="string" required>
  Claude model ID'si (örneğin `claude-sonnet-4-5`).
</ParamField>

<ParamField body="messages" type="array" required>
  `role` ve `content` içeren mesaj nesneleri dizisi.
</ParamField>

<ParamField body="max_tokens" type="integer" required>
  Oluşturulacak maksimum token sayısı.
</ParamField>

<ParamField body="system" type="string">
  Sistem istemi (`messages` dizisinden ayrı).
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Örnekleme sıcaklığı (0-1).
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Akış (streaming) yanıtlarını etkinleştirin.
</ParamField>

<ParamField body="thinking" type="object">
  Genişletilmiş düşünme yapılandırması (Claude Opus 4.5).

  - `type` (string): Etkinleştirmek için `"enabled"` olarak ayarlayın
  - `budget_tokens` (integer): Düşünme için token bütçesi
</ParamField>

<ParamField body="tools" type="array">
  Modelin kullanabileceği araçlar.
</ParamField>

<ParamField body="tool_choice" type="object">
  Modelin araçları nasıl kullanması gerektiği. Seçenekler: `auto`, `any`, `tool` (belirli bir araç).
</ParamField>

<ParamField body="top_p" type="number">
  Çekirdek örnekleme (Nucleus sampling) parametresi. Lütfen `temperature` veya `top_p` parametrelerinden yalnızca birini kullanın, ikisini birden kullanmayın.
</ParamField>

<ParamField body="top_k" type="integer">
  Her token için yalnızca en iyi K seçenek arasından örnekleme yapın.
</ParamField>

<ParamField body="stop_sequences" type="array">
  Modelin oluşturmayı durdurmasına neden olacak özel durdurma dizileri.
</ParamField>

<ParamField body="metadata" type="object">
  İsteğe eklenen meta veriler.
</ParamField>
