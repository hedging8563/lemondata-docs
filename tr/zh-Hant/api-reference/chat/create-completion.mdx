---
title: "Sohbet Tamamlama Oluştur (Create Chat Completion)"

---

## İstek Gövdesi (Request Body)

<ParamField body="model" type="string" required>
  Kullanılacak model ID'si. Mevcut seçenekleri görmek için lütfen [Models](https://lemondata.cc/zh-TW/models) sayfasına bakın.
</ParamField>

<ParamField body="messages" type="array" required>
  Sohbeti oluşturan mesajların listesi.

  Her mesaj nesnesi şunları içerir:
  - `role` (string): `system`, `user` veya `assistant`
  - `content` (string | array): Mesaj içeriği
</ParamField>

<ParamField body="temperature" type="number" default="1">
  0 ile 2 arasında bir örnekleme sıcaklığı. Daha yüksek değerler çıktıyı daha rastgele hale getirir.
</ParamField>

<ParamField body="max_tokens" type="integer">
  Oluşturulacak maksimum token sayısı.
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Eğer true ise, kısmi mesaj artışları SSE olayları olarak gönderilir.
</ParamField>

<ParamField body="stream_options" type="object">
  Akış seçenekleri. Akış bloklarında token kullanımını almak için `include_usage: true` olarak ayarlayın.
</ParamField>

<ParamField body="top_p" type="number" default="1">
  Çekirdek örnekleme (Nucleus sampling) parametresi. Her ikisini birden değil, bu parametreyi veya temperature değerini ayarlamanızı öneririz.
</ParamField>

<ParamField body="frequency_penalty" type="number" default="0">
  -2.0 ile 2.0 arasında bir değer. Pozitif değerler tekrarlanan token'ları cezalandırır.
</ParamField>

<ParamField body="presence_penalty" type="number" default="0">
  -2.0 ile 2.0 arasında bir değer. Pozitif değerler metinde zaten bulunan token'ları cezalandırır.
</ParamField>

<ParamField body="stop" type="string | array">
  API'nin token oluşturmayı durduracağı en fazla 4 dizi.
</ParamField>

<ParamField body="tools" type="array">
  Modelin çağırabileceği araçların listesi (function calling).
</ParamField>

<ParamField body="tool_choice" type="string | object">
  Modelin araçları nasıl kullanacağını kontrol eder. Seçenekler: `auto`, `none`, `required` veya belirli bir araç nesnesi.
</ParamField>

<ParamField body="parallel_tool_calls" type="boolean" default="true">
  Paralel fonksiyon çağrılarının etkinleştirilip etkinleştirilmeyeceği. Sıralı çağırmak için false olarak ayarlayın.
</ParamField>

<ParamField body="max_completion_tokens" type="integer">
  Tamamlama için maksimum token sayısı. `max_tokens` için bir alternatif olup o1/o3 gibi daha yeni modeller için önerilir.
</ParamField>

<ParamField body="reasoning_effort" type="string">
  o1/o3 modelleri için akıl yürütme (reasoning) çabası. Seçenekler: `low`, `medium`, `high`.
</ParamField>

<ParamField body="seed" type="integer">
  Belirleyici örnekleme için kullanılan rastgele tohum (seed).
</ParamField>

<ParamField body="n" type="integer" default="1">
  Oluşturulacak tamamlama sayısı (1-128).
</ParamField>

<ParamField body="logprobs" type="boolean">
  Log olasılıklarının (log probabilities) döndürülüp döndürülmeyeceği.
</ParamField>

<ParamField body="top_logprobs" type="integer">
  Döndürülecek en yüksek log olasılıklarının sayısı (0-20). `logprobs: true` ayarını gerektirir.
</ParamField>

<ParamField body="top_k" type="integer">
  Top-K örnekleme parametresi (Anthropic/Gemini modelleri için geçerlidir).
</ParamField>

<ParamField body="response_format" type="object">
  Yanıt formatı belirtimi. JSON modunu açmak için `{"type": "json_object"}` kullanın veya yapılandırılmış çıktı için `{"type": "json_schema", "json_schema": {...}}` kullanın.
</ParamField>

<ParamField body="logit_bias" type="object">
  Belirli token'ların görünme olasılığını değiştirir. Token ID'lerini (dize formatında) -100 ile 100 arasındaki sapma değerlerine eşler.
</ParamField>

<ParamField body="user" type="string">
  Kötüye kullanım izleme için son kullanıcınızı temsil eden benzersiz bir tanımlayıcı.
</ParamField>

<ParamField body="cache_control" type="object">
  LemonData önbellek (cache) kontrol seçenekleri.

  - `type` (string): Önbellek stratejisi - `default`, `no_cache`, `no_store`, `response_only`, `semantic_only`
  - `max_age` (integer): Saniye cinsinden önbellek yaşam süresi (TTL) (maksimum 86400)
</ParamField>

## Yanıt (Response)

<ResponseField name="id" type="string">
  Tamamlamanın benzersiz tanımlayıcısı.
</ResponseField>

<ResponseField name="object" type="string">
  Her zaman `chat.completion` değeridir.
</ResponseField>

<ResponseField name="created" type="integer">
  Tamamlamanın oluşturulduğu Unix zaman damgası.
</ResponseField>

<ResponseField name="model" type="string">
  Tamamlama için kullanılan model.
</ResponseField>

<ResponseField name="choices" type="array">
  Tamamlama seçeneklerinin listesi.

  Her seçenek şunları içerir:
  - `index` (integer): Seçenek dizini
  - `message` (object): Oluşturulan mesaj
  - `finish_reason` (string): Modelin durma nedeni (`stop`, `length`, `tool_calls`)
</ResponseField>

<ResponseField name="usage" type="object">
  Token kullanım istatistikleri.

  - `prompt_tokens` (integer): İstemdeki (prompt) token sayısı
  - `completion_tokens` (integer): Tamamlamadaki token sayısı
  - `total_tokens` (integer): Kullanılan toplam token sayısı
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"}
    ],
    temperature=0.7,
    max_tokens=1000
)

print(response.choices[0].message.content)
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateChatCompletion(
        context.Background(),
        openai.ChatCompletionRequest{
            Model: "gpt-4o",
            Messages: []openai.ChatCompletionMessage{
                {Role: openai.ChatMessageRoleSystem, Content: "You are a helpful assistant."},
                {Role: openai.ChatMessageRoleUser, Content: "Hello!"},
            },
            Temperature: 0.7,
            MaxTokens:   1000,
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Choices[0].Message.Content)
}
```
