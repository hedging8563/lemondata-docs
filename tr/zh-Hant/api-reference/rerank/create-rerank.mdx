---
title: "Dokümanları Yeniden Sırala"
openapi: "POST /v1/rerank"
description: "Dokümanları bir sorgu ile olan ilgilerine göre yeniden sıralayın"
---

Dokümanları yeniden sıralamak için anlamsal benzerlik modellerini kullanın. Arama sonuçlarını ve RAG uygulamalarını optimize etmek için uygundur.

## İstek Gövdesi

<ParamField body="model" type="string" required>
  Kullanılacak yeniden sıralama modeli ID'si (örneğin: `BAAI/bge-reranker-v2-m3`, `qwen3-rerank`).
</ParamField>

<ParamField body="query" type="string" required>
  Dokümanları sıralamak için kullanılan sorgu ifadesi.
</ParamField>

<ParamField body="documents" type="array" required>
  Yeniden sıralanacak dokümanların listesi (dizeler).
</ParamField>

<ParamField body="top_n" type="integer">
  Döndürülecek en iyi sonuçların sayısı. Varsayılan olarak tüm dokümanlardır.
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  Yanıtın orijinal doküman metnini içerip içermeyeceği.
</ParamField>

## Yanıt

<ResponseField name="results" type="array">
  Puanlanmış ve sıralanmış doküman listesi.

  Her sonuç şunları içerir:
  - `index` (integer): Orijinal doküman dizini
  - `relevance_score` (number): İlgi puanı (0-1)
  - `document` (string): Orijinal metin (eğer `return_documents=true` ise)
</ResponseField>

<ResponseField name="model" type="string">
  Yeniden sıralama için kullanılan model.
</ResponseField>

<ResponseField name="usage" type="object">
  Token kullanım istatistikleri.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.lemondata.cc/v1/rerank" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "BAAI/bge-reranker-v2-m3",
    "query": "What is machine learning?",
    "documents": [
      "Machine learning is a subset of AI",
      "The weather is nice today",
      "Deep learning uses neural networks"
    ],
    "top_n": 2,
    "return_documents": true
  }'
```
