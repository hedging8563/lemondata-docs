## Genel Bakış

Desteklenen herhangi bir dildeki sesi İngilizce metne çevirin. Transkripsiyondan (transcription) farklı olarak, bu uç nokta giriş dili ne olursa olsun her zaman İngilizce metin çıktısı verir.

## İstek Gövdesi

<ParamField body="file" type="file" required>
  Çevrilecek ses dosyası. Desteklenen formatlar: `flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`. Maksimum dosya boyutu 25 MB'dır.
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  Kullanılacak model. Şu anda yalnızca `whisper-1` desteklenmektedir.
</ParamField>

<ParamField body="prompt" type="string">
  Modelin stilini yönlendirmek veya önceki bölümleri devam ettirmek için kullanılan isteğe bağlı metin. İngilizce kullanılmalıdır.
</ParamField>

<ParamField body="response_format" type="string" default="json">
  Çıktı formatı. Seçenekler: `json`, `text`, `srt`, `verbose_json`, `vtt`.
</ParamField>

<ParamField body="temperature" type="number">
  Örnekleme sıcaklığı, 0 ile 1 arasında. Daha yüksek değerler (örneğin 0.8) daha rastgele çıktılar üretirken, daha düşük değerler (örneğin 0.2) çıktıyı daha odaklanmış ve belirleyici hale getirir.
</ParamField>

## Yanıt

<ResponseField name="text" type="string">
  Çevrilmiş İngilizce metin.
</ResponseField>

`verbose_json` formatı için yanıt ayrıca şunları içerir:

<ResponseField name="language" type="string">
  Algılanan giriş sesi dili.
</ResponseField>

<ResponseField name="duration" type="number">
  Giriş sesinin süresi (saniye cinsinden).
</ResponseField>

<ResponseField name="segments" type="array">
  Zaman damgalı çevrilmiş metin segmentleri.
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.lemondata.cc/v1"
)

with open("german_audio.mp3", "rb") as audio_file:
    response = client.audio.translations.create(
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/audio/translations');

$file = new CURLFile('german_audio.mp3', 'audio/mpeg', 'german_audio.mp3');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => [
        'file' => $file,
        'model' => 'whisper-1'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo $data['text'];
```

## Çeviri vs Transkripsiyon

| Özellik | Çeviri | Transkripsiyon |
|---------|-------------|---------------|
| Çıktı Dili | Her zaman İngilizce | Giriş ile aynı |
| Kullanım Durumu | Yabancı dildeki sesi İngilizceye dönüştürme | Orijinal dili koruma |
| Dil Parametresi | Uygulanamaz | İsteğe bağlı ipucu |

<Note>
  Çeviri uç noktası kaynak dili otomatik olarak algılar ve İngilizceye çevirir. Transkripsiyondaki `language` parametresi yoksayılacaktır.
</Note>
