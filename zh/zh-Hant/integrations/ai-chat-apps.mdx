## 概览

LemonData 可与任何兼容 OpenAI 的聊天应用配合使用。本指南涵盖了热门桌面端和移动端聊天应用的配置。

## 支持的应用

| 应用 | 平台 | 功能 |
|-------------|----------|----------|
| LobeChat | 网页、桌面端、自托管 | 多供应商、插件、RAG |
| CherryStudio | Windows, macOS, Linux | 丰富的插件、多个供应商 |
| Chatbox | Windows, macOS, Linux | 轻量、快速 |
| BotGem | iOS, macOS | 原生 Apple 体验 |
| TypingMind | 网页、桌面端 | 自定义人格、插件 |
| OpenCat | iOS, macOS | 原生、离线支持 |
| ChatWise | Windows, macOS | 团队协作 |

## 配置步骤

### 步骤 1：获取您的 API Key

1. 登录 [LemonData 控制台](https://lemondata.cc/dashboard)
2. 导航至 [API Keys](https://lemondata.cc/dashboard/api)
3. 创建并复制您的 API Key（格式：`sk-...`）

### 步骤 2：配置您的应用

<Tabs>
  <Tab title="LobeChat">
    **针对自托管 LobeChat：**

    配置环境变量：
    ```bash
    OPENAI_API_KEY=sk-your-lemondata-key
    OPENAI_PROXY_URL=https://api.lemondata.cc/v1
    ```

    **针对 LobeChat Cloud：**
    1. 打开 **Settings** → **AI Service Provider**
    2. 选择 **OpenAI**
    3. 输入您的 LemonData API Key
    4. 配置自定义端点：`https://api.lemondata.cc/v1`
  </Tab>
  <Tab title="CherryStudio">
    1. 打开 **Settings** → **Model Provider**
    2. 点击 **Add Provider**
    3. 选择 **OpenAI API Compatible**
    4. 输入：
       - **Name**: LemonData
       - **API Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    5. 添加模型：`gpt-4o`, `claude-sonnet-4-5` 等
    6. 保存并测试
  </Tab>
  <Tab title="Chatbox">
    1. 打开 **Settings** → **AI Model**
    2. 选择 **OpenAI API Compatible**
    3. 输入：
       - **API Host**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
       - **Model**: `gpt-4o`
    4. 点击 **Verify** 以测试连接
  </Tab>
  <Tab title="BotGem">
    1. 打开 **Settings** → **Providers**
    2. 点击 **Add Custom Provider**
    3. 输入：
       - **Name**: LemonData
       - **Base URL**: `https://api.lemondata.cc/v1`
       - **API Key**: `sk-your-lemondata-key`
    4. 添加您偏好的模型
    5. 保存配置
  </Tab>
  <Tab title="TypingMind">
    1. 点击 **Settings**（齿轮图标）
    2. 选择 **Custom Endpoint**
    3. 输入：
       - **Endpoint URL**: `https://api.lemondata.cc/v1/chat/completions`
       - **API Key**: `sk-your-lemondata-key`
    4. 配置模型并保存
  </Tab>
</Tabs>

### 步骤 3：添加模型

配置供应商后，添加您想要使用的模型：

| 类别 | 模型 |
|----------|--------|
| OpenAI | `gpt-4o`, `gpt-4o-mini`, `o1`, `o3-mini` |
| Anthropic | `claude-sonnet-4-5`, `claude-opus-4-5` |
| Google | `gemini-2.5-flash`, `gemini-2.5-pro` |
| DeepSeek | `deepseek-r1`, `deepseek-chat` |

## 配置参考

| 字段 | 值 |
|-------|-------|
| Base URL | `https://api.lemondata.cc/v1` |
| API Key | `sk-your-lemondata-key` |
| API 类型 | OpenAI Compatible |

**JSON 配置（适用于支持的应用）：**

```json
{
  "provider": "LemonData",
  "baseURL": "https://api.lemondata.cc/v1",
  "apiKey": "sk-your-lemondata-key",
  "models": [
    "gpt-4o",
    "gpt-4o-mini",
    "claude-sonnet-4-5",
    "gemini-2.5-flash"
  ]
}
```

## 故障排除

<AccordionGroup>
  <Accordion title="连接错误">
    - 验证 Base URL 是否准确为 `https://api.lemondata.cc/v1`
    - 检查结尾是否有斜杠（如有请移除）
    - 验证网络连接和防火墙设置
  </Accordion>

  <Accordion title="401 Unauthorized">
    - 再次检查您的 API Key
    - 在 [LemonData 控制台](https://lemondata.cc/dashboard/api) 中验证 Key 是否处于启用状态
    - 确保 Key 中没有多余的空格
  </Accordion>

  <Accordion title="402 Insufficient Balance">
    - 在控制台中检查您的账户余额
    - 为您的账户充值
  </Accordion>

  <Accordion title="404 Model Not Available">
    - 验证模型名称拼写是否正确
    - 在 [lemondata.cc/en/models](https://lemondata.cc/zh-TW/models) 检查模型可用性
    - 尝试不同的模型
  </Accordion>

  <Accordion title="响应缓慢">
    - 尝试更快的模型（例如使用 `gpt-4o-mini` 代替 `gpt-4o`）
    - 如果可行，请减少 `max_tokens` 参数
    - 检查您的网络连接
  </Accordion>
</AccordionGroup>
