## 这项技能可以做什么？

当您想在代码中使用 AI 功能（如 GPT-4、图像生成、语音识别等）时，此技能将：

1. ✅ **自动搜索** LemonData 的数百个 API
2. ✅ **找到最适合**您需求的 API
3. ✅ **生成完整、可运行的代码**
4. ✅ **配置您的 API Key**
5. ✅ **提供使用示例**和最佳实践

**简而言之：只需用自然语言描述您的需求，即可获得完整的 API 集成代码！**

## 步骤 1：安装技能

<Tabs>
  <Tab title="npx (推荐)">
    最简单的安装方式：

    ```bash
    npx add-skill hedging8563/lemondata-api-skill -y
    ```

    这将自动为所有检测到的编程助手（Claude Code, Cursor, Copilot 等）安装此技能。
  </Tab>
  <Tab title="分享 GitHub URL">
    1. 在您的终端或 IDE 中打开 **Claude Code**
    2. 在对话中粘贴此链接：
    ```
    https://github.com/hedging8563/lemondata-api-skill
    ```
    3. Claude Code 会将其识别为技能仓库并协助您安装
  </Tab>
  <Tab title="Git Clone">
    **个人安装**（适用于所有项目）：
    ```bash
    git clone https://github.com/hedging8563/lemondata-api-skill.git ~/.claude/skills/lemondata-api-integration
    ```

    **项目特定安装**（通过 git 与团队共享）：
    ```bash
    git clone https://github.com/hedging8563/lemondata-api-skill.git .claude/skills/lemondata-api-integration
    ```
  </Tab>
</Tabs>

### 验证安装

询问 Claude Code：
```
What skills are available?
```

如果您看到 `lemondata-api-integration`，则表示安装成功！
