---

## 概览

将任何支持语言的音频翻译为英文文本。与转录（transcription）不同，无论输入语言为何，此端点始终输出英文文本。

## 请求主体

<ParamField body="file" type="file" required>
  要翻译的音频文件。支持的格式：`flac`, `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `ogg`, `wav`, `webm`。文件大小上限为 25 MB。
</ParamField>

<ParamField body="model" type="string" default="whisper-1">
  要使用的模型。目前仅支持 `whisper-1`。
</ParamField>

<ParamField body="prompt" type="string">
  选填文本，用于引导模型的风格或延续之前的片段。应使用英文。
</ParamField>

<ParamField body="response_format" type="string" default="json">
  输出格式。选项：`json`, `text`, `srt`, `verbose_json`, `vtt`。
</ParamField>

<ParamField body="temperature" type="number">
  采样温度，介于 0 与 1 之间。较高的值（如 0.8）会产生更随机的输出，而较低的值（如 0.2）则会使输出更集中且具确定性。
</ParamField>

## 响应

<ResponseField name="text" type="string">
  翻译后的英文文本。
</ResponseField>

对于 `verbose_json` 格式，响应还包含：

<ResponseField name="language" type="string">
  检测到的输入音频语言。
</ResponseField>

<ResponseField name="duration" type="number">
  输入音频的时长（以秒为单位）。
</ResponseField>

<ResponseField name="segments" type="array">
  带有时间戳的翻译文本片段。
</ResponseField>

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("sk-your-api-key")
    config.BaseURL = "https://api.lemondata.cc/v1"
    client := openai.NewClientWithConfig(config)

    resp, err := client.CreateTranslation(
        context.Background(),
        openai.AudioRequest{
            Model:    openai.Whisper1,
            FilePath: "german_audio.mp3",
        },
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Text)
}
```

## 翻译 vs 转录

| 特性 | 翻译 | 转录 |
|---------|-------------|---------------|
| 输出语言 | 始终为英文 | 与输入相同 |
| 使用场景 | 将外语音频转换为英文 | 保留原始语言 |
| 语言参数 | 不适用 | 选填提示 |

<Note>
  翻译端点会自动检测源语言并翻译为英文。转录中的 `language` 参数将被忽略。
</Note>
