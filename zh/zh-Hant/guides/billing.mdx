## 概览

LemonData 采用 **按量计费 (pay-as-you-go)**。您只需为使用的部分付费，无需订阅或最低消费承诺。

## 账单运作方式

1. **充值额度** 到您的账户
2. **使用 API** - 费用按请求扣除
3. **监控用量** 在您的控制面板中
4. **余额不足时充值**

## Token 计数

Tokens 是文本处理的基本单位：

- ~4 个字符 = 1 token (英文)
- ~1-2 个字符 = 1 token (中文)
- 1 张图片 = 视尺寸和细节而定

### 估算 Tokens

```python
# Rough estimation
def estimate_tokens(text):
    return len(text) / 4  # Approximate for English

# Actual count (for OpenAI models)
```
