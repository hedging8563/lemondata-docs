---
title: "企业微信 (WeCom)"
description: "通过 webhook 回调将您的 Claw 实例连接到企业微信 (WeCom)"
---

将企业微信 (WeCom) 与您的 Claw 实例集成，为您的企业工作空间引入 AI。

## 前提条件

- 一个 [企业微信](https://work.weixin.qq.com/) 管理员账号
- 一个 Claw 实例（[在此处创建](https://lemondata.cc/dashboard/claw)）

## 设置指南

<Steps>
  <Step title="创建企业微信应用">
    1. 登录 [企业微信管理后台](https://work.weixin.qq.com/wework_admin/frame)
    2. 前往 **应用管理** → **创建应用**
    3. 输入应用名称，上传 logo，并设置可见范围
    4. 点击 **创建应用**
  </Step>
  <Step title="获取凭据">
    1. 记录您的 **Corp ID** —— 在 **我的企业** → **企业信息** 底部可以找到
    2. 在应用设置中，复制 **Secret** (Corp Secret)
  </Step>
  <Step title="配置回调 URL">
    1. 在应用设置中，前往 **接收消息** → **设置 API 接收**
    2. 当您展开企业微信部分时，LemonData 控制面板会显示您的回调 URL：
    ```
    https://claw-{your-subdomain}.lemondata.cc/webhook/wecom
    ```
    3. 设置一个 **Token**（随机字符串）和 **EncodingAESKey**（43 位随机字符串）
    4. 点击 **保存** —— 企业微信将验证回调 URL

    <Tip>
    回调 URL 显示在 LemonData 控制面板中，并配有复制按钮。在验证 URL 之前，请确保您的 Claw 实例正在运行。
    </Tip>
  </Step>
  <Step title="在 LemonData 控制面板中配置">
    1. 前往您的 [Claw 控制面板](https://lemondata.cc/dashboard/claw)
    2. 点击实例上的 **设置图标**
    3. 打开 **IM Integrations** 选项卡
    4. 展开 **WeCom** 部分
    5. 复制顶部显示的 **Callback URL**
    6. 填写：**Corp ID**、**Corp Secret**、**Token**、**EncodingAESKey**
    7. 点击 **Save**
  </Step>
  <Step title="开始聊天">
    1. 在企业微信中，在 **工作台** 中找到您的应用
    2. 向应用发送一条消息
    3. 完成配对过程
    4. 开始与 AI 聊天！
  </Step>
</Steps>

<Warning>
**Token** 和 **EncodingAESKey** 在企业微信管理后台和您的 Claw 配置之间必须完全一致。不匹配将导致消息验证失败。
</Warning>

## 故障排除

<AccordionGroup>
  <Accordion title="回调 URL 验证失败">
    - 确保您的 Claw 实例正在运行（绿色状态）
    - 验证 URL 格式：`https://claw-{subdomain}.lemondata.cc/webhook/wecom`
    - 在企业微信中验证 **之前**，请检查 Token 和 EncodingAESKey 是否已保存在 Claw 配置中
  </Accordion>
  <Accordion title="机器人没有响应">
    - 验证 Corp ID 和 Corp Secret 是否正确
    - 检查应用对您的用户账号是否可见（可见范围设置）
    - 确保企业微信和 Claw 之间的 Token 和 EncodingAESKey 一致
  </Accordion>
</AccordionGroup>
