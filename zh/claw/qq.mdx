使用官方的 QQ 开放平台机器人 API 将 QQ（中国最受欢迎的即时通讯应用）连接到你的 Claw 实例。

## 前提条件

- 一个 [QQ](https://im.qq.com/) 账号
- 访问 [QQ 开放平台](https://q.qq.com/) 的权限（开发者账号）
- 一个 Claw 实例（[在此处创建](https://lemondata.cc/dashboard/claw)）

## 设置指南

<Steps>
  <Step title="创建 QQ 机器人应用">
    1. 前往 [QQ 开放平台](https://q.qq.com/)
    2. 登录并前往 **机器人** 板块
    3. 点击 **创建机器人**
    4. 填写机器人信息（名称、头像、描述）
    5. 提交审核
  </Step>
  <Step title="获取凭据">
    1. 审核通过后，前往机器人的 **开发设置**
    2. 复制 **AppID**
    3. 复制 **AppSecret** (Client Secret)
  </Step>
  <Step title="配置沙箱（用于测试）">
    1. 在机器人设置中，前往 **沙箱配置**
    2. 将你的 QQ 号添加到沙箱白名单中
    3. 这允许你在正式发布前进行测试

    <Tip>
    建议先从沙箱模式开始，在提交生产审核前验证一切功能正常。
    </Tip>
  </Step>
  <Step title="在 LemonData 控制面板中进行配置">
    1. 前往你的 [Claw 控制面板](https://lemondata.cc/dashboard/claw)
    2. 点击实例上的 **设置图标**
    3. 打开 **IM 集成** 标签页
    4. 展开 **QQ** 板块
    5. 粘贴你的 **AppID** 和 **AppSecret**
    6. 点击 **保存**
  </Step>
  <Step title="开始聊天">
    1. 在 QQ 中搜索你的机器人或将其添加为好友
    2. 发送一条消息
    3. 第一个用户将 **自动配对** 为所有者 —— 无需代码！
    4. 开始与 AI 聊天！
  </Step>
</Steps>

## 支持的消息类型

| 类型 | 描述 |
|---|---|
| **C2C (私聊)** | 直接向机器人发送消息 |
| **群聊 @提及** | 在 QQ 群中 @提及机器人 |
| **频道** | QQ 频道中的消息 |

## 故障排查

<AccordionGroup>
  <Accordion title="机器人无响应">
    - 验证 AppID 和 AppSecret 是否正确
    - 检查你的 QQ 号是否在沙箱白名单中（如果正在测试）
    - 确保机器人应用已通过审核
  </Accordion>
  <Accordion title="'invalid appid' 错误">
    - 仔细检查 AppID —— 它应该是一个数字字符串
    - 确保你使用的是正确的 AppSecret（不是 token）
  </Accordion>
</AccordionGroup>
