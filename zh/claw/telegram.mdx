Telegram 是连接起来**最简单**的平台。您只需要从 BotFather 获取一个 bot token，即可在 2 分钟内开始与 AI 聊天。

## 先决条件

- 一个 [Telegram](https://telegram.org/) 账号
- 一个 Claw 实例（[在此处创建](https://lemondata.cc/dashboard/claw)）

## 设置指南

<Steps>
  <Step title="使用 BotFather 创建机器人">
    1. 打开 Telegram 并搜索 **@BotFather**
    2. 发送 `/newbot`
    3. 为您的机器人选择一个**显示名称**（例如，“我的 AI 助手”）
    4. 选择一个以 `bot` 结尾的**用户名**（例如，`my_ai_assistant_bot`）
    5. BotFather 会回复您的 **Bot Token** —— 看起来像这样：
    ```
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
    ```
    <Warning>
    请对您的 bot token 保密！任何拥有此 token 的人都可以控制您的机器人。
    </Warning>
  </Step>
  <Step title="在 LemonData 控制台中配置">
    1. 前往您的 [Claw 控制台](https://lemondata.cc/dashboard/claw)
    2. 点击实例上的**设置图标**
    3. 打开 **IM Integrations** 选项卡
    4. 展开 **Telegram** 部分
    5. 粘贴您的 **Bot Token**
    6. 点击 **Save**
  </Step>
  <Step title="开始聊天">
    1. 打开 Telegram 并通过用户名找到您的机器人
    2. 发送任何消息（例如，“Hello!”）
    3. 第一个用户将**自动配对**为所有者 —— 无需代码！
    4. 开始与 AI 聊天！
  </Step>
</Steps>

## 可选：自定义机器人资料

在创建机器人后，您可以使用 BotFather 对其进行自定义：

| 命令 | 用途 |
|---------|---------|
| `/setdescription` | 设置机器人的简介/描述 |
| `/setabouttext` | 设置“关于”部分 |
| `/setuserpic` | 上传个人资料照片 |
| `/setcommands` | 添加命令建议 |

## 在群组中使用

<Warning>
Telegram 机器人的 **Privacy Mode 默认启用**。在群聊中，机器人只能看到以下消息：
- 以 `/` 开头（命令）
- 直接 @提及机器人
- 是对机器人消息的回复

要让机器人查看**所有**群组消息，请通过 BotFather 禁用 Privacy Mode：发送 `/setprivacy` → 选择您的机器人 → 选择 **Disable**。注意：您必须**移除并重新添加**机器人到现有群组中，此更改才能生效。
</Warning>

### 群组设置

在控制台的 **IM Integrations** 选项卡中配置群组行为：

| 设置 | 选项 | 描述 |
|---------|---------|-------------|
| **Group Policy** | Open（默认） / Allowlist / Disabled | 控制谁可以在群组中触发机器人 |
| **Require @mention** | On / Off | 机器人仅在被 @提及时响应（推荐） |
| **Group IDs** | 聊天 ID 列表 | 限制在特定群组中（留空 = 所有群组） |
| **Group Allowed Users** | 用户 ID 列表 | 仅这些用户可以触发机器人（白名单模式） |

<Tip>
**Group Policy** 控制*谁*可以在群组中给机器人发消息，而 **Group IDs** 控制机器人响应*哪些*群组。结合使用两者以实现精细化控制。
</Tip>

## 私聊策略

控制谁可以向您的机器人发送私聊消息：

| 策略 | 行为 |
|--------|----------|
| **Pairing**（默认） | 新用户将获得配对码 —— 在 WebUI 中输入以进行授权 |
| **Allowlist** | 只有您指定的 用户 ID 可以向机器人发送消息 |
| **Open** | 任何人都可以向机器人发送消息（慎用 —— 会消耗您的 AI 额度） |

## 故障排除

<AccordionGroup>
  <Accordion title="机器人没有响应">
    - 确认 bot token 正确（没有多余的空格）
    - 检查您的实例是否正在运行（控制台中显示绿色状态）
    - 尝试从控制台重启实例
  </Accordion>
  <Accordion title="“配对码”未出现">
    - 确保您是向机器人发送**私聊消息**，而不是在群组中
    - 配对系统在收到新用户的第一条消息时激活
  </Accordion>
  <Accordion title="机器人响应缓慢">
    - 响应时间取决于所选的 AI 模型
    - 较大的模型（Claude Opus, GPT-5）可能需要几秒钟
    - 检查您的 LemonData 余额 —— 余额不足会导致错误
  </Accordion>
</AccordionGroup>
