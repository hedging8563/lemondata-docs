## 概览

[Cursor](https://cursor.sh) 是一款 AI 驱动的代码编辑器。您可以将 LemonData 作为自定义 API 提供商，以访问 300 多个模型。

## 配置

1. 打开 Cursor 设置 (`Cmd/Ctrl + ,`)
2. 导航至左侧边栏的 **Models**
3. 向下滚动到 **OpenAI API Key** 部分
4. 输入您的 LemonData API 密钥
5. 启用 **Override OpenAI Base URL**
6. 将 Base URL 设置为：

```
https://api.lemondata.cc/v1
```

## 模型选择

配置完成后，您可以在 Cursor 中选择任何 LemonData 模型：

### 推荐的编程模型

| 模型 | 最适合 |
|-------|----------|
| `claude-opus-4-5` | 复杂编程、架构 |
| `claude-sonnet-4-5` | 通用编程、调试 |
| `gpt-4o` | 代码生成、解释 |
| `deepseek-r1` | 重推理任务 |
| `gpt-4o-mini` | 快速补全、简单任务 |

## 功能特性

### Chat

配合任何 LemonData 模型使用 AI 聊天面板 (`Cmd/Ctrl + L`)：

- 代码解释
- Bug 修复
- 重构建议
- 文档生成

### Composer

Composer 功能 (`Cmd/Ctrl + I`) 可配合 LemonData 模型用于：

- 多文件编辑
- 功能实现
- 代码迁移

### Tab 补全

对于 Tab 补全，我们建议使用快速模型：

- `gpt-4o-mini`
- `claude-haiku-4-5`
- `gemini-2.5-flash`
