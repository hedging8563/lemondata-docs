## 概览

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) 是 Anthropic 官方推出的 AI 辅助编程 CLI 工具。你可以将其配置为使用 LemonData 作为 API 提供商。

## 配置

### 环境变量

设置以下环境变量：

```bash
export ANTHROPIC_API_KEY="sk-your-lemondata-key"
export ANTHROPIC_BASE_URL="https://api.lemondata.cc"
```

将这些添加到你的 shell 配置文件（`~/.bashrc`、`~/.zshrc` 等）中以实现持久化。

### 配置文件

或者，创建或编辑 `~/.claude/settings.json`：

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  }
}
```

## 使用方法

配置完成后，即可正常使用 Claude Code：

```bash
# 启动交互式会话
claude

# 运行单个命令
claude "Explain this code" < main.py

# 聊天模式
claude chat
```

## 可用模型

LemonData 支持所有 Claude 模型：

| 模型 | 描述 |
|-------|-------------|
| `claude-opus-4-5` | 最强大的模型，适用于复杂推理 |
| `claude-sonnet-4-5` | 性能均衡，非常适合编程 |
| `claude-haiku-4-5` | 速度最快，性价比高 |

## 模型选择

使用 `--model` 标志指定模型：

```bash
claude --model claude-sonnet-4-5 "Review this PR"
```

或者在配置中设置默认模型：

```json
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-your-lemondata-key",
    "ANTHROPIC_BASE_URL": "https://api.lemondata.cc"
  },
  "model": "claude-sonnet-4-5"
}
```

## 扩展思维

对于复杂的推理任务，请使用支持扩展思维（Extended Thinking）的模型：

```bash
claude --model claude-opus-4-5 "Design a microservices architecture for..."
```

## 故障排除

<AccordionGroup>
  <Accordion title="连接错误">
    验证基础 URL 是否准确为 `https://api.lemondata.cc`（Anthropic SDK 不需要 `/v1` 后缀）。
  </Accordion>

  <Accordion title="无效的 API 密钥">
    确保你的 LemonData API 密钥以 `sk-` 开头且余额充足。
  </Accordion>

  <Accordion title="未找到模型">
    检查模型名称是否正确。请使用 `claude-sonnet-4-5` 格式。
  </Accordion>
</AccordionGroup>
