---
title: "创建视频"
openapi: "POST /v1/videos/generations"
description: "创建一个视频生成任务"
---

## 概览

视频生成是异步的。您提交请求并接收一个任务 ID，然后轮询结果。

## 请求体

<ParamField body="model" type="string" default="kling-video">
  视频模型（例如：`kling-video`，`veo3.1`，`wan2.6-i2v`）。
</ParamField>

<ParamField body="prompt" type="string" required>
  要生成的视频的文本描述。
</ParamField>

<ParamField body="image_url" type="string">
  用于图生视频生成的可选起始图片。
</ParamField>

<ParamField body="duration" type="integer">
  视频时长，以秒为单位（取决于模型）。
</ParamField>

<ParamField body="aspect_ratio" type="string">
  宽高比（例如：`16:9`，`9:16`，`1:1`）。
</ParamField>

<ParamField body="resolution" type="string">
  视频分辨率（例如：`1080p`，`720p`）。
</ParamField>

<ParamField body="fps" type="integer">
  每秒帧数 (1-120)。
</ParamField>

<ParamField body="negative_prompt" type="string">
  视频生成中需要避免的内容。
</ParamField>

<ParamField body="seed" type="integer">
  用于可复现生成的随机种子。
</ParamField>

<ParamField body="cfg_scale" type="number">
  无分类器指导比例 (0-20)。
</ParamField>

<ParamField body="motion_strength" type="number">
  运动强度 (0-1)。
</ParamField>

<ParamField body="start_image" type="string">
  起始帧图片的 URL。
</ParamField>

<ParamField body="end_image" type="string">
  结束帧图片的 URL。
</ParamField>

<ParamField body="user" type="string">
  终端用户的唯一标识符。
</ParamField>

## 响应

<ResponseField name="task_id" type="string">
  用于轮询的唯一任务标识符。
</ResponseField>

<ResponseField name="status" type="string">
  初始状态：`pending`。
</ResponseField>

<ResponseField name="model" type="string">
  使用的模型。
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":    "kling-v2.6-pro",
        "prompt":   "A cat walking through a garden, cinematic lighting",
        "duration": 5,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/videos/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["task_id"])
}
```

## 图生视频

```python
response = requests.post(
    "https://api.lemondata.cc/v1/videos/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "wan2.6-i2v",
        "prompt": "The person starts walking forward",
        "image_url": "https://example.com/image.jpg"
    }
)
```

## 可用模型

| 模型 | 类型 | 描述 |
|-------|------|-------------|
| `kling-video` | T2V/I2V | 逼真的动态效果，多个版本 (v1.0-v2.6) |
| `veo3.1` | T2V | Google 最新模型 |
| `veo3.1-pro` | T2V | 专业级质量 |
| `minimax/video-01` | T2V | 高节奏感与稳定性 |
| `wan2.6-i2v` | I2V | 图生视频 |
