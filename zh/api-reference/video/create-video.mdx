---
title: "创建视频"
openapi: "POST /v1/videos/generations"
description: "创建一个视频生成任务"
---

## 概览

视频生成是异步的。您提交请求并获取一个任务 ID，然后轮询结果。

## 请求体

<ParamField body="model" type="string" default="sora-2">
  视频模型 ID。请参阅 [视频生成指南](/guides/video-generation) 以获取涵盖 11 个供应商的 80 多个模型的完整列表。
</ParamField>

<ParamField body="prompt" type="string" required>
  要生成的视频的文本描述。
</ParamField>

<ParamField body="image_url" type="string">
  用于图生视频 (I2V) 生成的起始图像 URL。
</ParamField>

<ParamField body="image" type="string">
  带有数据 URL 前缀的 Base64 编码图像（例如：`data:image/jpeg;base64,...`）。图生视频 (I2V) 生成中 `image_url` 的替代方案。
</ParamField>

<ParamField body="duration" type="integer">
  视频时长，以秒为单位（取决于模型）。
</ParamField>

<ParamField body="aspect_ratio" type="string">
  宽高比（例如：`16:9`、`9:16`、`1:1`）。
</ParamField>

<ParamField body="resolution" type="string">
  视频分辨率（例如：`1080p`、`720p`）。
</ParamField>

<ParamField body="fps" type="integer">
  每秒帧数 (1-120)。
</ParamField>

<ParamField body="negative_prompt" type="string">
  视频生成中需要避免的内容。
</ParamField>

<ParamField body="seed" type="integer">
  用于可复现生成的随机种子。
</ParamField>

<ParamField body="cfg_scale" type="number">
  无分类器指导比例 (0-20)。
</ParamField>

<ParamField body="motion_strength" type="number">
  运动强度 (0-1)。
</ParamField>

<ParamField body="start_image" type="string">
  起始帧图像的 URL。
</ParamField>

<ParamField body="end_image" type="string">
  结束帧图像的 URL。
</ParamField>

<ParamField body="user" type="string">
  终端用户的唯一标识符。
</ParamField>

## 响应

<ResponseField name="task_id" type="string">
  用于轮询的唯一任务标识符。
</ResponseField>

<ResponseField name="status" type="string">
  初始状态：`pending`。
</ResponseField>
