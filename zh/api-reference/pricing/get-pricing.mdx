## 查询参数

<ParamField query="model" type="string">
  要获取定价的模型 ID。如果未指定，则返回所有模型的定价。
</ParamField>

## 响应

<ResponseField name="object" type="string">
  始终为 `list`。
</ResponseField>

<ResponseField name="data" type="array">
  模型定价对象数组。

  每个对象包含：
  - `model` (string): 模型标识符
  - `pricing` (object): 定价信息
    - `input_per_1m` (string | null): 每 1M 输入 token 的美元价格
    - `output_per_1m` (string | null): 每 1M 输出 token 的美元价格
    - `per_request` (string | null): 每次请求的美元价格（适用于固定价格模型）
    - `is_lock_price` (boolean): 定价是否按每次请求固定
    - `currency` (string): 始终为 `USD`
  - `provider` (string): 模型提供商
</ResponseField>

## 定价类型

| 类型 | 描述 | 示例模型 |
|------|-------------|----------------|
| **基于 Token** | 按输入/输出 token 计费 | GPT-4o, Claude, Gemini |
| **按次计费** | 每次请求固定价格 | DALL-E 3, Sora, Suno |
