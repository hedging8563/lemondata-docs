---
title: "创建音乐"
openapi: "POST /v1/music/generations"
description: "使用 Suno 创建音乐生成任务"
---

## 授权

<ParamField header="Authorization" type="string" required>
  API 认证的 Bearer 令牌。格式：`Bearer sk-your-api-key`
</ParamField>

使用 AI 生成音乐和歌词。这是一个异步 API - 您将收到一个任务 ID 用于轮询完成状态。

## 请求体

<ParamField body="model" type="string" default="suno_music">
  使用的模型：`suno_music` 用于音乐生成，`suno_lyrics` 仅用于歌词。
</ParamField>

<ParamField body="prompt" type="string" required>
  要生成的音乐描述。可以包括风格、情绪、乐器等。
</ParamField>

<ParamField body="title" type="string">
  生成的歌曲标题。
</ParamField>

<ParamField body="tags" type="string">
  风格标签（例如 "pop, upbeat, electronic"）。
</ParamField>

<ParamField body="action" type="string">
  生成类型：`MUSIC`（默认）或 `LYRICS`。
</ParamField>

<ParamField body="mv" type="string">
  要使用的模型版本。
</ParamField>

<ParamField body="continue_clip_id" type="string">
  要续写的上一个片段的 ID。
</ParamField>

<ParamField body="continue_at" type="number">
  续写的起始时间戳（以秒为单位）。
</ParamField>

<ParamField body="user" type="string">
  终端用户的唯一标识符。
</ParamField>

## 响应

<ResponseField name="id" type="string">
  用于轮询状态的任务 ID。
</ResponseField>

<ResponseField name="status" type="string">
  任务状态：`pending`、`processing`、`completed` 或 `failed`。
</ResponseField>

<ResponseField name="created" type="integer">
  任务创建的 Unix 时间戳。
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "suno_music",
        "prompt": "An upbeat electronic dance track with heavy bass",
        "title":  "Night Drive",
        "tags":   "electronic, EDM, energetic",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.lemondata.cc/v1/music/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```

```php PHP
<?php
$ch = curl_init('https://api.lemondata.cc/v1/music/generations');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'suno_music',
        'prompt' => 'An upbeat electronic dance track with heavy bass',
        'title' => 'Night Drive',
        'tags' => 'electronic, EDM, energetic'
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo "Task ID: " . $data['id'];
```
